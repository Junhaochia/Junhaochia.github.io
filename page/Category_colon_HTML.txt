<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<p>The HTML control can be used to display HTML code just like an internet browser. Unlike <a href="/page/Category_colon_DHTML" title="Category:DHTML">DHTML</a>, this control cannot accept user input or complex interaction, such as retrieving image data.
</p><p><b>Blocked URL Extentions</b>
</p><p><i>This means your url cannot end in any of these extensions, e.g.: (<a rel="nofollow" class="external free" href="http://yourdomain.com/garry.exe">http://yourdomain.com/garry.exe</a>)</i>
</p>
<ul><li> .exe
</li><li> .bat
</li><li> .dll
</li><li> .zip
</li><li> .mp3
</li><li> .swf
</li></ul>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates an HTML control in the center of screen, and makes it display <a rel="nofollow" class="external free" href="/">http://wiki.garrysmod.com/</a>
</p>
<pre><code data-language="lua">HTMLTest = vgui.Create( &quot;HTML&quot; )
HTMLTest:SetPos( 50, 50 )
HTMLTest:SetSize( ScrW() - 100, ScrH() - 100 )
HTMLTest:OpenURL( &quot;wiki.garrysmod.com&quot; )</code></pre>
