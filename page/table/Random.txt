<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a random value from the supplied table.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function iterates over the given table <b>twice</b>, therefore with sequential tables you should instead use following:
</p>
<pre><code data-language="lua">mytable[ math.random( #mytable ) ]</code></pre>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>A simple example of this function using two tables.
</p>
<pre><code data-language="lua">color = { &quot;green&quot;, &quot;red&quot;, &quot;blue&quot;, &quot;yellow&quot; }
object = { &quot;car&quot;, &quot;house&quot;, &quot;bike&quot; }

print( &quot;I have a &quot; .. table.Random( color ) .. &quot; &quot; .. table.Random( object ) .. &quot;.&quot; )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">I have a green house.</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Example of using the alternative with sequential tables for performance reasons.
</p>
<pre><code data-language="lua">websites = {&quot;facepunch.com&quot;, &quot;google.com&quot;, &quot;steampowered.com&quot;}
print(&quot;I think the best website ever is &quot; .. websites[math.random(1, #websites)] .. &quot;.&quot;)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">I think the best website ever is google.com.</div>
