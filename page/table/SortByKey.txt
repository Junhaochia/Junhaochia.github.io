<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a list of keys sorted based on values of those keys.
</p><p>For normal sorting see <a href="/page/Category_colon_table" title="Category:table">table</a>.<a href="/page/table/sort" title="table/sort">sort</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> tab</span>
</p>
<div style="margin-left: 32px;">
Table to sort. All values of this table must be of same type.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> descending=false</span>
</p>
<div style="margin-left: 32px;">
Should the order be descending?</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
A table of keys sorted by values from supplied table.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage of the function.
</p>
<pre><code data-language="lua">local t = {}
t['h'] = 2 -- Lowest value
t['a'] = 150 -- Highest value
t['x'] = 30

PrintTable( table.SortByKey( t ) )
PrintTable( table.SortByKey( t , true ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">1 = a
2 = x
3 = h</code></pre>
<pre><code data-language="lua">1 = h
2 = x
3 = a</code></pre></div>
