<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Checks if a table has a value.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>This function is <b>very inefficient for large tables</b> (O(n)) and should probably not be called in things that run each frame. Instead, consider a table structure such as example 2 below.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>For optimization, functions that look for a value by sorting the table should never be needed if you work on a table that you built yourself.
</p>
</td></tr></table>

</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">

</p>
<div style="margin-left: 32px;">
Returns true if the table has that value, false otherwise</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a table with values "123" and "test" and checks to see it the table holds value "apple"
</p>
<pre><code data-language="lua">local mytable = {&quot;123&quot;, &quot;test&quot;}
print(table.HasValue(mytable, &quot;apple&quot;), table.HasValue(mytable, &quot;test&quot;))</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">false	true</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Example usage of O(1) associative array structure
</p>
<pre><code data-language="lua">local mytable = { [&quot;123&quot;] = true, test = true }
print(mytable[&quot;apple&quot;], mytable[&quot;test&quot;])</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">nil	true</div>
