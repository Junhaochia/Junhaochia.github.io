<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Empties the target table, and merges all values from the source table into it.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> source</span>
</p>
<div style="margin-left: 32px;">
The table to copy from.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> target</span>
</p>
<div style="margin-left: 32px;">
The table to write to.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Demonstrates the use of this function.
</p>
<pre><code data-language="lua">local Test1 = {A = &quot;String keys&quot;, B = &quot;Table 1&quot;}
local Test2 = {&quot;Numeric keys&quot;, &quot;Table 2&quot;}
table.CopyFromTo( Test2, Test1 )
PrintTable( Test1 )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><br />
<p>1 = Numeric keys<br />
</p>
2 = Table 2<br /></div>
