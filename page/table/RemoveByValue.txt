<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Removes the first instance of a given value from the specified table with <a href="/page/Category_colon_table" title="Category:table">table</a>.<a href="/page/table/remove" title="table/remove">remove</a>, then returns the key that the value was found at.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> tbl</span>
</p>
<div style="margin-left: 32px;">
The table that will be searched.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_any" title="Category:any">any</a> val</span>
</p>
<div style="margin-left: 32px;">
The value to find within the table.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_any" title="Category:any">any</a></span>
</p>
<div style="margin-left: 32px;">
The key at which the value was found, or false if the value was not found.</div>
</div>
