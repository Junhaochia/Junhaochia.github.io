<h1> <span class="mw-headline" id="Description">Description</span></h1>
<div class="deprecatedfunc"><span>This feature is deprecated.</span><br />
You should avoid using it as it may be removed in a future version.<br /><br />Instead, iterate your table with ipairs, storing the previous value and checking for the target. Non-numerically indexed tables are not ordered.</div>
<p>Returns the value positioned before the supplied value in a table. If it isn't found then the last element in the table is returned
</p>
