<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Adds the contents from one table into another. The target table will be modified.
</p><p>See also <a href="/page/Category_colon_table" title="Category:table">table</a>.<a href="/page/table/Inherit" title="table/Inherit">Inherit</a> and <a href="/page/Category_colon_table" title="Category:table">table</a>.<a href="/page/table/Merge" title="table/Merge">Merge</a>.
</p>

</p>

</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Demonstrates the use of this function. Note that duplicate values will be added.
</p>
<pre><code data-language="lua">local Test1 = {&quot;One&quot;,&quot;Two&quot;,&quot;Three&quot;, &quot;Four&quot;}
local Test2 = {&quot;Four&quot;, &quot;Five&quot;, &quot;Six&quot;}
table.Add( Test1, Test2 )
print( table.concat(Test1, &quot; &quot;) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><br />
One Two Three Four Four Five Six</div>
