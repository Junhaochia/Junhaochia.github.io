<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Copies any missing data from base to target, and sets the target's <i>BaseClass</i> member to the base table's pointer.
</p><p>See <a href="/page/Category_colon_table" title="Category:table">table</a>.<a href="/page/table/Merge" title="table/Merge">Merge</a>, which overrides existing values and doesn't add a BaseClass member.<br />
See also <a href="/page/Category_colon_table" title="Category:table">table</a>.<a href="/page/table/Add" title="table/Add">Add</a>, which simply adds values of one table to another.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>Sub-tables aren't inherited. The target's table value will take priority.
</p><p><b>Pull Request: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod/pull/1304">#1304</a></b>
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example of how this function works.
</p>
<pre><code data-language="lua">local table1 = { &quot;A&quot;, &quot;Golden&quot; }
local table2 = { &quot;Two&quot;, &quot;Orange&quot;, &quot;Apple&quot; }
table.Inherit( table1, table2 )
PrintTable( table1 )</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">1	=	A
2	=	Golden
3	=	Apple
BaseClass:
		1	=	Two
		2	=	Orange
		3	=	Apple</code></pre>
