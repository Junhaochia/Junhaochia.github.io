<h1> <span class="mw-headline" id="Description">Description</span></h1>
<div class="deprecatedfunc"><span>This feature is deprecated.</span><br />
You should avoid using it as it may be removed in a future version.<br /><br />This was deprecated in Lua 5.1 and removed in 5.2. You should use <a href="/page/Global/pairs" title="Global/pairs">pairs</a>() instead.</div> Iterates for each key-value pair in the table, calling the function with the key and value of the pair. If the function returns anything, the loop is broken.
<p>This is inherited from the original Lua implementation and is deprecated in Lua as of 5.1; see <a rel="nofollow" class="external text" href="http://lua-users.org/wiki/TableLibraryTutorial">here</a>. You should use <a href="/page/Global/pairs" title="Global/pairs">pairs</a>() instead. The GLua interpretation of this is <a href="/page/Category_colon_table" title="Category:table">table</a>.<a href="/page/table/ForEach" title="table/ForEach">ForEach</a>.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Demonstrates the use of this function.
</p>
<pre><code data-language="lua">local food = { &quot;Cake&quot;, &quot;Pies&quot;, Delicious = &quot;Cookies&quot;, Awesome = &quot;Pizza&quot; }
table.foreach( food, function( key, value )
 print( tostring(key) ..&quot; &quot;.. value)
end)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">
1 Cake
2 Pies
Awesome Pizza
Delicious Cookies</code></pre></div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Demonstrates the breaking effect if the callback returns a value.
</p>
<pre><code data-language="lua">local tbl = { &quot;One&quot;, &quot;Two&quot;, &quot;Three&quot;, &quot;Four&quot; }
table.foreach( tbl, function( key, value )
   print( key, value )
   if key == 2 then return true end
end)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">
1    One
2    Two</code></pre></div>
