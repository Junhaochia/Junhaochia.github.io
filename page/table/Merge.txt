<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Merges the contents of the second table with the content in the first one.
</p><p>See <a href="/page/Category_colon_table" title="Category:table">table</a>.<a href="/page/table/Inherit" title="table/Inherit">Inherit</a>, which doesn't override existing values.<br />
See also <a href="/page/Category_colon_table" title="Category:table">table</a>.<a href="/page/table/Add" title="table/Add">Add</a>, which simply adds values of one table to another.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>"Merges" the content of the second table with the first one, overwriting any matching key/value pairs in the destination with the source's version and prints the resulting merge.
</p>
<pre><code data-language="lua">local destination = {[1] = &quot;A&quot;, [2] = &quot;Golden&quot;, [3] = &quot;Apple&quot;}
local source = {[1] = &quot;Two&quot;, [2] = &quot;Orange&quot;}
table.Merge( destination, source )
PrintTable( destination )</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">1	=	Two
2	=	Orange
3	=	Apple</code></pre>
