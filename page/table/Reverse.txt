<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a reversed copy of a sequential table. Any non-sequential and non-numeric keyvalue pairs will not be copied.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> tbl</span>
</p>
<div style="margin-left: 32px;">
Table to reverse.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
A reversed copy of the table.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a table and reverses it.
</p>
<pre><code data-language="lua">local tbl = { &quot;One&quot;, &quot;Two&quot;, &quot;Three&quot;, four = 4, [5] = &quot;5&quot; }

PrintTable(tbl)
print(&quot;&quot;)
PrintTable(table.Reverse(tbl))</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">1	=	One
2	=	Two
3	=	Three
four	=	4
5	=	5

1	=	Three
2	=	Two
3	=	One
</code></pre></div>
