<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Changes all keys to sequential integers. This creates a new table object and does not affect the original.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> table</span>
</p>
<div style="margin-left: 32px;">
The original table to modify.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> saveKeys=false</span>
</p>
<div style="margin-left: 32px;">
Save the keys within each member table. This will insert a new field <i>__key</i> into each value, and should not be used if the table contains non-table values.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
Table with integer keys.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Changes all the table's keys to integer values
</p>
<pre><code data-language="lua">local Table = {One = &quot;A&quot;, Two = &quot;B&quot;, Three = &quot;C&quot;}
local Table2 = table.ClearKeys(Table)
PrintTable(Table2)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><br />
<p>1 = A<br />
2 = C<br />
</p>
3 = B<br /></div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Clears a table of its keys, and preserves the old key names within each member.
</p>
<pre><code data-language="lua">local tbl = {
	FirstMember = { Name = &quot;John Smith&quot;, Age  = 25 },
	SecondMember = { Name = &quot;Jane Doe&quot;, Age = 42 },
	ThirdMember = { Name = &quot;Joe Bloggs&quot;, Age = 39 }
}
print( &quot;===== Before =====&quot; )
PrintTable( tbl )
local tbl2 = table.ClearKeys( tbl, true )
print( &quot;===== After =====&quot; )
PrintTable( tbl2 )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">===== Before =====
FirstMember:
		Name	=	John Smith
		Age	=	25
SecondMember:
		Name	=	Jane Doe
		Age	=	42
ThirdMember:
		Name	=	Joe Bloggs
		Age	=	39
===== After =====
1:
		Age	=	25
		Name	=	John Smith
		__key	=	FirstMember
2:
		Age	=	39
		Name	=	Joe Bloggs
		__key	=	ThirdMember
3:
		Age	=	42
		Name	=	Jane Doe
		__key	=	SecondMember</code></pre></div>
