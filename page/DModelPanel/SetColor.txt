<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the color of the rendered entity.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This does not work on Garry's Mod player models since they use a different color system. To modify a player model color, see Example 2 on the <a href="/page/Category_colon_DModelPanel" title="Category:DModelPanel">DModelPanel</a> page
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> color</span>
</p>
<div style="margin-left: 32px;">
The render color of the entity.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a model panel with a red watermelon inside.
</p>
<pre><code data-language="lua">local panel = vgui.Create(&quot;DPanel&quot;)
panel:SetPos(20, 20)
panel:SetSize(200, 200)
	
local mdl = vgui.Create(&quot;DModelPanel&quot;, panel)
mdl:SetSize(panel:GetSize())
mdl:SetModel(&quot;models/props_junk/watermelon01.mdl&quot;)
mdl:SetLookAt(Vector(0, 0, 0))
mdl:SetCamPos(Vector(10, 10, 10))
mdl:SetColor(Color(255, 0, 0))</code></pre>
