<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets whether or not to animate the entity when the default <a href="/page/Category_colon_DModelPanel" title="Category:DModelPanel">DModelPanel</a>:<a href="/page/DModelPanel/LayoutEntity" title="DModelPanel/LayoutEntity">LayoutEntity</a> is called.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> animated</span>
</p>
<div style="margin-left: 32px;">
True to animate, false otherwise.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>A comparison between 2 model panels: the first one has <i>animated</i> set to false and the second one has it set to true. Both are using the default <a href="/page/Category_colon_DModelPanel" title="Category:DModelPanel">DModelPanel</a>:<a href="/page/DModelPanel/LayoutEntity" title="DModelPanel/LayoutEntity">LayoutEntity</a> method.
</p>
<pre><code data-language="lua">BGPanel = vgui.Create(&quot;DPanel&quot;)
BGPanel:SetPos(20, 20)
BGPanel:SetSize(400, 200)
	
-- Left panel
local mdl1 = vgui.Create(&quot;DModelPanel&quot;, BGPanel)
mdl1:SetPos(0, 0)
mdl1:SetSize(200, 200)
mdl1:SetModel(&quot;models/player/mossman.mdl&quot;)
mdl1:SetCamPos(Vector(40, 40, 40))

mdl1:SetAnimated(false)

-- Right panel
local mdl2 = vgui.Create(&quot;DModelPanel&quot;, BGPanel)
mdl2:SetPos(200, 0)
mdl2:SetSize(200, 200)
mdl2:SetModel(&quot;models/player/mossman.mdl&quot;)
mdl2:SetCamPos(Vector(40, 40, 40))

mdl2:SetAnimated(true)

-- Dance sequence	
local dance = mdl1:GetEntity():LookupSequence(&quot;taunt_dance&quot;)

-- Make both dance
mdl1:GetEntity():SetSequence(dance)
mdl2:GetEntity():SetSequence(dance)</code></pre>
