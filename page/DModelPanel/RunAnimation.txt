<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>This function is used in the <b>DModelPanel:LayoutEntity</b>. It will set the active model to the last set animation using <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetSequence" title="Entity/SetSequence">SetSequence</a>. By default, it is the walking animation.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Sets the model to alyx and puts her in a walking animation
</p>
<pre><code data-language="lua">local ModelPanel = vgui.Create( &quot;DModelPanel&quot;, Panel )
ModelPanel:SetModel( &quot;models/player/alyx.mdl&quot; )
function ModelPanel:LayoutEntity( ent )
     ModelPanel:RunAnimation()
end</code></pre>
