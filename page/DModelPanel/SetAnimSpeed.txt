<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the speed used by <a href="/page/Category_colon_DModelPanel" title="Category:DModelPanel">DModelPanel</a>:<a href="/page/DModelPanel/RunAnimation" title="DModelPanel/RunAnimation">RunAnimation</a> to advance frame on an entity sequence.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p><a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/FrameAdvance" title="Entity/FrameAdvance">FrameAdvance</a> doesn't seem to have any functioning arguments and therefore changing this will not have any affect on the panel entity's sequence speed without reimplementation. It only affects the value returned by <a href="/page/Category_colon_DModelPanel" title="Category:DModelPanel">DModelPanel</a>:<a href="/page/DModelPanel/GetAnimSpeed" title="DModelPanel/GetAnimSpeed">GetAnimSpeed</a>
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> animSpeed</span>
</p>
<div style="margin-left: 32px;">
The animation speed.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>A reimplementation of <a href="/page/Category_colon_DModelPanel" title="Category:DModelPanel">DModelPanel</a>:<a href="/page/DModelPanel/LayoutEntity" title="DModelPanel/LayoutEntity">LayoutEntity</a> which will modify the entity's sequence playback rate based on the set animation speed. This example has Alyx run twice as fast as normal.
</p>
<pre><code data-language="lua">local mdl = vgui.Create(&quot;DModelPanel&quot;)
mdl:SetPos(20, 20)
mdl:SetSize(200, 200)
mdl:SetModel(&quot;models/alyx.mdl&quot;)

-- Play sequence twice as fast
mdl:SetAnimSpeed(2)
	
-- Make Alyx run
mdl:GetEntity():SetSequence(mdl:GetEntity():LookupSequence(&quot;run_all&quot;))

-- Play animation
function mdl:LayoutEntity(ent)

	-- Playback rate based on anim speed
	ent:SetPlaybackRate(self:GetAnimSpeed())
	
	-- Advance frame
	ent:FrameAdvance()

end</code></pre>
