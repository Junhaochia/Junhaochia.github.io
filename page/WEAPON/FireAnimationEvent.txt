<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called before firing animation events, such as muzzle flashes or shell ejections.
</p><p>This will only be called serverside for 3000-range events, and clientside for 5000-range and other events.
</p>

<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> pos</span>
</p>
<div style="margin-left: 32px;">
Position of the effect.</div>
</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Disables muzzle flashes. Taken from tool gun source code.
</p>
<pre><code data-language="lua">function SWEP:FireAnimationEvent( pos, ang, event, options )
	
	-- Disables animation based muzzle event
	if ( event == 21 ) then return true end	

	-- Disable thirdperson muzzle flash
	if ( event == 5003 ) then return true end

end</code></pre>

<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Counter-Strike: Source like muzzle flashes.
</p>
<pre><code data-language="lua">function SWEP:FireAnimationEvent( pos, ang, event, options )

	if (&#160;!self.CSMuzzleFlashes ) then return end

	-- CS Muzzle flashes
	if ( event == 5001 or event == 5011 or event == 5021 or event == 5031 ) then
	
		local data = EffectData()
		data:SetFlags( 0 )
		data:SetEntity( self.Owner:GetViewModel() )
		data:SetAttachment( math.floor( ( event - 4991 ) / 10 ) )
		data:SetScale( 1 ) -- Change me

		if ( self.CSMuzzleX ) then
			util.Effect( &quot;CS_MuzzleFlash_X&quot;, data )
		else
			util.Effect( &quot;CS_MuzzleFlash&quot;, data )
		end
	
		return true
	end

end</code></pre>
