<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when weapon tries to holster.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This is called twice for every holster clientside, one in <a href="/page/Prediction" title="Prediction">prediction</a> and one not.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/2854">#2854</a></b>
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>Before <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a>:<a href="/page/WEAPON/OnRemove" title="WEAPON/OnRemove">OnRemove</a> is called, this function is only called serverside.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3133">#3133</a></b>
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This hook is <a href="/page/Prediction" title="Prediction">predicted</a>. This means that in singleplayer, it will <i>not</i> be called in the Client realm.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Returns the weapon switched to when when the weapon is being holstered.
</p>
<pre><code data-language="lua">function SWEP:Holster( wep )
	if not IsFirstTimePredicted() then return end
	LocalPlayer():ChatPrint(&quot;Switched to: &quot;..(wep.PrintName or wep:GetClass()))
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Switched to Toolgun
Switched to weapon_crowbar</div>
