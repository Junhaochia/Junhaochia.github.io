<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Translate a player's Activity into a weapon's activity, depending on how you want the player to be holding the weapon.
</p><p>For example, ACT_MP_RUN becomes ACT_HL2MP_RUN_PISTOL.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> act</span>
</p>
<div style="margin-left: 32px;">
The activity to translate</div>
</div>

</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Default action
</p>
<pre><code data-language="lua">function SWEP:TranslateActivity( act )

	if ( self.Owner:IsNPC() ) then
		if ( self.ActivityTranslateAI[ act ] ) then
			return self.ActivityTranslateAI[ act ]
		end
		return -1
	end

	if ( self.ActivityTranslate[ act ]&#160;!= nil ) then
		return self.ActivityTranslate[ act ]
	end

	return -1

end</code></pre>
