<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when the weapon entity is created.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p><a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetOwner" title="Entity/GetOwner">GetOwner</a> will return NULL at this point because the weapon is not equpped by a player or NPC yet. Use <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a>:<a href="/page/WEAPON/Equip" title="WEAPON/Equip">Equip</a> or <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a>:<a href="/page/WEAPON/Deploy" title="WEAPON/Deploy">Deploy</a> if you need the owner to be valid.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This is sometimes not called clientside. You can work around this by setting a variable in Initialize and check if it exists in <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a>:<a href="/page/WEAPON/Think" title="WEAPON/Think">Think</a>. See the example below.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/2732">#2732</a></b>
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This is not called serverside after a quicksave.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3015">#3015</a></b>
</p>
</td></tr></table>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Sets the weapon hold type to SWEP.HoldType.
</p>
<pre><code data-language="lua">function SWEP:Initialize()

	self:SetHoldType( self.HoldType )

end</code></pre>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Fixes the function not being called clientside.
</p>
<pre><code data-language="lua">function SWEP:Initialize()
	self.m_bInitialized = true

	-- Other code
end

function SWEP:Think()
	if (not self.m_bInitialized) then
		self:Initialize()
	end

	-- Other code
end</code></pre>
