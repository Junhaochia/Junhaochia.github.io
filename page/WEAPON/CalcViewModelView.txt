<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Allows overriding the position and angle of the viewmodel. This hook only works if you haven't overridden <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/CalcViewModelView" title="GM/CalcViewModelView">CalcViewModelView</a>.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>One way to change the viewmodel texture (skin) for your SWEP. Changing it from 0 to 1.
</p><p>Remember to define the base path (<a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/$cdmaterials">$cdmaterials</a>) for your model when e.g. using <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Crowbar">Crowbar</a> to compile .smd to .mdl; or else you won't get any texture(s) applied. (the game won't know where to look) <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Category:QC_Commands">QC Commands</a>
</p><p>And remember to use <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/$texturegroup">$texturegroup</a> for multiple skins.
</p>
<pre><code data-language="lua">function SWEP:CalcViewModelView(ViewModel, OldEyePos, OldEyeAng, EyePos, EyeAng)
    -- Set skin to the SWEPs second skin; 1. Starts at 0
    ViewModel:SetSkin(1)
end</code></pre>
