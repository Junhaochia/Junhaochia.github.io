<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the hold type of the weapon. This function also calls <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a>:<a href="/page/WEAPON/SetWeaponHoldType" title="WEAPON/SetWeaponHoldType">SetWeaponHoldType</a> and properly networks it to all clients.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This only works on scripted weapons.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>Using this function on weapons held by bots will not network holdtype changes to clients if the world model is set to an empty string (SWEP.WorldModel = "").
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> name</span>
</p>
<div style="margin-left: 32px;">
Name of the hold type. You can find all default hold types <a href="/page/Hold_Types" title="Hold Types">here</a></div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Puts a players hands down by its sides on <a href="/page/WEAPON/Reload" title="WEAPON/Reload">reload</a>.
</p>
<pre><code data-language="lua">function SWEP:Reload()
	self:SetHoldType( &quot;normal&quot; )
end</code></pre>
