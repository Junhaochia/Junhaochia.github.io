<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Allows you to adjust player view while this weapon in use.
</p><p>This hook is called from the default implementation of <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/CalcView" title="GM/CalcView">CalcView</a> (<a rel="nofollow" class="external autonumber" href="https://github.com/garrynewman/garrysmod/blob/master/garrysmod/gamemodes/base/gamemode/cl_init.lua#L376-L383">[1]</a>). Therefore, it will not be called if any other hook added to CalcView returns any value, or if the current gamemode overrides the default hook and does not call the SWEP function.
</p>
