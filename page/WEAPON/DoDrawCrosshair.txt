<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when the crosshair is about to get drawn, and allows you to override it.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function will not be called if <b>SWEP.DrawCrosshair</b> set to false.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function will not be called if player was affected by <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/CrosshairDisable" title="Player/CrosshairDisable">CrosshairDisable</a>.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>The arguments passed to this function are not affected by <b>SWEP.AccurateCrosshair</b>.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/2117">#2117</a></b>
</p>
</td></tr></table>

Y coordinate of the crosshair.</div>

<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">

</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Draws an outlined rectangle in place of the crosshair.
</p>
<pre><code data-language="lua">function SWEP:DoDrawCrosshair( x, y )
	surface.SetDrawColor( 0, 250, 255, 255 )
	surface.DrawOutlinedRect( x - 32, y - 32, 64, 64 )
	return true
end</code></pre>
