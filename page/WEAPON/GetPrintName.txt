<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the non-internal name of the weapon, that should be for displaying.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>If that returns an untranslated message (#HL2_XX), use <a href="/page/Category_colon_language" title="Category:language">language</a>.<a href="/page/language/GetPhrase" title="language/GetPhrase">GetPhrase</a> to see the "nice" name.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>If SWEP.PrintName is not set in the Weapon or the Weapon Base then "&lt;MISSING SWEP PRINT NAME&gt;" will be returned.
</p>
</td></tr></table>
