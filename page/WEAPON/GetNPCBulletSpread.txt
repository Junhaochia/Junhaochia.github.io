<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when the weapon is used by NPCs to determine how accurate the bullets fired should be.
</p><p>The inaccuracy is simulated by changing the <a href="/page/Category_colon_NPC" title="Category:NPC">NPC</a>:<a href="/page/NPC/GetAimVector" title="NPC/GetAimVector">GetAimVector</a> based on the value returned from this hook.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> proficiency</span>
</p>
<div style="margin-left: 32px;">
How proficient the NPC is with this gun. See <a href="/page/Enums/WEAPON_PROFICIENCY" title="Enums/WEAPON PROFICIENCY">WEAPON_PROFICIENCY_ Enums</a></div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
An amount of degrees the bullets should deviate from the NPC's <a href="/page/Category_colon_NPC" title="Category:NPC">NPC</a>:<a href="/page/NPC/GetAimVector" title="NPC/GetAimVector">GetAimVector</a>. Default is 15.</div>
</div>
