<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Adds a task to the schedule. See also <a href="/page/Category_colon_Schedule" title="Category:Schedule">Schedule</a>:<a href="/page/Schedule/AddTaskEx" title="Schedule/AddTaskEx">AddTaskEx</a> if you wish to customize task start and run function names.
</p><p>See also <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/StartSchedule" title="ENTITY/StartSchedule">StartSchedule</a>, <a href="/page/Category_colon_NPC" title="Category:NPC">NPC</a>:<a href="/page/NPC/StartEngineTask" title="NPC/StartEngineTask">StartEngineTask</a>, and <a href="/page/Category_colon_NPC" title="Category:NPC">NPC</a>:<a href="/page/NPC/RunEngineTask" title="NPC/RunEngineTask">RunEngineTask</a>.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This creates a new schedule with a task named "HelloWorld" that is defined to print the taskdata passed in.
</p>
<pre><code data-language="lua">local schdHello = ai_schedule.New( &quot;SayHello&quot; )
schdHello:AddTask( &quot;HelloWorld&quot;, &quot;HELLO&quot; )

-- Called when the task is initiated (started)
function ENT:TaskStart_HelloWorld( data )
    print(data)

    -- Set a variable that is 5 seconds in the future so the task can complete when we tick past it
    self.TaskEndTime = CurTime() + 5
end

-- Called every think until the task is completed
function ENT:Task_HelloWorld(data)
    print( data, &quot;again&quot; )

    -- Check if the 5 seconds have passed
    if CurTime() &lt; self.TaskEndTime then
        self:TaskComplete()
    end
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Prints "HELLO" in the console, then prints "HELLO again" on every NPC think until 5 seconds have passed.</div>
