<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Used to create a new category in the list inside of a spawnmenu ToolTab.
</p><p>You must call this function from <a href="/page/Category_colon_SANDBOX_Hooks" title="Category:SANDBOX Hooks">SANDBOX</a>:<a href="/page/SANDBOX/AddToolMenuCategories" title="SANDBOX/AddToolMenuCategories">AddToolMenuCategories</a> for it to work properly.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> tab</span>
</p>
<div style="margin-left: 32px;">
<p>The ToolTab name, as created with <a href="/page/Category_colon_spawnmenu" title="Category:spawnmenu">spawnmenu</a>.<a href="/page/spawnmenu/AddToolTab" title="spawnmenu/AddToolTab">AddToolTab</a>.
</p>
You can also use the default ToolTab names "Main" and "Utilities".</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> RealName</span>
</p>
<div style="margin-left: 32px;">
The identifier name</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> PrintName</span>
</p>
<div style="margin-left: 32px;">
The displayed name</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Adds the Constraints category to the Main ToolTab. See <i>lua\includes\modules\spawnmenu.lua</i>.
</p>
<pre><code data-language="lua">spawnmenu.AddToolCategory( &quot;Main&quot;, &quot;Constraints&quot;, &quot;#spawnmenu.tools.constraints&quot; )</code></pre>
