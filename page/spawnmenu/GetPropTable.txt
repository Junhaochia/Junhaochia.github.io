<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a table of all prop categories and their props in the spawnmenu.
</p><p>Note that if the spawnmenu has not been populated, this will return an empty table.
</p><p>This will not return spawnlists created by addons, see <a href="/page/Category_colon_spawnmenu" title="Category:spawnmenu">spawnmenu</a>.<a href="/page/spawnmenu/GetCustomPropTable" title="spawnmenu/GetCustomPropTable">GetCustomPropTable</a> for that.
</p>

<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
