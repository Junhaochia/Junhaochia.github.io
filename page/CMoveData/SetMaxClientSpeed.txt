<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the maximum player speed. Player won't be able to run or sprint faster then this value.<br />
This also automatically sets <a href="/page/Category_colon_CMoveData" title="Category:CMoveData">CMoveData</a>:<a href="/page/CMoveData/SetMaxSpeed" title="CMoveData/SetMaxSpeed">SetMaxSpeed</a> when used in the <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/SetupMove" title="GM/SetupMove">SetupMove</a> hook. You must set it manually in the <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/Move" title="GM/Move">Move</a> hook.<br />
This must be called on both client and server to avoid prediction errors.<br />
This will <b>not</b> reduce speed in air.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Setting this to 0 will not make the player stationary. It won't do anything.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Doesn't let the player to run or sprint faster than 100 units per second.
</p>
<pre><code data-language="lua">hook.Add(&quot;SetupMove&quot;,&quot;MySpeed&quot;, function( ply, mv )
    mv:SetMaxClientSpeed( 100 )
end )</code></pre>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Doubles the players speed properly.
</p>
<pre><code data-language="lua">hook.Add( &quot;Move&quot;, &quot;testestst&quot;, function( ply, mv, usrcmd )
	local speed = mv:GetMaxSpeed() * 2
	mv:SetMaxSpeed( speed )
	mv:SetMaxClientSpeed( speed )
end )</code></pre>
