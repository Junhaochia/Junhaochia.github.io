<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DModelPanel and sets its model to your playermodel.
</p>
<pre><code data-language="lua">local Panel = vgui.Create( &quot;DPanel&quot; )
Panel:SetPos( 10, 10 )
Panel:SetSize( 200, 200 )

local icon = vgui.Create( &quot;DModelPanel&quot;, Panel )
icon:SetSize( 200, 200 )
icon:SetModel( LocalPlayer():GetModel() )</code></pre>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Creates a DModelPanel and sets its model to the Alyx playermodel, then changes its player color to red. Also disables default rotation animation.
</p>
<pre><code data-language="lua">local Panel = vgui.Create( &quot;DPanel&quot; )
Panel:SetPos( 10, 10 )
Panel:SetSize( 200, 200 )

local icon = vgui.Create( &quot;DModelPanel&quot;, Panel )
icon:SetSize(200,200)
icon:SetModel( &quot;models/player/alyx.mdl&quot; ) -- you can only change colors on playermodels
function icon:LayoutEntity( Entity ) return end -- disables default rotation
function icon.Entity:GetPlayerColor() return Vector (1, 0, 0) end --we need to set it to a Vector not a Color, so the values are normal RGB values divided by 255.</code></pre>
