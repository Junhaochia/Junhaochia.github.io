<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Forces the associated player to select a weapon. This is used internally in the default HL2 weapon selection HUD.
</p><p>This may not work immediately if the current command is in prediction. Use <a href="/page/Category_colon_input" title="Category:input">input</a>.<a href="/page/input/SelectWeapon" title="input/SelectWeapon">SelectWeapon</a> to switch the weapon from the client when the next available command can do so.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is the ideal function to use to create a custom weapon selection HUD, as it allows prediction to run properly for <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a>:<a href="/page/WEAPON/Deploy" title="WEAPON/Deploy">Deploy</a> and <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PlayerSwitchWeapon" title="GM/PlayerSwitchWeapon">PlayerSwitchWeapon</a>
</p>
</td></tr></table>
