<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Removes all players that can see this PVS from the recipient filter.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> pos</span>
</p>
<div style="margin-left: 32px;">
Position that players may be able to see.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Adds the first player object to the recipient filter, then sends him a message.
</p>
<pre><code data-language="lua">function SendMessage()
	local filter = RecipientFilter()
	filter:AddPVS( Vector( 0,0,0 ) )
	filter:RemovePVS( Vector( 0,10,0 ) )
	umsg.Start( &quot;message&quot;, filter )
	umsg.End()
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Adds all players that can see the map's origin to the recipient filter, then removes all players who can see 10 units to the left of the origin, and sends the rest a message.</div>
