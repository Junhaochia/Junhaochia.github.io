<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Scales the vector by the given number, that means x, y and z are multiplied by that value.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> multiplier</span>
</p>
<div style="margin-left: 32px;">
The value to scale the vector with.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Scales a vector by 250.
</p>
<pre><code data-language="lua">a = Vector(1, 1, 1)
a:Mul(250)
print(a)</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">250 250 250</code></pre>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>If you don't want to set your vector to the result, and just return a new vector as the result. You can use a ' * ' operator to multiply a vector with a scalar.
</p>
<pre><code data-language="lua">a = Vector(1, 1, 1)
print(a*250)</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">250 250 250</code></pre>
