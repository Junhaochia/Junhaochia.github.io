<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns whenever the given vector is in a box created by the 2 other vectors.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Checks if player is within a certain area on the map.
</p>
<pre><code data-language="lua">-- Position to test, we get the position of first player on the server
local testPos = Entity( 1 ):GetPos()

-- Positions to test, in this case we test if the player is in spawn area of gm_construct
local pos1 = Vector( 1119, 895, 63 )
local pos2 = Vector( 656, -896, -144 )

-- This will return true if the player is within the tested area
print( testPos:WithinAABox( pos1, pos2 ) )</code></pre>
