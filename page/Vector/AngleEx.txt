<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the angle of the vector, but instead of assuming that up is <a href="/page/Global/Vector" title="Global/Vector">Vector</a>( 0, 0, 1 ) (Like <a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a>:<a href="/page/Vector/Angle" title="Vector/Angle">Angle</a> does) you can specify which direction is 'up' for the angle.
</p>

</p>

</div>

</div>
