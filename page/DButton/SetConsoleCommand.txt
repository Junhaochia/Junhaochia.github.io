<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets a console command to be called when the button is clicked.
</p><p>This overrides the button's <i>DoClick</i> method.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> command</span>
</p>
<div style="margin-left: 32px;">
The console command to be called.</div>
</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a button that makes the player say their name.
</p>
<pre><code data-language="lua">local button = vgui.Create( &quot;DButton&quot; )
button:SetSize( 100, 35 )
button:SetText( &quot;Say your nickname&quot; )
button:Center()
button:MakePopup()
button:SetConsoleCommand( &quot;say&quot;, LocalPlayer():Nick() )</code></pre>
