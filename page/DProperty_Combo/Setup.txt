<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets up a combo control.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> prop="Combo"</span>
</p>
<div style="margin-left: 32px;">
The name of DProperty sub control to add.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> data={ text = "Select..." }</span>
</p>
<div style="margin-left: 32px;">
<p>Data to use to set up the combo box control.
</p><p>Structure:
</p>
<ul><li> <a href="/page/Category_colon_string" title="Category:string">string</a> text - The default label for this combo box
</li><li> <a href="/page/Category_colon_table" title="Category:table">table</a> values - The values to add to the combo box</div>
</li></ul>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Setup a Combo control with a custom default text and two options.
</p>
<pre><code data-language="lua">local Combo = DP:CreateRow( &quot;Catergory&quot;, &quot;Hello World&quot; )
Combo:Setup( &quot;Combo&quot;, {
	text = &quot;Select me!&quot;,
	values = {
		[ &quot;Label 1&quot; ] = &quot;data 1&quot;,
		[ &quot;Label 2&quot; ] = 2,
	}
} )</code></pre>
