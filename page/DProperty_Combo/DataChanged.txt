<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Called after the user selects a new value.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_any" title="Category:any">any</a> data</span>
</p>
<div style="margin-left: 32px;">
The new data that was selected.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Click on the "Table" choice.
</p>
<pre><code data-language="lua">local choice = DP:CreateRow( &quot;Choices&quot;, &quot;Combo #2: Custom default text&quot; )
choice:Setup( &quot;Combo&quot;, &quot;Select type...&quot; )
choice:AddChoice( &quot;Table&quot;, {} )
choice:AddChoice( &quot;Function&quot;, function() end )
choice:AddChoice( &quot;String&quot;, &quot;Hello world&quot; )
choice.DataChanged = function( self, data )

   print( &quot;You selected: &quot;, data )

end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><code>You selected:	table: 0x8e05f3b8</code></div>
