<div style="float:right;font-size:10px; font-weight: 100; margin: 0px 0px 10px 10px;"><a href="/page/Special:WhatLinksHere/Structures/EntityCopyData" title="Special:WhatLinksHere/Structures/EntityCopyData">What links here?</a></div>Data structure used by the <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator library</a> to store and load entity data.
<p>It is created by <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/CopyEntTable" title="duplicator/CopyEntTable">CopyEntTable</a> and can be loaded by <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/CreateEntityFromTable" title="duplicator/CreateEntityFromTable">CreateEntityFromTable</a>.
</p><p>It also shows up in several other contexts:
</p>
<ul><li> <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/RegisterEntityClass" title="duplicator/RegisterEntityClass">RegisterEntityClass</a>
</li><li> <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/OnEntityCopyTableFinish" title="ENTITY/OnEntityCopyTableFinish">OnEntityCopyTableFinish</a>
</li><li> <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/OnDuplicated" title="ENTITY/OnDuplicated">OnDuplicated</a>
</li></ul>
<p>
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>In addition to all fields listed here, any field saved on the entity will be saved as-is to this table. For example, if you set ent.FavoriteFood = "Steak" before duplicating it, this structure will also contain the field FavoriteFood with the value "Steak".
</p>
</td></tr></table>
