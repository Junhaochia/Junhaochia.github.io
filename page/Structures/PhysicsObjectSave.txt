<div style="float:right;font-size:10px; font-weight: 100; margin: 0px 0px 10px 10px;"><a href="/page/Special:WhatLinksHere/Structures/PhysicsObjectSave" title="Special:WhatLinksHere/Structures/PhysicsObjectSave">What links here?</a></div>Structure used in storing/restoring physics object attributes.
<p><a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/GenericDuplicatorFunction" title="duplicator/GenericDuplicatorFunction">GenericDuplicatorFunction</a> and <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/DoGenericPhysics" title="duplicator/DoGenericPhysics">DoGenericPhysics</a> use a table of PhysicsObjectSaves with the struct's index in the table being the physics object ID.
</p><p>Pos and Angle are set by the following:
</p>
<pre><code data-language="lua">
Pos, Angle = WorldToLocal(phys:GetPos(), phys:GetAngle(), Vector( 0, 0, 0 ), Angle( 0, 0, 0 ))
</code></pre>
