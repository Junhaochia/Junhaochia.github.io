<td> <a href="/page/Category_colon_function" title="Category:function">function</a>
</td>
<td> Filter
</td>
<td>
<p>Used <b>clientside</b> to decide whether this property should be shown for an entity. Gets these arguments:
</p>
<ul><li> <a href="/page/Category_colon_table" title="Category:table">table</a> self - the property table
</li><li> <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent - the entity the player clicked
</li></ul>
<p>Return true if the property should be shown for this entity. It's good practice to call <a href="/page/Category_colon_SANDBOX_Hooks" title="Category:SANDBOX Hooks">SANDBOX</a>:<a href="/page/SANDBOX/CanProperty" title="SANDBOX/CanProperty">CanProperty</a> in this hook via <a href="/page/Category_colon_gamemode" title="Category:gamemode">gamemode</a>.<a href="/page/gamemode/Call" title="gamemode/Call">Call</a> or <a href="/page/Category_colon_hook" title="Category:hook">hook</a>.<a href="/page/hook/Run" title="hook/Run">Run</a>.
</p>
</td></tr>
<tr>
<td> <a href="/page/Category_colon_function" title="Category:function">function</a>
</td>
<td> Checked
</td>
<td>
<p>Required for toggle properties (clientside). Must return a boolean value to either check or uncheck the toggle property.
</p>
<ul><li> <a href="/page/Category_colon_table" title="Category:table">table</a> self - the property table
</li><li> <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent - the entity the player clicked
</li><li> <a href="/page/Category_colon_table" title="Category:table">table</a> tr - the player's eye trace
</li></ul>
</td></tr>
<tr>
<td> <a href="/page/Category_colon_function" title="Category:function">function</a>
</td>
<td> Action
</td>
<td>
<p>Called <b>clientside</b> when the property is clicked with these arguments:
</p>
<ul><li> <a href="/page/Category_colon_table" title="Category:table">table</a> self - the property table
</li><li> <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent - the entity the player clicked
</li><li> <a href="/page/Category_colon_table" title="Category:table">table</a> tr - the player's eye trace
</li></ul>
<p>When appropriate, within this function you can call <kbd>self:MsgStart()</kbd>, write data with the <a href="/page/Category_colon_net" title="Category:net">net library</a> 'Write' functions, and finish with <kbd>self:MsgEnd()</kbd>. This will activate the 'Receive' function on the server. In most cases, you will want to send the entity to the server, as it's not done by default.
</p>
</td></tr>
<tr>
<td> <a href="/page/Category_colon_function" title="Category:function">function</a>
</td>
<td> Receive
</td>
<td>
<p>Called <b>serverside</b> if the client sends a message in the 'Action' function (see above).
</p>
<ul><li> <a href="/page/Category_colon_table" title="Category:table">table</a> self - the property table
</li><li> <a href="/page/Category_colon_number" title="Category:number">number</a> len - the net message length, although this includes the property identifier used internally (the name of the property)
</li><li> <a href="/page/Category_colon_table" title="Category:table">table</a> ply - the player who clicked the property
</li></ul>
<p>You can read data received from the client with the <a href="/page/Category_colon_net" title="Category:net">net library</a> 'Read' functions. It's good practice to check <a href="/page/Category_colon_SANDBOX_Hooks" title="Category:SANDBOX Hooks">SANDBOX</a>:<a href="/page/SANDBOX/CanProperty" title="SANDBOX/CanProperty">CanProperty</a> here via <a href="/page/Category_colon_gamemode" title="Category:gamemode">gamemode</a>.<a href="/page/gamemode/Call" title="gamemode/Call">Call</a> or <a href="/page/Category_colon_hook" title="Category:hook">hook</a>.<a href="/page/hook/Run" title="hook/Run">Run</a>.
</p><p><b>Default:</b> nil
</p>
</td></tr>
<tr>
<td> <a href="/page/Category_colon_function" title="Category:function">function</a>
</td>
<td> MenuOpen
</td>
<td>
<p>Called <b>clientside</b> when the property option has been created in the right-click menu. This is not called for toggle properties!
</p>
<ul><li> <a href="/page/Category_colon_table" title="Category:table">table</a> self - the property table
</li><li> <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> option - the menu option
</li><li> <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent - the entity the player right-clicked
</li><li> <a href="/page/Category_colon_table" title="Category:table">table</a> tr - the player's eye trace
</li></ul>
<p><b>Default:</b> nil
</p>
</td></tr>

<td> OnCreate
</td>
<td>

</li><li> <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> menu - the property menu
</li><li> <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> option - the menu option
</li></ul>
<p><b>Default:</b> nil
</p>
</td></tr></table>
