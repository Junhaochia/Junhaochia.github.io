<td> SoundName
</td>
<td>
<p>The file path of the played sound.
</p><p>Sounds prefixed with ) tell the sound engine that it is a spatial sound; this allows the sound to emit from a specific location within the world.
</p><p>Sounds prefixed with ^ tell the sound engine that it is a distance based sound. The left channel of the .wav is the 'near' sound that will play when the sound originates close to you, and the right channel is the 'far' sound that will play when the sound originates far from you.
</p>
</td></tr>

<td>
<p>The original file path or soundscript name.
</p><p>Even if the SoundName value is changed, the OriginalSoundName value will remain the same.
</p><p>Sounds may be returned as soundscripts (for example, "Weapon_357.Single") instead of the sound's file path. You can use <a href="/page/Category_colon_sound" title="Category:sound">sound</a>.<a href="/page/sound/GetProperties" title="sound/GetProperties">GetProperties</a> to retrieve the actual file path.
</p>
