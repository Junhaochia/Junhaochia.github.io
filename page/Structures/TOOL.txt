<div style="float:right;font-size:10px; font-weight: 100; margin: 0px 0px 10px 10px;"><a href="/page/Special:WhatLinksHere/Structures/TOOL" title="Special:WhatLinksHere/Structures/TOOL">What links here?</a></div><div class="derivedonly"><span>This function or feature is only available in the Sandbox gamemode and its <a href="/page/Global/DeriveGamemode" title="Global/DeriveGamemode">derivatives</a>.</span></div>
<p>The TOOL table is used in Sandbox tool creation. You can find a list of callbacks on the <a href="/page/Category_colon_TOOL_Hooks" title="Category:TOOL Hooks">TOOL Hooks</a> page and a list of methods on the <a href="/page/Category_colon_Tool" title="Category:Tool">Tool Functions</a> page. Do note that some of the fields below have no effect on server-side operations.
</p><p>The tool information box drawn on the HUD while your tool is selected has 2 values that are set by <a href="/page/Category_colon_language" title="Category:language">language</a>.<a href="/page/language/Add" title="language/Add">Add</a>.<br />
"tool.[tool mode].name" - The tool name (Note this is NOT the same as TOOL.Name)<br />
"tool.[tool mode].desc" - The tool description
</p><p>Ensure that all tool file names are entirely lowercase. Including capital letters can lead to unintended behavior.
<br /><br />
</p>

<td> <a href="/page/Category_colon_function" title="Category:function">function</a>
</td>
<td> BuildCPanel
</td>
<td>
<p>The function that is called to build the context menu for your tool. It has one argument, namely the context menu's base panel to which all of your custom panels are going to be parented to.

</p>
