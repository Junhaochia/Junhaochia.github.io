<h1> <span class="mw-headline" id="DescriptionPanelHook">Description<span style="display: none;"><a href="/page/Form:PanelHook" title="Form:PanelHook">PanelHook</a></span></span></h1>
<p>Called when the color is changed after clicking a new value.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> newcol</span>
</p>
<div style="margin-left: 32px;">
The new color of the <a href="/page/Category_colon_DColorPalette" title="Category:DColorPalette">DColorPalette</a></div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DFrame with a color palette that prints values after being changed.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetSize( ScrW() - 100, ScrH() - 100 )
frame:Center()
frame:MakePopup()

local palette = vgui.Create( &quot;DColorPalette&quot;, frame )
palette:Dock( FILL )

function palette:OnValueChanged( newcol )
	PrintTable( newcol )
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">The color chosen on the palette.</div>
