<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Returns a fraction representing the current number selector value to number selector min/max range ratio. If argument <i>val</i> is supplied, that number will be computed instead.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> val</span>
</p>
<div style="margin-left: 32px;">
The fraction numerator.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints out some fractions based on a number selector with a min/max range of 0 to 255.
</p>
<pre><code data-language="lua">local numinput = vgui.Create(&quot;DNumberWang&quot;)
numinput:SetPos(5, 5)
numinput:SetSize(90, 20)

numinput:SetMinMax(0, 255)

numinput:SetValue(64)

print(numinput:GetFraction())	-- Should return ~0.25
print(numinput:GetFraction(128))	-- Should return ~0.5
print(numinput:GetFraction(192))	-- Should return ~0.75
print(numinput:GetFraction(255))	-- Should return 1</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">0.25098039215686
0.50196078431373
0.75294117647059
1</code></pre></div>
