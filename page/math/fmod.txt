<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the modulus of the specified values.
</p><p>While this is similar to the&#160;% operator, <b>it will return a negative value if the first argument is negative</b>, whereas the&#160;% operator will return a <i>positive</i> value <b>even if the first operand is negative</b>.
</p>
