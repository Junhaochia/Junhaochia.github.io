<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Converts an angle in degrees to it's equivalent in radians.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> degrees</span>
</p>
<div style="margin-left: 32px;">
The angle measured in degrees.</div>
</div>

<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
radians</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Convert various angles in degrees to their equivalent in radians.
</p>
<pre><code data-language="lua">print( &quot;Degrees: 360, Radians: &quot; .. math.rad( 360 ) ) -- 2*pi
print( &quot;Degrees: 180, Radians: &quot; .. math.rad( 180 ) ) -- pi
print( &quot;Degrees: 90, Radians: &quot; .. math.rad( 90 ) ) -- pi/2
print( &quot;Degrees: 1, Radians: &quot; .. math.rad( 1 ) ) -- pi/180</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><br />Degrees: 360, Radians: 6.2831853071796<br />
<p>Degrees: 180, Radians: 3.1415926535898<br />
Degrees: 90, Radians: 1.5707963267949<br />
</p>
Degrees: 1, Radians: 0.017453292519943</div>
