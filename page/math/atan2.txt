<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p><a href="https://en.wikipedia.org/wiki/atan2" class="extiw" title="wikipedia:atan2">atan2</a> functions like <a href="/page/Category_colon_math" title="Category:math">math</a>.<a href="/page/math/atan" title="math/atan">atan</a>(y / x), except it also takes into account the quadrant of the angle and so doesn't have a limited range of output.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>The Y argument comes first!
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>atan( 1 ) and atan2( 1, 1 ) are both math.pi / 4
</p><p>atan2( -1, -1 ) equals to ( (-3) * math.pi ) / 4
</p>
<pre><code data-language="lua">print( atan( 1 ) )
print( ata2( 1, 1 ) )
print( atan2( -1, -1 ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">0.7853981633974483
<p>0.7853981633974483
</p>
-2.356194490192345</div>
