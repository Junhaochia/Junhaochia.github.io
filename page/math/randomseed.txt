<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Seeds the random number generator. The same seed will guarantee the same sequence of numbers each time with <a href="/page/Category_colon_math" title="Category:math">math</a>.<a href="/page/math/random" title="math/random">random</a>.
</p><p>For shared random values across predicted realms, use <a href="/page/Category_colon_util" title="Category:util">util</a>.<a href="/page/util/SharedRandom" title="util/SharedRandom">SharedRandom</a>.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Incorrect usage of this function will affect <i>all</i> random numbers in the game.
</p>
</td></tr></table>
