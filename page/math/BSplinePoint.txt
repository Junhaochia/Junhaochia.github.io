<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Basic code for <a href="https://en.wikipedia.org/wiki/B%C3%A9zier_curve" class="extiw" title="wikipedia:Bézier curve">Bézier-Spline</a> algorithm.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> tDiff</span>
</p>
<div style="margin-left: 32px;">
From 0 to 1, where alongside the spline the point will be.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> tPoints</span>
</p>
<div style="margin-left: 32px;">
A table of <a href="/page/Category_colon_Vector" title="Category:Vector">Vectors</a>. The amount cannot be less than 4.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> tMax</span>
</p>
<div style="margin-left: 32px;">
Just leave this at 1.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a></span>
</p>
<div style="margin-left: 32px;">
Point on Bezier curve, related to tDiff.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage of the function, makes a black box moving along the beizer curve made out of 4 points
</p>
<pre><code data-language="lua">local points = { Vector( 100, 100, 0 ), Vector( 200, 200, 0 ), Vector( 300, 100, 0 ), Vector( 400, 200, 0 ) }
hook.Add( &quot;HUDPaint&quot;, &quot;BSplinePointExample&quot;, function()
	-- Draw the points
	for id, p in pairs( points ) do
		draw.RoundedBox( 0, p.x - 2, p.y - 2, 4, 4, color_white )
	end

	-- Draw the spline
	local pos = math.BSplinePoint( ( math.cos( CurTime() ) + 1 ) / 2, points, 1 )
	draw.RoundedBox( 0, pos.x - 2, pos.y - 2, 4, 4, Color( 0, 0, 0 ) )
end )</code></pre>
