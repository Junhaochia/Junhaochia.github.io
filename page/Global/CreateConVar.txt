<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a console variable (<a href="/page/Category_colon_ConVar" title="Category:ConVar">ConVar</a>), in general these are for things like gamemode/server settings.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> name</span>
</p>
<div style="margin-left: 32px;">
<p>Name of the convar.<br /><br />
</p>
This cannot be a name of an engine console command or console variable. It will silently fail if it is. If it is the same name as another lua ConVar, it will return that ConVar object.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> value</span>
</p>
<div style="margin-left: 32px;">
Default value of the convar. Can also be a number.</div>
</div>
