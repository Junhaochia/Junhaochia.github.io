<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets an integer that is shared between the server and all clients.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Running this function clientside will only set it clientside for the client it is called on!
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This function will not round decimal values as it actually networks a float internally.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3374">#3374</a></b>
</p>
</td></tr></table>
