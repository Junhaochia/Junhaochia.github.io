<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Convenience function that creates a DLabel, sets the text, and returns it
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> text</span>
</p>
<div style="margin-left: 32px;">
The string to set the label's text to</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> parent=nil</span>
</p>
<div style="margin-left: 32px;">
Optional. The panel to parent the DLabel to</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a></span>
</p>
<div style="margin-left: 32px;">
The created DLabel</div>
</div>

</p>
<pre><code data-language="lua">local lbl = Label( &quot;The quick brown fox&quot; )</code></pre>
