<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a <a href="/page/Structures/Color" title="Structures/Color">Color structure</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function creates a new table for the color. It's a good practice to localize your colors.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> r</span>
</p>
<div style="margin-left: 32px;">
An integer from 0-255 describing the red value of the color.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> g</span>
</p>
<div style="margin-left: 32px;">
An integer from 0-255 describing the green value of the color.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">

</p>
<div style="margin-left: 32px;">
An integer from 0-255 describing the blue value of the color.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>

An integer from 0-255 describing the alpha (transparency) of the color.</div>
</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a color and prints the components to the console.
</p>
<pre><code data-language="lua">PrintTable(Color(1, 2, 3, 4))</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">
a	=	4
b	=	3
g	=	2
r	=	1</code></pre></div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Color variables can have individual channels set using the arguments
</p>
<pre><code data-language="lua">local col = Color( 0, 255, 0, 255)

col.r = 255

PrintTable(col)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">
a	=	255
b	=	0
g	=	255
r	=	255</code></pre></div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_3">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_3">Example</span></h2>
<p>Transforms a color object to a string, then prints it.
</p>
<pre><code data-language="lua">local str = tostring( Color( 255, 0, 0 ) )
print( str )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">255 0 0</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_4">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_4">Example</span></h2>
<p>Prints "equal" if both colors are equal, otherwise "unequal" will be printed.
</p>
<pre><code data-language="lua">if Color( 255, 0, 0 ) == Color( 255, 0, 0 ) then
	print( &quot;equal&quot; )
else
	print( &quot;unequal&quot; )
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">equal</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_5">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_5">Example</span></h2>
<p>How to localize colors for hooks
</p>
<pre><code data-language="lua">local color1 = Color(100,100,100)
local color2 = Color(200,200,200)
hook.Add(&quot;HUDPaint&quot;,&quot;Example&quot;,function()
    draw.RoundedBox(4,100,100,100,40,color1)
    draw.RoundedBox(4,110,110,80,20,color2)
end)</code></pre>
