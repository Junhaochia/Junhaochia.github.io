<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Attempts to compile the given file. If successful, returns a function that can be called to perform the actual execution of the script.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> path</span>
</p>
<div style="margin-left: 32px;">
Path to the file, relative to the garrysmod/lua/ directory.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a></span>
</p>
<div style="margin-left: 32px;">
The function which executes the script.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Assuming our file is named example.lua and located in the garrysmod/lua/ directory, the following code would execute the script.
</p>
<pre><code data-language="lua">local example = CompileFile(&quot;example.lua&quot;)
example()</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Hello!</div>
