<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Callback function for when the client has joined a server. This function shows the server's loading URL by default.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> servername</span>
</p>
<div style="margin-left: 32px;">
Server's name.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> serverurl</span>
</p>
<div style="margin-left: 32px;">
Server's loading screen URL, or "" if the URL is not set.</div>
</div>

</p>

Server's current map's name.</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints GameDetails of the server you join to console, and preserves default behavior.
</p>
<pre><code data-language="lua">local OldGameDetails = GameDetails
function GameDetails( servername, serverurl, mapname, maxplayers, steamid, gamemode )
	print( 1, servername )
	print( 2, serverurl )
	print( 3, mapname )
	print( 4, maxplayers )
	print( 5, steamid )
	print( 6, gamemode )
	OldGameDetails( servername, serverurl, mapname, maxplayers, steamid, gamemode )
end</code></pre>
