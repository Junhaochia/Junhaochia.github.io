<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sends the specified Lua code to all connected clients and executes it.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> code</span>
</p>
<div style="margin-left: 32px;">
The code to be executed. Capped at length of 254 characters.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Print "Hello World!" in the clients' console
</p>
<pre><code data-language="lua">BroadcastLua( [[print( &quot;Hello World!&quot; )]] )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Hello World!</div>
