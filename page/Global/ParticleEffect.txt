<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a particle effect.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>The particle effect must be precached with <a href="/page/Global/PrecacheParticleSystem" title="Global/PrecacheParticleSystem">PrecacheParticleSystem</a> and the file its from must be added via <a href="/page/Category_colon_game" title="Category:game">game</a>.<a href="/page/game/AddParticles" title="game/AddParticles">AddParticles</a> before it can be used!
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> particleName</span>
</p>
<div style="margin-left: 32px;">
The name of the particle effect.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> position</span>
</p>
<div style="margin-left: 32px;">
The start position of the effect.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Angle" title="Category:Angle">Angle</a> angles</span>
</p>
<div style="margin-left: 32px;">
The orientation of the effect.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> parent=NULL</span>
</p>
<div style="margin-left: 32px;">
If set, the particle will be parented to the entity.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage of the function. Precaches <b>ExplosionCore_wall</b> particle from <b>particles/explosion.pcf</b>, a Team Fortress 2 particle file.
</p><p>You can then test the particle by using the <b>particleitup</b> console command.
</p><p>You can find a list of particles inside a .pcf file using the <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Particle_Editor">Particle Editor Tool</a>
</p>
<pre><code data-language="lua">game.AddParticles( &quot;particles/explosion.pcf&quot; )
PrecacheParticleSystem( &quot;ExplosionCore_wall&quot; )

if ( SERVER ) then
	-- A test console command to see if the particle works, spawns the particle where the player is looking at. 
	concommand.Add( &quot;particleitup&quot;, function( ply, cmd, args )
		local tr = ply:GetEyeTrace()
		ParticleEffect( &quot;ExplosionCore_wall&quot;, tr.HitPos, tr.HitNormal:Angle() )
	end )
end</code></pre>
