<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns an iterator function that can be used to loop through a table in order of its <b>values</b>.
</p><p>To sort by specific <b>value member</b>, use <a href="/page/Global/SortedPairsByMemberValue" title="Global/SortedPairsByMemberValue">SortedPairsByMemberValue</a>.<br />
To sort by <b>keys</b>, use <a href="/page/Global/SortedPairs" title="Global/SortedPairs">SortedPairs</a>.
</p>
