<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Used to select single values from a vararg or get the count of values in it.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_any" title="Category:any">any</a> parameter</span>
</p>
<div style="margin-left: 32px;">
<p>Can be a <a href="/page/Category_colon_number" title="Category:number">number</a> or <a href="/page/Category_colon_string" title="Category:string">string</a>.
</p>
<ul><li> If it's a string and starts with "#", the function will return the amount of values in the vararg (ignoring the rest of the string).
</li><li> If it's a positive number, the function will return all values starting from the given index.
</li><li> If the number is negative, it will return the amount specified from the end instead of the beginning. This mode will not be compiled by LuaJIT.</div>
</li></ul>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_vararg" title="Category:vararg">vararg</a> vararg</span>
</p>
<div style="margin-left: 32px;">
The vararg. These are the values from which you want to select.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_any" title="Category:any">any</a></span>
</p>
<div style="margin-left: 32px;">
Returns a <a href="/page/Category_colon_number" title="Category:number">number</a> or <a href="/page/Category_colon_vararg" title="Category:vararg">vararg</a>, depending on the select method.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This code shows how it works with the "#" modifier:
</p>
<pre><code data-language="lua">print( select( '#', 'a', true, false, {}, 1 ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">"5", which is the count of parameters passed excluding the modifier (the "#")</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>This prints from the 2nd vararg passed to the last
</p>
<pre><code data-language="lua">print( select( 2, 1, 2, 3, 4, 5 ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">"2 3 4 5" in the console</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_3">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_3">Example</span></h2>
<p>This prints the last 2 arguments passed
</p>
<pre><code data-language="lua">print( select( -2, 1, 2, 3, 4, 5 ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">"4 5" in the console</div>
