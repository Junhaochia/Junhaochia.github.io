<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Linear interpolation between two vectors. It is commonly used to smooth movement between two vectors.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">

</p>

Fraction ranging from 0 to 1</div>
</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Get the middle point (50%) between two vectors.
</p>
<pre><code data-language="lua">local output = LerpVector( 0.5, Vector( 0, 0, 100 ), Vector( 0, 0, 200 ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Vector( 0, 0, 150 )</div>
<p>If you do it each frame to smooth positions, you should couple it with FrameTime()
</p>
