<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Performs a linear interpolation from the start number to the end number.
</p><p>This function provides a very efficient and easy way to smooth out movements.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> t</span>
</p>
<div style="margin-left: 32px;">
The fraction for finding the result. This number is clamped between 0 and 1.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> from</span>
</p>
<div style="margin-left: 32px;">
The starting number. The result will be equal to this if delta is 0.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> to</span>
</p>
<div style="margin-left: 32px;">
The ending number. The result will be equal to this if delta is 1.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
The result of the linear interpolation, (1 - t) * from + t * to.</div>
</div>
