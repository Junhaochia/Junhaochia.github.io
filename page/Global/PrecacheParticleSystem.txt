<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Precaches the particle with the specified name.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> particleSystemName</span>
</p>
<div style="margin-left: 32px;">
The name of the particle system.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage of the function. Precaches "ExplosionCore_wall" particle from "particles/explosion.pcf", a Team Fortress 2 particle file.
</p><p>You can find a list of particles inside a .pcf file using the <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Particle_Editor">Particle Editor Tool</a>
</p>
<pre><code data-language="lua">game.AddParticles( &quot;particles/explosion.pcf&quot; )
PrecacheParticleSystem( &quot;ExplosionCore_wall&quot; )

if ( SERVER ) then
	-- A test console command to see if the particle works, spawns the particle where the player is looking at. 
	concommand.Add( &quot;particleitup&quot;, function( ply, cmd, args )
		local tr = ply:GetEyeTrace()
		ParticleEffect( &quot;ExplosionCore_wall&quot;, tr.HitPos, tr.HitNormal:Angle() )
	end )
end</code></pre>
