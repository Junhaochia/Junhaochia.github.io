<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns whether an object is valid or not. (Such as <a href="/page/Category_colon_Entity" title="Category:Entity">Entitys</a>, <a href="/page/Category_colon_Panel" title="Category:Panel">Panels</a>, custom <a href="/page/Category_colon_table" title="Category:table">table</a> objects and more)<br />Checks that an object is not nil, has an IsValid method and if this method returns true.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Due to vehicles being technically valid the moment they're spawned, also use <a href="/page/Category_colon_Vehicle" title="Category:Vehicle">Vehicle</a>:<a href="/page/Vehicle/IsValidVehicle" title="Vehicle/IsValidVehicle">IsValidVehicle</a> to make sure they're fully initialized
</p>
</td></tr></table>
