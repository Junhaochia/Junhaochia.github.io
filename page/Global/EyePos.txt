<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the origin of the current render context as calculated by <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/CalcView" title="GM/CalcView">CalcView</a>.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Ensuring EyePos returns the correct value outside of render hooks.
</p>
<pre><code data-language="lua">hook.Add(&quot;PreDrawTranslucentRenderables&quot;, &quot;FixEyePos&quot;, function() EyePos() end)
hook.Add( &quot;Think&quot;, &quot;Use Eyepos outside of render function&quot;, function()
	local start = EyePos()
	local dir = gui.ScreenToVector(gui.MousePos())
	local trace = util.TraceLine{
		start = start,
		endpos = start + ( dir * 10000 ),
		filter = { ply },
	}
	debugoverlay.Box( trace.HitPos, Vector( -5, -5, -5 ), Vector( 5, 5, 5 ) )
end )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Draws a white box in the world where you point the mouse to. Requires the <b>developer</b> convar to be set to 1.</div>
