<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the meta table for the class with the matching name.
</p><p>Internally returns <a href="/page/Category_colon_debug" title="Category:debug">debug</a>.<a href="/page/debug/getregistry" title="debug/getregistry">getregistry</a>()[metaName]
</p><p>You can learn more about meta tables on the <a href="/page/Meta_Tables" title="Meta Tables">Meta Tables</a> page.
</p><p>You can find a list of meta tables that can be retrieved with this function on <a href="/page/Enums/TYPE" title="Enums/TYPE">TYPE_ Enums</a>. The name in the description is the string to use with this function.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> metaName</span>
</p>
<div style="margin-left: 32px;">
The object type to retrieve the meta table of.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">

</p>
<div style="margin-left: 32px;">
The corresponding meta table.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Adds a very simple function for checking if a player is sick to the player metatable.
</p>
<pre><code data-language="lua">local meta = FindMetaTable(&quot;Player&quot;)

function meta:IsSick()
	return true
end

-- Sometime later...
local ply = Entity(1)
if ( ply:IsSick() ) then
	ply:ChatPrint( &quot;Get well soon, &quot; .. ply:Nick() .. &quot;!&quot; )
	ply:ChatPrint( &quot;I just don't understand how you're always sick...&quot; )
end</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">Get well soon, Player1!
I just don't understand how you're always sick...</code></pre>
