<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Runs a menu command. Equivalent to <a href="/page/Global/RunConsoleCommand" title="Global/RunConsoleCommand">RunConsoleCommand</a>( "gamemenucommand", command ) unless the command starts with the "engine" keyword in which case it is equivalent to <a href="/page/Global/RunConsoleCommand" title="Global/RunConsoleCommand">RunConsoleCommand</a>( command ).
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> command</span>
</p>
<div style="margin-left: 32px;">
<p>The menu command to run
</p><p>Should be one of the following:
</p>
<ul><li> Disconnect - Disconnects from the current server.
</li><li> OpenBenchmarkDialog - Opens the "Video Hardware Stress Test" dialog.
</li><li> OpenChangeGameDialog - Does not work in GMod.
</li><li> OpenCreateMultiplayerGameDialog - Opens the Source dialog for creating a listen server.
</li><li> OpenCustomMapsDialog - Does nothing.
</li><li> OpenFriendsDialog - Does nothing.
</li><li> OpenGameMenu - Does not work in GMod.
</li><li> OpenLoadCommentaryDialog - Opens the "Developer Commentary" selection dialog. Useless in GMod.
</li><li> OpenLoadDemoDialog - Does nothing.
</li><li> OpenLoadGameDialog - Opens the Source "Load Game" dialog.
</li><li> OpenNewGameDialog - Opens the "New Game" dialog. Useless in GMod.
</li><li> OpenOptionsDialog - Opens the options dialog.
</li><li> OpenPlayerListDialog - Opens the "Mute Players" dialog that shows all players connected to the server and allows to mute them.
</li><li> OpenSaveGameDialog - Opens the Source "Save Game" dialog.
</li><li> OpenServerBrowser - Opens the legacy server browser.
</li><li> Quit - Quits the game <i>without</i> confirmation (unlike other Source games).
</li><li> QuitNoConfirm - Quits the game without confirmation (like other Source games).
</li><li> ResumeGame - Closes the menu and returns to the game.
</li><li> engine &lt;concommand&gt; - Runs a console command. Equivalent to <a href="/page/Global/RunConsoleCommand" title="Global/RunConsoleCommand">RunConsoleCommand</a>( &lt;concommand&gt; ).</div>
</li></ul>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Opens the options dialog.
</p>
<pre><code data-language="lua">RunGameUICommand( &quot;OpenOptionsDialog&quot; )</code></pre>
