<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns an iterator function(<a href="/page/Global/next" title="Global/next">next</a>) for a for loop that will return the values of the specified table in an arbitrary order.
</p><p>For alphabetical <b>key</b> order use <a href="/page/Global/SortedPairs" title="Global/SortedPairs">SortedPairs</a>.<br />
For alphabetical <b>value</b> order use <a href="/page/Global/SortedPairsByValue" title="Global/SortedPairsByValue">SortedPairsByValue</a>.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Iterates through all players on the server and prints their names.
</p>
<pre><code data-language="lua">for k, v in pairs( player.GetAll() ) do
	print( v:Nick() )
end</code></pre>

</p>
<div style="margin:2em;">A list of players in console.</div>
