<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>This function can be used in a for loop instead of <a href="/page/Global/pairs" title="Global/pairs">pairs</a>. It sorts all <b>keys</b> alphabetically.
</p><p>For sorting by specific <b>value member</b>, use <a href="/page/Global/SortedPairsByMemberValue" title="Global/SortedPairsByMemberValue">SortedPairsByMemberValue</a>.<br />
For sorting by <b>value</b>, use <a href="/page/Global/SortedPairsByValue" title="Global/SortedPairsByValue">SortedPairsByValue</a>.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example of usage.
</p>
<pre><code data-language="lua">for id, text in SortedPairs( { &quot;e&quot;, &quot;b&quot;, &quot;d&quot;, &quot;c&quot;, &quot;a&quot; } ) do
    print(id, text)
end

print( &quot;---&quot; )

for id, text in SortedPairs( { e = 1, b = 2, d = 3, c = 4, a = 5 } ) do
    print(id, text)
end</code></pre>

</p>
<div style="margin:2em;"><pre><code data-language="lua">1 e
2 b
3 d
4 c
5 a
---
a 5
b 2
c 4
d 3
e 1</code></pre></div>
