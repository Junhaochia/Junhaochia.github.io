<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Just like <a href="/page/Global/Msg" title="Global/Msg">Msg</a>, except it can also print colored text, just like <a href="/page/Category_colon_chat" title="Category:chat">chat</a>.<a href="/page/chat/AddText" title="chat/AddText">AddText</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_vararg" title="Category:vararg">vararg</a> args</span>
</p>
<div style="margin-left: 32px;">
Values to print. If you put in a color, all text after that color will be printed in that color.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints "Hello World!" in red to the console.
</p>
<pre><code data-language="lua">MsgC( Color( 255, 0, 0 ), &quot;Hello World!&quot; )</code></pre>
