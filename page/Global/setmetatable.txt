<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets, changes or removes a table's metatable. Returns Tab (the first argument).
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a metatable and assigns it to a table.
</p>
<pre><code data-language="lua">local Pupil_meta = {
	GetName = function(self)
		return self.name
	end
}
Pupil_meta.__index = Pupil_meta
-- If a key cannot be found in an object, it will look in it's metatable's __index metamethod.

local Pupil = {
	name = &quot;John Doe&quot;
}

setmetatable(Pupil, Pupil_meta)

print( Pupil:GetName() )
-- This will look for the &quot;GetName&quot; key in Pupil, but it doesn't have one. So it will look in it's metatable (Pupil_meta) __index key instead.</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">"John Doe"</div>
