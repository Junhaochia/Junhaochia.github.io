<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns an iterator function that can be used to loop through a table in order of member values, when the values of the table are also tables and contain that member.
</p><p>To sort by <b>value</b>, use <a href="/page/Global/SortedPairsByValue" title="Global/SortedPairsByValue">SortedPairsByValue</a>.<br />
To sort by <b>keys</b>, use <a href="/page/Global/SortedPairs" title="Global/SortedPairs">SortedPairs</a>.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a table and prints its contents in order of the age member descending
</p>
<pre><code data-language="lua">local tab = {
	{
		Name = &quot;Adam&quot;,
		Age = 16
	},
	{
		Name = &quot;Charles&quot;,
		Age = 18
	}
}

for k, v in SortedPairsByMemberValue(tab, &quot;Age&quot;, true) do
	print(v.Name)
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">Charles
Adam</code></pre></div>
