<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Evaluates and executes the given code, will throw an error on failure.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Local variables are not passed to the given code.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> code</span>
</p>
<div style="margin-left: 32px;">
The code to execute.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> identifier="RunString"</span>
</p>
<div style="margin-left: 32px;">
The name that should appear in any error messages caused by this code.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">

</p>
<div style="margin-left: 32px;">
If false, this function will return a string containing any error messages instead of throwing an error.</div>
</div>
