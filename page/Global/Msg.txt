<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Writes every given argument to the console.
</p><p>Automatically attempts to convert each argument to a string. (See <a href="/page/Global/tostring" title="Global/tostring">tostring</a>)
</p><p>Unlike <a href="/page/Global/print" title="Global/print">print</a>, arguments are not separated by anything. They are simply concatenated.
</p><p>Additionally, a newline isn't added automatically to the end, so subsequent Msg or print operations will continue the same line of text in the console. See <a href="/page/Global/MsgN" title="Global/MsgN">MsgN</a> for a version that does add a newline.
</p><p>The text is blue on the server, orange on the client, and green on the menu: <a href="/page/File:msg_server_client_colors.png" class="image"><img alt="msg server client colors.png" src="/images/e/ed/msg_server_client_colors.png" width="37" height="17" /></a>
</p>
