<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Moves the given model to the given position and calculates appropriate camera parameters for rendering the model to an icon.
</p><p>The output table interacts nicely with <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/RebuildSpawnIconEx" title="Panel/RebuildSpawnIconEx">RebuildSpawnIconEx</a> with a few key renames.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> model</span>
</p>
<div style="margin-left: 32px;">
Model that is being rendered to the spawn icon</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> position</span>
</p>
<div style="margin-left: 32px;">
Position that the model is being rendered at</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> noAngles</span>
</p>
<div style="margin-left: 32px;">
If true the function won't reset the angles to 0 for the model.</div>
</div>

</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a clientside model and then PositionSpawnIcon is used to figure out the appropriate camera parameters for rendering this model.
</p>
<pre><code data-language="lua">local ent = ClientsideModel(&quot;models/props_wasteland/cafeteria_table001a.mdl&quot;, RENDERGROUP_BOTH)
local tab = PositionSpawnIcon(ent, vector_origin)
ent:Remove()

PrintTable(tab)</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">angles	=	25.000 220.000 0.000
fov	=	8.4436927451273
origin	=	1156.924316 970.773010 704.184265
zfar	=	1888.5692536466
znear	=	1</code></pre>
