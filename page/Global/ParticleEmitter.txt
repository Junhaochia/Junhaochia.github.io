<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a new <a href="/page/Category_colon_CLuaEmitter" title="Category:CLuaEmitter">CLuaEmitter</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Do not forget to delete the emitter with <a href="/page/Category_colon_CLuaEmitter" title="Category:CLuaEmitter">CLuaEmitter</a>:<a href="/page/CLuaEmitter/Finish" title="CLuaEmitter/Finish">Finish</a> once you are done with it
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> position</span>
</p>
<div style="margin-left: 32px;">
<p>The start position of the emitter.
</p>
This is only used to determine particle drawing order for translucent particles.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> use3D</span>
</p>
<div style="margin-left: 32px;">
Whenever to render the particles in 2D or 3D mode.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_CLuaEmitter" title="Category:CLuaEmitter">CLuaEmitter</a></span>
</p>
<div style="margin-left: 32px;">
The new particle emitter.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a simple spark particle effect 100 units above where the local player is looking at.
</p>
<pre><code data-language="lua">local tr =  LocalPlayer():GetEyeTrace()
local pos = tr.HitPos + tr.HitNormal * 100 -- The origin position of the effect

local emitter = ParticleEmitter( pos ) -- Particle emitter in this position

for i = 0, 100 do -- Do 100 particles
	local part = emitter:Add( &quot;effects/spark&quot;, pos ) -- Create a new particle at pos
	if ( part ) then
		part:SetDieTime( 1 ) -- How long the particle should &quot;live&quot;

		part:SetStartAlpha( 255 ) -- Starting alpha of the particle
		part:SetEndAlpha( 0 ) -- Particle size at the end if its lifetime

		part:SetStartSize( 5 ) -- Starting size
		part:SetEndSize( 0 ) -- Size when removed

		part:SetGravity( Vector( 0, 0, -250 ) ) -- Gravity of the particle
		part:SetVelocity( VectorRand() * 50 ) -- Initial velocity of the particle
	end
end

emitter:Finish()</code></pre>
