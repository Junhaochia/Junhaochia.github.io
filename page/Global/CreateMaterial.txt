<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a new material with the specified name and shader.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Materials created with this function can be used in <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetMaterial" title="Entity/SetMaterial">SetMaterial</a> and <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetSubMaterial" title="Entity/SetSubMaterial">SetSubMaterial</a> by prepending a "!" to their material name argument.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>.pngs must be loaded with <a href="/page/Global/Material" title="Global/Material">Material</a> before being used with this function.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/1531">#1531</a></b>
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This does not work with <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Patch">patch materials</a>.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/2511">#2511</a></b>
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This will not create a new material if another material object with the same name already exists.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3103">#3103</a></b>
</p>
</td></tr></table>

<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> materialData</span>
</p>
<div style="margin-left: 32px;">
<p>Key-value table that contains shader parameters and proxies.
</p>
<ul><li>See: <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Category:List_of_Shader_Parameters">List of Shader Parameters on Valve Developers Wiki</a> and each shader's page from <a href="/page/Category_colon_Shaders" title="Category:Shaders">Category: Shaders</a>.
</li></ul>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Unlike <a href="/page/Category_colon_IMaterial" title="Category:IMaterial">IMaterial</a>:<a href="/page/IMaterial/SetTexture" title="IMaterial/SetTexture">SetTexture</a>, this table will not accept <a href="/page/Category_colon_ITexture" title="Category:ITexture">ITexture</a> values. Instead, use the texture's name (see <a href="/page/Category_colon_ITexture" title="Category:ITexture">ITexture</a>:<a href="/page/ITexture/GetName" title="ITexture/GetName">GetName</a>).
</p>
</td></tr></table></div>
