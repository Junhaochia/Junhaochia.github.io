<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets the associated type ID of the variable. Unlike <a href="/page/Global/type" title="Global/type">type</a>, this does not work with <a href="/page/Category_colon_no_value" title="Category:no value">no value</a> - an argument must be provided.
</p>
