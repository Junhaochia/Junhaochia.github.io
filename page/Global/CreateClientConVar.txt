<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Makes a clientside-only console variable
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function is a wrapper of <a href="/page/Global/CreateConVar" title="Global/CreateConVar">CreateConVar</a>, with the difference being that FCVAR_ARCHIVE and FCVAR_USERINFO are added automatically when <b>shouldsave</b> and <b>userinfo</b> are true, respectively.
</p>
</td></tr></table>
<p>Although this function is shared, it should only be used clientside.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> name</span>
</p>
<div style="margin-left: 32px;">
<p>Name of the ConVar to be created and able to be accessed.<br /><br />
</p>
This cannot be a name of existing console command or console variable. It will silently fail if it is.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> default</span>
</p>
<div style="margin-left: 32px;">
Default value of the ConVar.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> shouldsave=true</span>
</p>
<div style="margin-left: 32px;">
Should the ConVar be saved across sessions</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> userinfo=false</span>
</p>
<div style="margin-left: 32px;">
Should the ConVar and its containing data be sent to the server when it has changed. This make the convar accessible from server using <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/GetInfoNum" title="Player/GetInfoNum">GetInfoNum</a> and similar functions.</div>
</div>
