<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a <a href="/page/Category_colon_DMenu" title="Category:DMenu">DMenu</a> and closes any current menus.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> parent</span>
</p>
<div style="margin-left: 32px;">
The panel to parent the created menu to.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a></span>
</p>
<div style="margin-left: 32px;">
The created <a href="/page/Category_colon_DMenu" title="Category:DMenu">DMenu</a></div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DMenu with buttons to commit suicide or close it.
</p>
<pre><code data-language="lua">local menu = DermaMenu() 
menu:AddOption(&quot;Die&quot;, function() RunConsoleCommand(&quot;kill&quot;) end)
menu:AddOption(&quot;Close&quot;, function() print(&quot;Close pressed&quot;) end) -- The menu will remove itself, we don't have to do anything.
menu:Open()</code></pre>
