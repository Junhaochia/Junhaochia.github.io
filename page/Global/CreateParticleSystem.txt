<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a new particle system.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>The particle effect must be precached with <a href="/page/Global/PrecacheParticleSystem" title="Global/PrecacheParticleSystem">PrecacheParticleSystem</a> and the file its from must be added via <a href="/page/Category_colon_game" title="Category:game">game</a>.<a href="/page/game/AddParticles" title="game/AddParticles">AddParticles</a> before it can be used!
</p>
</td></tr></table>
