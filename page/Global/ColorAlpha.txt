<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a new <a href="/page/Structures/Color" title="Structures/Color">Color structure</a> with the RGB components of the given <a href="/page/Structures/Color" title="Structures/Color">Color structure</a> and the alpha value specified.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> color</span>
</p>
<div style="margin-left: 32px;">
The <a href="/page/Structures/Color" title="Structures/Color">Color structure</a> from which to take RGB values. This color will not be modified.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> alpha</span>
</p>
<div style="margin-left: 32px;">
The new alpha value, a number between 0 and 255. Values above 255 will be clamped.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
The new <a href="/page/Structures/Color" title="Structures/Color">Color structure</a> with the modified alpha value</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<pre><code data-language="lua">local red = Color( 255, 0, 0, 255 )
local red2 = ColorAlpha( red, 125 )
print( red.r, red.g, red.b, red.a )
print( red2.r, red2.g, red2.b, red2.a )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">255	0	0	255
255	0	0	125</code></pre></div>
