<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Draws the bloom shader, which creates a glowing effect from bright objects.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> Darken</span>
</p>
<div style="margin-left: 32px;">
Determines how much to darken the effect. A lower number will make the glow come from lower light levels. A value of 1 will make the bloom effect unnoticeable. Negative values will make even pitch black areas glow.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> Multiply</span>
</p>
<div style="margin-left: 32px;">
Will affect how bright the glowing spots are. A value of 0 will make the bloom effect unnoticeable.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> SizeX</span>
</p>
<div style="margin-left: 32px;">
The size of the bloom effect along the horizontal axis.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> SizeY</span>
</p>
<div style="margin-left: 32px;">
The size of the bloom effect along the vertical axis.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_5">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> Passes</span>
</p>
<div style="margin-left: 32px;">
Determines how much to exaggerate the effect.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_6">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> ColorMultiply</span>
</p>
<div style="margin-left: 32px;">
Will multiply the colors of the glowing spots, making them more vivid.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_7">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> Red</span>
</p>
<div style="margin-left: 32px;">
How much red to multiply with the glowing color. Should be between 0 and 1</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_8">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> Green</span>
</p>
<div style="margin-left: 32px;">
How much green to multiply with the glowing color. Should be between 0 and 1</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_9">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> Blue</span>
</p>
<div style="margin-left: 32px;">
How much blue to multiply with the glowing color. Should be between 0 and 1</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Draws bloom effect with default settings.
</p>
<pre><code data-language="lua">function GM:RenderScreenspaceEffects()
	DrawBloom( 0.65, 2, 9, 9, 1, 1, 1, 1, 1 )
end</code></pre>
