<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Callback function for when the client's language changes. Called by the engine.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> lang</span>
</p>
<div style="margin-left: 32px;">
The new language code.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints the new language code whenever the language changes.
</p>
<pre><code data-language="lua">local OldLanguageChanged = LanguageChanged
function LanguageChanged( lang )
	print( &quot;New language: &quot; .. lang )
	OldLanguageChanged( lang )
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">New language: en</div>
