<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a new derma animation.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> name</span>
</p>
<div style="margin-left: 32px;">
Name of the animation to create</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> panel</span>
</p>
<div style="margin-left: 32px;">
Panel to run the animation on</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> func</span>
</p>
<div style="margin-left: 32px;">
<p>Function to call to process the animation<br />
Arguments:
</p>
<ul><li> <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> pnl - the panel passed to Derma_Anim
</li><li> <a href="/page/Category_colon_table" title="Category:table">table</a> anim - the anim table
</li><li> <a href="/page/Category_colon_number" title="Category:number">number</a> delta - the fraction of the progress through the animation
</li><li> <a href="/page/Category_colon_any" title="Category:any">any</a> data - optional data passed to the run metatable method</div>
</li></ul>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
<p>A lua metatable containing four methods:
</p>
<ul><li> Run() - Should be called each frame you want the animation to be ran.
</li><li> Active() - Returns if the animation is currently active (has not finished and stop has not been called)
</li><li> Stop() - Halts the animation at its current progress.
</li><li> Start( Length, Data ) - Prepares the animation to be ran for Length seconds. Must be called once before calling Run(). The data parameter will be passed to the func function.</div>
</li></ul>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Applies an <a rel="nofollow" class="external text" href="http://easings.net/#easeInQuad">easeInQuad</a> easing to the panel to make it glide naturally across the screen.
</p>
<pre><code data-language="lua">local function inQuad(fraction, beginning, change)
	return change * (fraction ^ 2) + beginning
end

local main = vgui.Create(&quot;DFrame&quot;)
main:SetTitle(&quot;Derma_Anim Example&quot;)
main:SetSize(250, 200)
main:SetPos(200)
main:MakePopup()
local anim = Derma_Anim(&quot;EaseInQuad&quot;, main, function(pnl, anim, delta, data)
	pnl:SetPos(inQuad(delta, 200, 600), 300) -- Change the X coordinate from 200 to 200+600
end)
anim:Start(2) -- Animate for two seconds
main.Think = function(self)
	if anim:Active() then
		anim:Run()
	end
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Panel naturally glides across the screen from 200 x to 800 x</div>
