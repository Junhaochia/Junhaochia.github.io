<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a number based on the Size argument and your screen's width. The screen's width is always equal to size 640. This function is primarily used for scaling font sizes.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> Size</span>
</p>
<div style="margin-left: 32px;">
The number you want to scale.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints a scaled number based on the number 96, 400 and 640.
</p>
<pre><code data-language="lua">print( ScreenScale(96) )
print( ScreenScale(400) )
print( ScreenScale(640) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">252, 1050, 1680 (This will differ depending on your screen width. Here the screen width is 1680.)</div>
