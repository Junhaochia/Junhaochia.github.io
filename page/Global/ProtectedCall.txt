<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Runs a function without stopping the whole script on error.
</p><p>This function is similar to <a href="/page/Global/pcall" title="Global/pcall">pcall</a> and <a href="/page/Global/xpcall" title="Global/xpcall">xpcall</a> except the errors are still printed and sent to the error handler (i.e. sent to server console if clientside and <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/OnLuaError" title="GM/OnLuaError">OnLuaError</a> called).
</p>
