<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Adds the specified vector to the PVS which is currently building. This allows all objects in visleafs visible from that vector to be drawn.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> position</span>
</p>
<div style="margin-left: 32px;">
The origin to add.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Adds an RTCamera's current position to all player's PVS, causing props near it to always render on an rtscreen
</p>
<pre><code data-language="lua">hook.Add(&quot;SetupPlayerVisibility&quot;, &quot;AddRTCamera&quot;, function(pPlayer, pViewEntity)
	-- Adds any view entity
	if (pViewEntity:IsValid()) then
		AddOriginToPVS(pViewEntity:GetPos())
	end
end)</code></pre>
