<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a particle effect with specialized parameters.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>The particle effect must be precached with <a href="/page/Global/PrecacheParticleSystem" title="Global/PrecacheParticleSystem">PrecacheParticleSystem</a> and the file its from must be added via <a href="/page/Category_colon_game" title="Category:game">game</a>.<a href="/page/game/AddParticles" title="game/AddParticles">AddParticles</a> before it can be used!
</p>
</td></tr></table>

</p>
<div style="margin-left: 32px;">
The name of the particle effect.</div>
</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>A console commands that attaches a "Rocket_Smoke" particle effect from "particles/rocket_fx.pcf" to the entity the player is looking at.
</p>
<pre><code data-language="lua">game.AddParticles( &quot;particles/rocket_fx.pcf&quot; )
PrecacheParticleSystem( &quot;Rocket_Smoke&quot; )

if ( SERVER ) then
	-- A test console command to see if the particle works, attaches the particle to the entity the player is looking at.
	concommand.Add( &quot;particle_attach&quot;, function( ply, cmd, args )
		local tr = ply:GetEyeTrace()
		ParticleEffectAttach( &quot;Rocket_Smoke&quot;, PATTACH_ABSORIGIN_FOLLOW, tr.Entity, 0 )
	end )
end</code></pre>
