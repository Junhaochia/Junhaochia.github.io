<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a sine value that fluctuates based on <a href="/page/Global/CurTime" title="Global/CurTime">CurTime</a>. The value returned will be between the start value plus/minus the range value.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>The range arguments don't work as intended. The existing (bugged) behavior is documented below.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>

<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> frequency</span>
</p>
<div style="margin-left: 32px;">
The frequency of fluctuation, in <a href="https://en.wikipedia.org/wiki/hertz" class="extiw" title="wikipedia:hertz">hertz</a></div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> origin</span>
</p>
<div style="margin-left: 32px;">
The center value of the sine wave.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> max</span>
</p>
<div style="margin-left: 32px;">
This argument's distance from origin defines the size of the full range of the sine wave. For example, if origin is 3 and max is 5, then the full range of the sine wave is 5-3 = 2. 3 is the center point of the sine wave, so the sine wave will range between 2 and 4.</div>
</div>

</p>
<div style="margin-left: 32px;">
Offset variable that doesn't affect the rate of change, but causes the returned value to be offset by time</div>
</div>
