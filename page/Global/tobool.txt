<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Attempts to return an appropriate boolean for the given value
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_any" title="Category:any">any</a> val</span>
</p>
<div style="margin-left: 32px;">
The object to be converted to a boolean</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
<b>false</b> for the boolean false.<br /><b>false</b> for "false".<br /><b>false</b> for "0".<br /><b>false</b> for numeric 0.<br /><b>false</b> for nil.<br /><br /><b>true</b> otherwise.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Demonstrate the output of this function with various values.
</p>
<pre><code data-language="lua">print(&quot;boolean true:&quot;, tobool(true))
print(&quot;boolean false:&quot;, tobool(false))
print(&quot;string true:&quot;, tobool(&quot;true&quot;))
print(&quot;string false:&quot;, tobool(&quot;false&quot;))
print(&quot;numeric 0:&quot;, tobool(0))
print(&quot;string 0:&quot;, tobool(&quot;0&quot;))
print(&quot;string 1:&quot;, tobool(&quot;1&quot;))
print(&quot;nil:&quot;, tobool(nil))
print(&quot;text string:&quot;, tobool(&quot;not a boolean&quot;))
print(&quot;empty string:&quot;, tobool(&quot;&quot;))</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><code>boolean true: true
boolean false: false
string true: true
string false: false
numeric 0: false
string 0: false
string 1: true
nil: false
text string: true
empty string: true</code></div>
