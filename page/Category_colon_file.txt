<p>The file library provides functions for finding, reading and writing to files. <br />
The following path values are most commonly used:
</p>
<ul><li> “LUA” searches the lua files (in /lua/, in your gamemodes, in all the addons).
</li><li> “GAME” searches all the mounted content (main folder, addons, mounted games etc).
</li><li> “MOD” searches only the garrysmod folder.
</li><li> “DATA” searches in the data folder.
</li></ul>
<p>For the full list of path values, type “path” in the console.
</p>
