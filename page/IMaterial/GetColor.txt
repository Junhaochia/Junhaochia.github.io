<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the color of the specified pixel of the $basetexture, only works for materials created from PNG files.
</p><p>Basically identical to <a href="/page/Category_colon_ITexture" title="Category:ITexture">ITexture</a>:<a href="/page/ITexture/GetColor" title="ITexture/GetColor">GetColor</a> used on <a href="/page/Category_colon_IMaterial" title="Category:IMaterial">IMaterial</a>:<a href="/page/IMaterial/GetTexture" title="IMaterial/GetTexture">GetTexture</a>( "$basetexture" ).
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>The returned color will not have the color metatable.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/2407">#2407</a></b>
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Identical functionality.
</p>
<pre><code data-language="lua">local m = Material( &quot;gui/colors_dark.png&quot; )
local t = m:GetTexture(&quot;$basetexture&quot;)

PrintTable( t:GetColor( 5, 5 ) )
PrintTable( m:GetColor( 5, 5 ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Both printouts will return identical color, which at the time of testing is RGBA - 255, 244, 242, 255.</div>
