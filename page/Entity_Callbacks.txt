</td>
<td>Called whenever the entity angles change.
<p>The arguments are:
</p>
<ul><li> <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent - Our entity with new angles
</li><li> <a href="/page/Category_colon_Angle" title="Category:Angle">Angle</a> ang - The new angles
</li></ul>
</td></tr>

<td>BuildBonePositions
</td>
<td>Called whenever the bones are being built.
<p>The arguments are:
</p>
<ul><li> <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent - Our entity
</li><li> <a href="/page/Category_colon_number" title="Category:number">number</a> numbones - The max amount of bones this entity has, same as calling <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetBoneCount" title="Entity/GetBoneCount">GetBoneCount</a>
</li></ul>
</td></tr>
<tr>
<td>PhysicsCollide
</td>
<td>Called when we collide with something else.
<p>The arguments are:
</p>
<ul><li> <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent - Our entity
</li><li> <a href="/page/Category_colon_table" title="Category:table">table</a> data - The collision data. See <a href="/page/Structures/CollisionData" title="Structures/CollisionData">CollisionData structure</a>
</li></ul>
</td></tr>
<tr>
<td>BuildFlexWeights
</td>
<td>Called whenever the flexes are being built.
<p>The arguments are:
</p>
<ul><li> <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent - Our entity
</li><li> <a href="/page/Category_colon_number" title="Category:number">number</a> numFlexes - Number of flexes in the table
</li><li> <a href="/page/Category_colon_table" title="Category:table">table</a> data - A number-indexed table of flex values starting at index 0
</li></ul>
<p>Return the edited <b>data</b> table to apply any changes.
</p>
</td></tr></table>
