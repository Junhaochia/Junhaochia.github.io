<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns whether a mouse key was initially pressed in the same frame this function was called.<br />
If <a href="/page/Category_colon_input" title="Category:input">input</a>.<a href="/page/input/WasMouseDoublePressed" title="input/WasMouseDoublePressed">WasMouseDoublePressed</a> returns true, this function will return false.
</p><p>This function only works in Move hooks, and will detect mouse events even in main menu or when a typing in a text field.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> key</span>
</p>
<div style="margin-left: 32px;">
The key, see <a href="/page/Enums/MOUSE" title="Enums/MOUSE">MOUSE_ Enums</a></div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
True if the mouse key was initially pressed the same frame that this function was called, false otherwise.</div>
</div>
