<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets the button name from a numeric button code. The name needs to be translated with <a href="/page/Category_colon_language" title="Category:language">language</a>.<a href="/page/language/GetPhrase" title="language/GetPhrase">GetPhrase</a> before being displayed.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Despite the name of the function, this also works for the full range of keys in <a href="/page/Enums/BUTTON_CODE" title="Enums/BUTTON CODE">BUTTON_CODE_ Enums</a>.
</p>
</td></tr></table>
