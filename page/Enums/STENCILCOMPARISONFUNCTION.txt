<p>Enumerations for use with <a href="/page/Category_colon_render" title="Category:render">render</a>.<a href="/page/render/SetStencilCompareFunction" title="render/SetStencilCompareFunction">SetStencilCompareFunction</a>.
</p><p>The comparison is between the reference value set by <a href="/page/Category_colon_render" title="Category:render">render</a>.<a href="/page/render/SetStencilReferenceValue" title="render/SetStencilReferenceValue">SetStencilReferenceValue</a>, and the value of each pixel in the stencil buffer.
</p><p>Clientside only.
