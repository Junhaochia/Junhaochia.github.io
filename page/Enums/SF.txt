<p>Enumerations describing certain spawnflags. Everything except for SF_PHYS* is serverside only.
</p><p>Spawnflags are set using <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetKeyValue" title="Entity/SetKeyValue">SetKeyValue</a> with <b>"spawnflags"</b> as the key.
</p>
<ul><li> SF_CITIZEN_* spawnflags represent spawnflags only usable on <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Npc_citizen">npc_citizen</a>.
</li><li> SF_NPC_* - Usable on all NPCs
</li><li> SF_PHYSBOX_* - Usable on <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Func_physbox">func_physbox</a>
</li><li> SF_PHYSPROP_* - Usable on <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Prop_physics">prop_physics</a> entities
</li></ul>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is not a full list of available spawnflags, there are <i>a lot</i> more, each unique to each entity, you can find out more on the <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Main_Page">Valve Developer Community</a> website for the entities in question.
</p>
</td></tr></table><br /><br />
