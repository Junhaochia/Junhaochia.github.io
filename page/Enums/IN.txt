<p>Unlike <a href="/page/Enums/BUTTON_CODE" title="Enums/BUTTON CODE">BUTTON_CODE_ Enums</a>, these enums are abstracted to allow the user to bind actions to any key they might prefer.
</p><p>Keybinds using these actions work with two console commands, one starting with a plus and one with a minus symbol. A key press or release will call either the plus or minus command, adding or removing the corresponding enum in the current <a href="/page/Category_colon_CUserCmd" title="Category:CUserCmd">CUserCmd</a>.
