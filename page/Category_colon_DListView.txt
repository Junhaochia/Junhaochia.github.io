<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DListView and populates it with two columns and three items, only one of which can be selected at a time.
</p><p>Selecting a row will print a console message containing the text of the row and its index.
</p>
<pre><code data-language="lua">local f = vgui.Create( &quot;DFrame&quot; )
f:SetSize( 500, 500 )
f:Center()
f:MakePopup()

local AppList = vgui.Create( &quot;DListView&quot;, f )
AppList:Dock( FILL )
AppList:SetMultiSelect( false )
AppList:AddColumn( &quot;Application&quot; )
AppList:AddColumn( &quot;Size&quot; )

AppList:AddLine( &quot;PesterChum&quot;, &quot;2mb&quot; )
AppList:AddLine( &quot;Lumitorch&quot;, &quot;512kb&quot; )
AppList:AddLine( &quot;Troj-on&quot;, &quot;661kb&quot; )

AppList.OnRowSelected = function( lst, index, pnl )
	print( &quot;Selected &quot; .. pnl:GetColumnText( 1 ) .. &quot; ( &quot; .. pnl:GetColumnText( 2 ) .. &quot; ) at index &quot; .. index )
end</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">Selected Troj-on ( 661kb ) at index 3
Selected Lumitorch ( 512kb ) at index 2</code></pre>
