<h1> <span class="mw-headline" id="File_Search_Paths"> File Search Paths </span></h1>
<p>Here is a list of file search paths for use by the <a href="/page/Category_colon_file" title="Category:file">file library</a>.
</p>
<table class="wikitable">
<caption> Search Paths
</caption>
<tr>
<th>Path
</th>
<th>Description
</th></tr>
<tr>
<td>"GAME"
</td>
<td>Structured like base folder (garrysmod/), searches all the mounted content (main folder, addons, mounted games, etc)
</td></tr>
<tr>
<td>"LUA"
</td>
<td>All Lua folders (lua/) including gamemodes and addons
</td></tr>
<tr>
<td>"lcl"
</td>
<td>All Lua files and subfolders (lua/) visible to the client <a href="/page/States" title="States">state</a>.
</td></tr>
<tr>
<td>"lsv"
</td>
<td>All Lua files and subfolders (lua/) visible to the server <a href="/page/States" title="States">state</a>.
</td></tr>
<tr>
<td>"LuaMenu"
</td>
<td>All Lua files and subfolders (lua/) visible to the menu <a href="/page/States" title="States">state</a>.
</td></tr>
<tr>
<td>"DATA"
</td>
<td>Data folder (garrysmod/data)
</td></tr>
<tr>
<td>"DOWNLOAD"
</td>
<td>Downloads folder (garrysmod/download/)
</td></tr>
<tr>
<td>"MOD"
</td>
<td>Strictly the game folder (garrysmod/), ignores mounting.
</td></tr>
<tr>
<td>"BASE_PATH"
</td>
<td>Location of the root folder (where hl2.exe, bin and etc. are)
</td></tr>
<tr>
<td>"EXECUTABLE_PATH"
</td>
<td>Bin folder and root folder
</td></tr>
<tr>
<td>"THIRDPARTY"
</td>
<td>Contains content from installed addons or gamemodes.
</td></tr>
<tr>
<td>"WORKSHOP"
</td>
<td>Contains only the content from workshop addons. This includes all mounted .gma files.
</td></tr>
<tr>
<td>&lt;mounted folder&gt;
</td>
<td>Strictly within the folder of a mounted game specified, e.g. "cstrike"
</td></tr>
<tr>
<td>&lt;mounted Workshop addon title&gt;
</td>
<td><div class="deprecatedfunc"><span>This feature is deprecated.</span><br />
You should avoid using it as it may be removed in a future version.</div> Strictly within the specified Workshop addon. See <a href="/page/Category_colon_engine" title="Category:engine">engine</a>.<a href="/page/engine/GetAddons" title="engine/GetAddons">GetAddons</a>.
</td></tr></table>
<p>For a more comprehensive list of paths accessible via the above Search Paths, plus additional paths such as "episodic", "hl2mp", "lostcoast", "tf", etc.. depending on mounted content and addons, use the console command: "path" which outputs a large list of all of the paths accessible for each id.
</p>
