<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Called when a line in the DListView is double clicked.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> lineID</span>
</p>
<div style="margin-left: 32px;">
The line number of the double clicked line.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> line</span>
</p>
<div style="margin-left: 32px;">
The double clicked <a href="/page/Category_colon_DListView_Line" title="Category:DListView Line">DListView_Line</a>.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Demonstrates the use of this function.
</p>
<pre><code data-language="lua">local DList = vgui.Create(&quot;DListView&quot;)
DList:SetPos(5,50)
DList:SetSize(150, 245)
DList:AddColumn(&quot;Player Names&quot;)

for k, v in pairs( player.GetAll() ) do
	DList:AddLine(v:Name())
end

function DList:DoDoubleClick( lineID, line )
	MsgN( &quot;Line &quot; .. lineID .. &quot; was double clicked!&quot; )
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">The double clicked line number is printed, e.g.<br /><br /><samp>Line 6 was double clicked!</samp></div>
