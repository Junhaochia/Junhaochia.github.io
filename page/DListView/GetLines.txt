The lines added to the DListView.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Loops through all of the lines of a DListView and prints their first value.
</p>
<pre><code data-language="lua">local list = vgui.Create( &quot;DListView&quot; )
list:AddColumn( &quot;Fruit&quot; )

local lines = { &quot;Apple&quot;, &quot;Orange&quot;, &quot;Banana&quot; }

for _, line in pairs( lines ) do
    list:AddLine( line )
end

for k, line in pairs( list:GetLines() ) do
    print( k, line:GetValue( 1 ) )
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">1 Apple
2 Orange
3 Banana</code></pre></div>
