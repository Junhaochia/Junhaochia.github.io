<h1> <span class="mw-headline" id="DescriptionPanelHook">Description<span style="display: none;"><a href="/page/Form:PanelHook" title="Form:PanelHook">PanelHook</a></span></span></h1>
<p>Called internally by <a href="/page/Category_colon_DListView" title="Category:DListView">DListView</a>:<a href="/page/DListView/OnClickLine" title="DListView/OnClickLine">OnClickLine</a> when a line is selected. This is the function you should override to define the behavior when a line is selected.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> rowIndex</span>
</p>
<div style="margin-left: 32px;">
The index of the row/line that the user clicked on.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> row</span>
</p>
<div style="margin-left: 32px;">
The <a href="/page/Category_colon_DListView_Line" title="Category:DListView Line">DListView_Line</a> that the user clicked on.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints the first two column values of the row that was clicked. In this example it's the nickname and kills of the player selected in the list
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetSize( 300, 300 )
frame:MakePopup()

local listView = frame:Add( &quot;DListView&quot; )
listView:Dock( FILL )

listView:AddColumn( &quot;Nick&quot; )
listView:AddColumn( &quot;Frags&quot; )

for _, v in pairs( player.GetAll() ) do
	listView:AddLine( v:Nick(), v:Frags() )
end
listView.OnRowSelected = function( panel, rowIndex, row )
	print( row:GetValue( 1 ) )
	print( row:GetValue( 2 ) )
end</code></pre>
