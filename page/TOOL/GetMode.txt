<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>The Toolgun weapon has a similar function, checking weapon class is strongly recommended as GetMode() is not available on all weapons.
</p>
<pre><code data-language="lua">local wep = Entity( 1 ):GetActiveWeapon()
if ( IsValid( wep ) &amp;&amp; wep:GetClass() == &quot;gmod_tool&quot; ) then
	print( wep:GetMode() )
end</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">remover</code></pre>
