<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Checks whether the tool is allowed on the server. This will always return true clientside and will not be affected by <a href="/page/Category_colon_SANDBOX_Hooks" title="Category:SANDBOX Hooks">SANDBOX</a>:<a href="/page/SANDBOX/CanTool" title="SANDBOX/CanTool">CanTool</a>.
</p><p>This function uses <b>TOOL.AllowedCVar</b> which is a <a href="/page/Category_colon_ConVar" title="Category:ConVar">ConVar</a> object pointing to <b>toolmode_allow_<i>*toolname*</i></b> convar.
</p>
