<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called after the default tool screen has been drawn from <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a>:<a href="/page/WEAPON/RenderScreen" title="WEAPON/RenderScreen">RenderScreen</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>If this method exists on the TOOL object table, the default scrolling text will not be drawn
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Materials rendered in this hook require $ignorez parameter to draw properly.
</p>
</td></tr></table>

<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> width</span>

The height of the tool's screen in pixels.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>White text that says "Hello world!" on a black background.
</p>
<pre><code data-language="lua">function TOOL:DrawToolScreen( width, height )
	-- Draw black background
	surface.SetDrawColor( Color( 20, 20, 20 ) )
	surface.DrawRect( 0, 0, width, height )
	
	-- Draw white text in middle
	draw.SimpleText( &quot;Hello world!&quot;, &quot;DermaLarge&quot;, width / 2, height / 2, Color( 200, 200, 200 ), TEXT_ALIGN_CENTER, TEXT_ALIGN_CENTER )
end</code></pre>
