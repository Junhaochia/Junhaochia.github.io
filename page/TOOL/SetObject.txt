<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Stores an <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> for later use in the tool.
</p><p>The stored values can be retrieved by <a href="/page/Category_colon_Tool" title="Category:Tool">Tool</a>:<a href="/page/Tool/GetEnt" title="Tool/GetEnt">GetEnt</a>, <a href="/page/Category_colon_Tool" title="Category:Tool">Tool</a>:<a href="/page/Tool/GetPos" title="Tool/GetPos">GetPos</a>, <a href="/page/Category_colon_Tool" title="Category:Tool">Tool</a>:<a href="/page/Tool/GetLocalPos" title="Tool/GetLocalPos">GetLocalPos</a>, <a href="/page/Category_colon_Tool" title="Category:Tool">Tool</a>:<a href="/page/Tool/GetPhys" title="Tool/GetPhys">GetPhys</a>, <a href="/page/Category_colon_Tool" title="Category:Tool">Tool</a>:<a href="/page/Tool/GetBone" title="Tool/GetBone">GetBone</a> and <a href="/page/Category_colon_Tool" title="Category:Tool">Tool</a>:<a href="/page/Tool/GetNormal" title="Tool/GetNormal">GetNormal</a>
</p>
