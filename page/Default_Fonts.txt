<p>This page describes a list of fonts available for use by default on all clients.
</p><p>You can create new custom fonts using <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/CreateFont" title="surface/CreateFont">CreateFont</a>. Note that you cannot use the fonts in this page directly as font names for <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/CreateFont" title="surface/CreateFont">CreateFont</a>! You must use the font resource name instead, as explained in: <a href="/page/Structures/FontData" title="Structures/FontData">FontData structure</a>.
</p>
