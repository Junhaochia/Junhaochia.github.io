<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the color value of the property.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> color</span>
</p>
<div style="margin-left: 32px;">
Sets the color to use in a <a href="/page/Category_colon_DProperty_VectorColor" title="Category:DProperty VectorColor">DProperty_VectorColor</a>.</div>
</div>

<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>

</p>
<pre><code data-language="lua">local color = DP:CreateRow( &quot;Category&quot;, &quot;Select Color&quot; )
color:Setup( &quot;VectorColor&quot;, {} )
color:SetValue( Vector( 0.39, 1, 1 ) )</code></pre>
