<p>Garry's Mod adds a bunch of additional console commands and variables alongside with the Source Engine default ones.
</p>
<h2> <span class="mw-headline" id="Lua_related_console_variables">Lua related console variables</span></h2>
<table class="wikitable">
<tr>
<th>Command
</th>
<th>Default
</th>
<th>Description
</th></tr>
<tr>
<td>lua_error_url
</td>
<td>""
</td>
<td>Allows to override where to send all occurred Lua errors. If not set, it will send them to <a rel="nofollow" class="external text" href="http://errors.garrysmod.com">Garry's Mod Errors</a> website.
</td></tr>
<tr>
<td>sv_allowcslua
</td>
<td>1
</td>
<td>Allow clients on the server to run lua_openscript_cl and lua_run_cl.
</td></tr>
<tr>
<td>sv_kickerrornum
</td>
<td>0
</td>
<td>If a client's amount of clientside errors reaches this value, the client is automatically disconnected from the server. A value of 0 means disabling this functionality.
</td></tr></table>
