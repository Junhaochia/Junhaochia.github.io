<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a clientside only prop. See also <a href="/page/Global/ClientsideModel" title="Global/ClientsideModel">ClientsideModel</a>.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>Clientside entities are not garbage-collected, thus you must store a reference to the object and call <a href="/page/Category_colon_CSEnt" title="Category:CSEnt">CSEnt</a>:<a href="/page/CSEnt/Remove" title="CSEnt/Remove">Remove</a> manually.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/1387">#1387</a></b>
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>Projected textures do not work on clientside props under specific conditions (see <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3268#issuecomment-324821962">this explanation</a>).
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3268">#3268</a></b>
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a clientside prop at the player location.
</p>
<pre><code data-language="lua">function GhostBarrel( ply )
	local c_Model = ents.CreateClientProp()
	c_Model:SetPos( ply:GetPos() )
	c_Model:SetModel( &quot;models/props_borealis/bluebarrel001.mdl&quot; )
	c_Model:SetParent( ply )
	c_Model:Spawn()
end</code></pre>
