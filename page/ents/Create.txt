<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates an entity. This function will fail and return NULL if the networked-edict limit is hit (around 8176), or the provided entity class doesn't exist.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Do not use before <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/InitPostEntity" title="GM/InitPostEntity">InitPostEntity</a> has been called, otherwise the server will crash!<br />If you need to perform entity creation when the game starts, create a hook for <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/InitPostEntity" title="GM/InitPostEntity">InitPostEntity</a> and do it there
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a gmod_button entity near 0,0,0
</p>
<pre><code data-language="lua">local button = ents.Create( &quot;gmod_button&quot; )
if (&#160;!IsValid( button ) ) then return end // Check whether we successfully made an entity, if not - bail
button:SetModel( &quot;models/dav0r/buttons/button.mdl&quot; )
button:SetPos( Vector( 0, 0, 100 ) )
button:Spawn()</code></pre>
