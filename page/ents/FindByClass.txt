<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets all entities with the given class, supports wildcards. This works internally by iterating over <a href="/page/Category_colon_ents" title="Category:ents">ents</a>.<a href="/page/ents/GetAll" title="ents/GetAll">GetAll</a>. Even if internally ents.GetAll is used, It is faster to use ents.FindByClass than ents.GetAll with a single class comparison.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Asterisks (*) are the only wildcard supported.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints the location of every prop on the map.
</p>
<pre><code data-language="lua">for k, v in pairs( ents.FindByClass( &quot;prop_*&quot; ) ) do
	print( v:GetPos() )
end</code></pre>
