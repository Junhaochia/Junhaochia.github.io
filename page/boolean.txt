<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Demonstrates how conditional statements treat booleans.
</p>
<pre>if false then print(&quot;Value is false!&quot;) end
if true then print(&quot;Value is true!&quot;) end</pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Value is true!</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Inverting a boolean is often very useful. The easiest way to do that is to use the "not" operator.
</p>
<pre>local bool = false
if bool then
	print(&quot;Boolean is false, this code is not run!&quot;)
end

// Boolean is now being inverted ( false -&gt; true )
bool = not bool

if bool then 
	print(&quot;Boolean has been inverted and is now true!&quot;)
end

// This also works backwards. Note how &quot;!&quot; can be used in place of &quot;not&quot;
bool =&#160;!bool

if bool then 
	print(&quot;Boolean is false again, and won't be printed!&quot;)
end</pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Boolean has been inverted and is now true!</div>
