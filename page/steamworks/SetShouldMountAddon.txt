<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets if an addon should be enabled or disabled. Call <a href="/page/Category_colon_steamworks" title="Category:steamworks">steamworks</a>.<a href="/page/steamworks/ApplyAddons" title="steamworks/ApplyAddons">ApplyAddons</a> afterwards to update.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> workshopItemID</span>
</p>
<div style="margin-left: 32px;">
The ID of the Steam Workshop item we should enable/disable</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> shouldMount</span>
</p>
<div style="margin-left: 32px;">
true to enable the item, false to disable.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Enables the Gm_construct_Beta Steam Workshop addon and reloads addons afterwards
</p>
<pre><code data-language="lua">steamworks.SetShouldMountAddon( 21197, true )
steamworks.ApplyAddons()</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Enabled the gm_construct_beta addon, if installed.</div>
