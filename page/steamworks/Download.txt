<h1> <span class="mw-headline" id="Description">Description</span></h1>
<div class="deprecatedfunc"><span>This feature is deprecated.</span><br />
You should avoid using it as it may be removed in a future version.<br /><br />Use <a href="/page/Category_colon_steamworks" title="Category:steamworks">steamworks</a>.<a href="/page/steamworks/DownloadUGC" title="steamworks/DownloadUGC">DownloadUGC</a> instead to download addons/content to use with <a href="/page/Category_colon_game" title="Category:game">game</a>.<a href="/page/game/MountGMA" title="game/MountGMA">MountGMA</a>.</div>
<p>Downloads a file from the supplied addon and saves it as a .cache file in garrysmod/cache folder.
</p><p>This is mostly used to download the preview image of the addon, but the game seems to also use it to download replays and saves.
</p><p>In case the retrieved file is an image and you need the <a href="/page/Category_colon_IMaterial" title="Category:IMaterial">IMaterial</a>, use <a href="/page/Global/AddonMaterial" title="Global/AddonMaterial">AddonMaterial</a> with the path supplied from the callback.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> workshopPreviewID</span>
</p>
<div style="margin-left: 32px;">
The Preview ID of workshop item.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> uncompress</span>
</p>
<div style="margin-left: 32px;">
<p>Whether to uncompress the file or not, assuming it was compressed with LZMA.<br />
</p>
You will usually want to set this to true.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> resultCallback</span>
</p>
<div style="margin-left: 32px;">
The function to process retrieved data. The first and only argument is a string, containing path to the saved file.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Downloads and saves icon of Gm_construct_Beta Steam Workshop addon.
</p>
<pre><code data-language="lua">steamworks.FileInfo( 21197, function( result )
	steamworks.Download( result.previewid, true, function( name )
		print( name )
	end) 
end)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Something like this will be printed into console: cache/559813303754221947.cache</div>
