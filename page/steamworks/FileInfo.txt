<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Retrieves info about supplied Steam Workshop addon.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> workshopItemID</span>
</p>
<div style="margin-left: 32px;">
The ID of Steam Workshop item.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> resultCallback</span>
</p>
<div style="margin-left: 32px;">
<p>The function to process retrieved data, with the following arguments:
</p>
<ul><li> <a href="/page/Category_colon_table" title="Category:table">table</a> data - The data about the item, if the request succeeded, nil otherwise. See <a href="/page/Structures/UGCFileInfo" title="Structures/UGCFileInfo">UGCFileInfo structure</a>.</div>
</li></ul>
</div>
