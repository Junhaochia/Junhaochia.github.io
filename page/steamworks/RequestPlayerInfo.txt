<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Requests information of the player with SteamID64 for later use with <a href="/page/Category_colon_steamworks" title="Category:steamworks">steamworks</a>.<a href="/page/steamworks/GetPlayerName" title="steamworks/GetPlayerName">GetPlayerName</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> steamID64</span>
</p>
<div style="margin-left: 32px;">
The 64bit Steam ID of player.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> callback</span>
</p>
<div style="margin-left: 32px;">
A callback function with only 1 argument - <a href="/page/Category_colon_string" title="Category:string">string</a> name.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Gets and prints the steam name of the local player
</p>
<pre><code data-language="lua">steamworks.RequestPlayerInfo( LocalPlayer():SteamID64(), function(steamName)
		print( steamName )
end )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">The local player's name</div>
