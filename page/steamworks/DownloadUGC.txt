<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Downloads a Steam Workshop file by its ID and returns a path to it.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> workshopID</span>
</p>
<div style="margin-left: 32px;">
The ID of workshop item to download. <b>NOT</b> a file ID.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> resultCallback</span>
</p>
<div style="margin-left: 32px;">
The function to process retrieved data. The first and only argument is a string, containing path to the saved file, or nil if the download failed to any reason.</div>
</div>
