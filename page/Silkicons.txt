<p>In Garry's Mod 13, all silkicons are included in the game. They can be found in materials/icon16/*.png
</p><p>A full list of all silkicons which can be used in the game can be <a rel="nofollow" class="external text" href="http://www.famfamfam.com/lab/icons/silk/previews/index_abc.png">viewed here</a> or with search feature <a rel="nofollow" class="external text" href="http://www.famfamfam.com/lab/icons/silk/preview.php">here</a>
</p><p>Also included are flag icons. They can be found in materials/flags16/*.png
</p><p>The flag icon filenames follow the <a rel="nofollow" class="external text" href="https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2">ISO 3166-1 alpha-2 country codes</a> and include all flags
</p>
<h1> <span class="mw-headline" id="Example_of_using_silkicons">Example of using silkicons</span></h1>
<p>This line of code makes a DermaMenu option have a silkicon logo on its left side.
</p>
<pre><code data-language="lua">MENU:AddOption("Say hi", function() RunConsoleCommand("say", "hi") end ):SetImage( "icon16/user.png" )
</code></pre>
<h1> <span class="mw-headline" id="Rendering_silkicons">Rendering silkicons</span></h1>
<p>You no longer can use <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/GetTextureID" title="surface/GetTextureID">GetTextureID</a> and <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/SetTexture" title="surface/SetTexture">SetTexture</a> with silkicons, because they are now .png files. Now, to draw them, you must use <a href="/page/Global/Material" title="Global/Material">Material</a> and
<a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/SetMaterial" title="surface/SetMaterial">SetMaterial</a>. Since <a href="/page/Global/Material" title="Global/Material">Material</a> is slower than <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/GetTextureID" title="surface/GetTextureID">GetTextureID</a>, it is not recommended to call it in a drawing hook, which are ran every frame, as that will cause lag. Instead, save the material as a variable outside of the draw hook.
</p>
