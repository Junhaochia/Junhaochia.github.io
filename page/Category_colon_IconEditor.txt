<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<p>An icon editor that permits a user to modify a <a href="/page/Category_colon_SpawnIcon" title="Category:SpawnIcon">SpawnIcon</a> and re-render it. This is used by the spawn menu and is what is shown when you right-click an icon and select <i>Edit Icon</i>.
</p><p>This makes use of the <a href="/page/Category_colon_DAdjustableModelPanel" title="Category:DAdjustableModelPanel">DAdjustableModelPanel</a> element.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This panel is only available in Sandbox and Sandbox derived gamemodes!
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates an SpawnIcon with model <tt>"models/props_borealis/bluebarrel001.mdl"</tt> and IconEditor to modify it.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; ) -- Container for the SpawnIcon
frame:SetPos( 200, 200 )
frame:SetSize( 200, 200 )
frame:SetTitle( &quot;Icon Editor Example&quot; )
frame:MakePopup()

local icon = vgui.Create( &quot;SpawnIcon&quot; , frame ) -- SpawnIcon, with blue barrel model
icon:Center()
-- It is important below to include the SkinID (0 = default skin); the IconEditor will not work otherwise
icon:SetModel( &quot;models/props_borealis/bluebarrel001.mdl&quot;, 0 )

local editor = vgui.Create( &quot;IconEditor&quot; ) -- Create IconEditor
editor:SetIcon( icon ) -- Set the SpawnIcon to modify
editor:Refresh() -- Sets up the internal DAdjustableModelPanel and SpawnIcon
editor:MakePopup()
editor:Center()</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">A <a href="/page/Category_colon_DFrame" title="Category:DFrame">DFrame</a> containing a <a href="/page/Category_colon_SpawnIcon" title="Category:SpawnIcon">SpawnIcon</a>, and a window identical to that in the <a href="#Preview">preview</a> above.</div>
