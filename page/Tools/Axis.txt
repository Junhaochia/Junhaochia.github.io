<h1> <span class="mw-headline" id="Axis_Tool"> Axis Tool </span></h1>
<h2> <span class="mw-headline" id="Description"> Description </span></h2>
<p>This tool makes an object rotate around a point, in a circle.
</p>
<h2> <span class="mw-headline" id="Usage"> Usage </span></h2>
<ol><li> Spawn the props you want to use. <div class="center"><div class="floatnone"><a href="/page/File:AxisTool_1.png" class="image"><img alt="AxisTool 1.png" src="/images/1/19/AxisTool_1.png" width="423" height="237" /></a></div></div>
</li><li> Click on one prop, and then the other. <div class="center"><div class="floatnone"><a href="/page/File:AxisTool_2.png" class="image"><img alt="AxisTool 2.png" src="/images/7/71/AxisTool_2.png" width="476" height="246" /></a></div></div>
</li><li> Click again to finish! <div class="center"><div class="floatnone"><a href="/page/File:AxisTool_3.png" class="image"><img alt="AxisTool 3.png" src="/images/5/55/AxisTool_3.png" width="409" height="264" /></a></div></div>
</li></ol>
<h2> <span class="mw-headline" id="Settings"> Settings </span></h2>
<div class="floatright"><a href="/page/File:AxisToolB.png" class="image"><img alt="AxisToolB.png" src="/images/4/4b/AxisToolB.png" width="316" height="368" /></a></div>
<p><b>Force Limit:</b> How much force this constraint can take before breaking. Useful for contraptions that can fall apart. Set to 0 to never break.
</p><p><b>Torque Limit:</b> How much torque this constraint can take before breaking. Set to 0 to never break.
</p><p><b>Hinge Friction:</b> How much friction is in the hinge. The more friction, the more the prop will slow down when spun.
</p><p><b>No Collide:</b> Tick to stop the two props from colliding.
</p>
