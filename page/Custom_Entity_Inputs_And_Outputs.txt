<h2> <span class="mw-headline" id="Description"> Description </span></h2>
<p>This page describes Garry's Mod specific Hammer inputs and outputs for built-in Source Engine entities.<br />
Some of the features are ported from newer Source Engine versions.
</p><p>These inputs can be fired from Lua by the use of <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/Fire" title="Entity/Fire">Fire</a> and <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/Input" title="Entity/Input">Input</a>.<br />
Key-values and spawnflags can be set via <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetKeyValue" title="Entity/SetKeyValue">SetKeyValue</a>.
</p>
<h2> <span class="mw-headline" id="Entities"> Entities </span></h2>
<h3> <span class="mw-headline" id="All_NPCs"> All NPCs </span></h3>
<ul><li> Inputs
<ul><li> <a href="/page/Category_colon_number" title="Category:number">number</a> SetMaxLookDistance - Sets the maximum distance the NPC can see at. Default is 2048, 6000 for long range NPCs.
</li></ul>
</li></ul>
<h3> <span class="mw-headline" id="prop_dynamic"> prop_dynamic </span></h3>
<ul><li> Key-values
<ul><li> <a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> HoldAnimation - If set, the prop will not loop its animation, but hold the last frame. (Ported from CS:GO)
</li><li> <a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> AnimateEveryFrame - Force this prop to animate every frame. Usually this doesn't need to be touched. (Ported from CS:GO)
</li></ul>
</li><li> Inputs
<ul><li> <a href="/page/Category_colon_string" title="Category:string">string</a> SetAnimationNoReset - Force the prop to play an animation unless the prop is already playing the animation. The parameter should be the name of the animation.(Ported from CS:GO)
</li><li> <a href="/page/Category_colon_nil" title="Category:nil">nil</a> BecomeRagdoll - Kills the entity and creates a client-side ragdoll from the model if supported.(Ported from CS:GO)
</li><li> <a href="/page/Category_colon_nil" title="Category:nil">nil</a> FadeAndKill - Fade out and remove itself(Ported from CS:GO)
</li></ul>
</li></ul>
<h3> <span class="mw-headline" id="beam_spotlight_.28Entity_ported_from_CS:GO.29"> beam_spotlight (Entity ported from CS:GO) </span></h3>
<ul><li> Inputs
<ul><li> <a href="/page/Category_colon_Color" title="Category:Color">Color</a> SetColor - Sets the color of the beam effect
</li></ul>
</li></ul>
<h3> <span class="mw-headline" id="point_spotlight"> point_spotlight </span></h3>
<ul><li> Inputs
<ul><li> <a href="/page/Category_colon_Color" title="Category:Color">Color</a> SetColor - Sets the color of the beam effect (Ported from Alien Swarm)
</li><li> <a href="/page/Category_colon_nil" title="Category:nil">nil</a> ForceUpdate - Forces the spotlight to update (Ported from Alien Swarm)
</li></ul>
</li></ul>
<h3> <span class="mw-headline" id="prop_door_rotating"> prop_door_rotating </span></h3>
<ul><li> Inputs
<ul><li> <a href="/page/Category_colon_number" title="Category:number">number</a> MoveToRotationDistance - Opens the door to a specific angle (Ported from CS:GO)
</li></ul>
</li><li> Outputs
<ul><li> <a href="/page/Category_colon_nil" title="Category:nil">nil</a> OnRotationDone - Fired when the door stops moving (Ported from CS:GO)
</li></ul>
</li><li> Spawnflags
<ul><li> 524288 - Start breakable (unset by default) (Ported from CS:GO)
</li></ul>
</li></ul>

</li></ul>
</li></ul>

</li></ul>
</li></ul>
