<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Loads next block of data to be read inside current block. Blocks <b>must</b> be ended with <a href="/page/Category_colon_IRestore" title="Category:IRestore">IRestore</a>:<a href="/page/IRestore/EndBlock" title="IRestore/EndBlock">EndBlock</a>.
</p>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a></span>
</p>
<div style="margin-left: 32px;">
The name of the next data block to be read.</div>
</div>

</p>
<pre><code data-language="lua">saverestore.AddRestoreHook( &quot;HookNameHere&quot;, function( save )

end )</code></pre>
