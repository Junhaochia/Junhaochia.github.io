<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Adds text to the local player's chat box (which only they can read).
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_vararg" title="Category:vararg">vararg</a> arguments</span>
</p>
<div style="margin-left: 32px;">
<p>The arguments. Arguments can be:
</p>
<ul><li> <a href="/page/Category_colon_table" title="Category:table">table</a> - <a href="/page/Structures/Color" title="Structures/Color">Color structure</a>. Will set the color for all following strings until the next Color argument.
</li><li> <a href="/page/Category_colon_string" title="Category:string">string</a> - Text to be added to the chat box.
</li><li> <a href="/page/Category_colon_Player" title="Category:Player">Player</a> - Adds the name of the player in the player's team color to the chat box.
</li><li> <a href="/page/Category_colon_any" title="Category:any">any</a> - Any other type, such as <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> will be converted to string and added as text.</div>
</li></ul>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints the player's name and current weapon to their chat area.
</p>
<pre><code data-language="lua">local ply = LocalPlayer()
chat.AddText( Color( 100, 100, 255 ), ply, &quot;, you are holding &quot;, Color( 100, 255, 100 ), ply:GetActiveWeapon():GetClass() )</code></pre>
