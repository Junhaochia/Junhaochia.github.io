<p>A brush surface object returned by <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetBrushSurfaces" title="Entity/GetBrushSurfaces">GetBrushSurfaces</a>.
</p><p>Brush surfaces are a part of what makes up the map geometry aside from displacements. Any primitive brush model, including most breakable windows, can be represented by this object.
</p>

<tr>
<td>NOTE
</td>
<td>
<p>This object internally uses the SurfaceHandle_t class, which is an alias for msurface2_t*.
