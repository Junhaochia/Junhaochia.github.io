<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Copies the entity, and all of its constraints and entities, then returns them in a table.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent</span>
</p>
<div style="margin-left: 32px;">
The entity to duplicate. The function will automatically copy all constrained entities.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> tableToAdd={}</span>
</p>
<div style="margin-left: 32px;">
<p>A preexisting table to add entities and constraints in from.
</p>
Uses the same table format as the table returned from this function.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
<p>A table containing duplication info which includes the following members:
</p>
<ul><li> <a href="/page/Category_colon_table" title="Category:table">table</a> Entities
</li><li> <a href="/page/Category_colon_table" title="Category:table">table</a> Constraints
</li><li> <a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> Mins
</li><li> <a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> Maxs
</li></ul>
The values of Mins &amp; Maxs from the table are returned from <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/WorkoutSize" title="duplicator/WorkoutSize">WorkoutSize</a></div>
</div>
