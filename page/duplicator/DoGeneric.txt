<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>"Applies generic every-day entity stuff for ent from table data."<br />
Depending on the values of Model, Angle, Pos, Skin, Flex, Bonemanip, ModelScale, ColGroup, Name, and BodyG (<a href="/page/Category_colon_table" title="Category:table">table</a> of multiple values) in the data table, this calls <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetModel" title="Entity/SetModel">SetModel</a>, <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetAngles" title="Entity/SetAngles">SetAngles</a>, <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetPos" title="Entity/SetPos">SetPos</a>, <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetSkin" title="Entity/SetSkin">SetSkin</a>, <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/DoFlex" title="duplicator/DoFlex">DoFlex</a>, <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/DoBoneManipulator" title="duplicator/DoBoneManipulator">DoBoneManipulator</a>, <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetModelScale" title="Entity/SetModelScale">SetModelScale</a>, <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetCollisionGroup" title="Entity/SetCollisionGroup">SetCollisionGroup</a>, <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetName" title="Entity/SetName">SetName</a>, <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetBodygroup" title="Entity/SetBodygroup">SetBodygroup</a> on ent.<br />
If ent has a RestoreNetworkVars function, it is called with data.DT.
</p>
