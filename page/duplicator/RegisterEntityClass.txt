<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>This allows you to specify a specific function to be run when your SENT is pasted with the duplicator, instead of relying on the generic automatic functions.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Automatically calls <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/Allow" title="duplicator/Allow">Allow</a> for the entity class.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> name</span>
</p>
<div style="margin-left: 32px;">
The ClassName of the entity you wish to register a factory for</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> function</span>
</p>
<div style="margin-left: 32px;">
The factory function you want to have called. It should have the arguments (Player, ...) where ... is whatever arguments you request to be passed.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_vararg" title="Category:vararg">vararg</a> args</span>
</p>
<div style="margin-left: 32px;">
Strings of the names of arguments you want passed to function the from the <a href="/page/Structures/EntityCopyData" title="Structures/EntityCopyData">EntityCopyData structure</a>. As a special case, "Data" will pass the whole structure.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints the datatable and then lets the duplicator do it's job
</p>
<pre><code data-language="lua">duplicator.RegisterEntityClass(&quot;prop_physics&quot;, function(ply, data)
	PrintTable(data)
	duplicator.GenericDuplicatorFunction(ply, data)
end, &quot;Data&quot;)</code></pre>
