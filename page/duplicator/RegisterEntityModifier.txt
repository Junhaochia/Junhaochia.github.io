<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>This allows you to register tweaks to entities. For instance, if you were making an "unbreakable" addon, you would use this to enable saving the "unbreakable" state of entities between duplications.
</p><p>This function registers a piece of generic code that is run on all entities with this modifier. In order to have it actually run, use <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/StoreEntityModifier" title="duplicator/StoreEntityModifier">StoreEntityModifier</a>.
</p><p>This function does nothing when run clientside.
</p>

An identifier for your modification. This is not limited, so be verbose. "Person's 'Unbreakable' mod" is far less likely to cause conflicts than "unbreakable"</div>
