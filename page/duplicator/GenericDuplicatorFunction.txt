<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>"Generic function for duplicating stuff" <br />
This is called when <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/CreateEntityFromTable" title="duplicator/CreateEntityFromTable">CreateEntityFromTable</a> can't find an entity factory to build with. It calls <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/DoGeneric" title="duplicator/DoGeneric">DoGeneric</a> and <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/DoGenericPhysics" title="duplicator/DoGenericPhysics">DoGenericPhysics</a> to apply standard duplicator stored things such as the model and position.
</p>
