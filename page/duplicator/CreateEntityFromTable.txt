<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>"Create an entity from a table." <br />
This creates an entity using the data in EntTable.<br />
If an entity factory has been registered for the entity's Class, it will be called. <br />
Otherwise, <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/GenericDuplicatorFunction" title="duplicator/GenericDuplicatorFunction">GenericDuplicatorFunction</a> will be called instead.
</p>
