<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>"Given entity list and constraint list, create all entities and return their tables"
</p><p>Calls <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/CreateEntityFromTable" title="duplicator/CreateEntityFromTable">CreateEntityFromTable</a> on each sub-table of EntityList. If an entity is actually created, it calls <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/OnDuplicated" title="ENTITY/OnDuplicated">OnDuplicated</a> with the entity's duplicator data, then <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/ApplyEntityModifiers" title="duplicator/ApplyEntityModifiers">ApplyEntityModifiers</a>, <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/ApplyBoneModifiers" title="duplicator/ApplyBoneModifiers">ApplyBoneModifiers</a> and finally <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/PostEntityPaste" title="ENTITY/PostEntityPaste">PostEntityPaste</a> is called.
</p><p>The constraints are then created with <a href="/page/Category_colon_duplicator" title="Category:duplicator">duplicator</a>.<a href="/page/duplicator/CreateConstraintFromTable" title="duplicator/CreateConstraintFromTable">CreateConstraintFromTable</a>.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Code used for a TOOL to copy duplication data on right click and paste it with its original info on left click.
</p>
<pre><code data-language="lua">function TOOL:LeftClick( trace )
	if (SERVER) then
		duplicator.Paste(self:GetOwner(),Dupe.Entities,Dupe.Constraints)
		print(&quot;PASTED&quot;)
	end
	return true
end
 
function TOOL:RightClick( trace )
	if (SERVER and IsValid(trace.Entity)) then
		Dupe = duplicator.Copy(trace.Entity)
		print(&quot;COPIED&quot;)
	end
	return true
end</code></pre>
