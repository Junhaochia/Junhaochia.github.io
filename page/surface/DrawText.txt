<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Draw the specified text on the screen, using the previously set position, font and color.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function does not handle newlines properly
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is a rendering function that requires a 2D rendering context.<br />This means that it will only work in <a href="/page/Category_colon_2D_Rendering_Hooks" title="Category:2D Rendering Hooks">hooks with a 2D rendering context</a>.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> text</span>
</p>
<div style="margin-left: 32px;">
The text to be rendered.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Draws 'Hello World' on the screen. All functions in this example must be called for the draw to work flawlessly.
</p>
<pre><code data-language="lua">hook.Add( &quot;HUDPaint&quot;, &quot;drawsometext&quot;, function()
	surface.SetFont( &quot;Default&quot; )
	surface.SetTextColor( 255, 255, 255 )
	surface.SetTextPos( 128, 128 ) 
	surface.DrawText( &quot;Hello World&quot; )
end )</code></pre>
