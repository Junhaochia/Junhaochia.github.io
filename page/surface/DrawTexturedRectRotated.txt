<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Draw a textured rotated rectangle with the given position and dimensions and angle on the screen, using the current active texture.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is a rendering function that requires a 2D rendering context.<br />This means that it will only work in <a href="/page/Category_colon_2D_Rendering_Hooks" title="Category:2D Rendering Hooks">hooks with a 2D rendering context</a>.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>A function that allows you to override the origin of rotation.
</p><p>x0 and y0 are relative to the center of the rectangle.
</p>
<pre><code data-language="lua">function surface.DrawTexturedRectRotatedPoint( x, y, w, h, rot, x0, y0 )
	
	local c = math.cos( math.rad( rot ) )
	local s = math.sin( math.rad( rot ) )
	
	local newx = y0 * s - x0 * c
	local newy = y0 * c + x0 * s
	
	surface.DrawTexturedRectRotated( x + newx, y + newy, w, h, rot )
	
end</code></pre>

<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Draws a simple red forever rotating box.
</p>
<pre><code data-language="lua">function draw.RotatedBox( x, y, w, h, ang, color )
	draw.NoTexture()
	surface.SetDrawColor( color or color_white )
	surface.DrawTexturedRectRotated( x, y, w, h, ang )
end

hook.Add( &quot;HUDPaint&quot;, &quot;my_rotated_box&quot;, function()
	draw.RotatedBox( 100, 100, 100, 100, CurTime()&#160;% 360, Color( 255, 0, 0) )
end )</code></pre>
