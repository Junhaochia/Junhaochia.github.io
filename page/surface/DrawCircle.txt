<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Draws a hollow circle, made of dots. For a filled circle, see examples for <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/DrawPoly" title="surface/DrawPoly">DrawPoly</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is a rendering function that requires a 2D rendering context.<br />This means that it will only work in <a href="/page/Category_colon_2D_Rendering_Hooks" title="Category:2D Rendering Hooks">hooks with a 2D rendering context</a>.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage. Draws an orange circle at position 500, 500 with a varying/animated radius of 50 to 150.
</p>
<pre><code data-language="lua">hook.Add( &quot;HUDPaint&quot;, &quot;DrawCircleExample&quot;, function()

	surface.DrawCircle( 500, 500, 100 + math.sin( CurTime() ) * 50, Color( 255, 120, 0 ) )

end )</code></pre>
