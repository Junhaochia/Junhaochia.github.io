<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Draw a textured rectangle with the given position and dimensions on the screen, using the current active texture set with <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/SetMaterial" title="surface/SetMaterial">SetMaterial</a>. It is also affected by <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/SetDrawColor" title="surface/SetDrawColor">SetDrawColor</a>.
</p><p>See also <a href="/page/Category_colon_render" title="Category:render">render</a>.<a href="/page/render/SetMaterial" title="render/SetMaterial">SetMaterial</a> and <a href="/page/Category_colon_render" title="Category:render">render</a>.<a href="/page/render/DrawScreenQuadEx" title="render/DrawScreenQuadEx">DrawScreenQuadEx</a>.<br />
See also <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/DrawTexturedRectUV" title="surface/DrawTexturedRectUV">DrawTexturedRectUV</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is a rendering function that requires a 2D rendering context.<br />This means that it will only work in <a href="/page/Category_colon_2D_Rendering_Hooks" title="Category:2D Rendering Hooks">hooks with a 2D rendering context</a>.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Draws a 512x512 textured rectangle with the wireframe texture.
</p>
<pre><code data-language="lua">local ourMat = Material( &quot;models/wireframe&quot; ) -- Calling Material() every frame is quite expensive

hook.Add( &quot;HUDPaint&quot;, &quot;example_hook&quot;, function()
	surface.SetDrawColor( 255, 255, 255, 255 )
	surface.SetMaterial( ourMat  ) -- If you use Material, cache it!
	surface.DrawTexturedRect( 0, 0, 512, 512 )
end )</code></pre>
