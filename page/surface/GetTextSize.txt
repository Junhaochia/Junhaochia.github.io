<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the width and height (in pixels) of the given text, but only if the font has been set with <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/SetFont" title="surface/SetFont">SetFont</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> text</span>
</p>
<div style="margin-left: 32px;">
The string to check the size of.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
Width of the provided text</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns_2">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
Height of the provided text</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints out the size of "Hello World" in the Trebuchet24 font.
</p>
<pre><code data-language="lua">surface.SetFont(&quot;Trebuchet24&quot;)

local message = &quot;Hello World&quot;
local width, height = surface.GetTextSize(message)

print(&quot;Text width: &quot; .. width .. &quot;, text height: &quot; .. height)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Text width: 100, text height: 24</div>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Get text height very quickly. No tables or spare variables used.
</p>
<pre><code data-language="lua">local message = &quot;Hello World&quot;

surface.SetFont(&quot;Trebuchet24&quot;)
local height = select(2, surface.GetTextSize(message))

print(height)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">24</div>
