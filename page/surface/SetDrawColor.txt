<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Set the color of any future shapes to be drawn, can be set by either using r, g, b, a as separate values or by a <a href="/page/Structures/Color" title="Structures/Color">Color structure</a>. Using a color structure is not recommended to be created procedurally.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> r</span>
</p>
<div style="margin-left: 32px;">
The red value of color, or a <a href="/page/Structures/Color" title="Structures/Color">Color structure</a>.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> g</span>
</p>
<div style="margin-left: 32px;">
The green value of color. Unused if a <a href="/page/Structures/Color" title="Structures/Color">Color structure</a> was given.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> b</span>
</p>
<div style="margin-left: 32px;">
The blue value of color. Unused if a <a href="/page/Structures/Color" title="Structures/Color">Color structure</a> was given.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> a=255</span>
</p>
<div style="margin-left: 32px;">
The alpha value of color. Unused if a <a href="/page/Structures/Color" title="Structures/Color">Color structure</a> was given.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Draws a 512x512 textured rectangle with the wireframe material
</p>
<pre><code data-language="lua">local ourMat = Material( &quot;models/wireframe&quot; ) -- Calling Material() every frame is quite expensive

hook.Add( &quot;HUDPaint&quot;, &quot;example_hook&quot;, function()
	surface.SetDrawColor( 255, 0, 0, 255 ) -- Set the color to red
	surface.SetMaterial( ourMat ) -- If you use Material, cache it!
	surface.DrawTexturedRect( 0, 0, 512, 512 )
end )</code></pre>
