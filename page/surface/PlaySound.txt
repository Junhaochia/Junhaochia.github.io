<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Play a sound file directly on the client (such as UI sounds, etc).
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> soundfile</span>
</p>
<div style="margin-left: 32px;">
The path to the sound file, which must be relative to the sound/ folder.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Plays a sound with a given name. The game will look for the file in following places in that order:
</p>
<ul><li> garrysmod/addons/myaddon/sound/mysound.wav
</li><li> garrysmod/gamemodes/mygamemode/content/sound/mysound.wav
</li><li> garrysmod/sound/mysound.wav
</li></ul>
<pre><code data-language="lua">surface.PlaySound( &quot;mysound.wav&quot; )</code></pre>
