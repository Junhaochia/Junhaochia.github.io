<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Set the current font to be used for text operations later.
</p><p>The fonts must first be created with <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/CreateFont" title="surface/CreateFont">CreateFont</a> or be one of the <a href="/page/Default_Fonts" title="Default Fonts">Default Fonts</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> fontName</span>
</p>
<div style="margin-left: 32px;">
The name of the font to use.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Draws 'Hello World' on the screen, with the 'Default' font.
</p>
<pre><code data-language="lua">hook.Add( &quot;HUDPaint&quot;, &quot;HUDPaint_DrawABox&quot;, function()
	surface.SetDrawColor( 0, 0, 0, 128 ) -- Set color for background
	surface.DrawRect( 100, 100, 128, 20 ) -- Draw background

	surface.SetTextColor( 255, 255, 255 ) -- Set text color
	surface.SetTextPos( 136, 104 ) -- Set text position, top left corner
	surface.SetFont( &quot;Default&quot; ) -- Set the font
	surface.DrawText( &quot;Hello World&quot; ) -- Draw the text
end )</code></pre>
