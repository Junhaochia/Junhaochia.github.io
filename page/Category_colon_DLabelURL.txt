<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<p>Underlined link label without a DoClick function. When a valid URL is set and the label is clicked, it will open a browser window and navigate to the address.
</p><p>This panel uses <a href="/page/Category_colon_gui" title="Category:gui">gui</a>.<a href="/page/gui/OpenURL" title="gui/OpenURL">OpenURL</a> internally and its restrictions apply.
</p>

</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DLabelURL that opens the wiki when clicked.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetSize(400,400)
frame:SetTitle(&quot;Test panel&quot;)
frame:Center()
frame:MakePopup()

local lbl = vgui.Create( &quot;DLabelURL&quot;, frame )
lbl:SetPos(50, 50)
lbl:SetSize(100, 50)
lbl:SetColor(Color(255,255,255,255)) 
lbl:SetText(&quot;I Love Gmod&quot;) 
lbl:SetURL(&quot;http://steamcommunity.com/groups/glua&quot;)</code></pre>
