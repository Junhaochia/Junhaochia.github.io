<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<p>The DFrame control is the foundation for any Derma menu. It holds all of your controls.
</p>
<h1> <span class="mw-headline" id="Inherited_Functions"> Inherited Functions </span></h1>
<p>Inherits the following functions from <a href="/page/Category_colon_EditablePanel" title="Category:EditablePanel">EditablePanel</a> and its parents up to 3 levels deep:
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DFrame.
</p>
<pre><code data-language="lua">local DermaPanel = vgui.Create( &quot;DFrame&quot; )
DermaPanel:SetPos( 100, 100 )
DermaPanel:SetSize( 300, 200 )
DermaPanel:SetTitle( &quot;My new Derma frame&quot; )
DermaPanel:SetDraggable( true )
DermaPanel:MakePopup()</code></pre>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Creates a DFrame with only close button.
</p>
<pre><code data-language="lua">local DermaPanel = vgui.Create( &quot;DFrame&quot; )
DermaPanel:SetPos( 100, 100 )
DermaPanel:SetSize( 300, 200 )
DermaPanel:SetTitle( &quot;My new Derma frame&quot; )
DermaPanel:SetDraggable( true )
DermaPanel:MakePopup()
DermaPanel.btnMinim:SetVisible(false)
DermaPanel.btnMaxim:SetVisible(false)</code></pre>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_3">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_3">Example</span></h2>
<p>Creates a DFrame with custom title color.
</p>
<pre><code data-language="lua">local DermaPanel = vgui.Create( &quot;DFrame&quot; )
DermaPanel:SetPos( 100, 100 )
DermaPanel:SetSize( 300, 200 )
DermaPanel:SetTitle( &quot;My new Derma frame&quot; )
DermaPanel:SetDraggable( true )
DermaPanel:MakePopup()
DermaPanel.lblTitle.UpdateColours = function( label ) 
	label:SetTextStyleColor( color_green ) 
end</code></pre>
