<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a simple timer that runs the given function after a specified delay.
</p><p>For a more advanced version that you can control after creation, see <a href="/page/Category_colon_timer" title="Category:timer">timer</a>.<a href="/page/timer/Create" title="timer/Create">Create</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Timers use <a href="/page/Global/CurTime" title="Global/CurTime">CurTime</a> for timing.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Timers won't advance while the client is timing out from the server.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> delay</span>
</p>
<div style="margin-left: 32px;">
How long until the function should be ran (in seconds). Use 0 to have the function run in the next <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/Think" title="GM/Think">Think</a>.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> func</span>
</p>
<div style="margin-left: 32px;">
The function to run after the specified delay.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Print "Hello World" after 5 seconds.
</p>
<pre><code data-language="lua">timer.Simple( 5, function() print( &quot;Hello World&quot; ) end )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Hello World</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Spawns 5 zombies and creates a timer.Simple that removes them in 11, 12, 13, 14, and 15 seconds.
</p>
<pre><code data-language="lua">for i = 1, 5 do
	local zombie = ents.Create(&quot;npc_zombie&quot;)
	zombie:SetPos( Vector( i*40, 0 , 250 ) )
	zombie:Spawn()

	timer.Simple( 10 + i, function() zombie:Remove() end  )
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">--11 seconds into game 1st zombie disappears
<p>--12 seconds into game 2nd zombie disappears
</p>
etc.</div>
