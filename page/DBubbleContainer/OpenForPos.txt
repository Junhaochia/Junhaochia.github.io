<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the speech bubble position and size along with the dialog point position.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> x</span>
</p>
<div style="margin-left: 32px;">
The x position of the dialog point. If this is set to a value greater than half of the set width, the entire bubble container will be moved in addition to the dialog point.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> y</span>
</p>
<div style="margin-left: 32px;">
The y position of the bubble container. Has no effect unless set to a value greater than the set height + 64 pixels.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> w</span>
</p>
<div style="margin-left: 32px;">
The width of the bubble container.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> h</span>
</p>
<div style="margin-left: 32px;">
The height of the bubble container.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a bubble container that properly fits a 200x200 background panel.
</p>
<pre><code data-language="lua">-- Length and width of background panel
local size = 200
		
-- Background panel
BGPanel = vgui.Create(&quot;DPanel&quot;)
BGPanel:SetSize(size, size)
BGPanel:Center()


local bubble1 = vgui.Create(&quot;DBubbleContainer&quot;, BGPanel)

-- x = 100 (Set pointer in the middle of the speech bubble)
-- y = 0 (Don't adjust height)
-- w = 180 (20 pixel right margin)
-- h = 184 (16 pixel bottom margin)
bubble1:OpenForPos(size/2, 0, size-20, size-16)</code></pre>
