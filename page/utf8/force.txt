<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Forces a string to contain only valid UTF-8 data. Invalid sequences are replaced with U+FFFD (the Unicode replacement character).
</p><p>This is a lazy way for users to ensure a string contains only valid UTF-8 data.
</p>
