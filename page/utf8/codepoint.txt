<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the codepoints (as numbers) from all characters in the given string that start between byte position startPos and endPos. It raises an error if it meets any invalid byte sequence. This functions similarly to <a href="/page/Category_colon_string" title="Category:string">string</a>.<a href="/page/string/byte" title="string/byte">byte</a>.
</p>

</p>

The string that you will get the code(s) from.</div>
</div>
