<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the number of UTF-8 sequences in the given string between positions startPos and endPos (both inclusive). If it finds any invalid UTF-8 byte sequence, returns false as well as the position of the first invalid byte.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> string</span>
</p>
<div style="margin-left: 32px;">
The string to calculate the length of.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> startPos=1</span>
</p>
<div style="margin-left: 32px;">
The starting position to get the length from.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> endPos=-1</span>
</p>
<div style="margin-left: 32px;">
The ending position to get the length from.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
The number of UTF-8 characters in the string. If there are invalid bytes, this will be false.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns_2">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
The position of the first invalid byte. If there were no invalid bytes, this will be nil.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Demonstrates output of this function compared to <a href="/page/Category_colon_string" title="Category:string">string</a>.<a href="/page/string/len" title="string/len">len</a> when given a string that contains Russian text.
</p>
<pre><code data-language="lua">print( string.len( &quot;Мёнём&quot; ) )
print( utf8.len( &quot;Мёнём&quot; ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><code>11
5</code></div>
