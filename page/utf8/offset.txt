<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the byte-index of the n'th UTF-8-character after the given startPos (nil if none). startPos defaults to 1 when n is positive and -1 when n is negative. If n is zero, this function instead returns the byte-index of the UTF-8-character startPos lies within.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> string</span>
</p>
<div style="margin-left: 32px;">
The string that you will get the byte position from.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> n</span>
</p>
<div style="margin-left: 32px;">
The position to get the beginning byte position from.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> startPos=1 when n&gt;=0, -1 otherwise</span>
</p>
<div style="margin-left: 32px;">
The offset for n.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>

</p>
<div style="margin-left: 32px;">
Starting byte-index of the given position.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Returns the byte-index where the character at the 5th byte begins.
</p>
<pre><code data-language="lua">print(utf8.offset(&quot;( ͡° ͜ʖ ͡°)&quot;, 5))</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">7</div>
