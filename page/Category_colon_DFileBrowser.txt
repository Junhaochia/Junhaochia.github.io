<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<p>A tree and list-based file browser.
</p><p>It allows filtering by folder (directory) name and file extension, and can display models as <a href="/page/Category_colon_SpawnIcon" title="Category:SpawnIcon">SpawnIcons</a>.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DFileBrowser and displays the data/persist folder. Any file clicked is printed to the console.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetSize( 500, 250 )
frame:SetSizable( true )
frame:Center()
frame:MakePopup()
frame:SetTitle( &quot;DFileBrowser Example&quot; )

local browser = vgui.Create( &quot;DFileBrowser&quot;, frame )
browser:Dock( FILL )

browser:SetPath( &quot;GAME&quot; ) -- The access path i.e. GAME, LUA, DATA etc.
browser:SetBaseFolder( &quot;data&quot; ) -- The root folder
browser:SetOpen( true ) -- Open the tree to show sub-folders
browser:SetCurrentFolder( &quot;persist&quot; ) -- Show files from persist

function browser:OnSelect( path, pnl ) -- Called when a file is clicked
	print( path )
end</code></pre>

<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Creates a DFileBrowser that can spawn models from <i>props_</i> folders.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Uses the same <a href="/page/Category_colon_DFrame" title="Category:DFrame">DFrame</a> as above
</p>
</td></tr></table>
<pre><code data-language="lua">local browser = vgui.Create( &quot;DFileBrowser&quot;, frame )
browser:Dock( FILL )

browser:SetPath( &quot;GAME&quot; ) -- The access path i.e. GAME, LUA, DATA etc.
browser:SetBaseFolder( &quot;models&quot; ) -- The root folder
browser:SetName( &quot;Props_ Models&quot; ) -- Name to display in tree
browser:SetSearch( &quot;props_&quot; ) -- Search folders starting with &quot;props_&quot;
browser:SetFileTypes( &quot;*.mdl&quot; ) -- File type filter
browser:SetOpen( true ) -- Opens the tree (same as double clicking)
browser:SetCurrentFolder( &quot;props_badlands&quot; ) -- Set the folder to use

function browser:OnSelect( path, pnl ) -- Called when a file is clicked
	RunConsoleCommand( &quot;gm_spawn&quot;, path ) -- Spawn the model we clicked
	frame:Close()
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><i>See Preview</i>.</div>
