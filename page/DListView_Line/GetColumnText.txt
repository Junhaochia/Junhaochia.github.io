<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Gets the string held in the specified column of a <a href="/page/Category_colon_DListView_Line" title="Category:DListView Line">DListView_Line</a> panel.
</p><p>This is the same thing as doing <a href="/page/Category_colon_DListView_Line" title="Category:DListView Line">DListView_Line</a>:<a href="/index.php?title=DListView_Line/GetValue&amp;action=edit&amp;redlink=1" class="new" title="DListView Line/GetValue (page does not exist)">GetValue</a>( column_number ).
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> column</span>
</p>
<div style="margin-left: 32px;">
The number of the column to retrieve the text from, starts with 1.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a></span>
</p>
<div style="margin-left: 32px;">
The contents of the specified column.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<pre><code data-language="lua">local dframe = vgui.Create(&quot;DFrame&quot;)
dframe:SetSize(450, 350)
dframe:Center()

local dlist = vgui.Create(&quot;DListView&quot;, dframe)
dlist:Dock(FILL)
dlist:SetMultiSelect(false)
dlist:AddColumn(&quot;Name&quot;)
dlist:AddColumn(&quot;SteamID&quot;)
dlist:AddLine(&quot;Stalker&quot;, &quot;STEAM_0:1:18093014&quot;)
function dlist:DoDoubleClick(linenumber, lineinfo)
	print(lineinfo:GetColumnText(2))
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Prints "STEAM_0:1:18093014" when the line containing Stalker is double clicked.</div>
