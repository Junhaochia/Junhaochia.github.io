<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Removes the hook with the supplied identifier from the given event.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> eventName</span>
</p>
<div style="margin-left: 32px;">
The event name.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_any" title="Category:any">any</a> identifier</span>
</p>
<div style="margin-left: 32px;">
The unique identifier of the hook to remove, usually a string.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Darkens the player's screen for 15 seconds.
</p>
<pre><code data-language="lua">hook.Add( &quot;HUDPaint&quot;, &quot;my_hook_identifier&quot;, function()
	
	surface.SetDrawColor( 0, 0, 0, 150 )
	surface.DrawRect( 0, 0, ScrW(), ScrH() )
	
end )

timer.Simple( 15, function()

	hook.Remove( &quot;HUDPaint&quot;, &quot;my_hook_identifier&quot; )

end )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">After the hook gets removed, the dark overlay doesn't appear anymore.</div>
