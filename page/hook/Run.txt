<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Calls hooks associated with the given event.
</p><p>Calls all hooks until one returns something other than nil and then returns that data.<br />
If no hook returns any data, it will try to call the GAMEMODE:&lt;eventName&gt; alternative, if one exists.
</p><p>This function internally calls <a href="/page/Category_colon_hook" title="Category:hook">hook</a>.<a href="/page/hook/Call" title="hook/Call">Call</a>.
</p><p>See also: <a href="/page/Category_colon_gamemode" title="Category:gamemode">gamemode</a>.<a href="/page/gamemode/Call" title="gamemode/Call">Call</a> - same as this, but does not call hooks if the gamemode hasn't defined the function.
</p>
