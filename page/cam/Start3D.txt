<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets up a new 3D rendering context. Must be finished by <a href="/page/Category_colon_cam" title="Category:cam">cam</a>.<a href="/page/cam/End3D" title="cam/End3D">End3D</a>.
</p><p>For more advanced settings such as an orthographic view, use <a href="/page/Category_colon_cam" title="Category:cam">cam</a>.<a href="/page/cam/Start" title="cam/Start">Start</a> instead.
</p><p>
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is a function that starts a 3D rendering context.<br />This means that the only rendering functions will work after it are <a href="/page/Category_colon_3D_Rendering_Functions" title="Category:3D Rendering Functions">functions with a 3D rendering context</a>.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Set up a 3D rendering environment in a 2D rendering hook to render models on HUD.
</p>
<pre><code data-language="lua">hook.Add( &quot;HUDPaint&quot;, &quot;3d_camera_example&quot;, function()
	cam.Start3D()
		for id, ply in pairs( player.GetAll() ) do
			ply:DrawModel()
		end
	cam.End3D()
end )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">All players can be seen through walls.</div>
