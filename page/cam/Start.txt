<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets up a new rendering context. This is an extended version of <a href="/page/Category_colon_cam" title="Category:cam">cam</a>.<a href="/page/cam/Start3D" title="cam/Start3D">Start3D</a> and <a href="/page/Category_colon_cam" title="Category:cam">cam</a>.<a href="/page/cam/Start2D" title="cam/Start2D">Start2D</a>. Must be finished by <a href="/page/Category_colon_cam" title="Category:cam">cam</a>.<a href="/page/cam/End3D" title="cam/End3D">End3D</a> or <a href="/page/Category_colon_cam" title="Category:cam">cam</a>.<a href="/page/cam/End2D" title="cam/End2D">End2D</a>.
</p>
