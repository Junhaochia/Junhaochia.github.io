<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Function which prints out a list of idle NPCs to the server console.
</p>
<pre><code data-language="lua">function ReportIdleNPCs()

	for _, npc in pairs(ents.FindByClass(&quot;npc_*&quot;)) do
	
		if(IsValid(npc) &amp;&amp; npc:IsNPC() &amp;&amp;
			npc:GetNPCState() == NPC_STATE_IDLE) then
		
			print(&quot;Ent #&quot;..npc:EntIndex()..&quot;: &quot;..npc:GetClass()..&quot; is idle.&quot;)
			
		end
		
	end

end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">(To server console)<br />
<p>Ent #111: npc_citizen is idle.<br />
Ent #120: npc_citizen is idle.<br />
Ent #122: npc_citizen is idle.<br />
</p>
Ent #124: npc_citizen is idle.</div>
