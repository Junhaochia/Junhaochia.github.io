<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Makes the NPC like, hate, feel neutral towards, or fear the entity in question. If you want to setup relationship towards a certain entity <i>class</i>, use <a href="/page/Category_colon_NPC" title="Category:NPC">NPC</a>:<a href="/page/NPC/AddRelationship" title="NPC/AddRelationship">AddRelationship</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>NPCs do not see <a href="/page/Category_colon_NextBot" title="Category:NextBot">NextBots</a> by default. This can be fixed by adding the <a href="/page/Enums/FL" title="Enums/FL">FL_OBJECT</a> flag to the NextBot.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Find wanted NPC class name and make them hate the entity.
</p>
<pre><code data-language="lua">function ENT:Think()
local enemy = ents.FindByClass(&quot;npc_*&quot;) --Find any spawned entity in map with class beginning at npc
    for _, x in pairs(enemy) do --for every found entity do
       if&#160;!x:IsNPC() then return end -- if found entity is not NPC then do nothing 
       if x:GetClass()&#160;!= self:GetClass() then -- if found entity is not self entity then continue
        x:AddEntityRelationship( self, D_HT, 99 ) -- found entity will hate self entity
        self:AddEntityRelationship( x, D_HT, 99 ) -- self entity will hate found entity          
       end
    end 
end</code></pre>
