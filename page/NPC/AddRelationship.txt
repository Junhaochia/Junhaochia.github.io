<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Changes how an NPC feels towards another NPC. If you want to setup relationship towards a certain <i>entity</i>, use <a href="/page/Category_colon_NPC" title="Category:NPC">NPC</a>:<a href="/page/NPC/AddEntityRelationship" title="NPC/AddEntityRelationship">AddEntityRelationship</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> relationstring</span>
</p>
<div style="margin-left: 32px;">
<p>A string representing how the relationship should be set up.
</p>
Should be formatted as "npc_class <a href="/page/Enums/D" title="Enums/D">D_ Enums</a> numberPriority".</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Spawns a manhack and makes it hate floor turrets.
</p>
<pre><code data-language="lua">local hack = ents.Create( &quot;npc_manhack&quot; )
hack:Spawn()
hack:AddRelationship( &quot;npc_turret_floor D_HT 99&quot; )</code></pre>
