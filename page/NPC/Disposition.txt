<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the way the NPC "feels" about the entity.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent</span>
</p>
<div style="margin-left: 32px;">
The entity to get the disposition from.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
The NPCs disposition, see <a href="/page/Enums/D" title="Enums/D">D_ Enums</a>.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>If a player is hurt by a friendly NPC, announce it.
</p>
<pre><code data-language="lua">function FriendlyFireAnnouncement( ply, atk )
   if atk:IsNPC() and atk:Disposition(ply) == D_LI then --like
      PrintMessage(&quot;A &quot;..atk:GetClass()..&quot; attacked &quot;..ply:Nick()..&quot;!&quot;, HUD_PRINTTALK)
   end
end

hook.Add( &quot;PlayerHurt&quot;, &quot;FriendlyFire&quot;, FriendlyFireAnnouncement )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Prints "A ____ attacked ____!" to everyones chat.</div>
