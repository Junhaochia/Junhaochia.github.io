<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<div class="internalfunc"><span>This is an internal function or feature.</span><br />
This means you will be able to use it, but you really shouldn't.<br /><br />Use <a href="/page/Category_colon_DScrollPanel" title="Category:DScrollPanel">DScrollPanel</a>.</div>A scrollbar used in <a href="/page/Category_colon_DScrollPanel" title="Category:DScrollPanel">DScrollPanel</a>.
<p>The scrollbar notably contains <b>DVScrollBar.btnUp, DVScrollBar.btnDown, and DVScrollBar.btnGrip</b> which are the up button, down button, and grip respectively. Editing the paint functions of these allows custom scrollbar styling. See <a href="/page/Category_colon_DScrollPanel" title="Category:DScrollPanel">DScrollPanel</a>:<a href="/page/DScrollPanel/GetVBar" title="DScrollPanel/GetVBar">GetVBar</a> for an example.
</p>
