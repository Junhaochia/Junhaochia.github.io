<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<p>A really simple grid layout panel.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This panel will set its size automatically based on set column count. This makes it play badly with <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/Dock" title="Panel/Dock">Dock</a> and cause a <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/PerformLayout" title="PANEL/PerformLayout">PerformLayout</a> call every frame.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a simple grid with numbered buttons.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetPos( 500, 500 )
frame:SetSize( 200, 300 )
frame:SetTitle( &quot;Frame&quot; )
frame:MakePopup()
 
local grid = vgui.Create( &quot;DGrid&quot;, frame )
grid:SetPos( 10, 30 )
grid:SetCols( 5 )
grid:SetColWide( 36 )
 
for i = 1, 30 do
	local but = vgui.Create( &quot;DButton&quot; )
	but:SetText( i )
	but:SetSize( 30, 20 )
	grid:AddItem( but )
end</code></pre>
