<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Adds a restore/load hook for the Half-Life 2 save system.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> identifier</span>
</p>
<div style="margin-left: 32px;">
The unique identifier for this hook.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> callback</span>
</p>
<div style="margin-left: 32px;">
<p>The function to be called when an engine save is being loaded. It has one argument:<br />
<a href="/page/Category_colon_IRestore" title="Category:IRestore">IRestore</a> save - The restore object to be used to read data from save file that is being loaded<br />
<br />
You can also use those functions to read data:<br />
<a href="/page/Category_colon_saverestore" title="Category:saverestore">saverestore</a>.<a href="/page/saverestore/ReadVar" title="saverestore/ReadVar">ReadVar</a><br />
<a href="/page/Category_colon_saverestore" title="Category:saverestore">saverestore</a>.<a href="/page/saverestore/ReadTable" title="saverestore/ReadTable">ReadTable</a><br />
<a href="/page/Category_colon_saverestore" title="Category:saverestore">saverestore</a>.<a href="/page/saverestore/LoadEntity" title="saverestore/LoadEntity">LoadEntity</a><br />
</p>
<br /></div>
</div>
