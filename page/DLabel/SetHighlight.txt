<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<div class="deprecatedfunc"><span>This feature is deprecated.</span><br />
You should avoid using it as it may be removed in a future version.<br /><br />You really should use <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/SetTextColor" title="DLabel/SetTextColor">SetTextColor</a>.</div>
<p>Sets the color of the text to the highlight text color defined in the skin.
</p><p>For the default Derma skin this makes the label red.
</p><p>See also <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/SetBright" title="DLabel/SetBright">SetBright</a> and <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/SetDark" title="DLabel/SetDark">SetDark</a>.
</p><p>You should only consider using this if you are using background elements that are not manually painted and are using the skin colors.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> highlight</span>
</p>
<div style="margin-left: 32px;">
true to set the label's color to skins's text highlight color, false otherwise.</div>
