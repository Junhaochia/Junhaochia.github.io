<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the text color of the <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>. This will take precedence over <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/SetTextStyleColor" title="DLabel/SetTextStyleColor">SetTextStyleColor</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> color</span>
</p>
<div style="margin-left: 32px;">
The text color. Uses the <a href="/page/Structures/Color" title="Structures/Color">Color structure</a>.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Changes the text color to red.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetTitle( &quot;Text Color Example&quot; )
frame:SetSize( 300, 100 )
frame:Center()
frame:MakePopup()

local frame_label = vgui.Create( &quot;DLabel&quot;, frame )
frame_label:SetPos( 10, 30 )
frame_label:SetTextColor( Color( 255, 0, 0) )</code></pre>
