<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Called when the label is right clicked (on key release) by the player.
</p><p>This will be called after <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/OnDepressed" title="DLabel/OnDepressed">OnDepressed</a> and <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/OnReleased" title="DLabel/OnReleased">OnReleased</a>.
</p><p>See also <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/DoClick" title="DLabel/DoClick">DoClick</a>, <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/DoMiddleClick" title="DLabel/DoMiddleClick">DoMiddleClick</a> and <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/DoDoubleClick" title="DLabel/DoDoubleClick">DoDoubleClick</a>.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a label in the center of the screen, that prints <i>I was right clicked!</i> to the console and disappears when right clicked.
</p>
<pre><code data-language="lua">local lbl = vgui.Create( &quot;DLabel&quot; )
lbl:SetFont( &quot;DermaLarge&quot; )
lbl:SetText( &quot;Click me!&quot; )
lbl:SizeToContents()
lbl:Center()
lbl:SetMouseInputEnabled( true )
function lbl:DoRightClick()
	print(&quot;I was right clicked!&quot;)
	self:Remove()
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">I was clicked!</code></pre> When the label is right clicked.</div>
