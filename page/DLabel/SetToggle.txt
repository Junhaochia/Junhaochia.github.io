<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the toggle state of the label. This can be retrieved with <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/GetToggle" title="DLabel/GetToggle">GetToggle</a> and toggled with <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/Toggle" title="DLabel/Toggle">Toggle</a>.
</p><p>In order to use toggle functionality, you must first call <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/SetIsToggle" title="DLabel/SetIsToggle">SetIsToggle</a> with <i>true</i>, as it is disabled by default.
</p>

</p>

The toggle state to be set.</div>
