<h1> <span class="mw-headline" id="DescriptionPanelHook">Description<span style="display: none;"><a href="/page/Form:PanelHook" title="Form:PanelHook">PanelHook</a></span></span></h1>
<p>Called when the player releases any mouse button on the label. This is always called after <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/OnDepressed" title="DLabel/OnDepressed">OnDepressed</a>.
</p><p>This works as an alternative to <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/OnMouseReleased" title="PANEL/OnMouseReleased">OnMouseReleased</a> as that hook is used heavily by <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a> and overriding it will break functionality.
</p><p>See also <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/DoClick" title="DLabel/DoClick">DoClick</a>, <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/DoMiddleClick" title="DLabel/DoMiddleClick">DoMiddleClick</a>, <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/DoRightClick" title="DLabel/DoRightClick">DoRightClick</a> and <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/DoDoubleClick" title="DLabel/DoDoubleClick">DoDoubleClick</a>.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Changes the text of the label when the hook is called.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetTitle( &quot;OnDepressed/Released Example&quot; )
frame:SetSize( 300, 100 )
frame:Center()
frame:MakePopup()

local frame_label = vgui.Create( &quot;DLabel&quot;, frame )
frame_label:SetPos( 10, 30 )
frame_label:SetTextColor( Color( 255, 255, 255 ) )
frame_label:SetText( &quot;Click me!&quot; )
frame_label:SizeToContents()
frame_label:SetMouseInputEnabled( true )
frame_label.OnDepressed = function( s )
	s:SetText( &quot;OnDepressed&quot; )
	frame_label:SizeToContents()
end
frame_label.OnReleased = function( s )
	s:SetText( &quot;OnReleased&quot; )
	frame_label:SizeToContents()
end</code></pre>
