<h1> <span class="mw-headline" id="DescriptionPanelHook">Description<span style="display: none;"><a href="/page/Form:PanelHook" title="Form:PanelHook">PanelHook</a></span></span></h1>
<p>Called when the label is left clicked (on key release) by the player.
</p><p>This will be called after <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/OnDepressed" title="DLabel/OnDepressed">OnDepressed</a> and <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/OnReleased" title="DLabel/OnReleased">OnReleased</a>.
</p><p>This can be overridden; by default, it calls <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/Toggle" title="DLabel/Toggle">Toggle</a>.
</p><p>See also <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/DoRightClick" title="DLabel/DoRightClick">DoRightClick</a>, <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/DoMiddleClick" title="DLabel/DoMiddleClick">DoMiddleClick</a> and <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/DoDoubleClick" title="DLabel/DoDoubleClick">DoDoubleClick</a>.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a label in the center of the screen, that prints <i>I was clicked!</i> to the console and disappears when clicked.
</p>
<pre><code data-language="lua">local lbl = vgui.Create( &quot;DLabel&quot; ) -- Creates our label
lbl:SetFont( &quot;DermaLarge&quot; )
lbl:SetText( &quot;Click me!&quot; )
lbl:SizeToContents()
lbl:Center()
lbl:SetMouseInputEnabled( true ) -- We must accept mouse input
function lbl:DoClick() -- Defines what should happen when the label is clicked
	print(&quot;I was clicked!&quot;)
	self:Remove()
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">I was clicked!</code></pre> When the label is clicked.</div>
