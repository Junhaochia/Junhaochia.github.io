<h1> <span class="mw-headline" id="Description">Description</span></h1>

</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Adds an NPC to the spawnmenu NPC list with the name "Fisherman", classname of "npc_fisherman" and the default weapon of "weapon_oldmanharpoon"
</p>
<pre><code data-language="lua">-- Lost Coast
list.Set(&quot;NPC&quot;, &quot;npc_fisherman&quot;, {
	Name = &quot;Fisherman&quot;,
	Class = &quot;npc_fisherman&quot;,
	Weapons = { &quot;weapon_oldmanharpoon&quot; },
	Category = Category
})</code></pre>
