<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Adds a callback to be called when the named convar changes.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This does not callback convars in the menu state.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/1440">#1440</a></b>
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This does not callback convars on the client with FCVAR_GAMEDLL and convars on the server without FCVAR_GAMEDLL.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3503">#3503</a></b>
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This does not callback convars on the client with FCVAR_REPLICATED.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3740">#3740</a></b>
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> name</span>
</p>
<div style="margin-left: 32px;">
The name of the convar to add the change callback to.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> callback</span>
</p>
<div style="margin-left: 32px;">
<p>The function to be called when the convar changes. The arguments passed are:
</p>
<ul><li> <a href="/page/Category_colon_string" title="Category:string">string</a> convar - The name of the convar.
</li><li> <a href="/page/Category_colon_string" title="Category:string">string</a> oldValue - The old value of the convar.
</li><li> <a href="/page/Category_colon_string" title="Category:string">string</a> newValue - The new value of the convar.</div>
</li></ul>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> identifier=nil</span>
</p>
<div style="margin-left: 32px;">
If set, you will be able to remove the callback using <a href="/page/Category_colon_cvars" title="Category:cvars">cvars</a>.<a href="/page/cvars/RemoveChangeCallback" title="cvars/RemoveChangeCallback">RemoveChangeCallback</a>.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Called when convar changes.
</p>
<pre><code data-language="lua">cvars.AddChangeCallback(&quot;convar name&quot;, function(convar_name, value_old, value_new)
    print(convar_name, value_old, value_new)
end)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">"convar name" 2 5</div>
