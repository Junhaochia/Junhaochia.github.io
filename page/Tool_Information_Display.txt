<p><b>CLEAN ME UP</b>
</p><p>You can find source code for it <a rel="nofollow" class="external text" href="https://github.com/garrynewman/garrysmod/blob/master/garrysmod/gamemodes/sandbox/entities/weapons/gmod_tool/cl_init.lua#L103-L164">here</a>.
</p><p>You cannot have more than two key combinations displayed.
</p><p>Here's an example:
</p>
<pre>
if CLIENT then

	TOOL.Information = {

		{ name = &quot;info&quot;, stage = 1 },
		{ name = &quot;left&quot; },
		{ name = &quot;right&quot; },
		{ name = &quot;right_use&quot;, icon2 = &quot;gui/e.png&quot; },
		{ name = &quot;reload&quot; },
		{ name = &quot;reload_use&quot;, icon2 = &quot;gui/e.png&quot; },

	}

	language.Add( &quot;tool.example.1&quot;, &quot;See information in the context menu&quot; )
	language.Add( &quot;tool.example.left&quot;, &quot;Select an object&quot; )
	language.Add( &quot;tool.example.right&quot;, &quot;Select next mode&quot; )
	language.Add( &quot;tool.example.right_use&quot;, &quot;Select previous mode&quot; )
	language.Add( &quot;tool.example.reload&quot;, &quot;Select yourself&quot; )
	language.Add( &quot;tool.example.reload_use&quot;, &quot;Select your view model&quot; )

end</pre>
<p>For each table inside the TOOL.Information a new line of text is added on HUD. It will display "#tool.&lt;toolclassname&gt;.&lt;namefromthename&gt;" by default.
</p>
<ul><li> &lt;toolclassname&gt; is the filename of your tool.<br /><br />
</li></ul>
<ul><li> <b>icon</b> key defines a path to the icon
</li><li> <b>icon2</b> key defines a a second icon path
</li><li> certain <b>name</b>s have a default fallback icon
</li><li> <b>stage</b> key makes the text only appear for given stage ( <a href="/page/Category_colon_Tool" title="Category:Tool">Tool</a>:<a href="/page/Tool/SetStage" title="Tool/SetStage">SetStage</a> )
</li><li> <b>op</b> key makes the text only appear for given operation ( <a href="/page/Category_colon_Tool" title="Category:Tool">Tool</a>:<a href="/page/Tool/SetOperation" title="Tool/SetOperation">SetOperation</a> )<br /><br />
</li></ul>
<ul><li> Default name is "info", which will display tool.example.&lt;toolstage&gt;, as all tools that do not use TOOL.Information do.
</li><li> If your name ends on "use" and you do not override the icon2, icon2 will be E key. ( gui/e.png )
</li><li> If your name starts with "info" - default icon is "gui/info"
</li><li> If your name starts with "left" - default icon is LMB icon ( gui/lmb.png )
</li><li> If your name starts with "right" - default icon is RMB icon ( gui/rmb.png )
</li><li> If your name starts with "reload" - default icon is R key icon ( gui/r.png )
</li></ul>
