<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Snaps the angle to nearest interval of degrees.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This will modify the original angle too!
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> axis</span>
</p>
<div style="margin-left: 32px;">
The component/axis to snap. Can be either "p"/"pitch", "y"/"yaw" or "r"/"roll".</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> target</span>
</p>
<div style="margin-left: 32px;">
The target angle snap interval</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_Angle" title="Category:Angle">Angle</a></span>
</p>
<div style="margin-left: 32px;">
The snapped angle.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage
</p>
<pre><code data-language="lua">print( Angle( 0, 92, 0 ):SnapTo( &quot;y&quot;, 90 ) )
print( Angle( 0, 115, 0 ):SnapTo( &quot;y&quot;, 45 ) )
print( Angle( 12, 98, 167 ):SnapTo( &quot;p&quot;, 30 ):SnapTo( &quot;y&quot;, 45 ):SnapTo( &quot;r&quot;, 45 ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">Angle( 0, 90, 0 )
Angle( 0, 135, 0 )
Angle( 0, 90, -180 )</code></pre></div>
