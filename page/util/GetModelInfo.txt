<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a table containing the info about the model. It seems to be not working serverside, but still exists serverside.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function will silently fail if used on models with following strings in them:
</p>
<ul><li> _shared
</li><li> _anims
</li><li> _gestures
</li><li> _anim
</li><li> _postures
</li><li> _gst
</li><li> _pst
</li><li> _shd
</li><li> _ss
</li><li> _anm
</li><li> _include
</li></ul>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> mdl</span>
</p>
<div style="margin-left: 32px;">
Model path</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
<p>The model info as a table with the following keys:
</p>
<ul><li><a href="/page/Category_colon_number" title="Category:number">number</a> SkinCount - Identical to <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SkinCount" title="Entity/SkinCount">SkinCount</a>.
</li><li><a href="/page/Category_colon_string" title="Category:string">string</a> KeyValues - Valve key-value formatted info about the model's physics (Constraint Info, etc). This is limited to 4096 characters.
</li><li><a href="/page/Category_colon_string" title="Category:string">string</a> ModelKeyValues - Valve key-value formatted info about the model ($keyvalues command in the .qc of the model), if present</div>
</li></ul>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>

</p>
<pre><code data-language="lua">local ModelInfo = util.GetModelInfo(&quot;models/combine_gate_vehicle.mdl&quot; )
print( ModelInfo.SkinCount ) // The skin count
PrintTable( util.KeyValuesToTablePreserveOrder( ModelInfo.KeyValues ) ) // Physics data</code></pre>
