<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Checks if the specified model is valid.
</p><p>A model is considered invalid in following cases:
</p>
<ul><li> Starts with a space or <b>maps</b>
</li><li> Doesn't start with <b>models</b>
</li><li> Contains any of the following:
<ul><li> _gestures
</li><li> _animations
</li><li> _postures
</li><li> _gst
</li><li> _pst
</li><li> _shd
</li><li> _ss
</li><li> _anm
</li><li> .bsp
</li><li> cs_fix
</li></ul>
</li><li> On server: If the model isn't precached, if the model file doesn't exist on the disk
</li><li> If precache failed
</li><li> Model is the error model
</li></ul>

<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> modelName</span>
</p>
<div style="margin-left: 32px;">
Name/Path of the model to check.</div>
</div>

<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>

Whether the model is valid or not. Returns false clientside if the model is not precached by the server.</div>
</div>
