<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Decompresses the given string using <a href="https://en.wikipedia.org/wiki/LZMA" class="extiw" title="wikipedia:LZMA">LZMA</a> algorithm. Used to decompress strings previously compressed with <a href="/page/Category_colon_util" title="Category:util">util</a>.<a href="/page/util/Compress" title="util/Compress">Compress</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> compressedString</span>
</p>
<div style="margin-left: 32px;">
The compressed string to decompress.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> maxSize=nil</span>
</p>
<div style="margin-left: 32px;">
The maximal size in bytes it will decompress.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a></span>
</p>
<div style="margin-left: 32px;">
The original, decompressed string or an empty string on failure or invalid input.</div>
</div>
