<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates an effect with the specified data.
</p><p>You can find a list of built-in engine effects <a href="/page/Effects" title="Effects">here</a>. You can create your own. Example effects can be found <a rel="nofollow" class="external text" href="https://github.com/garrynewman/garrysmod/tree/master/garrysmod/gamemodes/sandbox/entities/effects">here</a> and <a rel="nofollow" class="external text" href="https://github.com/garrynewman/garrysmod/tree/master/garrysmod/gamemodes/base/entities/effects">here</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>When dispatching an effect from the server, some values may be clamped for networking optimizations. Visit the Set accessors on <a href="/page/Category_colon_CEffectData" title="Category:CEffectData">CEffectData</a> to see which ones are affected.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>You will need to couple this function with <a href="/page/Global/IsFirstTimePredicted" title="Global/IsFirstTimePredicted">IsFirstTimePredicted</a> should you use this in a <a href="/page/Category_colon_Predicted_Hooks" title="Category:Predicted Hooks">predicted hook</a>.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> effectName</span>
</p>
<div style="margin-left: 32px;">
The name of the effect to create.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_CEffectData" title="Category:CEffectData">CEffectData</a> effectData</span>
</p>
<div style="margin-left: 32px;">
The effect data describing the effect.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> allowOverride=true</span>
</p>
<div style="margin-left: 32px;">
Whether Lua-defined effects should override engine-defined effects with the same name for this/single function call.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_any" title="Category:any">any</a> ignorePredictionOrRecipientFilter=nil</span>
</p>
<div style="margin-left: 32px;">
<p>Can either be a boolean to ignore the prediction filter or a <a href="/page/Category_colon_CRecipientFilter" title="Category:CRecipientFilter">CRecipientFilter</a>.
</p>
Set this to true if you wish to call this function in multiplayer from server.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a HelicopterMegaBomb effect at the origin of the map. ( 0, 0, 0 )
</p>
<pre><code data-language="lua">local vPoint = Vector( 0, 0, 0 )
local effectdata = EffectData()
effectdata:SetOrigin( vPoint )
util.Effect( &quot;HelicopterMegaBomb&quot;, effectdata )</code></pre>
