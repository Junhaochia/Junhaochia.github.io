<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Adds a trail to the specified entity.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent</span>
</p>
<div style="margin-left: 32px;">
Entity to attach trail to</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> attachmentID</span>
</p>
<div style="margin-left: 32px;">
Attachment ID of the entitiys model to attach trail to. If you are not sure, set this to 0</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> color</span>
</p>
<div style="margin-left: 32px;">
Color of the trail, use <a href="/page/Global/Color" title="Global/Color">Color</a></div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> additive</span>
</p>
<div style="margin-left: 32px;">
Should the trail be additive or not</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_5">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> startWidth</span>
</p>
<div style="margin-left: 32px;">
Start width of the trail</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_6">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> endWidth</span>
</p>
<div style="margin-left: 32px;">
End width of the trail</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_7">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> lifetime</span>
</p>
<div style="margin-left: 32px;">
How long it takes to transition from startWidth to endWidth</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_8">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> textureRes</span>
</p>
<div style="margin-left: 32px;">
The resolution of trails texture. A good value can be calculated using this formula: 1 / ( startWidth + endWidth ) * 0.5</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_9">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> texture</span>
</p>
<div style="margin-left: 32px;">
Path to the texture to use as a trail.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a></span>
</p>
<div style="margin-left: 32px;">
Entity of created trail (<a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Env_spritetrail">env_spritetrail</a>)</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>A console command that gives the player a red trail.
</p>
<pre><code data-language="lua">concommand.Add( &quot;givetrail&quot;, function( ply )
	local trail = util.SpriteTrail( ply, 0, Color( 255, 0, 0 ), false, 15, 1, 4, 1 / ( 15 + 1 ) * 0.5, &quot;trails/plasma&quot; )
	print( trail )
end )</code></pre>
