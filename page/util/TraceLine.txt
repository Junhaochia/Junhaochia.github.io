<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Performs a trace with the given trace data.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Clientside entities will not be hit by traces.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> TraceData</span>
</p>
<div style="margin-left: 32px;">
The trace data to use. See <a href="/page/Structures/Trace" title="Structures/Trace">Trace structure</a></div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
<p>Trace result. See <a href="/page/Structures/TraceResult" title="Structures/TraceResult">TraceResult structure</a>.
</p>
Can return nil if <a href="/page/Category_colon_game" title="Category:game">game</a>.<a href="/page/game/GetWorld" title="game/GetWorld">GetWorld</a> or its <a href="/page/Entity/GetPhysicsObject" title="Entity/GetPhysicsObject">physics object</a> is invalid. This will be the case for any traces done before <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/InitPostEntity" title="GM/InitPostEntity">InitPostEntity</a> is called.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Using a function callback as filter.
</p>
<pre><code data-language="lua">local tr = util.TraceLine( {
	start = LocalPlayer():EyePos(),
	endpos = LocalPlayer():EyePos() + EyeAngles():Forward() * 10000,
	filter = function( ent ) if ( ent:GetClass() == &quot;prop_physics&quot; ) then return true end end
} )

print( tr.HitPos, tr.Entity )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">The trace will only hit prop_physics or world.</div>
