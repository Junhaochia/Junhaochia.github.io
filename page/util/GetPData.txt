<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets PData of an offline player using their SteamID
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>This function internally uses <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/UniqueID" title="Player/UniqueID">UniqueID</a>, which can cause collisions (two or more players sharing the same PData entry). It's recommended that you don't use it. See the related wiki page for more information.
</p>
</td></tr></table>
