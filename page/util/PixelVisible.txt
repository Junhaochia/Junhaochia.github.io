<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the visibility of a sphere in the world.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> position</span>
</p>
<div style="margin-left: 32px;">
The center of the visibility test.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> radius</span>
</p>
<div style="margin-left: 32px;">
The radius of the sphere to check for visibility.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_pixelvis_handle_t" title="Category:pixelvis handle t">pixelvis handle t</a> PixVis</span>
</p>
<div style="margin-left: 32px;">
<p>The PixVis handle created with <a href="/page/Category_colon_util" title="Category:util">util</a>.<a href="/page/util/GetPixelVisibleHandle" title="util/GetPixelVisibleHandle">GetPixelVisibleHandle</a>.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Don't use the same handle twice per tick or it will give unpredictable results.
</p>
</td></tr></table></div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
Visibility, ranges from 0-1. 0 when none of the area is visible, 1 when all of it is visible.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Draws a box when the center of the map is visible on your screen.
</p>
<pre><code data-language="lua">local PixVis
function SetupPixVis()
	PixVis = util.GetPixelVisibleHandle()
end
hook.Add(&quot;Initialize&quot;, &quot;SetupPixVis&quot;, SetupPixVis)
 
function TestPixelVisibility()
	local visible = util.PixelVisible(Vector(0, 0, 0), 16, PixVis)
 
	if visible &amp;&amp; visible&#160;!= 0 then
		draw.RoundedBox( 10, 0, 0, 100, 100, Color(0, 0, 0, 200 * visible) )
	end
end
hook.Add(&quot;HUDPaint&quot;, &quot;TestPixelVisibility&quot;, TestPixelVisibility)</code></pre>
