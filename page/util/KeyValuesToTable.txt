<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Converts a KeyValue string to a Lua table.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Table keys will not repeat, see <a href="/page/Category_colon_util" title="Category:util">util</a>.<a href="/page/util/KeyValuesToTablePreserveOrder" title="util/KeyValuesToTablePreserveOrder">KeyValuesToTablePreserveOrder</a>.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage and output of this function. Note how there's only one "solid" key in the table despite the fact that input string contains multiple.
</p>
<pre><code data-language="lua">local ModelInfo = util.GetModelInfo(&quot;models/combine_gate_vehicle.mdl&quot; )
PrintTable( util.KeyValuesToTable( ModelInfo.KeyValues ) )</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">editparams:
		rootname	=	
		totalmass	=	1
solid:
		damping	=	0
		index	=	4
		inertia	=	1
		mass	=	1
		name	=	Vehicle_Gate.Gate1_L
		rotdamping	=	0
		surfaceprop	=	metal
		volume	=	68522.8828125</code></pre>
