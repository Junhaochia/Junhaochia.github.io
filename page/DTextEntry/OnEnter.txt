<h1> <span class="mw-headline" id="DescriptionPanelHook">Description<span style="display: none;"><a href="/page/Form:PanelHook" title="Form:PanelHook">PanelHook</a></span></span></h1>
<p>Called whenever enter is pressed on a DTextEntry.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p><a href="/page/Category_colon_DTextEntry" title="Category:DTextEntry">DTextEntry</a>:<a href="/page/DTextEntry/IsEditing" title="DTextEntry/IsEditing">IsEditing</a> will still return true in this callback!
</p>
</td></tr></table>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<pre><code data-language="lua">local TextEntry = vgui.Create( &quot;DTextEntry&quot; )
TextEntry:SetTextColor( Color( 255, 0, 0, 255 ) )

TextEntry.OnEnter = function()
    TextEntry:SetTextColor( Color( 0, 0, 255, 255 ) )
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Changes the text color when enter is pressed.</div>
