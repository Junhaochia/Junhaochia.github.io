<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Called internally when the text changes of the <a href="/page/Category_colon_DTextEntry" title="Category:DTextEntry">DTextEntry</a> are applied. (And set to the attached console variable, if one is given)
</p><p>See also <a href="/page/Category_colon_DTextEntry" title="Category:DTextEntry">DTextEntry</a>:<a href="/page/DTextEntry/OnChange" title="DTextEntry/OnChange">OnChange</a> for a function that is called on every text change, even if the console variable is not updated.
</p><p>You should override this function to define custom behavior when the text changes.
</p><p>This method is called:
</p>
<ul><li> When Enter is pressed after typing
</li><li> When <a href="/page/Category_colon_DTextEntry" title="Category:DTextEntry">DTextEntry</a>:<a href="/page/DTextEntry/SetValue" title="DTextEntry/SetValue">SetValue</a> is used
</li><li> For every key typed - only if <a href="/page/Category_colon_DTextEntry" title="Category:DTextEntry">DTextEntry</a>:<a href="/page/DTextEntry/SetUpdateOnType" title="DTextEntry/SetUpdateOnType">SetUpdateOnType</a> was set to true (default is false)
</li></ul>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> value</span>
</p>
<div style="margin-left: 32px;">
The DTextEntry text.</div>
