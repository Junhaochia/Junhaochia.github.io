<h1> <span class="mw-headline" id="DescriptionPanelHook">Description<span style="display: none;"><a href="/page/Form:PanelHook" title="Form:PanelHook">PanelHook</a></span></span></h1>
<p>Called whenever a valid character is typed while the text entry is focused.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This hook only works on panels derived from <a href="/page/Category_colon_TextEntry" title="Category:TextEntry">TextEntry</a>, such as <a href="/page/Category_colon_DTextEntry" title="Category:DTextEntry">DTextEntry</a>.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> keyCode</span>
</p>
<div style="margin-left: 32px;">
They key code of the key pressed, see <a href="/page/Enums/KEY" title="Enums/KEY">KEY_ Enums</a>.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Whether you've handled the key press. Returning true prevents the default text entry behavior from occurring.</div>
</div>
