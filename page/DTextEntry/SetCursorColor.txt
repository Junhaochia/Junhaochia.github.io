<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the cursor's color in DTextEntry (the blinking line).
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> color</span>
</p>
<div style="margin-left: 32px;">
The color to set the cursor to.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a panel and a DTextEntry, and then sets the cursor color.
(As seen on <a rel="nofollow" class="external free" href="https://maurits.tv/data/garrysmod/wiki/wiki.garrysmod.com/index9051.html">https://maurits.tv/data/garrysmod/wiki/wiki.garrysmod.com/index9051.html</a>)
</p>
<pre><code data-language="lua">local ParentPanel = vgui.Create(&quot;DFrame&quot;)
	ParentPanel:SetSize(ScrW()/7, ScrH()/12)
	ParentPanel:Center()
	ParentPanel:SetTitle(&quot;Cursor Color Test&quot;)
	ParentPanel:SetDeleteOnClose(true)
	ParentPanel:MakePopup()
 
local TextEntry = vgui.Create( &quot;DTextEntry&quot;, ParentPanel )
	TextEntry:SetSize(ScrW()/9, ScrH()/30)
	TextEntry:SetValue(&quot;Cursor Color Test &quot;)
	TextEntry:SetPos(ParentPanel:GetWide()/2-TextEntry:GetWide()/2,
	ParentPanel:GetTall()/2-TextEntry:GetTall()/5)
	TextEntry:SetEnterAllowed(false)
 
	-- Uses Simple RGBA (Red, Green, Blue, Alpha) Colors. --
	TextEntry:SetCursorColor(Color(255,0,0,255))
 
	TextEntry:RequestFocus()</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Makes the text entry's cursor color red.</div>
