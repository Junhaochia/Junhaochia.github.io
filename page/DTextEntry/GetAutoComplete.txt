<h1> <span class="mw-headline" id="DescriptionPanelHook">Description<span style="display: none;"><a href="/page/Form:PanelHook" title="Form:PanelHook">PanelHook</a></span></span></h1>
<p>Called by the DTextEntry when a list of autocompletion options is requested. Meant to be overridden.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> inputText</span>
</p>
<div style="margin-left: 32px;">
Player's current input.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
If a table is returned, the values of the table will show up as autocomplete suggestions for the user.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Shows a list of players to choose from.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetSize( 300, 300 )
frame:SetTitle( &quot;Autocompletion Example&quot; )
frame:Center()
frame:MakePopup()

local label = vgui.Create( &quot;DLabel&quot;, frame )
label:SetText( &quot;Type a player...&quot; )
label:Dock( TOP )

local textentry = vgui.Create( &quot;DTextEntry&quot;, frame )
textentry:Dock( TOP )
function textentry:GetAutoComplete( text )
    local suggestions = {}

    for k,v in pairs( player.GetAll() ) do -- For every player,
        if string.StartWith( v:Nick(), text ) then -- if the player's name starts with it...
            table.insert( suggestions, v:Nick() ) -- ...insert it into the list.
        end
    end

	return suggestions
end</code></pre>
