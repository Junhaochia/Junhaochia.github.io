<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets whether pressing the Enter key will cause the DTextEntry to lose focus or not, provided it is not multiline. This is true by default.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> allowEnter</span>
</p>
<div style="margin-left: 32px;">
If set to false, pressing Enter will not cause the panel to lose focus and will never call <a href="/page/Category_colon_DTextEntry" title="Category:DTextEntry">DTextEntry</a>:<a href="/page/DTextEntry/OnEnter" title="DTextEntry/OnEnter">OnEnter</a>.</div>
</div>
