<h1> <span class="mw-headline" id="DescriptionPanelHook">Description<span style="display: none;"><a href="/page/Form:PanelHook" title="Form:PanelHook">PanelHook</a></span></span></h1>
<p>Called whenever the DTextEntry gains focus.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<pre><code data-language="lua">local dTextEntry = vgui.Create(&quot;DTextEntry&quot;)
dTextEntry:SetText(&quot;Enter text here&quot;)

dTextEntry.OnGetFocus = function()
    self:SetText(&quot;&quot;) -- Clear the text box for user input
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Text entry clears when user clicks it to begin typing</div>
