<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Called whenever the value of the panel has been updated (whether by user input or otherwise).
</p><p>It allows you to determine whether a user can modify the <a href="/page/Category_colon_TextEntry" title="Category:TextEntry">TextEntry</a>'s text.
</p><p>By default, this only checks whether the panel disallows numeric characters, preventing it from being edited if the value contains any.
</p><p>This is actually an engine hook that only works on <a href="/page/Category_colon_TextEntry" title="Category:TextEntry">TextEntry</a> derived elements.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> char</span>
</p>
<div style="margin-left: 32px;">
The last character entered into the panel.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Return true to prevent the value from changing, false to allow it.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prevents the user from editing the text entirely.
</p>
<pre><code data-language="lua">local TextEntry = vgui.Create( &quot;DTextEntry&quot; )
TextEntry.AllowInput = function( self, stringValue )
	return true
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">The panel does not allow any alterations.</div>
