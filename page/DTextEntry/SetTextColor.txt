<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the text color of the <a href="/page/Category_colon_DTextEntry" title="Category:DTextEntry">DTextEntry</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> color</span>
</p>
<div style="margin-left: 32px;">
The text color. Uses the <a href="/page/Structures/Color" title="Structures/Color">Color structure</a>.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Changes the text color inside the DTextEntry to the color red.
</p>
<pre><code data-language="lua">local Frame = vgui.Create(&quot;DFrame&quot;)
Frame:SetSize(200,200)
Frame:Center()
Frame:MakePopup()

local TextEntry = vgui.Create(&quot;DTextEntry&quot;, Frame)
TextEntry:SetSize(180,20)
TextEntry:SetPos(10,80)
TextEntry:SetTextColor(Color(255,20,20))
TextEntry:SetValue(&quot;Basic Text&quot;)</code></pre>
