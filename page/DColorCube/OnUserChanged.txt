<h1> <span class="mw-headline" id="DescriptionPanelHook">Description<span style="display: none;"><a href="/page/Form:PanelHook" title="Form:PanelHook">PanelHook</a></span></span></h1>
<p>Function which is called when the color cube slider is moved (through user input). Meant to be overridden.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> color</span>
</p>
<div style="margin-left: 32px;">
The new color, uses <a href="/page/Structures/Color" title="Structures/Color">Color structure</a>.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a color cube which controls the blue saturation and value of a ball image.
</p>
<pre><code data-language="lua">-- Frame
MainFrame = vgui.Create(&quot;DFrame&quot;)
MainFrame:SetSize(320, 200)
MainFrame:Center()
MainFrame:SetTitle(&quot;Choose the saturation and value&quot;)

-- Image of a ball
local ball_img = vgui.Create(&quot;DImage&quot;, MainFrame)
ball_img:SetPos(20, 45)
ball_img:SetSize(128, 128)

ball_img:SetImage(&quot;sprites/sent_ball&quot;)

-- Color cube
local color_cube = vgui.Create(&quot;DColorCube&quot;, MainFrame)
color_cube:SetPos(160, 40)
color_cube:SetSize(150, 150)

-- Set color to blue
color_cube:SetColor(Color(0, 0, 255))

-- Called when slider is moved by user
function color_cube:OnUserChanged(col)
	
	-- Update ball color
	ball_img:SetImageColor(col)

end</code></pre>
