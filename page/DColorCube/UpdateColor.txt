<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<div class="internalfunc"><span>This is an internal function or feature.</span><br />
This means you will be able to use it, but you really shouldn't.</div>
<p>Updates the color cube RGB based on the given x and y position. Similar to <a href="/page/Category_colon_DColorCube" title="Category:DColorCube">DColorCube</a>:<a href="/page/DColorCube/TranslateValues" title="DColorCube/TranslateValues">TranslateValues</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> x=nil</span>
</p>
<div style="margin-left: 32px;">
The x position to set color to/the percentage of saturation to remove from the color (ranges from 0.0 to 1.0).</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> y=nil</span>
</p>
<div style="margin-left: 32px;">
The y position to set color to/the percentage of brightness or value to remove from the color (ranges from 0.0 to 1.0).</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a yellow color cube and updates/prints out the color at the (0.1, 0.6) position.
</p>
<pre><code data-language="lua">local color_cube = vgui.Create(&quot;DColorCube&quot;)
color_cube:SetSize(200, 200)
color_cube:Center()

-- Set base color to yellow
color_cube:SetColor(Color(255, 255, 0))

-- 10% less saturation, 60% darker
color_cube:UpdateColor(0.1, 0.6)

-- Get new color
local new_color = color_cube:GetRGB()

-- Print new color
print(&quot;Color( &quot;..new_color.r..&quot;, &quot;..new_color.g..&quot;, &quot;..new_color.b..&quot;, &quot;..new_color.a..&quot; )&quot;)</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">Color( 102, 102, 10, 255 )</code></pre>
