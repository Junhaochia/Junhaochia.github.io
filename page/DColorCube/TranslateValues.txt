<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<div class="internalfunc"><span>This is an internal function or feature.</span><br />
This means you will be able to use it, but you really shouldn't.</div>
<p>Updates the color cube RGB based on the given x and y position and returns its arguments. Similar to <a href="/page/Category_colon_DColorCube" title="Category:DColorCube">DColorCube</a>:<a href="/page/DColorCube/UpdateColor" title="DColorCube/UpdateColor">UpdateColor</a>.
</p>

<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> x</span>
</p>
<div style="margin-left: 32px;">
The x position to sample color from/the percentage of saturation to remove from the color (ranges from 0.0 to 1.0).</div>
</div>

The y position to sample color from/the percentage of brightness or value to remove from the color (ranges from 0.0 to 1.0).</div>
</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a green color cube and prints out the color at the (0.2, 0.4) position.
</p>
<pre><code data-language="lua">local color_cube = vgui.Create(&quot;DColorCube&quot;)
color_cube:SetSize(200, 200)
color_cube:Center()

-- Set base color to green
color_cube:SetColor(Color(0, 255, 0))

-- 20% desaturated, 40% darker
color_cube:TranslateValues(0.2, 0.4)

-- Get new color
local new_color = color_cube:GetRGB()

-- Update slider position
color_cube:SetColor(new_color)

-- Print out new color
print(&quot;Color( &quot;..new_color.r..&quot;, &quot;..new_color.g..&quot;, &quot;..new_color.b..&quot;, &quot;..new_color.a..&quot; )&quot;)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">Color( 30, 153, 30, 255 )</code></pre>
