<h1> <span class="mw-headline" id="DescriptionPanelHook">Description<span style="display: none;"><a href="/page/Form:PanelHook" title="Form:PanelHook">PanelHook</a></span></span></h1>
<p>Function which is called when the cursor is clicked and/or moved on the color picker. Meant to be overridden.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> col</span>
</p>
<div style="margin-left: 32px;">
The color that is selected on the color picker (<a href="/page/Structures/Color" title="Structures/Color">Color structure</a> form).</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a color picker which controls the color of a ball image.
</p>
<pre><code data-language="lua">-- Frame
MainFrame = vgui.Create(&quot;DFrame&quot;)
MainFrame:SetSize(200, 200)
MainFrame:Center()
MainFrame:SetTitle(&quot;Pick a color&quot;)

-- Image of a ball
local ball_img = vgui.Create(&quot;DImage&quot;, MainFrame)
ball_img:SetPos(20, 45)
ball_img:SetSize(128, 128)

ball_img:SetImage(&quot;sprites/sent_ball&quot;)

-- Vertical color picker
local color_picker = vgui.Create(&quot;DRGBPicker&quot;, MainFrame)
color_picker:SetPos(165, 30)
color_picker:SetSize(25, 150)

function color_picker:OnChange(col)

	ball_img:SetImageColor(col)
	
end</code></pre>
