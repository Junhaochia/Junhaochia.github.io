<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets whether or not the <a href="/page/Category_colon_DFrame" title="Category:DFrame">DFrame</a> can be resized by the user.
</p><p>This is achieved by clicking and dragging in the bottom right corner of the frame.
</p><p>You can set the minimum size using <a href="/page/Category_colon_DFrame" title="Category:DFrame">DFrame</a>:<a href="/page/DFrame/SetMinWidth" title="DFrame/SetMinWidth">SetMinWidth</a> and <a href="/page/Category_colon_DFrame" title="Category:DFrame">DFrame</a>:<a href="/page/DFrame/SetMinHeight" title="DFrame/SetMinHeight">SetMinHeight</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> sizeable</span>
</p>
<div style="margin-left: 32px;">
Whether the frame should be resizeable or not.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>A snippet of code that makes a frame resizable and sets the minimum size to its current size.
</p>
<pre><code data-language="lua">frame:SetSizable( true )
frame:SetMinWidth( frame:GetWide() )
frame:SetMinHeight( frame:GetTall() )</code></pre>
