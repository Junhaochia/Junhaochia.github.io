<h1> <span class="mw-headline" id="DescriptionPanelHook">Description<span style="display: none;"><a href="/page/Form:PanelHook" title="Form:PanelHook">PanelHook</a></span></span></h1>
<p>Called when the "checked" state is changed.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> bVal</span>
</p>
<div style="margin-left: 32px;">
Whether the CheckBox is checked or not.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a metamethod on the DCheckBox class to print any changes to the console.
</p>
<pre><code data-language="lua">function DCheckBox:OnChange(bVal)
	if (bVal) then
		print(&quot;Checked!&quot;)
	else
		print(&quot;Unchecked!&quot;)
	end
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">All new checkboxes created will print <i>Checked!</i> or <i>Unchecked!</i> when their checked state is changed.</div>
