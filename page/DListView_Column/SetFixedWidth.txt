<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the fixed width of the column.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> width</span>
</p>
<div style="margin-left: 32px;">
The number value which will determine a fixed width.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DListView and populates it with two columns and two items, only one of which can be selected at a time.
</p><p>The first column will be longer than the second column.
</p>
<pre><code data-language="lua">local Frame = vgui.Create( &quot;DFrame&quot; )
Frame:SetSize( 500, 500 )
Frame:Center()
Frame:MakePopup()

local SimpleList = vgui.Create( &quot;DListView&quot;, Frame )
SimpleList:Dock( FILL )
SimpleList:SetMultiSelect( false )

SimpleList:AddColumn( &quot;Column 1&quot; )
SimpleList:AddColumn( &quot;Column 2&quot; ):SetFixedWidth(125)

SimpleList:AddLine( &quot;First&quot;, &quot;Column&quot; )
SimpleList:AddLine( &quot;Second&quot;, &quot;Column&quot; )</code></pre>
