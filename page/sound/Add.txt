<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a sound script. It can also override sounds, which seems to only work when set on the server.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> soundData</span>
</p>
<div style="margin-left: 32px;">
The sounds properties. See <a href="/page/Structures/SoundData" title="Structures/SoundData">SoundData structure</a></div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a sound script. It will automatically vary in pitch and be played in a given audio channel to better control which sounds mix with which sounds and how.
</p><p>In this example, the file location could be: ( Choose one )
</p>
<ul><li> garrysmod/sound/vehicles/enzo/idle.wav
</li><li> garrysmod/gamemodes/MyCoolGameMode/content/sound/vehicles/enzo/idle.wav
</li><li> garrysmod/addons/myCoolAddon/sound/vehicles/enzo/idle.wav
</li><li> garrysmod/addons/myCoolAddon/gamemodes/MyCoolGameMode/content/sound/vehicles/enzo/idle.wav
</li></ul>
<pre><code data-language="lua">sound.Add( {
	name = &quot;enzo_engine_idle&quot;,
	channel = CHAN_STATIC,
	volume = 1.0,
	level = 80,
	pitch = {95, 110},
	sound = &quot;vehicles/enzo/idle.wav&quot;
} )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">You can now play your custom <b>sound script</b> with <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/EmitSound" title="Entity/EmitSound">EmitSound</a> like so:
<code>Entity( 1 ):EmitSound( "enzo_engine_idle" )</code></div>
