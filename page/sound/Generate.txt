<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a sound from a function.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This function cannot generate sounds that have a duration of less than 1 second.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3360">#3360</a></b>
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>Sounds persist between disconnects.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/4082">#4082</a></b>
</p>
</td></tr></table>

<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> samplerate</span>
</p>
<div style="margin-left: 32px;">
The sample rate of the sound. Must be 11025, 22050 or 44100.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> length</span>
</p>
<div style="margin-left: 32px;">
The length in seconds of the sound to generate.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> callback</span>
</p>
<div style="margin-left: 32px;">
A function which will be called to generate every sample on the sound. This function gets the current sample number passed as the first argument. The return value must be between -1.0 and 1.0. Other values will wrap back to the -1 to 1 range and basically clip. There are 65535 possible quantifiable values between -1 and 1.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Plays a 2000 Hz sine wave at maximum volume
</p>
<pre><code data-language="lua">local frequency = 2000 -- Hz
local samplerate = 44100

local function data(t)
    return math.sin( t * math.pi * 2 / samplerate * frequency )
end

test_sound_id = test_sound_id and test_sound_id + 1 or 10

sound.Generate   (&quot;testgen&quot; .. test_sound_id, samplerate, 2, data)
surface.PlaySound(&quot;testgen&quot; .. test_sound_id)</code></pre>
