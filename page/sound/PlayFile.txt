<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Plays a file from GMod directory. You can find a list of all error codes <a rel="nofollow" class="external text" href="https://www.un4seen.com/doc/#bass/BASS_ErrorGetCode.html">here</a>
</p><p>For external file/stream playback, see <a href="/page/Category_colon_sound" title="Category:sound">sound</a>.<a href="/page/sound/PlayURL" title="sound/PlayURL">PlayURL</a>.
</p>

<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> path</span>
</p>
<div style="margin-left: 32px;">
<p>The path to the file to play.
</p>
Unlike other sound functions and structures, the path is relative to <i>garrysmod/</i> instead of <i>garrysmod/sound/</i></div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> flags</span>
</p>
<div style="margin-left: 32px;">
<p>Flags for the sound. Can be one or more of following, separated by a space (" "):
</p>
<ul><li> 3d - Makes the sound 3D, so you can set its position
</li><li> mono - Forces the sound to have only one channel
</li><li> noplay - Forces the sound not to play as soon as this function is called
</li><li> noblock - Disables streaming in blocks. It is more resource-intensive, but it is required for <a href="/page/Category_colon_IGModAudioChannel" title="Category:IGModAudioChannel">IGModAudioChannel</a>:<a href="/page/IGModAudioChannel/SetTime" title="IGModAudioChannel/SetTime">SetTime</a>.<br /><br />
</li></ul>
If you don't want to use any of the above, you can just leave it as "".</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> callback</span>
</p>
<div style="margin-left: 32px;">
<p>Callback function that is called as soon as the the stream is loaded. It has next arguments:<br />
<a href="/page/Category_colon_IGModAudioChannel" title="Category:IGModAudioChannel">IGModAudioChannel</a> soundchannel - The sound channel. Will be nil if an error occured.<br />
<a href="/page/Category_colon_number" title="Category:number">number</a> errorID - ID of an error if an error has occured. Will be nil, otherwise.<br />
<a href="/page/Category_colon_string" title="Category:string">string</a> errorName - Name of an error if an error has occured. Will be nil, otherwise.<br />
</p>
<br /></div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Plays a piece of music from Half-Life 2.
</p>
<pre><code data-language="lua">sound.PlayFile( &quot;sound/music/hl2_song12_long.mp3&quot;, &quot;noplay&quot;, function( station, errCode, errStr )
	if ( IsValid( station ) ) then
		station:Play()
	else
		print( &quot;Error playing sound!&quot;, errCode, errStr )
	end
end )</code></pre>
