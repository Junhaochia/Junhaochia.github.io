<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Translates the <a href="/page/Structures/Color" title="Structures/Color">Color structure</a> into a <a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a>, losing the alpha channel.
This will also range the values from 0 - 255 to 0 - 1
</p><p>r / 255 -&gt; x<br />
g / 255 -&gt; y<br />
b / 255 -&gt; z
</p><p>This is the opposite of <a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a>:<a href="/page/Vector/ToColor" title="Vector/ToColor">ToColor</a>
</p>

The created <a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a></div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Useful when setting player colors, since the function requires a vector as argument.
</p>
<pre><code data-language="lua">Entity( 1 ):SetPlayerColor( Color( 220, 20, 60 ):ToVector() )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Sets the player color for Player1 (only works if they are using a colorable player model)</div>
