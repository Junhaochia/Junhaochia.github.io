<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<p>DScrollPanel is a VGUI Element similar to DPanel however it has a vertical scrollbar docked to the right which can be used to put more content in a smaller area. DScrollPanels are essentially <a href="/page/Category_colon_DPanel" title="Category:DPanel">DPanels</a> with the Scroll Bar.
</p><p>This can be used to replace the <a href="/page/Category_colon_DPanelList" title="Category:DPanelList">DPanelList</a> with <a href="/page/Category_colon_DPanelList" title="Category:DPanelList">DPanelList</a>:<a href="/page/DPanelList/EnableVerticalScrollbar" title="DPanelList/EnableVerticalScrollbar">EnableVerticalScrollbar</a> and get similar functionality as well as using a non-deprecated element.
</p><p>If you would like to paint or edit the elements of the scrollbar use <a href="/page/DScrollPanel/GetVBar" title="DScrollPanel/GetVBar">GetVBar</a>. If you want to see if the scrollbar is visible then use the VBar.Enabled variable on the scrollbar's VBar.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p><a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/DockPadding" title="Panel/DockPadding">DockPadding</a> will not have an effect on children of this panel. Use the function on <strong class="selflink">DScrollPanel</strong>:<a href="/page/DScrollPanel/GetCanvas" title="DScrollPanel/GetCanvas">GetCanvas</a> instead.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a <strong class="selflink">DScrollPanel</strong> and adds 100 <a href="/page/Category_colon_DButton" title="Category:DButton">DButtons</a> to it.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetSize( 500, 500 )
frame:Center()
frame:MakePopup()

local DScrollPanel = vgui.Create( &quot;DScrollPanel&quot;, frame )
DScrollPanel:Dock( FILL )

for i=0, 100 do
	local DButton = DScrollPanel:Add( &quot;DButton&quot; )
	DButton:SetText( &quot;Button #&quot; .. i )
	DButton:Dock( TOP )
	DButton:DockMargin( 0, 0, 0, 5 )
end</code></pre>
