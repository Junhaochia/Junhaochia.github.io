<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Moves the panel to the specified position using animation.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> posX</span>
</p>
<div style="margin-left: 32px;">
The target x coordinate of the panel.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> posY</span>
</p>
<div style="margin-left: 32px;">
The target y coordinate of the panel.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> time</span>
</p>
<div style="margin-left: 32px;">
The time to perform the animation within.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> delay=0</span>
</p>
<div style="margin-left: 32px;">
The delay before the animation starts.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_5">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> ease=-1</span>
</p>
<div style="margin-left: 32px;">
The easing of the start and/or end speed of the animation. See <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/NewAnimation" title="Panel/NewAnimation">NewAnimation</a> for how this works.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_6">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> callback</span>
</p>
<div style="margin-left: 32px;">
<p>The function to be called once the animation finishes. Arguments are:
</p>
<ul><li> <a href="/page/Category_colon_table" title="Category:table">table</a> animData - The <a href="/page/Structures/AnimationData" title="Structures/AnimationData">AnimationData structure</a> that was used.
</li><li> <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> pnl - The panel object that was moved.</div>
</li></ul>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Move panel to center
</p>
<pre><code data-language="lua">local frame = vgui.Create(&quot;DFrame&quot;)
frame:SetSize(ScrW() / 4, ScrH() / 4)
frame:SetPos(ScrW() / 4, ScrH() / 2)
frame:SetTitle(&quot;MoveTo Example&quot;)

local btn = vgui.Create(&quot;DButton&quot;, frame)
btn:SetSize(frame:GetWide() / 2, frame:GetTall() / 3)
btn:Center()
btn:SetText(&quot;Move&#160;!&quot;)
btn.DoClick = function(self)
	frame:MoveTo(ScrW() / 2 - frame:GetWide() / 2, ScrH() / 2 - frame:GetTall() / 2, 1, 0, -1, function()
		self:SetText(&quot;Yeah&#160;!&quot;)
	end)
end</code></pre>
