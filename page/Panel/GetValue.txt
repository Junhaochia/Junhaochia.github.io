<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the value the panel holds.
</p><p>In engine is only implemented for <a href="/page/Category_colon_CheckButton" title="Category:CheckButton">CheckButton</a>, <a href="/page/Category_colon_Label" title="Category:Label">Label</a> (limited to 8192 characters) and <a href="/page/Category_colon_TextEntry" title="Category:TextEntry">TextEntry</a> as a string.
</p><p>If using with <a href="/page/Category_colon_DTextEntry" title="Category:DTextEntry">DTextEntry</a>, you should use <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/GetText" title="Panel/GetText">GetText</a> instead.
</p>

</p>
<div style="margin-left: 32px;">
The value the panel holds.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Returns the string typed in a TextEntry.
</p>
<pre><code data-language="lua">local TextEntry = vgui.Create( &quot;TextEntry&quot; )
TextEntry:SetText( &quot;Hello world!&quot; )
print( TextEntry:GetValue() )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">"Hello world!"</div>
