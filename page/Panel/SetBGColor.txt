<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the background color of a panel such as a <a href="/page/Category_colon_RichText" title="Category:RichText">RichText</a>, <a href="/page/Category_colon_Label" title="Category:Label">Label</a> or <a href="/page/Category_colon_DColorCube" title="Category:DColorCube">DColorCube</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This doesn't apply to all VGUI elements and its function varies between them
</p>
</td></tr></table>
<p>For <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a> elements, you must use <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/SetPaintBackgroundEnabled" title="Panel/SetPaintBackgroundEnabled">SetPaintBackgroundEnabled</a>( true ) before applying the color.
</p><p>This will not work on setup of the panel - you should use this function in a hook like <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/ApplySchemeSettings" title="PANEL/ApplySchemeSettings">ApplySchemeSettings</a> or <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/PerformLayout" title="PANEL/PerformLayout">PerformLayout</a>.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a RichText panel that mimics a <a rel="nofollow" class="external text" href="https://en.wikipedia.org/wiki/Blue_Screen_of_Death">blue screen of death</a>.
</p>
<pre><code data-language="lua">-- Create a window frame
TextFrame = vgui.Create(&quot;DFrame&quot;)
TextFrame:SetSize(300, 100)
TextFrame:Center()
TextFrame:SetTitle(&quot;Windows XP Blue Screen&quot;)
TextFrame:MakePopup()

-- RichText panel
local richtext = vgui.Create(&quot;RichText&quot;, TextFrame)
richtext:Dock(FILL)

-- Sample text
richtext:SetText(&quot;A problem has been detected and Windows has been shut down to prevent damage to your computer.\n\nMOTHERBOARD_FRIED&quot;)

-- When the panel is ready for layout, set the background color to blue
function richtext:PerformLayout()
	
	self:SetBGColor(Color(0, 0, 255))
	
end</code></pre>
