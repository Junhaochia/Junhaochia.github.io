<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the target area for dropping when an object is being dragged around this panel using the <a href="/page/Category_colon_dragndrop" title="Category:dragndrop">dragndrop library</a>.
</p><p>This draws a target box of the specified size and position, until <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/DragHoverEnd" title="Panel/DragHoverEnd">DragHoverEnd</a> is called. It uses <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/DrawDragHover" title="Panel/DrawDragHover">DrawDragHover</a> to draw this area.
</p>
