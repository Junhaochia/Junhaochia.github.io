<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the cursor position local to the position of the panel (usually the upper-left corner).
</p>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
The x coordinate</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns_2">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
The y coordinate</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Create and center a label panel and update its text with the local cursor position.
</p>
<pre><code data-language="lua">TestLabel = vgui.Create(&quot;DLabel&quot;)
TestLabel:SetSize(100, 20)
TestLabel:Center()
TestLabel:SetPaintBackgroundEnabled(true)
TestLabel:SetColor(Color(255, 0, 0))

local x, y = 0, 0

function TestLabel:PerformLayout()

	x, y = self:LocalCursorPos()

	self:SetText(&quot; X: &quot;..x..&quot; , Y: &quot;..y)
	
end</code></pre>
