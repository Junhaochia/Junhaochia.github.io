<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Re-renders a spawn icon with customized cam data.
</p><p><a href="/page/Global/PositionSpawnIcon" title="Global/PositionSpawnIcon">PositionSpawnIcon</a> can be used to easily calculate the necessary camera parameters.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function does <b>not</b> accept the standard <a href="/page/Structures/CamData" title="Structures/CamData">CamData structure</a>.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> data</span>
</p>
<div style="margin-left: 32px;">
<p>A four-membered table containing the information needed to re-render:
</p>
<ul><li> <a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> cam_pos - The relative camera position the model is viewed from.
</li><li> <a href="/page/Category_colon_Angle" title="Category:Angle">Angle</a> cam_ang - The camera angle the model is viewed from.
</li><li> <a href="/page/Category_colon_number" title="Category:number">number</a> cam_fov - The camera's field of view (FOV).
</li><li> <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent - The entity object of the model.
</li></ul>
See the example below for how to retrieve these values.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>The <i>RenderIcon</i> method used by <a href="/page/Category_colon_IconEditor" title="Category:IconEditor">IconEditor</a>. <i>SpawnIcon</i> is a <a href="/page/Category_colon_SpawnIcon" title="Category:SpawnIcon">SpawnIcon</a> and <i>ModelPanel</i> is a <a href="/page/Category_colon_DAdjustableModelPanel" title="Category:DAdjustableModelPanel">DAdjustableModelPanel</a>.
</p>
<pre><code data-language="lua">function PANEL:RenderIcon()
	
	local ent = self.ModelPanel:GetEntity()
	
	local tab = {}
	tab.ent		= ent
	tab.cam_pos = self.ModelPanel:GetCamPos()
	tab.cam_ang = self.ModelPanel:GetLookAng()
	tab.cam_fov = self.ModelPanel:GetFOV()

	self.SpawnIcon:RebuildSpawnIconEx( tab )
end</code></pre>
