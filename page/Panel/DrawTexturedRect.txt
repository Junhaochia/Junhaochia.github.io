<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Draws a textured rectangle to fill the panel object this method is called on. The texture is set using <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/SetTexture" title="surface/SetTexture">SetTexture</a> or <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/SetMaterial" title="surface/SetMaterial">SetMaterial</a>. This should only be called within the object's <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/Paint" title="PANEL/Paint">Paint</a> or <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/PaintOver" title="PANEL/PaintOver">PaintOver</a> hooks, as a shortcut for <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/DrawTexturedRect" title="surface/DrawTexturedRect">DrawTexturedRect</a>.
</p>
