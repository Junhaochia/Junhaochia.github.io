<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when an object is dragged and hovered over this panel for 0.1 seconds.
</p><p>This is used by <a href="/page/Category_colon_DPropertySheet" title="Category:DPropertySheet">DPropertySheet</a> and <a href="/page/Category_colon_DTree" title="Category:DTree">DTree</a>, for example to open a tab or expand a node when an object is hovered over it.
</p>

<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> hoverTime</span>
</p>
<div style="margin-left: 32px;">
The time the object was hovered over this panel.</div>
