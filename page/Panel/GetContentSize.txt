<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets the size of the content/children within a panel object.
</p><p>Only works with <a href="/page/Category_colon_Label" title="Category:Label">Label</a> derived panels by default such as <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>.<br />
Will also work on any panel that manually implements this method.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Demonstrates how to implement this function in your own panel.
</p>
<pre><code data-language="lua">local PANEL = {}

function PANEL:GetContentSize()
	surface.SetFont( self:GetFont() )
	return surface.GetTextSize( self:GetText() )
end

vgui.Register( &quot;DTextEntry_Edit&quot;, PANEL, &quot;DTextEntry&quot; )

-- Somewhere else, to test the newly created panel
local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetSize( 500, 200 )
frame:Center()
frame:MakePopup()

local txt = vgui.Create( &quot;DTextEntry_Edit&quot;, frame )
txt:SetPos( 5, 25 )
txt:SetSize( 100, 10 )
txt:SetText( &quot;Really long string that is bigger than 100 pixels&quot; )
txt:SizeToContentsX( 5 ) -- Must be called after setting the text
txt:SizeToContentsY( 5 ) -- These two functions will not have effect on a normal DTextEntry</code></pre>
