<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Configures a text input to allow user to type characters that are not included in the US-ASCII (7-bit ASCII) character set.
</p><p>Characters not included in US-ASCII are multi-byte characters in UTF-8. They can be accented characters, non-Latin characters and special characters.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> allowed</span>
</p>
<div style="margin-left: 32px;">
Set to true in order not to restrict input characters.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Replaces the default vgui.Create() function to always allow non US-ASCII characters for text inputs.
</p>
<pre><code data-language="lua">if vgui.CreateStdRestrict == nil then
	vgui.CreateStdRestrict = vgui.Create
end
function vgui.Create( classname, parent, name )
	local vgui_elt = vgui.CreateStdRestrict(classname, parent, name)
	if classname == &quot;DTextEntry&quot; or classname == &quot;RichText&quot; or classname == &quot;TextEntry&quot; then
		vgui_elt:SetAllowNonAsciiCharacters(true)
	end
	return vgui_elt
end</code></pre>
