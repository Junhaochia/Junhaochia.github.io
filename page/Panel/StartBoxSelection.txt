<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Begins a box selection, enables mouse capture for the panel object, and sets the start point of the selection box to the mouse cursor's position, relative to this object. For this to work, either the object or its parent must be enabled as a selection canvas. This is set using <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/SetSelectionCanvas" title="Panel/SetSelectionCanvas">SetSelectionCanvas</a>.
</p>
