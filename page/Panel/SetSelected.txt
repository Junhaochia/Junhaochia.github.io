<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the selected state of a selectable panel object. This functionality is set with <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/SetSelectable" title="Panel/SetSelectable">SetSelectable</a> and checked with <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/IsSelectable" title="Panel/IsSelectable">IsSelectable</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> selected=false</span>
</p>
<div style="margin-left: 32px;">
Whether the object should be selected or deselected. <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/IsSelected" title="Panel/IsSelected">IsSelected</a> can be used to determine the selected state of the object.</div>
