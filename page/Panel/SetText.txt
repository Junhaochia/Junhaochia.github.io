<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the text value of a panel object containing text, such as a <a href="/page/Category_colon_Label" title="Category:Label">Label</a>, <a href="/page/Category_colon_TextEntry" title="Category:TextEntry">TextEntry</a> or <a href="/page/Category_colon_RichText" title="Category:RichText">RichText</a> and their derivatives, such as <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>, <a href="/page/Category_colon_DTextEntry" title="Category:DTextEntry">DTextEntry</a> or <a href="/page/Category_colon_DButton" title="Category:DButton">DButton</a>.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>When used on a <a href="/page/Category_colon_Label" title="Category:Label">Label</a> or its derivatives ( <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a> and <a href="/page/Category_colon_DButton" title="Category:DButton">DButton</a> ), it will automatically call <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/InvalidateLayout" title="Panel/InvalidateLayout">InvalidateLayout</a>, meaning that you should avoid running this function every frame on these panels to avoid unnecessary performance loss.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> text</span>
</p>
<div style="margin-left: 32px;">
The text value to set.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a <a href="/page/Category_colon_RichText" title="Category:RichText">RichText</a> element and sets the text to a localized string; the default VAC rejection message.
</p>
<pre><code data-language="lua">-- Window frame for the RichText
TextFrame = vgui.Create(&quot;DFrame&quot;)
TextFrame:SetSize(250, 150)
TextFrame:Center()
TextFrame:SetTitle(&quot;#VAC_ConnectionRefusedTitle&quot;) -- Results in &quot;Connection Refused - VAC&quot;

-- RichText panel
local richtext = vgui.Create(&quot;RichText&quot;, TextFrame)
richtext:Dock(FILL)

-- Set the text to the message you get when VAC banned
richtext:SetText(&quot;#VAC_ConnectionRefusedDetail&quot;)</code></pre>
