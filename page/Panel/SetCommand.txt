<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the action signal command that's fired when a <a href="/page/Category_colon_Button" title="Category:Button">Button</a> is clicked. The hook <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/ActionSignal" title="PANEL/ActionSignal">ActionSignal</a> is called as the click response.
</p><p>This has no effect on buttons unless it has had its <i>AddActionSignalTarget</i> method called (an internal function not available by default in Garry's Mod LUA).
</p><p>A better alternative is calling <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/Command" title="Panel/Command">Command</a> when a <a href="/page/Category_colon_DButton" title="Category:DButton">DButton</a> is clicked.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates an engine-based <a href="/page/Category_colon_Frame" title="Category:Frame">Frame</a> panel and changes the command that's fired when you click the white close button (by default the command is set to "Close").
</p>
<pre><code data-language="lua">-- Create a regular Frame panel
TestFrame = vgui.Create(&quot;Frame&quot;)
TestFrame:SetSize(200, 200)
TestFrame:Center()
TestFrame:MakePopup()
TestFrame:SetVisible(true)

local lbl = vgui.Create(&quot;DLabel&quot;, TestFrame)
lbl:Dock(FILL)
lbl:DockMargin(10, 10, 10, 10)
lbl:SetText(&quot;Click the white button in the upper right corner of this window.&quot;)
lbl:SetFont(&quot;ChatFont&quot;)
lbl:SetWrap(true)

-- Create a background panel so we can see the Frame's internal buttons
local bg = vgui.Create(&quot;DPanel&quot;, TestFrame)
bg:Dock(FILL)
bg:SetBackgroundColor(Color(64, 64, 64, 192))
bg:MoveToBack()

-- Loop through Frame's internal components
for _, child in pairs(TestFrame:GetChildren()) do

	-- Disable frame sizing
	if(string.find(child:GetName(), &quot;Grip&quot;)) then
	
		child:SetMouseInputEnabled(false)
		
	-- Fire the &quot;Testing&quot; command when we click the white button
	elseif(child:GetName() == &quot;frame_close&quot;) then
	
		child:SetCommand(&quot;Testing&quot;)
		
	end
	
end

function TestFrame:ActionSignal(signalName, signalValue)
	
	-- Show the signal name
	lbl:SetText(&quot;Received command: \&quot;&quot;..signalName..&quot;\&quot;&quot;)

end</code></pre>
