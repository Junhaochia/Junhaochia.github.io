<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets whether text wrapping should be enabled or disabled on <a href="/page/Category_colon_Label" title="Category:Label">Label</a> and <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a> panels.
Use <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>:<a href="/page/DLabel/SetAutoStretchVertical" title="DLabel/SetAutoStretchVertical">SetAutoStretchVertical</a> to automatically correct vertical size; <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/SizeToContents" title="Panel/SizeToContents">SizeToContents</a> will not set the correct height.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> wrap</span>
</p>
<div style="margin-left: 32px;">
True to enable text wrapping, false otherwise.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates two labels in a panel and sets the text wrapping to false and true respectively.
</p>
<pre><code data-language="lua">-- Background panel
BGPanel = vgui.Create(&quot;DPanel&quot;)
BGPanel:SetSize(300, 130)
BGPanel:Center()
BGPanel:SetBackgroundColor(Color(0, 0, 0))
		
-- Label with no text wrapping
local lbl_nowrap = vgui.Create(&quot;DLabel&quot;, BGPanel)
lbl_nowrap:SetPos(10, 10)
lbl_nowrap:SetSize(280, 50)		
lbl_nowrap:SetFont(&quot;GModNotify&quot;)
lbl_nowrap:SetText(&quot;This is a label that has text wrapping disabled.&quot;)

lbl_nowrap:SetWrap(false)

-- Label with text wrapping
local lbl_wrap = vgui.Create(&quot;DLabel&quot;, BGPanel)
lbl_wrap:SetPos(10, 70)
lbl_wrap:SetSize(280, 50)
lbl_wrap:SetFont(&quot;GModNotify&quot;)
lbl_wrap:SetText(&quot;This is a label that has text wrapping enabled.&quot;)

lbl_wrap:SetWrap(true)</code></pre>
