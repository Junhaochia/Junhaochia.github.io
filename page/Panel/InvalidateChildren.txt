<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Invalidates the layout of this panel object and all its children. This will cause these objects to re-layout immediately, calling <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/PerformLayout" title="PANEL/PerformLayout">PerformLayout</a>. If you want to perform the layout in the next frame, you will have loop manually through all children, and call <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/InvalidateLayout" title="Panel/InvalidateLayout">InvalidateLayout</a> on each.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> recursive=false</span>
</p>
<div style="margin-left: 32px;">
If <i>true</i>, the method will recursively invalidate the layout of all children. Otherwise, only immediate children are affected.</div>
</div>
