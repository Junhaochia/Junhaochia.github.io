<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the minimum dimensions of the panel or object.
</p><p>You can restrict either or both values.
</p><p>Calling the function without arguments will remove the minimum size.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> minW=nil</span>
</p>
<div style="margin-left: 32px;">
The minimum width of the object.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> minH=nil</span>
</p>
<div style="margin-left: 32px;">
The minimum height of the object.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Restricting height but not width
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetSize( 800, 600 )
frame:SetSizable( true )
frame:SetMinimumSize( nil, 300 )
frame:MakePopup()</code></pre>
