<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the foreground color of a panel.
</p><p>For a <a href="/page/Category_colon_Label" title="Category:Label">Label</a> or <a href="/page/Category_colon_RichText" title="Category:RichText">RichText</a>, this is the color of its text.
</p><p>This function calls <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/SetFGColorEx" title="Panel/SetFGColorEx">SetFGColorEx</a> internally.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This doesn't apply to all VGUI elements (such as <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a>) and its function varies between them
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a <a href="/page/Category_colon_Label" title="Category:Label">Label</a> and sets its text color to white.
</p>
<pre><code data-language="lua">local label = vgui.Create( &quot;Label&quot; )

label:SetFGColor( Color( 255, 255, 255, 255 ) )</code></pre>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Sets the foreground color of a <a href="/page/Category_colon_RichText" title="Category:RichText">RichText</a> to match the chat box format.
</p>
<pre><code data-language="lua">-- Create a window frame
TextFrame = vgui.Create(&quot;DFrame&quot;)
TextFrame:SetSize(200, 50)
TextFrame:Center()
TextFrame:SetTitle(&quot;This is a color test&quot;)
TextFrame:MakePopup()

-- RichText panel
local richtext = vgui.Create(&quot;RichText&quot;, TextFrame)
richtext:Dock(FILL)

-- Sample text
richtext:SetText(&quot;Here is some light green text.&quot;)

-- When the panel is ready for layout, make the text light green
function richtext:PerformLayout()
	self:SetFGColor(Color(153, 255, 153))
end</code></pre>
