<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Draws a coloured rectangle to fill the panel object this method is called on. The colour is set using <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/SetDrawColor" title="surface/SetDrawColor">SetDrawColor</a>. This should only be called within the object's <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/Paint" title="PANEL/Paint">Paint</a> or <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/PaintOver" title="PANEL/PaintOver">PaintOver</a> hooks, as a shortcut for <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/DrawRect" title="surface/DrawRect">DrawRect</a>.
</p>
