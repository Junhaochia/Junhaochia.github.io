<p>Sets this panel's convar. When the convar changes this panel will update automatically.
</p><p>For developer implementation, see <a href="/page/Global/Derma_Install_Convar_Functions" title="Global/Derma Install Convar Functions">Derma_Install_Convar_Functions</a>.
</p>

</p>
<div style="margin-left: 32px;">
The console variable to check.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Makes a checkbox linked to the <b>cl_drawhud</b> convar.
</p>
<pre><code data-language="lua">local DermaPanel = vgui.Create( &quot;DFrame&quot; )
DermaPanel:SetPos( 100, 100 )
DermaPanel:SetSize( 300, 200 )
DermaPanel:SetTitle( &quot;My new Derma frame&quot; )
DermaPanel:SetDraggable( true )
DermaPanel:MakePopup()

local Checkbox = vgui.Create( &quot;DCheckBoxLabel&quot;, DermaPanel )
Checkbox:SetConVar( &quot;cl_drawhud&quot; )
Checkbox:SetText( &quot;Enable HUD?&quot; )
Checkbox:SetPos( 5, 25 )
Checkbox:SizeToContents()</code></pre>
