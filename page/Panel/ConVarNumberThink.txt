<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>A think hook for <a href="/page/Category_colon_Panel" title="Category:Panel">Panels</a> using ConVars as a value. Call it in the Think hook. Sets the panel's value should the convar change.
</p><p>This function is best for: checkboxes, sliders, number wangs
</p><p>For a string alternative, see <a href="/page/Panel/ConVarStringThink" title="Panel/ConVarStringThink">Panel.ConVarStringThink</a>.
</p><p><i>Important</i>: Make sure your Panel has a SetValue function, else you may get errors.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>How it should be implemented into your input.
</p>
<pre><code data-language="lua">function PANEL:Think()
	self:ConVarNumberThink()
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Panel's value is changed when the convar changes.</div>
