<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when the panel should generate example use case / example code to use for this panel. Used in the panel opened by <b>derma_controls</b> console command.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> class</span>
</p>
<div style="margin-left: 32px;">
The classname of the panel to generate example for. This will be the class name of your panel.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> dpropertysheet</span>
</p>
<div style="margin-left: 32px;">
A <a href="/page/Category_colon_DPropertySheet" title="Category:DPropertySheet">DPropertySheet</a> to add your example to. See examples below.</div>
</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage of this hook from DButton's code.
</p>
<pre><code data-language="lua">function PANEL:GenerateExample( ClassName, PropertySheet, Width, Height )

	local ctrl = vgui.Create( ClassName )
	ctrl:SetText( &quot;Example Button&quot; )
	ctrl:SetWide( 200 )

	PropertySheet:AddSheet( ClassName, ctrl, nil, true, true )

end

derma.DefineControl( &quot;DButton&quot;, &quot;A standard Button&quot;, PANEL, &quot;DLabel&quot; )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">A tab named "DButton" will appear in <b>derma_controls</b> menu.</div>
