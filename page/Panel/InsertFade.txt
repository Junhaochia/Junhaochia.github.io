<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Begins a text fade for a <a href="/page/Category_colon_RichText" title="Category:RichText">RichText</a> element where the last appended text segment is fully faded out after a specific amount of time, at a specific speed.
</p><p>The alpha of the text at any given time is determined by the text's base alpha * ((<i>sustain</i> - <a href="/page/Global/CurTime" title="Global/CurTime">CurTime</a>) / <i>length</i>) where <a href="/page/Global/CurTime" title="Global/CurTime">CurTime</a> is added to <i>sustain</i> when this method is called.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> sustain</span>
</p>
<div style="margin-left: 32px;">
The number of seconds the text remains visible.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> length</span>
</p>
<div style="margin-left: 32px;">
<p>The number of seconds it takes the text to fade out.
</p><p>If set <b>lower</b> than <i>sustain</i>, the text will not begin fading out until (<i>sustain</i> - <i>length</i>) seconds have passed.
</p><p>If set <b>higher</b> than <i>sustain</i>, the text will begin fading out immediately at a fraction of the base alpha.
</p>
If set to <b>-1</b>, the text doesn't fade out.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a Rich Text panel that sustains visibility for 6 seconds with a 2 second long fade-out.
</p>
<pre><code data-language="lua">-- Create a window frame
TextFrame = vgui.Create(&quot;DFrame&quot;)
TextFrame:SetSize(200, 200)
TextFrame:Center()
TextFrame:SetTitle(&quot;Fading Text&quot;)
TextFrame:MakePopup()

-- RichText panel
local richtext = vgui.Create(&quot;RichText&quot;, TextFrame)
richtext:Dock(FILL)

-- Sample text
richtext:SetText(&quot;This is an example of a Rich Text panel using a fade-out with:\n\n6 seconds of sustain\n\n2 second fade-out length&quot;)
	
-- When the panel is ready for layout, begin the fade
function richtext:PerformLayout()
	
	self:SetFontInternal(&quot;Trebuchet18&quot;)
	self:SetBGColor(Color(64, 64, 92))
	
	-- Wait 6 seconds, then fade out in 2 seconds
	self:InsertFade(6, 2)
	
end</code></pre>

<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Create a Rich Text panel where Dr. Kleiner reads a fading message in sync with text.
</p>
<pre><code data-language="lua">-- Create a window frame
TextFrame = vgui.Create(&quot;DFrame&quot;)
TextFrame:SetSize(350, 100)
TextFrame:Center()
TextFrame:SetTitle(&quot;Kleiner says:&quot;)
TextFrame:MakePopup()

-- RichText panel
local richtext = vgui.Create(&quot;RichText&quot;, TextFrame)
richtext:Dock(FILL)	

-- Red text
richtext:InsertColorChange(200, 60, 32, 255)
richtext:SetVerticalScrollbarEnabled(false)
	
local words = {&quot;There's&quot;, &quot;only&quot;, &quot;one&quot;, &quot;hedy...&quot;}
local delay = 0

-- Display each word in half second interval
for w, txt in pairs(words) do

	if(w == 1) then delay = 0.2
	else delay = (w-1)*0.45 end

	timer.Simple(delay, function()
	
		richtext:AppendText(txt..&quot; &quot;)
		richtext:InsertFade(2, 1)	-- Sustain for 2 seconds while fading out after 1 second
		
		richtext:SetBGColor(Color(0, 0, 0))
		richtext:SetFontInternal(&quot;DermaLarge&quot;)
	
	end)

end

-- Kleiner read along
LocalPlayer():EmitSound(&quot;vo/k_lab2/kl_onehedy.wav&quot;)</code></pre>
