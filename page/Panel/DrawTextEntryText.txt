<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Used to draw the text in a <a href="/page/Category_colon_DTextEntry" title="Category:DTextEntry">DTextEntry</a> within a derma skin. This should be called within the <a href="/index.php?title=Category:SKIN_Hooks&amp;action=edit&amp;redlink=1" class="new" title="Category:SKIN Hooks (page does not exist)">SKIN</a>:<a href="/index.php?title=SKIN/PaintTextEntry&amp;action=edit&amp;redlink=1" class="new" title="SKIN/PaintTextEntry (page does not exist)">PaintTextEntry</a> skin hook.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>The paint function used in the default derma skin.
</p>
<pre><code data-language="lua">function SKIN:PaintTextEntry( panel, w, h )

	if ( panel.m_bBackground ) then
	
		if ( panel:GetDisabled() ) then
			self.tex.TextBox_Disabled( 0, 0, w, h )
		elseif ( panel:HasFocus() ) then
			self.tex.TextBox_Focus( 0, 0, w, h )
		else
			self.tex.TextBox( 0, 0, w, h )
		end
	
	end
	
	panel:DrawTextEntryText( panel.m_colText, panel.m_colHighlight, panel.m_colCursor )
	
end</code></pre>
