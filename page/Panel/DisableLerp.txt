<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Resets the panel object's <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/SetPos" title="Panel/SetPos">SetPos</a> method and removes its animation table (<i>Panel.LerpAnim</i>). This effectively undoes the changes made by <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/LerpPositions" title="Panel/LerpPositions">LerpPositions</a>.
</p><p>In order to use Lerp animation again, you must call <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/Stop" title="Panel/Stop">Stop</a> before setting its <i>SetPosReal</i> property to 'nil'. See the example below.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a function for changing the Lerp animation speed of a panel.
</p>
<pre><code data-language="lua">function ChangeLerpSpeed( pnl, newSpeed, ease )
	
	if&#160;!ispanel(pnl) or&#160;!IsValid(pnl) then return end -- Make sure panel is valid
	
	pnl:DisableLerp()
	pnl:Stop()
	pnl.SetPosReal = nil
	
	pnl:LerpPositions( newSpeed, ease )
	
end</code></pre>
