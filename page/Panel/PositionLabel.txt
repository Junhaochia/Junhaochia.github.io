<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the width and position of a <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a> and places the passed panel object directly to the right of it. Returns the <i>y</i> value of the bottom of the tallest object. The panel on which this method is run is not relevant; only the passed objects are affected.
</p>

</p>
<div style="margin-left: 32px;">
The width to set the label to.</div>
</div>
