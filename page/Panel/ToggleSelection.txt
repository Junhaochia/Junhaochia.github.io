<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Toggles the selected state of a selectable panel object. This functionality is set with <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/SetSelectable" title="Panel/SetSelectable">SetSelectable</a> and checked with <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/IsSelectable" title="Panel/IsSelectable">IsSelectable</a>. To check whether the object is selected or not, <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/IsSelected" title="Panel/IsSelected">IsSelected</a> is used.
</p>
