<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Stores a string in the named cookie using <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/GetCookieName" title="Panel/GetCookieName">GetCookieName</a> as prefix.
</p><p>You can also retrieve and modify this cookie by using the <a href="/page/Category_colon_cookie" title="Category:cookie">cookie library</a>. Cookies are stored in this format:
<code>panelCookieName.cookieName</code>
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>The panel's cookie name MUST be set for this function to work. See <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/SetCookieName" title="Panel/SetCookieName">SetCookieName</a>.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> cookieName</span>
</p>
<div style="margin-left: 32px;">
The unique name used to retrieve the cookie later.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> value</span>
</p>
<div style="margin-left: 32px;">
The value to store in the cookie. This can be retrieved later as a <a href="/page/Category_colon_string" title="Category:string">string</a> or <a href="/page/Category_colon_number" title="Category:number">number</a>.</div>
</div>
