<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called whenever the panel should be drawn.
</p><p>You can create panels with a customized appearance by overriding their Paint() function, which will prevent the default appearance from being drawn.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Render operations from the <a href="/page/Category_colon_surface" title="Category:surface">surface library</a> (and consequentially the <a href="/page/Category_colon_draw" title="Category:draw">draw library</a>) are always offset by the global position of this panel, as seen in the example below
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This hook will not run if the panel is completely off the screen. The hook will still run however if any parts of the panel are still on screen.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DPanel and overrides its Paint() function to draw a 100x100 pixel black rounded box in the center of the screen.
</p>
<pre><code data-language="lua">local panel = vgui.Create( &quot;DPanel&quot; )
panel:SetSize( 100, 100 )
panel:SetPos( ScrW() / 2 - 50, ScrH() / 2 - 50 )

function panel:Paint( w, h )
    draw.RoundedBox( 8, 0, 0, w, h, Color( 0, 0, 0 ) )
end</code></pre>
