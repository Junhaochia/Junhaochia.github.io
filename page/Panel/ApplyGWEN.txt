<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Used by <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/LoadGWENFile" title="Panel/LoadGWENFile">LoadGWENFile</a> and <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/LoadGWENString" title="Panel/LoadGWENString">LoadGWENString</a> to apply a GWEN controls table to a panel object.
</p><p>You can do this manually using <a href="/page/Category_colon_file" title="Category:file">file</a>.<a href="/page/file/Read" title="file/Read">Read</a> and <a href="/page/Category_colon_util" title="Category:util">util</a>.<a href="/page/util/JSONToTable" title="util/JSONToTable">JSONToTable</a> to import and create a GWEN table structure from a <i>.gwen</i> file.
