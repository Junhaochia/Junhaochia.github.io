<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Starts the insertion of clickable text for a <a href="/page/Category_colon_RichText" title="Category:RichText">RichText</a> element. Any text appended with <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/AppendText" title="Panel/AppendText">AppendText</a> between this call and <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/InsertClickableTextEnd" title="Panel/InsertClickableTextEnd">InsertClickableTextEnd</a> will become clickable text.
</p><p>The hook <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/ActionSignal" title="PANEL/ActionSignal">ActionSignal</a> is called when the text is clicked, with "TextClicked" as the signal name and <i>signalValue</i> as the signal value.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>The clickable text is a separate Derma panel which will not inherit the current font from the <i>RichText</i>.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> signalValue</span>
</p>
<div style="margin-left: 32px;">
The text passed as the action signal's value.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a panel with some information on Rich Text panels along with a click-able link to the <a href="/page/Category_colon_RichText" title="Category:RichText">RichText</a> page.
</p>
<pre><code data-language="lua">-- Create a window frame
TextFrame = vgui.Create(&quot;DFrame&quot;)
TextFrame:SetSize(250, 150)
TextFrame:Center()
TextFrame:SetTitle(&quot;RichText&quot;)
TextFrame:MakePopup()

-- RichText panel
local richtext = vgui.Create(&quot;RichText&quot;, TextFrame)
richtext:Dock(FILL)

-- First segment
richtext:InsertColorChange(255, 255, 255, 255)
richtext:AppendText(&quot;This is a Rich Text panel — a panel used in Source MP's default chat box and developer console.\n\nSee the &quot;)

-- Second segment
richtext:InsertColorChange(192, 192, 255, 255)
richtext:InsertClickableTextStart(&quot;OpenWiki&quot;)	-- Make incoming text fire the &quot;OpenWiki&quot; value when clicked
richtext:AppendText(&quot;Garry's Mod Wiki&quot;)
richtext:InsertClickableTextEnd()	-- End clickable text here

-- Third segment
richtext:InsertColorChange(255, 255, 255, 255)
richtext:AppendText(&quot; for information on how to use a Rich Text panel.&quot;)

-- Background color
function richtext:PerformLayout() self:SetBGColor(Color(32, 32, 46)) end

-- Handle any commands we get from the panel
function richtext:ActionSignal(signalName, signalValue)

	-- Some clickable text was clicked
	if (signalName == &quot;TextClicked&quot;) then
	
		-- Open the wiki
		if (signalValue == &quot;OpenWiki&quot;) then
		
			gui.OpenURL(&quot;http://wiki.garrysmod.com/page/Category_colon_RichText&quot;)
		
		end
	
	end

end</code></pre>
