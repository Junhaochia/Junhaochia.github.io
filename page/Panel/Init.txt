<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when the panel is created. This is called for each base type that the panel has.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Shows how this method is called recursively for each base type a panel has.
</p>
<pre><code data-language="lua">local BASE = {}
function BASE:Init()
	print(&quot;Base Init Called&quot;)
end


local PANEL = {}
function PANEL:Init()
	print(&quot;Panel Init Called&quot;)
end


vgui.Register(&quot;MyBase&quot;, BASE, &quot;DFrame&quot;)
vgui.Register(&quot;MyPanel&quot;, PANEL, &quot;MyBase&quot;)

local panel = vgui.Create(&quot;MyPanel&quot;)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Base Init Called
Panel Init Called</div>
