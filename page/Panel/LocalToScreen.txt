<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets the absolute screen position of the position specified relative to the panel.
</p><p>See also <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/ScreenToLocal" title="Panel/ScreenToLocal">ScreenToLocal</a>.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>This function uses a cached value for the screen position of the panel, computed at the end of the last VGUI Think/Layout pass.<br />
ie. inaccurate results may be returned if the panel or any of its ancestors have been repositioned outside of <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/Think" title="PANEL/Think">Think</a> or <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/PerformLayout" title="PANEL/PerformLayout">PerformLayout</a> within the last frame.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>If the panel uses <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/Dock" title="Panel/Dock">Dock</a>, this function will return 0, 0 when the panel was created. The position will be updated in the next frame.
</p>
</td></tr></table>
