<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Allows the panel to receive drag and drop events. Can be called multiple times with different names to receive multiple different draggable panel events.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> name</span>
</p>
<div style="margin-left: 32px;">
Name of DnD panels to receive. This is set on the drag'n'drop-able panels via <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/Droppable" title="Panel/Droppable">Droppable</a></div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> func</span>
</p>
<div style="margin-left: 32px;">
<p>This function is called whenever a panel with valid name is hovering above and dropped on this panel. It has next arguments:
</p>
<ul><li> <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> pnl - The receiver panel
</li><li> <a href="/page/Category_colon_table" title="Category:table">table</a> tbl - A table of panels dropped onto receiver panel
</li><li> <a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> dropped - False if hovering over, true if dropped onto
</li><li> <a href="/page/Category_colon_number" title="Category:number">number</a> menuIndex - Index of clicked menu item from third argument of <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<strong class="selflink">Receiver</strong>
</li><li> <a href="/page/Category_colon_number" title="Category:number">number</a> x - Cursor pos, relative to the receiver
</li><li> <a href="/page/Category_colon_number" title="Category:number">number</a> y - Cursor pos, relative to the receiver</div>
</li></ul>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> menu</span>
</p>
<div style="margin-left: 32px;">
A table of strings that will act as a menu if drag'n'drop was performed with a right click</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>A very simple drag'n'drop example without using <a href="/page/Category_colon_DDragBase" title="Category:DDragBase">DDragBase</a>.
</p>
<pre><code data-language="lua">local function DoDrop( self, panels, bDoDrop, Command, x, y )
	if ( bDoDrop ) then
		for k, v in pairs( panels ) do
			self:AddItem( v )
		end
	end
end

concommand.Add( &quot;test2&quot;, function()

	local frame = vgui.Create( &quot;DFrame&quot; )
	frame:SetSize( 500, 300 )
	frame:SetTitle( &quot;Frame&quot; )
	frame:MakePopup()
	frame:Center()

	local left = vgui.Create( &quot;DScrollPanel&quot;, frame )
	left:Dock( LEFT )
	left:SetWidth( frame:GetWide() / 2 - 7 )
	left:SetPaintBackground( true )
	left:DockMargin( 0, 0, 4, 0 )
	left:Receiver( &quot;myDNDname&quot;, DoDrop ) -- Make the panel a receiver for drag and drop events

	local right = vgui.Create( &quot;DScrollPanel&quot;, frame )
	right:Dock( FILL )
	right:SetPaintBackground( true )
	right:Receiver( &quot;myDNDname&quot;, DoDrop )

	for i = 1, 30 do
		local but = vgui.Create( &quot;DButton&quot; )
		but:SetText( i )
		but:SetSize( 36, 24 )
		but:Dock( TOP )
		but:Droppable( &quot;myDNDname&quot; ) -- make the panel be able to be drag'n'dropped onto other panels
		right:AddItem( but )
	end

end )</code></pre>
