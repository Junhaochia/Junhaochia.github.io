<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the number of lines in a <a href="/page/Category_colon_RichText" title="Category:RichText">RichText</a>. You must wait a couple frames before calling this after using <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/AppendText" title="Panel/AppendText">AppendText</a> or <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/SetText" title="Panel/SetText">SetText</a>, otherwise it will return the number of text lines before the text change.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Even though this function can be called on any panel, it will only work with <a href="/page/Category_colon_RichText" title="Category:RichText">RichText</a>
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a rich text panel with a block of text and prints out the number of text lines before and after <i>PerformLayout</i> is called.
</p>
<pre><code data-language="lua">-- Create a window frame
TextFrame = vgui.Create(&quot;DFrame&quot;)
TextFrame:SetSize(200, 224)
TextFrame:Center()
TextFrame:SetTitle(&quot;Generic Frame&quot;)

-- RichText panel
local richtext = vgui.Create(&quot;RichText&quot;, TextFrame)
richtext:Dock(FILL)

-- Throw some text in the panel
richtext:SetText(&quot;This is a block of text demonstrating how line wrapping and panel size relates to the number of lines shown inside of a RichText panel.&quot;)

-- Keep track of PerformLayout calls
richtext.layoutCount = 0

-- Custom function for this example
function richtext:NumLinesExample()
	print(&quot;PerformLayout called &quot;..self.layoutCount..&quot; times: &quot;..richtext:GetNumLines()..&quot; line(s) returned&quot;)	
end

-- Print # of lines before any layouts
richtext:NumLinesExample()

-- Render update
function richtext:PerformLayout()

	self.layoutCount = self.layoutCount + 1
	self:NumLinesExample()	-- Print current # of lines

end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">The panel shows 5 lines of text, but the number 5 isn't returned until PerformLayout has been called 2 times.
<pre><code data-language="lua">PerformLayout called 0 times: 1 line(s) returned
PerformLayout called 1 times: 1 line(s) returned
PerformLayout called 2 times: 5 line(s) returned</code></pre>
