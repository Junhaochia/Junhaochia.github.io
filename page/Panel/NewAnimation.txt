<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> length</span>
</p>
<div style="margin-left: 32px;">
The length of the animation in seconds.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> delay=0</span>
</p>
<div style="margin-left: 32px;">
The delay before the animation starts.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> ease=-1</span>
</p>
<div style="margin-left: 32px;">
<p>The power/index to use for easing.
</p>
<ul><li>Positive values greater than 1 will ease in; the higher the number, the sharper the curve's gradient (less linear).
</li><li>A value of 1 removes all easing.
</li><li>Positive values between 0 and 1 ease out; values closer to 0 increase the curve's gradient (less linear).
</li><li>A value of 0 will break the animation and should be avoided.
</li><li>Any value less than zero will ease in/out; the value has no effect on the gradient.</div>
</li></ul>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> callback=nil</span>
</p>
<div style="margin-left: 32px;">
<p>The function to be called when the animation ends. Arguments passed are:
</p>
<ul><li> <a href="/page/Category_colon_table" title="Category:table">table</a> animTable - The <a href="/page/Structures/AnimationData" title="Structures/AnimationData">AnimationData structure</a> that was used.
</li><li> <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> tgtPanel - The panel object that was animated.</div>
</li></ul>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>

<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
<p>Partially filled <a href="/page/Structures/AnimationData" title="Structures/AnimationData">AnimationData structure</a> with members:
</p>
<ul><li><a href="/page/Category_colon_number" title="Category:number">number</a> EndTime - Equal to <i>length</i> and <i>delay</i> arguments added together, plus either the <a href="/page/Global/SysTime" title="Global/SysTime">SysTime</a> if there is no other animation queued or the end time of the last animation in the queue.
</li><li><a href="/page/Category_colon_number" title="Category:number">number</a> StartTime - Equal to the <i>delay</i> argument, plus either the <a href="/page/Global/SysTime" title="Global/SysTime">SysTime</a> if there is no other animation queued or the end time of the last animation in the queue.
</li><li><a href="/page/Category_colon_number" title="Category:number">number</a> Ease - Equal to the <i>ease</i> argument.
</li><li><a href="/page/Category_colon_function" title="Category:function">function</a> OnEnd - Equal to the <i>callback</i> argument.</div>
</li></ul>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example on how to use this function, makes a button go around in a circle in a DFrame.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetSize( 500, 500 )
frame:Center()
frame:MakePopup()

local butt = frame:Add( &quot;DButton&quot; )
butt:SetPos( 5, 30 )
butt:SetSize( 100, 40 )

function butt:doAnim()
	local anim = self:NewAnimation( 10, 0, 1, function( anim, pnl )
		self:doAnim()
	end )

	anim.Think = function( anim, pnl, fraction )
		local radius = 200
		pnl:SetPos( 250 + math.sin( Lerp( fraction, -math.pi, math.pi ) ) * radius - pnl:GetWide() / 2,
					250 + math.cos( Lerp( fraction, -math.pi, math.pi ) ) * radius - pnl:GetTall() / 2 )

		pnl:SetText( &quot;Frac: &quot; .. fraction .. &quot;\nTime: &quot; .. ( SysTime() - anim.StartTime ) )
	end
end
butt:doAnim()</code></pre>
