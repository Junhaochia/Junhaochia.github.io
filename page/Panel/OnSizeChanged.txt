<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called just after the panel size changes.
</p><p>All size functions will return the new values when this hook is called.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Changing the panel size in this hook will cause an infinite loop!
</p>
</td></tr></table>
