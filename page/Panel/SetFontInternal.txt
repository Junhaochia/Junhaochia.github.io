<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the font used to render this panel's text.
</p><p>To retrieve the font used by a panel, call <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/GetFont" title="Panel/GetFont">GetFont</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> fontName</span>
</p>
<div style="margin-left: 32px;">
<p>The name of the font.
</p><p>See <a href="/page/Default_Fonts" title="Default Fonts">here</a> for a list of existing fonts.
</p>
Alternatively, use <a href="/page/Category_colon_surface" title="Category:surface">surface</a>.<a href="/page/surface/CreateFont" title="surface/CreateFont">CreateFont</a> to create your own custom font.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Sets the font of a <a href="/page/Category_colon_RichText" title="Category:RichText">RichText</a> element to match the chat box font.
</p>
<pre><code data-language="lua">-- Create a window frame
TextFrame = vgui.Create(&quot;DFrame&quot;)
TextFrame:SetSize(200, 100)
TextFrame:Center()
TextFrame:SetTitle(&quot;This is a font test&quot;)
TextFrame:MakePopup()

-- RichText panel
local richtext = vgui.Create(&quot;RichText&quot;, TextFrame)
richtext:Dock(FILL)

-- Sample text
richtext:SetText(&quot;This is a sample of text using the chat box font.&quot;)

-- Ensure font and text color changes are applied
function richtext:PerformLayout()

	self:SetFontInternal(&quot;ChatFont&quot;)
	self:SetFGColor(Color(255, 255, 255))
	
end</code></pre>
