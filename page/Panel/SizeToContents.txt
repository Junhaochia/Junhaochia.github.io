<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Resizes the panel so that its width and height fit all of the content inside.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Only works on <a href="/page/Category_colon_Label" title="Category:Label">Label</a> derived panels such as <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabel</a> by default, and on any panel that manually implemented the <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<strong class="selflink">SizeToContents</strong> method, such as <a href="/page/Category_colon_DNumberWang" title="Category:DNumberWang">DNumberWang</a> and <a href="/page/Category_colon_DImage" title="Category:DImage">DImage</a>.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>You must call this function <b>AFTER</b> setting text/font, adjusting child panels or otherwise altering the panel.
</p>
</td></tr></table>
