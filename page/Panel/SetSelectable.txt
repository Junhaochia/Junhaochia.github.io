<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets whether the panel object can be selected or not (like icons in the Spawn Menu, holding <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">⇧ Shift</span>). If enabled, this will affect the function of a <a href="/page/Category_colon_DButton" title="Category:DButton">DButton</a> whilst <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">⇧ Shift</span> is pressed. <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/SetSelected" title="Panel/SetSelected">SetSelected</a> can be used to select/deselect the object.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> selectable</span>
</p>
<div style="margin-left: 32px;">
Whether the panel object should be selectable or not.</div>
</div>
