<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the bitwise OR of all values specified.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> value1</span>
</p>
<div style="margin-left: 32px;">
The first value.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_vararg" title="Category:vararg">vararg</a> ...</span>
</p>
<div style="margin-left: 32px;">
Extra values to be evaluated. (must all be numbers)</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
The bitwise OR result between all numbers.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Performs the bitwise OR operation between three values
</p>
<pre><code data-language="lua">local a = math.BinToInt(&quot;1000&quot;)
local b = math.BinToInt(&quot;0100&quot;)
local c = math.BinToInt(&quot;0001&quot;)

local result = bit.bor(a, b, c)

print(math.IntToBin(result))
print(result)</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">1101
13</code></pre>
