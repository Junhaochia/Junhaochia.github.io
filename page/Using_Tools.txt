<h1> <span class="mw-headline" id="Tools"> Tools </span></h1>
<p>Tools are used for construction. With it, you can weld (or stick) props together, rope one prop to another, give props another color or material and remove props. Tools can also be used for posing purposes.
</p><p><b>For a list of tools, please scroll to the bottom of the page.</b>
</p>
<h2> <span class="mw-headline" id="Using_tools"> Using tools </span></h2>
<ol><li> Select your tool. Open your spawn menu by holding down <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">Q</span> and click on the tool you want to use. Here, we'll be using the <b>Winch</b> tool.<div class="center"><div class="floatnone"><a href="/page/File:tutorial_tools_1.png" class="image"><img alt="tutorial tools 1.png" src="/images/7/7e/tutorial_tools_1.png" width="720" height="405" /></a></div></div><br />
</li><li> Configure your tool. Most tools come with one or more settings; you can set these up however you like! It's worth noting that you can type a number into the slider boxes.<div class="center"><div class="floatnone"><a href="/page/File:tutorial_tools_2.png" class="image"><img alt="tutorial tools 2.png" src="/images/thumb/d/dc/tutorial_tools_2.png/720px-tutorial_tools_2.png" width="720" height="405" /></a></div></div><br />
</li><li> Aim at the first prop and <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;"><span title="Left mouse button"><img alt="LMB" src="/images/f/ff/Lmb.png" width="16" height="16" /></span></span> click!<div class="center"><div class="floatnone"><a href="/page/File:tutorial_tools_3.png" class="image"><img alt="tutorial tools 3.png" src="/images/e/eb/tutorial_tools_3.png" width="720" height="405" /></a></div></div><br />
</li><li> Aim at the second prop and <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;"><span title="Left mouse button"><img alt="LMB" src="/images/f/ff/Lmb.png" width="16" height="16" /></span></span> click!<div class="center"><div class="floatnone"><a href="/page/File:tutorial_tools_4.png" class="image"><img alt="tutorial tools 4.png" src="/images/thumb/2/2f/tutorial_tools_4.png/720px-tutorial_tools_4.png" width="720" height="405" /></a></div></div><br />
</li><li> Sit back and admire your mastery of roping things together. <div class="center"><div class="floatnone"><a href="/page/File:tutorial_tools_5.png" class="image"><img alt="tutorial tools 5.png" src="/images/1/15/tutorial_tools_5.png" width="720" height="405" /></a></div></div>
</li></ol>
<h2> <span class="mw-headline" id="Notes"> Notes </span></h2>
<ul><li> Not all tools require two props, some only work on one prop, some can only be applied to the world and some require 4 points (The pulley tool, for example).
</li><li> Servers may limit the usage of props to a certain maximum or not at all. Keep this in mind when building your vehicle.
</li><li> As of now, there is no maximum number of welds, so you can weld forever.
</li></ul>
