<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Converts the specified screen position to a <b>direction</b> vector local to the player's view. A related function is <a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a>:<a href="/page/Vector/ToScreen" title="Vector/ToScreen">ToScreen</a>, which translates a 3D position to a screen coordinate.
</p><p><a href="/page/Category_colon_util" title="Category:util">util</a>.<a href="/page/util/AimVector" title="util/AimVector">AimVector</a> is a more generic version of this, using a custom view instead of the player's current view.
</p>

</p>

</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This will do a trace from the mouse position to the world
</p>
<pre><code data-language="lua">local tr = util.QuickTrace(LocalPlayer():GetShootPos(), gui.ScreenToVector(gui.MousePos()),LocalPlayer())
print(tr.HitPos)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">The worldpos the cursor is pointing at will be printed</div>
