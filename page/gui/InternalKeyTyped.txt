<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Simulates an ASCII symbol writing.
Use to write text in the chat or in VGUI.
Doesn't work while the main menu is open!
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> code</span>
</p>
<div style="margin-left: 32px;">
ASCII code of symbol, see <a rel="nofollow" class="external free" href="http://www.mikroe.com/img/publication/spa/pic-books/programming-in-basic/chapter/04/fig4-24.gif">http://www.mikroe.com/img/publication/spa/pic-books/programming-in-basic/chapter/04/fig4-24.gif</a></div>
</div>
