<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Opens specified URL in the steam overlay browser. The URL has to start with either http:// or https://
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>User will be asked for confirmation before the website will open.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>You can't click the confirmation if a modal panel has focus.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3383">#3383</a></b>
</p>
</td></tr></table>

<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> url</span>
</p>
<div style="margin-left: 32px;">

</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Opens a page when a button is clicked.
</p>
<pre><code data-language="lua">local button = vgui.Create( &quot;DButton&quot; )
button:SetSize( 125, 90 )
button:Center() 
button:SetText( &quot;Join our Steam Group!&quot; )
button.DoClick = function()
    gui.OpenURL( &quot;http://steamcommunity.com/groups/glua&quot; )
end</code></pre>
