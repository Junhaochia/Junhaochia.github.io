<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DCheckBox and sets its initial value.
</p>
<pre><code data-language="lua">local window = vgui.Create( &quot;DFrame&quot; )
window:SetSize( 300, 200 )
window:Center()
window:MakePopup()

local checkbox = window:Add( &quot;DCheckBox&quot; ) -- Create the checkbox
checkbox:SetPos( 25, 50 ) -- Set the position
checkbox:SetValue( true ) -- Initial &quot;ticked&quot; value</code></pre>
