<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets whether or not the items should be sorted alphabetically in the dropdown menu of the <a href="/page/Category_colon_DComboBox" title="Category:DComboBox">DComboBox</a>. If set to false, items will appear in the order they were added by <a href="/page/Category_colon_DComboBox" title="Category:DComboBox">DComboBox</a>:<a href="/page/DComboBox/AddChoice" title="DComboBox/AddChoice">AddChoice</a> calls.
</p><p>This is enabled by default.
</p>

<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> sort</span>
</p>
<div style="margin-left: 32px;">
true to enable, false to disable</div>
