<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the text shown in the combo box when the menu is not collapsed.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> value</span>
</p>
<div style="margin-left: 32px;">
The text in the DComboBox.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>A simple feedback combo box which has the value set to a thank you message once a choice is clicked.
</p>
<pre><code data-language="lua">BGPanel = vgui.Create(&quot;DPanel&quot;)
BGPanel:SetPos(20, 20)
BGPanel:SetSize(200, 30)

local cbox = vgui.Create(&quot;DComboBox&quot;, BGPanel)
cbox:SetPos(5, 5)
cbox:SetSize(190, 20)

cbox:SetValue(&quot;What do you think of this server?&quot;)

-- Responses
cbox:AddChoice(&quot;It's the best server of all time!&quot;)
cbox:AddChoice(&quot;It's pretty good.&quot;)
cbox:AddChoice(&quot;It's okay.&quot;)
cbox:AddChoice(&quot;It's not that good.&quot;)
cbox:AddChoice(&quot;Don't bother me with this.&quot;)

function cbox:OnSelect(index, value, data)

	-- Clear combo box and set a thank you message
	self:Clear()
	self:SetText(&quot;Thank you for your feedback!&quot;)
	
	-- Here you would send the feedback to the server using a net message
	-- The choice is stored in the 'data' variable

end</code></pre>
