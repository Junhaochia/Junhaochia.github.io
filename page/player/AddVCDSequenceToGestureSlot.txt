<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Plays a sequence directly from a sequence number, similar to <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/AnimRestartGesture" title="Player/AnimRestartGesture">AnimRestartGesture</a>. This function has the advantage to play sequences that haven't been bound to an existing <a href="/page/Enums/ACT" title="Enums/ACT">ACT_ Enums</a>
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> slot</span>
</p>
<div style="margin-left: 32px;">
Gesture slot using <a href="/page/Enums/GESTURE_SLOT" title="Enums/GESTURE SLOT">GESTURE_SLOT_ Enums</a></div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> sequenceId</span>
</p>
<div style="margin-left: 32px;">
The sequence ID to play, can be retrieved with <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/LookupSequence" title="Entity/LookupSequence">LookupSequence</a>.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> cycle</span>
</p>
<div style="margin-left: 32px;">
The cycle to start the animation at, ranges from 0 to 1.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> autokill=false</span>
</p>
<div style="margin-left: 32px;">
If the animation should not loop. true = stops the animation, false = the animation keeps playing.</div>
</div>
