<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets all the current players in the server (not including connecting clients).
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function returns bots as well as human players. See <a href="/page/Category_colon_player" title="Category:player">player</a>.<a href="/page/player/GetBots" title="player/GetBots">GetBots</a> and <a href="/page/Category_colon_player" title="Category:player">player</a>.<a href="/page/player/GetHumans" title="player/GetHumans">GetHumans</a>.
</p>
</td></tr></table>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
All <a href="/page/Category_colon_Player" title="Category:Player">Players</a> currently in the server.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints all the players currently in the server.
</p>
<pre><code data-language="lua">PrintTable( player.GetAll() )</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">1 = [Player][1][Player1]
2 = [Player][2][Bot01]
3 = [Player][3][Bot02]</code></pre>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Prints the number of players in the server. The same output can be achieved more efficiently with <a href="/page/Category_colon_player" title="Category:player">player</a>.<a href="/page/player/GetCount" title="player/GetCount">GetCount</a>.
</p>
<pre><code data-language="lua">print(#player.GetAll())</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">3</code></pre>
