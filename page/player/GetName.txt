<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the player's name, this is an alias of <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/Nick" title="Player/Nick">Nick</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function overrides <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetName" title="Entity/GetName">GetName</a> (in the Lua metatable, not in c++), keep it in mind when dealing with <a href="/page/Category_colon_ents" title="Category:ents">ents</a>.<a href="/page/ents/FindByName" title="ents/FindByName">FindByName</a> or any engine function which requires the mapping name.
</p>
</td></tr></table>
