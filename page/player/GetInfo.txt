<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Retrieves the value of a client-side <a href="/page/Category_colon_ConVar" title="Category:ConVar">ConVar</a>. The <a href="/page/Category_colon_ConVar" title="Category:ConVar">ConVar</a> must have a <a href="/page/Enums/FCVAR" title="Enums/FCVAR">FCVAR_USERINFO</a> flag for this to work.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>The returned value is truncated to 31 bytes.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> cVarName</span>
</p>
<div style="margin-left: 32px;">
The name of the client-side <a href="/page/Category_colon_ConVar" title="Category:ConVar">ConVar</a></div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a></span>
</p>
<div style="margin-left: 32px;">
The value of the <a href="/page/Category_colon_ConVar" title="Category:ConVar">ConVar</a></div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates clientside <a href="/page/Category_colon_ConVar" title="Category:ConVar">ConVar</a> 'Apple' and retrieves value of it.
</p>
<pre><code data-language="lua">if CLIENT then
	CreateConVar( &quot;Apple&quot;, &quot;ILikeApples&quot;, FCVAR_USERINFO )
else
	MsgN( Entity( 1 ):GetInfo( &quot;Apple&quot; ) )
end</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">ILikeApples</code></pre>
