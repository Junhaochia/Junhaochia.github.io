<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the <b>desired</b> view offset which equals the difference between the players actual position and their view when standing.
</p><p>If you want to set <b>actual</b> view offset, use <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/SetCurrentViewOffset" title="Player/SetCurrentViewOffset">SetCurrentViewOffset</a>
</p><p>See also <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/SetViewOffsetDucked" title="Player/SetViewOffsetDucked">SetViewOffsetDucked</a> for <b>desired</b> view offset when crouching.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> viewOffset</span>
</p>
<div style="margin-left: 32px;">
The new desired view offset when standing.</div>
</div>
