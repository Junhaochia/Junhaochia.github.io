<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the last command which was sent by the specified player. Can only be used in a <a href="/page/Category_colon_Predicted_Hooks" title="Category:Predicted Hooks">Predicted Hook</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>

See the note on the page.
</p>
