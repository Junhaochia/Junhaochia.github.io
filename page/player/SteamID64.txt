<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the player's 64bit SteamID aka CommunityID.
</p><p>In singleplayer, this will return no value serverside.
</p><p>In a multirun environment, this will return no value serverside for all "copies" of a player.
</p><p>For bots, this will return 90071996842377216 (equivalent to STEAM_0:0:0) for the first bot to join.<br />
For each additional bot, the number increases by 1. So the next bot will be 90071996842377217 (STEAM_0:1:0) then 90071996842377218 (STEAM_0:0:1) and so on.
</p><p>On the client it returns no value for bots.
</p><p>Use <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/AccountID" title="Player/AccountID">AccountID</a> for a shorter version of the SteamID.
</p>
