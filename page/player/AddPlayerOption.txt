<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets up the voting system for the player.
This is a really barebone system. By calling this a vote gets started, when the player presses 0-9 the callback function gets called along with the key the player pressed. Use the draw callback to draw the vote panel.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> name</span>
</p>
<div style="margin-left: 32px;">
Name of the vote</div>
</div>

</p>

Time until the vote expires</div>
</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Simple example. Prints player's choice in chat.
</p>
<pre><code data-language="lua">function AfterChoice(num) -- This is callback after we press number (Argument #3)
	chat.AddText(&quot;Your rate is &quot;..num..&quot;. Thanks!&quot;) 
	return true -- Return true to close vote
end

function VisualVote() -- This is drawing function (Argument #4)
    draw.RoundedBox(4,ScrW()/2-300,ScrH()/2-25,600,50,Color(0,0,0,200))
    draw.SimpleText(&quot;Rate our server by scale of zero to nine. Use number line to vote.&quot;,&quot;Trebuchet24&quot;,ScrW()/2,ScrH()/2,Color(255,255,255),1,1)
end

LocalPlayer():AddPlayerOption(&quot;SelectWeapon&quot;,30,AfterChoice,VisualVote) -- Creates new vote</code></pre>
