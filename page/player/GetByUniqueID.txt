<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets the player with the specified uniqueID (not recommended way to identify players).
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>It is highly recommended to use <a href="/page/Category_colon_player" title="Category:player">player</a>.<a href="/page/player/GetByAccountID" title="player/GetByAccountID">GetByAccountID</a>, <a href="/page/Category_colon_player" title="Category:player">player</a>.<a href="/page/player/GetBySteamID" title="player/GetBySteamID">GetBySteamID</a> or <a href="/page/Category_colon_player" title="Category:player">player</a>.<a href="/page/player/GetBySteamID64" title="player/GetBySteamID64">GetBySteamID64</a> instead as this function can have collisions ( be same for different people ) while SteamID is guaranteed to unique to each player.
</p>
</td></tr></table>
