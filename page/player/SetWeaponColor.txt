<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the player weapon's color. The part of the model that is colored is determined by the model itself, and is different for each model.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> Color</span>
</p>
<div style="margin-left: 32px;">
This is the color to be set. The format is Vector(r,g,b), and each color should be between 0 and 1.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>When a player spawns their weapon's color will be red.
</p>
<pre><code data-language="lua">function GM:PlayerSpawn( ply )
 ply:SetWeaponColor( Vector(1,0,0) )
end</code></pre>
