<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Show/Hide the player's weapon's viewmodel.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> draw</span>
</p>
<div style="margin-left: 32px;">
Should draw</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> vm=0</span>
</p>
<div style="margin-left: 32px;">
Which view model to show/hide, 0-2.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Create a <a href="/page/Category_colon_ConVar" title="Category:ConVar">ConVar</a> object on the client realm to enable/disable the viewmodel.
</p>
<pre><code data-language="lua">local cvarObject = CreateClientConVar(&quot;nogun&quot;, &quot;1&quot;, true, false, &quot;Hide the current viewmodel.&quot;) -- Create the ConVar object

hook.Add(&quot;HUDPaint&quot;, &quot;draw_viewmodel&quot;, function()
	LocalPlayer():DrawViewModel(cvarObject:GetBool()) -- Call ConVar:GetBool() inside the parameters of DrawViewModel
end)</code></pre>
