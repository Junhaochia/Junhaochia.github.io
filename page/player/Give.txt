<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gives the player a weapon.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>While this function is meant for weapons/pickupables only, it is <b>not</b> restricted to weapons. Any entity can be spawned using this function, including NPCs and SENTs.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> weaponClassName</span>
</p>
<div style="margin-left: 32px;">
Class name of weapon to give the player</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> bNoAmmo=false</span>
</p>
<div style="margin-left: 32px;">
Set to true to not give any ammo on weapon spawn. (Reserve ammo set by DefaultClip)</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_Weapon" title="Category:Weapon">Weapon</a></span>
</p>
<div style="margin-left: 32px;">
The weapon given to the player, if one was given. It will return NULL if the player already has the weapon, or the weapon entity (entity with given classname) doesn't exist.</div>
</div>

<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Gives the player a toolgun
</p>
<pre><code data-language="lua">Entity( 1 ):Give( &quot;gmod_tool&quot; )</code></pre>
<p><br />
</p>
