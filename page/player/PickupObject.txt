<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>This makes the player hold ( same as pressing <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">E</span> on a small prop ) the provided entity.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Don't get this confused with picking up items like ammo or health kits
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This picks up the passed entity regardless of its mass or distance from the player
</p>
</td></tr></table>

</p>
<div style="margin-left: 32px;">

</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>An extra function to make sure the object isn't held before being picked up.
</p>
<pre><code data-language="lua">function PlayerPickupObject(ply, obj)
	if ( obj:IsPlayerHolding() ) then return end
	ply:PickupObject( obj )
end</code></pre>
