<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates the player's death ragdoll entity and deletes the old one.
</p><p>This is normally used when a player dies, to create their death ragdoll.
</p><p>The ragdoll will be created with the player's properties such as <a href="/page/Entity/GetPos" title="Entity/GetPos">Position</a>, <a href="/page/Entity/GetAngles" title="Entity/GetAngles">Angles</a>, <a href="/page/Player/GetPlayerColor" title="Player/GetPlayerColor">PlayerColor</a>, <a href="/page/Entity/GetVelocity" title="Entity/GetVelocity">Velocity</a> and <a href="/page/Entity/GetModel" title="Entity/GetModel">Model</a>.
</p><p>You can retrieve the entity this creates with <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/GetRagdollEntity" title="Player/GetRagdollEntity">GetRagdollEntity</a>.
</p>
