<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Kills a player and calls <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PlayerDeath" title="GM/PlayerDeath">PlayerDeath</a>.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>When the user types "/respawn" the user will be killed and then respawned.
</p>
<pre><code data-language="lua">hook.Add( &quot;PlayerSay&quot;, &quot;RespawnCommand&quot;, function( ply, text, public )
	if ( string.lower( text ) == &quot;/respawn&quot; ) then
		ply:Kill()
		ply:Spawn()
		return &quot;&quot;
	end
end )</code></pre>
