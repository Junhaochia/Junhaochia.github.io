<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets the amount of ammo the player has.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_any" title="Category:any">any</a> ammotype</span>
</p>
<div style="margin-left: 32px;">
The ammunition type. Can be either <a href="/page/Category_colon_number" title="Category:number">number</a> ammo ID or <a href="/page/Category_colon_string" title="Category:string">string</a> ammo name.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
The amount of ammo player has in reserve.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>A function that returns the ammo for the weapon the player is currently holding.
</p>
<pre><code data-language="lua">function GetAmmoForCurrentWeapon( ply )
	if (&#160;!IsValid( ply ) ) then return -1 end

	local wep = ply:GetActiveWeapon()
	if (&#160;!IsValid( wep ) ) then return -1 end
 
	return ply:GetAmmoCount( wep:GetPrimaryAmmoType() )
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">31</div>
