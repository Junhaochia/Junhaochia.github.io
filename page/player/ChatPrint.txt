<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Prints a string to the chatbox of the client.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Just like the <a href="/page/Category_colon_usermessage" title="Category:usermessage">usermessage library</a>, this function is affected by the 255 byte limit!
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> message</span>
</p>
<div style="margin-left: 32px;">
String to be printed</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints "Hello World" to chat of all players
</p>
<pre><code data-language="lua">for k, ply in pairs(player.GetAll()) do
	ply:ChatPrint(&quot;Hello World&quot;)
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Hello World (In chatbox)</div>
