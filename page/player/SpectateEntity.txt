<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Makes the player spectate the entity
</p><p>To get the applied spectated entity, use <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/GetObserverTarget" title="Player/GetObserverTarget">GetObserverTarget</a>().
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>The player's position will not update while spectating, causing area portals and other map optimisations to not work properly. You can fix this by setting the player's position to the spectated entity's each tick.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3267">#3267</a></b>
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a entity, spectates it and after 5 seconds, stops spectating it.
</p>
<pre><code data-language="lua">local ent = ents.Create( &quot;prop_physics&quot; )
ent:SetModel( &quot;models/hunter/misc/sphere025x025.mdl&quot; )
ent:SetPos( Vector( 0, 0, 0 ) )
ent:Spawn()
 
for _, ply in pairs( player.GetAll() ) do
	ply:Spectate( OBS_MODE_CHASE )
	ply:SpectateEntity( ent )
	ply:StripWeapons()

	timer.Simple( 5, function()
		ply:UnSpectate()
		ply:Spawn()
	end )
end</code></pre>
