<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the player's normal walking speed. Not sprinting, not slow walking <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">+walk</span>.
</p><p>There currently is no way to modify the slow walking <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">+walk</span> speed.
</p><p>See also <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/GetWalkSpeed" title="Player/GetWalkSpeed">GetWalkSpeed</a>, <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/SetCrouchedWalkSpeed" title="Player/SetCrouchedWalkSpeed">SetCrouchedWalkSpeed</a>, <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/SetMaxSpeed" title="Player/SetMaxSpeed">SetMaxSpeed</a> and <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/SetRunSpeed" title="Player/SetRunSpeed">SetRunSpeed</a>.
</p>
