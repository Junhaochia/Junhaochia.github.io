<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Pauses the active coroutine and passes all additional variables to the call of <a href="/page/Category_colon_coroutine" title="Category:coroutine">coroutine</a>.<a href="/page/coroutine/resume" title="coroutine/resume">resume</a> that resumed the coroutine last time, and returns all additional variables that were passed to the previous call of resume.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_vararg" title="Category:vararg">vararg</a> returnValue</span>
</p>
<div style="margin-left: 32px;">
Arguments to be returned by the last call of <a href="/page/Category_colon_coroutine" title="Category:coroutine">coroutine</a>.<a href="/page/coroutine/resume" title="coroutine/resume">resume</a></div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_vararg" title="Category:vararg">vararg</a></span>
</p>
<div style="margin-left: 32px;">
Arguments that were set previously by <a href="/page/Category_colon_coroutine" title="Category:coroutine">coroutine</a>.<a href="/page/coroutine/resume" title="coroutine/resume">resume</a></div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Demonstrates the use of using varargs as a return value.
</p>
<pre><code data-language="lua">local co = coroutine.create( function()
	coroutine.yield( &quot;Hello world!&quot; )
end )
print( coroutine.resume( co ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">true, "Hello world!"</div>
