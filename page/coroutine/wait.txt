<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Yields the coroutine for the given duration before continuing.
</p><p>This only works inside a coroutine.
</p><p>This function uses <a href="/page/Global/CurTime" title="Global/CurTime">CurTime</a> instead of <a href="/page/Global/RealTime" title="Global/RealTime">RealTime</a>.
</p>
