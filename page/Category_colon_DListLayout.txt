<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<p>A vertical list of panels that optionally allows child elements to be rearranged with the mouse cursor.
</p><p>Child panels' widths are set to the width of the DListLayout, and it resizes vertically to accommodate the heights of all children. You can place this inside a <a href="/page/Category_colon_DScrollPanel" title="Category:DScrollPanel">DScrollPanel</a> when adding many panels.
</p><p>To enable the drag &amp; drop rearrangement functionality, call <a href="/page/Category_colon_DDragBase" title="Category:DDragBase">DDragBase</a>:<a href="/page/DDragBase/MakeDroppable" title="DDragBase/MakeDroppable">MakeDroppable</a> on the DListLayout with a unique identifier. All panels added following this will be moveable.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DListLayout within a <a href="/page/Category_colon_DFrame" title="Category:DFrame">DFrame</a> and adds 8 <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabels</a>.
</p>
<pre><code data-language="lua">local frame = vgui.Create(&quot;DFrame&quot;)
frame:SetSize(250, 250)
frame:SetTitle(&quot;DListLayout Example&quot;)
frame:MakePopup()
frame:Center()

local layout = vgui.Create(&quot;DListLayout&quot;, frame)
layout:SetSize(100, 100)
layout:SetPos(20, 50)

--Draw a background so we can see what it's doing
layout:SetPaintBackground(true)
layout:SetBackgroundColor(Color(0, 100, 100))

layout:MakeDroppable(&quot;unique_name&quot;) -- Allows us to rearrange children

for i = 1, 8 do
	layout:Add(Label(&quot; Label &quot; .. i))
end</code></pre>
