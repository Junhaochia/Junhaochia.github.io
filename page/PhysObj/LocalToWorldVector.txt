<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Rotate a vector from the local frame of the physics object to world frame.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function only rotates the vector, without any translation operation.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> LocalVec</span>
</p>
<div style="margin-left: 32px;">
A vector in the physics object's local frame</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a></span>
</p>
<div style="margin-left: 32px;">
The corresponding vector in world frame</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Given the angle of the physics object is angle ( 0.045, 89.952, 89.99 ) ( This can be derived by PhysObj:GetAngles() ), convert the vector ( 1 ,2, 3 ) in this physics object's local frame into world frame.
</p>
<pre><code data-language="lua">print( &quot;Angle of the PhysObj: &quot;, PhysObj:GetAngles() )
print( &quot;Vector in world frame: &quot;, PhysObj:LocalToWorld( Vector( 1, 2, 3 ) ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Angle of the PhysObj: 0.045 89.952 89.993;
Vector in world frame: 3.000585 0.999039 1.999603;</div>
