<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets whether the physobject should be able to move or not.
</p><p>This is the exact method the Physics Gun uses to freeze props. If a motion-disabled physics object is grabbed with the physics gun, the object will be able to move again. To disallow this, use <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PhysgunPickup" title="GM/PhysgunPickup">PhysgunPickup</a>.
</p>

</p>
<div style="margin-left: 32px;">
True to enable, false to disable.</div>
