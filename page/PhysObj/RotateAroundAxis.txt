<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Shows that it is the same as <a href="/page/Category_colon_Angle" title="Category:Angle">Angle</a>:<a href="/page/Angle/RotateAroundAxis" title="Angle/RotateAroundAxis">RotateAroundAxis</a>.
</p>
<pre><code data-language="lua">local phys = Entity(1):GetEyeTrace().Entity:GetPhysicsObject() -- Our physics object

print( phys:RotateAroundAxis( Vector( 1, 0, 0 ), 20 ) )

local a = phys:GetAngles()
a:RotateAroundAxis( Vector( 1, 0, 0 ), 20 )
print( a )</code></pre>

</p>
