<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<p>The DHTML control wraps the internal Awesomium framework, supports calling Javascript functions from Lua, as well as running Lua from within the HTML. Running Lua code is disabled by default.
</p>
<h1> <span class="mw-headline" id="Inherited_Functions"> Inherited Functions </span></h1>
<p>Inherits the following functions from <a href="/page/Category_colon_Awesomium" title="Category:Awesomium">Awesomium</a> and its parents up to 3 levels deep:
</p>
<div style="float: left; width: 33%">
<ul>
<li><a href="/page/Panel/LoadTGAImage" title="Panel/LoadTGAImage">Panel/LoadTGAImage</a></li>
</ul>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DHTML.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetSize( 300, 200 )
frame:SetTitle( &quot;My new Derma frame&quot; )
frame:SetVisible( true )
frame:SetDraggable( true )
frame:Center()
--Fill the form with a html page
local html = vgui.Create( &quot;DHTML&quot; , frame )
html:Dock( FILL )
html:SetHTML( [[
    &lt;input type='submit' onclick='console.log(&quot;RUNLUA:print(\&quot;This is called in Lua context\&quot;)&quot;)' /&gt;
]] )

--Enable the webpage to call lua code
html:SetAllowLua( true )

frame:MakePopup()</code></pre>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Creates a DHTML and opens Google inside.
</p>
<pre><code data-language="lua">local frame = vgui.Create(&quot;DFrame&quot;)
frame:SetSize(ScrW() * 0.5, ScrH() * 0.5)
frame:SetTitle(&quot;HTML Example!&quot;)
frame:Center()
frame:MakePopup()

local html = vgui.Create(&quot;DHTML&quot;, frame)
html:Dock(FILL)
html:OpenURL(&quot;https://www.google.com/&quot;)</code></pre>
