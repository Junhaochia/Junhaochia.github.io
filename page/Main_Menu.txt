</p>

<h2> <span class="mw-headline" id="Overview">Overview</span></h2>
<h3> <span class="mw-headline" id="1._Start_New_Game">1. Start New Game</span></h3>
<p>Opens a map list to start a single player or local multiplayer game. By switching the gamemode with the Game Mode Selector, you can start the game in different gamemodes.
</p>
<h3> <span class="mw-headline" id="2._Find_Multiplayer_Game">2. Find Multiplayer Game</span></h3>
<p>Opens the <a href="/page/Finding_a_Multiplayer_Game" title="Finding a Multiplayer Game">Garry's Mod Server Browser</a>.
</p>
<h3> <span class="mw-headline" id="3._Addons">3. Addons</span></h3>
<p>Opens the <a href="/page/Addons" title="Addons">Addons</a> menu, which allows you download, enable, disable and delete <a rel="nofollow" class="external text" href="https://steamcommunity.com/workshop/browse?appid=4000">Steam Workshop</a> addons from your computer.
</p>
<h3> <span class="mw-headline" id="4._Demos">4. Demos</span></h3>
<p>Opens the Demos menu, which lets you see all of your saved demos, as well as demos uploaded to steam.
</p>
<h3> <span class="mw-headline" id="5._Saves">5. Saves</span></h3>
<p>Opens the Saves menu, which lets you see all of your saved saves, as well as saves uploaded to steam.
</p>
<h3> <span class="mw-headline" id="6._Options">6. Options</span></h3>
<p>Opens the Options menu.
</p>
<h3> <span class="mw-headline" id="7._Quit">7. Quit</span></h3>
<p>Closes Garry's Mod.
</p>
<h3> <span class="mw-headline" id="8._Installed_Games_List">8. Installed Games List</span></h3>
<p>Opens a list that shows all steam games that are compatible with Garry's Mod, and shows if they are installed, disabled (unmounted), or enabled(mounted).
</p><p>Red cross means you do not own that game, folder with a small minus sign means the game is not installed.
</p>
<h3> <span class="mw-headline" id="9._Language_Selector">9. Language Selector</span></h3>
<p>Opens a menu containing flags from different countries representing languages. Clicking on a flag changes the current language.
</p>
<h3> <span class="mw-headline" id="10._Game_Mode_Selector">10. Game Mode Selector</span></h3>
<p>Switches the current gamemode. The default gamemode is <b>Sandbox</b>. On installation, the only other gamemode available is <b>Trouble In Terrorist Town</b>. More gamemodes can be added through the use of the addons menu.
</p>
