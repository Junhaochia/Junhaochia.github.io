<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Draws a texture with a table structure.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is a rendering function that requires a 2D rendering context.<br />This means that it will only work in <a href="/page/Category_colon_2D_Rendering_Hooks" title="Category:2D Rendering Hooks">hooks with a 2D rendering context</a>.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> texturedata</span>
</p>
<div style="margin-left: 32px;">
The texture properties. See <a href="/page/Structures/TextureData" title="Structures/TextureData">TextureData structure</a></div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage
</p>
<pre><code data-language="lua">local TexturedQuadStructure = {
	texture = surface.GetTextureID( 'phoenix_storms/amraam' ),
	color   = Color( 255, 0, 255, 255 ),
	x 	= 0,
	y 	= 0,
	w 	= 512,
	h 	= 512
}
draw.TexturedQuad( TexturedQuadStructure )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Renders the texture.</div>
