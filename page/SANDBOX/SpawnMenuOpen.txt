<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when spawnmenu is trying to be opened.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Hiding the spawnmenu will not stop people from being able to use the various console commands to spawn in items, etc. See GM:PlayerSpawn* hooks for blocking actual spawning.
</p>
</td></tr></table>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Return false to dissallow opening the spawnmenu</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Only allow the people you want to open the spawn menu.
</p>
<pre><code data-language="lua">local allowed = {
	[&quot;STEAM_0:0:00000000&quot;] = true,
	[&quot;STEAM_0:0:10000000&quot;] = true
}

hook.Add( &quot;SpawnMenuOpen&quot;, &quot;SpawnMenuWhitelist&quot;, function()
	if (&#160;!allowed[ LocalPlayer():SteamID() ] ) then
		return false
	end
end )</code></pre>
