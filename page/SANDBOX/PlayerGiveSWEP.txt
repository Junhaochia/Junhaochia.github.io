<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when a player attempts to give themselves a weapon from the Q menu. ( Left mouse clicks on an icon )
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Player" title="Category:Player">Player</a> ply</span>
</p>
<div style="margin-left: 32px;">
The player who attempted to give themselves a weapon.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> weapon</span>
</p>
<div style="margin-left: 32px;">
Class name of the weapon the player tried to give themselves.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> swep</span>
</p>
<div style="margin-left: 32px;">
The swep table of this weapon, see <a href="/page/Structures/SWEP" title="Structures/SWEP">SWEP structure</a></div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Can the SWEP be given to the player</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Stops non-admins from giving themselves weapons.
</p>
<pre><code data-language="lua">hook.Add( &quot;PlayerGiveSWEP&quot;, &quot;BlockPlayerSWEPs&quot;, function( ply, class, swep )
	if ( not ply:IsAdmin() ) then
		return false
	end
end )</code></pre>
