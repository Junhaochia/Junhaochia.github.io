<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>This hook is used to add new tool tabs to spawnmenu.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Add a new tab and a few categories into it.
</p>
<pre><code data-language="lua">hook.Add( &quot;AddToolMenuTabs&quot;, &quot;myHookClass&quot;, function()
	spawnmenu.AddToolTab(&quot;myTab&quot;, &quot;My Tab&quot;, &quot;icon16/shield.png&quot;) -- Add a new tab

	spawnmenu.AddToolCategory(&quot;myTab&quot;, &quot;myCategory&quot;, &quot;My Category&quot;) -- Add a category into that new tab

	spawnmenu.AddToolMenuOption( &quot;myTab&quot;, &quot;myCategory&quot;, &quot;myEntry&quot;, &quot;My Entry&quot;, &quot;&quot;, &quot;&quot;, function( panel )
		panel:AddControl( &quot;Header&quot;, { Text = &quot;Hello!&quot; } )
	end ) -- Add an entry to our new category
end)</code></pre>
