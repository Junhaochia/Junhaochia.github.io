<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>This hook is used to add new categories to spawnmenu tool tabs.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Adds default categories to Utilities tab in spawnmenu.
</p>
<pre><code data-language="lua">local function CreateUtilitiesCategories()

        spawnmenu.AddToolCategory( &quot;Utilities&quot;, &quot;User&quot;, &quot;#spawnmenu.utilities.user&quot; )
        spawnmenu.AddToolCategory( &quot;Utilities&quot;, &quot;Admin&quot;, &quot;#spawnmenu.utilities.admin&quot; )

end        
hook.Add( &quot;AddToolMenuCategories&quot;, &quot;CreateUtilitiesCategories&quot;, CreateUtilitiesCategories )</code></pre>
