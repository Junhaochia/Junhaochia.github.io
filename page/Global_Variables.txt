<h1> <span class="mw-headline" id="GLOBAL_TABLES">GLOBAL TABLES</span></h1>
<h2> <span class="mw-headline" id="GAMEMODE">GAMEMODE</span></h2>
<p>The table of the currently active gamemode, outside the gamemode files.
</p>
<h2> <span class="mw-headline" id="GM">GM</span></h2>
<p>Same as GAMEMODE, but only exists gamemode files. (gamemodes/&lt;gamemodeName&gt;/gamemode/*.lua)
</p>
<h2> <span class="mw-headline" id="ENT">ENT</span></h2>
<p>Similar to GM, but for Scripted Entities. Exists only in the files defining the entity. (lua/entities/*.lua)
</p>
<h2> <span class="mw-headline" id="SWEP">SWEP</span></h2>
<p>Similar to ENT, but for Scripted Weapons. Exists only in the files defining the weapon. (lua/weapons/*.lua)
</p>
<h2> <span class="mw-headline" id="EFFECT">EFFECT</span></h2>
<p>Similar to ENT, but for Scripted Effects. Exists only in the files defining the effect. (lua/effects/*.lua)
</p>
<h2> <span class="mw-headline" id="G">_G</span></h2>
<p>This table contains all global objects, including itself.
</p>
<h2> <span class="mw-headline" id="MODULES">_MODULES</span></h2>
<p>Contains a list all modules loaded from /modules/.
</p>
<h1> <span class="mw-headline" id="NON_CONSTANTS">NON CONSTANTS</span></h1>
<h2> <span class="mw-headline" id="CLIENT">CLIENT</span></h2>
<p>This is true whenever the current script is executed on the client. ( client and menu states ) See <a href="/page/States" title="States">States</a>.
</p>
<h2> <span class="mw-headline" id="CLIENT_DLL">CLIENT_DLL</span></h2>
<p>This is true whenever the current script is executed on the client state. See <a href="/page/States" title="States">States</a>.
</p>
<h2> <span class="mw-headline" id="SERVER">SERVER</span></h2>
<p>This is true whenever the current script is executed on the server state. See <a href="/page/States" title="States">States</a>.
</p>
<h2> <span class="mw-headline" id="GAME_DLL">GAME_DLL</span></h2>
<p>This is true whenever the current script is executed on the server state.
</p>
<h2> <span class="mw-headline" id="MENU_DLL">MENU_DLL</span></h2>
<p>This is true when the script is being executed in the menu state. See <a href="/page/States" title="States">States</a>.
</p>
<h2> <span class="mw-headline" id="GAMEMODE_NAME">GAMEMODE_NAME</span></h2>
<p>Contains the name of the current active gamemode
</p>
<h2> <span class="mw-headline" id="NULL">NULL</span></h2>
<p>Represents a non existent entity
</p>
<h2> <span class="mw-headline" id="VERSION">VERSION</span></h2>
<p>Returns the version number of GMod.
</p>
<h2> <span class="mw-headline" id="VERSIONSTR">VERSIONSTR</span></h2>
<p>Returns a nicely formatted version of GMod.
</p>
<h2> <span class="mw-headline" id="BRANCH">BRANCH</span></h2>
<p>Returns the branch the game is running on.
</p>
<h2> <span class="mw-headline" id="VERSION_2">_VERSION</span></h2>
<p>Current Lua version. This returns "Lua 5.1" in GMod at the moment.
</p>
