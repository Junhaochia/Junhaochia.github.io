<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sends an asynchronous POST request to a HTTP server.
</p><p>HTTP requests returning a status code &gt;= 400 are still considered a success and will call the onSuccess callback.
</p><p>The onFailure callback is usually only called on DNS or TCP errors (e.g. the website is unavailable or the domain does not exist)
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>

<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> url</span>
</p>
<div style="margin-left: 32px;">
The url to of the website to fetch.</div>
</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Write a file in PHP, and invoke it from Lua. The output below is written in the file, not in the console.
</p>
<pre class="plain">&lt;?php
	$p = $_POST[&quot;p&quot;];
	$a = $_POST[&quot;a&quot;];

	$f = fopen(&quot;write.html&quot;, &quot;w&quot;);

	fwrite($f, &quot;This is a test. $p $a\n&quot;);

	fclose($f);
?&gt;</code></pre>
<pre><code data-language="lua">http.Post(&quot;http://localhost/post.php&quot;, { p = &quot;Gmod&quot;, a = &quot;Test&quot; }, function(result)
	if result then print(&quot;Done!&quot;) end
end, function(failed)
	print(failed)
end)</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">This is a test. Gmod Test</code></pre>
