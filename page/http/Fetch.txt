<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Launches a GET request.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> url</span>
</p>
<div style="margin-left: 32px;">
The URL of the website to fetch.</div>
</div>

<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> onSuccess=nil</span>
</p>
<div style="margin-left: 32px;">
<p>Function to be called on success. Arguments are
</p>
<ul><li> <a href="/page/Category_colon_string" title="Category:string">string</a> body
</li><li> <a href="/page/Category_colon_string" title="Category:string">string</a> size - equal to <a href="/page/Category_colon_string" title="Category:string">string</a>.<a href="/page/string/len" title="string/len">len</a>(body)
</li><li> <a href="/page/Category_colon_table" title="Category:table">table</a> headers
</li><li> <a href="/page/Category_colon_number" title="Category:number">number</a> code - The HTTP success code</div>
</li></ul>
</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Shows the typical usage to get the HTML of a webpage.
</p>
<pre><code data-language="lua">local TheReturnedHTML = &quot;&quot; -- Blankness

http.Fetch( &quot;http:&#47;&#47;www.google.com&quot;,
	function( body, len, headers, code )
		-- The first argument is the HTML we asked for.
		TheReturnedHTML = body
	end,
	function( error )
		-- We failed. =(
	end
)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">If it successfully fetched the page, the variable 'TheReturnedHTML' should contain the returned HTML in plain text.</div>
