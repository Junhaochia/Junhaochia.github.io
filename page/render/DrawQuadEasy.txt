<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Draws a quad.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is a rendering function that requires a 3D rendering context.<br />This means that it will only work in <a href="/page/Category_colon_3D_Rendering_Hooks" title="Category:3D Rendering Hooks">hooks with a 3D rendering context</a>.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> position</span>
</p>
<div style="margin-left: 32px;">
Origin of the sprite.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> normal</span>
</p>
<div style="margin-left: 32px;">
The face direction of the quad.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> width</span>
</p>
<div style="margin-left: 32px;">
The width of the quad.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> height</span>
</p>
<div style="margin-left: 32px;">
The height of the quad.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_5">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> color</span>
</p>
<div style="margin-left: 32px;">
The color of the quad. Uses the <a href="/page/Structures/Color" title="Structures/Color">Color structure</a>.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_6">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> rotation=0</span>
</p>
<div style="margin-left: 32px;">
The rotation of the quad counter-clockwise in degrees around the normal axis. In other words, the quad will always face the same way but this will rotate its corners.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage of this function.
</p>
<pre><code data-language="lua">local mat = Material( &quot;sprites/sent_ball&quot; )
local mat2 = Material( &quot;models/wireframe&quot; )
hook.Add(&quot;PostDrawTranslucentRenderables&quot;, &quot;DrawQuadEasyExample&quot;, function()

	-- Draw a rotating circle under local player
	render.SetMaterial( mat )
	local pos = LocalPlayer():GetPos()
	render.DrawQuadEasy( pos + Vector( 0, 0, 1 ), Vector( 0, 0, 1 ), 64, 64, Color( 255, 255, 255, 200 ), ( CurTime() * 50 )&#160;% 360 )

	-- Draw 3 rotating wireframe quads where local player is looking at
	render.SetMaterial( mat2 )
	local tr = LocalPlayer():GetEyeTrace()
	render.DrawQuadEasy( tr.HitPos + tr.HitNormal, tr.HitNormal, 64, 64, Color( 255, 255, 255 ), ( CurTime() * 50 )&#160;% 360 )

	local dir = tr.HitNormal:Angle()
	dir:RotateAroundAxis( tr.HitNormal, ( CurTime() * 50 )&#160;% 360 )
	dir = dir:Up()

	-- We need to call this function twice, once for each side
	render.DrawQuadEasy( tr.HitPos + tr.HitNormal * 32, dir, 64, 64, Color( 255, 255, 255 ), 0 )
	render.DrawQuadEasy( tr.HitPos + tr.HitNormal * 32, -dir, 64, 64, Color( 255, 255, 255 ), 0 )
end )</code></pre>
