<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets a scissoring rect which limits the drawing area.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> startX</span>
</p>
<div style="margin-left: 32px;">
X start coordinate of the scissor rect.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> startY</span>
</p>
<div style="margin-left: 32px;">
Y start coordinate of the scissor rect.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">

</p>

X end coordinate of the scissor rect.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>

Y end coordinate of the scissor rect.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_5">Arguments</span></h1>
<div class="argument">

</p>
<div style="margin-left: 32px;">
Enable or disable the scissor rect.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Shows how to use this function. This will cut the white rectangle from full screen to 512x512 box in top left corner
</p>
<pre><code data-language="lua">render.SetScissorRect( 0, 0, 512, 512, true ) -- Enable the rect
	draw.RoundedBox( 4, 0, 0, ScrW(), ScrH(), color_white ) -- Draw a white rectangle over the whole screen
render.SetScissorRect( 0, 0, 0, 0, false ) -- Disable after you are done</code></pre>
