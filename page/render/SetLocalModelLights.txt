<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets up the local lighting for any upcoming render operation. Up to 4 local lights can be defined, with one of three different types (point, directional, spot).
</p><p>Disables all local lights if called with no arguments.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> lights={}</span>
</p>
<div style="margin-left: 32px;">
A table containing up to 4 tables for each light source that should be set up. Each of these tables should contain the properties of its associated light source, see <a href="/page/Structures/LocalLight" title="Structures/LocalLight">LocalLight structure</a>.</div>
</div>
