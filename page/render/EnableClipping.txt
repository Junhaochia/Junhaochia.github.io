<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the status of the clip renderer, returning previous state.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>To prevent unintended rendering behavior of other mods/the game, you must reset the clipping state to its previous value.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>Reloading the map does not reset the previous value of this function.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3105">#3105</a></b>
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Clips the lower half of your custom entity
</p>
<pre><code data-language="lua">function ENT:Draw()
    local normal = self:GetUp() -- Everything &quot;behind&quot; this normal will be clipped
    local position = normal:Dot( self:GetPos() ) -- self:GetPos() is the origin of the clipping plane

    local oldEC = render.EnableClipping( true )
    render.PushCustomClipPlane( normal, position )

    self:DrawModel()

    render.PopCustomClipPlane()
    render.EnableClipping( oldEC )
end</code></pre>
