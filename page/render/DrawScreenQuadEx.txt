<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Draws the the current material set by <a href="/page/Category_colon_render" title="Category:render">render</a>.<a href="/page/render/SetMaterial" title="render/SetMaterial">SetMaterial</a> to the area specified. Color cannot be customized.
</p><p>See also <a href="/page/Category_colon_render" title="Category:render">render</a>.<a href="/page/render/DrawScreenQuad" title="render/DrawScreenQuad">DrawScreenQuad</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is a rendering function that requires a 2D rendering context.<br />This means that it will only work in <a href="/page/Category_colon_2D_Rendering_Hooks" title="Category:2D Rendering Hooks">hooks with a 2D rendering context</a>.
</p>
</td></tr></table>

<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage, draws a 256x256 rectangle with the wireframe material.
</p>
<pre><code data-language="lua">local ourMat = Material( &quot;models/wireframe&quot; )
hook.Add( &quot;HUDPaint&quot;, &quot;example_hook&quot;, function()
	render.SetMaterial( ourMat )
	render.DrawScreenQuadEx( 100, 100, 256, 256 )
end )</code></pre>
