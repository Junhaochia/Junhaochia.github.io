<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Captures a part of the current render target and returns the data as a binary string in the given format.
</p><p>Since the pixel buffer clears itself every frame, this will return a black screen outside of <a href="/page/Category_colon_Render_Hooks" title="Category:Render Hooks">render hooks</a>. To capture the user's final view, use <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PostRender" title="GM/PostRender">PostRender</a>. This will not capture the Steam overlay or third-party injections (such as the Discord overlay, Overwolf, and advanced cheats) on the user's screen.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>How you could use this to save a picture of your screen.
</p>
<pre><code data-language="lua">local ScreenshotRequested = false
function RequestAScreenshot()
	ScreenshotRequested = true
end

-- For the sake of this example, we use a console command to request a screenshot
concommand.Add( &quot;make_screenshot&quot;, RequestAScreenshot )

hook.Add( &quot;PostRender&quot;, &quot;example_screenshot&quot;, function()
	if (&#160;!ScreenshotRequested ) then return end
	ScreenshotRequested = false

	local data = render.Capture( {
		format = &quot;jpeg&quot;,
		quality = 70, //100 is max quality, but 70 is good enough.
		h = ScrH(),
		w = ScrW(),
		x = 0,
		y = 0,
	} )
	local f = file.Open( &quot;Image.jpg&quot;, &quot;wb&quot;, &quot;DATA&quot; )
	f:Write( data )
	f:Close()
end )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">You should now have Image.jpg in your garrysmod/garrysmod/data folder, containing a screenshot.</div>
