<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Draws 2 connected triangles. Expects material to be set by <a href="/page/Category_colon_render" title="Category:render">render</a>.<a href="/page/render/SetMaterial" title="render/SetMaterial">SetMaterial</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is a rendering function that requires a 3D rendering context.<br />This means that it will only work in <a href="/page/Category_colon_3D_Rendering_Hooks" title="Category:3D Rendering Hooks">hooks with a 3D rendering context</a>.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Draw a red half transparent quad facing upwards 150 units below the 0,0,0 of gm_construct.
</p>
<pre><code data-language="lua">local ourMat = Material( &quot;vgui/white&quot; ) -- Calling Material() every frame is quite expensive
hook.Add( &quot;PostDrawTranslucentRenderables&quot;, &quot;DrawQuad_Example&quot;, function()

	render.SetMaterial( ourMat ) -- If you use Material, cache it!
	render.DrawQuad( Vector( 0, 0, -150 ), Vector( 0, 100, -150 ),Vector( 100, 100, -150 ), Vector( 100, 0, -150 ), Color( 255, 0, 0, 128 ) )

end)</code></pre>
