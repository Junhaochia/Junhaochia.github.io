<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets a material to override a model's default material. Similar to <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetMaterial" title="Entity/SetMaterial">SetMaterial</a> except it uses an <a href="/page/Category_colon_IMaterial" title="Category:IMaterial">IMaterial</a> argument and it can be used to change materials on models which are part of the world geometry.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_IMaterial" title="Category:IMaterial">IMaterial</a> material</span>
</p>
<div style="margin-left: 32px;">
The material override.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Sets the material of all opaque models to the red flashing "ERROR" texture.
</p>
<pre><code data-language="lua">local error_mat = Material(&quot;models/error/new light1&quot;)

function GM:PreDrawOpaqueRenderables(bDrawingDepth, bDrawingSkybox)

	render.ModelMaterialOverride(error_mat)
	
end</code></pre>
