<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Swaps the frame buffers/cycles the frame. In other words, this updates the screen.
</p><p>If you take a really long time during a single frame render, it is a good idea to use this and let the user know that the game isn't stuck.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Code from <a rel="nofollow" class="external text" href="https://github.com/garrynewman/garrysmod/blob/master/garrysmod/lua/postprocess/super_dof.lua#L219-L236">Super DOF</a>; Push the current progress of rendering onto the screen and display a percentage of completion near the bottom right.
</p>
<pre><code data-language="lua">-- Restore RT
render.SetRenderTarget( OldRT )

-- Render our result buffer to the screen
matFSB:SetFloat( &quot;$alpha&quot;, 1 )
matFSB:SetTexture( &quot;$basetexture&quot;, texFSB )

render.SetMaterial( matFSB )
render.DrawScreenQuad()

cam.Start2D()
	local add = ( i / ( math.pi*2 ) ) * ( 1 / passes )
	local percent = string.format( &quot;%.1f&quot;, ( mul - ( 1 / passes ) + add ) * 100 )
	draw.DrawText( percent .. &quot;%&quot;, &quot;GModWorldtip&quot;, view.w - 100, view.h - 100, Color( 0, 0, 0, 255 ), TEXT_ALIGN_CENTER )
	draw.DrawText( percent .. &quot;%&quot;, &quot;GModWorldtip&quot;, view.w - 101, view.h - 101, Color( 255, 255, 255, 255 ), TEXT_ALIGN_CENTER )
cam.End2D()

render.Spin()</code></pre>
