<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the compare function of the stencil.
</p><p>Pixels which fail the stencil comparison function are not written to the render target. The operation to be performed on the stencil buffer values for these pixels can be set using <a href="/page/Category_colon_render" title="Category:render">render</a>.<a href="/page/render/SetStencilFailOperation" title="render/SetStencilFailOperation">SetStencilFailOperation</a>.
</p><p>Pixels which pass the stencil comparison function are written to the render target unless they fail the depth buffer test (where applicable). The operation to perform on the stencil buffer values for these pixels can be set using <a href="/page/Category_colon_render" title="Category:render">render</a>.<a href="/page/render/SetStencilPassOperation" title="render/SetStencilPassOperation">SetStencilPassOperation</a> and <a href="/page/Category_colon_render" title="Category:render">render</a>.<a href="/page/render/SetStencilZFailOperation" title="render/SetStencilZFailOperation">SetStencilZFailOperation</a>.
</p>
