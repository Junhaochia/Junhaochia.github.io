<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Changes the view port position and size. The values will be clamped to the game's screen resolution.
</p><p>If you are looking to render something to a texture (render target), you should use <a href="/page/Category_colon_render" title="Category:render">render</a>.<a href="/page/render/PushRenderTarget" title="render/PushRenderTarget">PushRenderTarget</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function will override values of <a href="/page/Global/ScrW" title="Global/ScrW">ScrW</a> and <a href="/page/Global/ScrH" title="Global/ScrH">ScrH</a> with the ones you set.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Renders a screen with a dimension of 32 X 32 and resets the render system to normal.
</p>
<pre><code data-language="lua">local oldW, oldH = ScrW(), ScrH()
local oldRT = render.GetRenderTarget()

render.SetRenderTarget(RTName)
render.Clear(0,0,0,255)
render.SetViewPort(0,0,32,32)
	render.RenderView(CamData)
render.SetRenderTarget(oldRT)
render.SetViewPort(0,0,oldW,oldH)</code></pre>
