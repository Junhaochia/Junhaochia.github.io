<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DAlphaBar and sets its value to 25%.
</p>
<pre><code data-language="lua">local DAlphaBar = vgui.Create( &quot;DAlphaBar&quot; )
DAlphaBar:SetPos( 20, 30 )
DAlphaBar:SetSize( 25, 125 )
DAlphaBar:SetValue( 0.25 )
DAlphaBar.OnChange = function( newvalue )
	print( &quot;DAlphaBar value changed to &quot; .. newvalue )
end</code></pre>
