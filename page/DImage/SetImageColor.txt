<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the image's color override.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> col</span>
</p>
<div style="margin-left: 32px;">
The color override of the image. Uses the <a href="/page/Structures/Color" title="Structures/Color">Color structure</a>.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a frame with a Portal box inside and sets the color to green.
</p>
<pre><code data-language="lua">-- Frame
MainFrame = vgui.Create(&quot;DFrame&quot;)
MainFrame:SetSize(200, 180)
MainFrame:Center()
MainFrame:SetTitle(&quot;Color example&quot;)

-- Image of a Portal box
local metalbox_img = vgui.Create(&quot;DImage&quot;, MainFrame)
metalbox_img:SetPos(35, 35)
metalbox_img:SetSize(128, 128)
metalbox_img:SetImage(&quot;spawnicons/models/props/metal_box_128.png&quot;)

metalbox_img:SetImageColor(Color(128, 255, 0, 255))</code></pre>
