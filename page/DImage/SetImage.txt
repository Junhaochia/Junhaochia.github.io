<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the image to load into the frame. If the first image can't be loaded and <i>strBackup</i> is set, that image will be loaded instead.
</p><p>This eventually calls <a href="/page/Category_colon_DImage" title="Category:DImage">DImage</a>:<a href="/page/DImage/SetMaterial" title="DImage/SetMaterial">SetMaterial</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> strImage</span>
</p>
<div style="margin-left: 32px;">
The path of the image to load. When no file extension is supplied the VMT file extension is used.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> strBackup=nil</span>
</p>
<div style="margin-left: 32px;">
The path of the backup image.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a frame with a map of de_inferno inside, with the default avatar image loaded as a backup.
</p>
<pre><code data-language="lua">-- Frame
MainFrame = vgui.Create( &quot;DFrame&quot; )
MainFrame:SetSize( 300, 300 )
MainFrame:Center()
MainFrame:SetTitle( &quot;Map of de_inferno&quot; )

-- Map of de_inferno (requires CS:S)
local css_map = vgui.Create( &quot;DImage&quot;, MainFrame )
css_map:SetPos( 25, 40 )
css_map:SetSize( 250, 250 )

-- Set image to de_inferno map
-- If it can't be loaded, load the [?] avatar image instead
css_map:SetImage(&quot;overviews/de_inferno&quot;, &quot;vgui/avatar_default&quot;)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">If CS:S is mounted then the left image is shown, otherwise the right image is shown.<br /><br />
