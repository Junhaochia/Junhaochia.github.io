<div class="internalfunc"><span>This is an internal function or feature.</span><br />
This means you will be able to use it, but you really shouldn't.</div>
<p>"Fixes" the current material of the <a href="/page/Category_colon_DImage" title="Category:DImage">DImage</a> if it has VertexLit shader by creating a new one with the same name and a prefix of "_DImage" and automatically calling <a href="/page/Category_colon_DImage" title="Category:DImage">DImage</a>:<a href="/page/DImage/SetMaterial" title="DImage/SetMaterial">SetMaterial</a> with the new material.
</p><p>This fixes the problem where using materials using shaders that expect lighting information causing "weird" flickering when displayed in 2D/Unlit environment.
