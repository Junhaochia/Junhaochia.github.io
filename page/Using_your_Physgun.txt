<h3> <span class="mw-headline" id="Overview"> Overview </span></h3>
<p>The Physics Gun (most commonly known as The Physgun) is a tool/weapon that allows its user to manipulate almost everything in the game, with the exception of players and the map itself. It allows the user to lift, rotate the object in the air, place, and even freeze an object completely still, whether it's in midair or not.
</p>
<h3> <span class="mw-headline" id="Abilities_and_usage"> Abilities and usage </span></h3>
<p>Holding down your <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;"><span title="Left mouse button"><img alt="LMB" src="/images/f/ff/Lmb.png" width="16" height="16" /></span></span> left mouse button makes the Physgun emit a long blue beam from the front of the gun. Anything that touches the beam or vice versa will be grabbed by the gun and can then be manipulated, by you, into different positions. <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;"><span title="Right mouse button"><img alt="RMB" src="/images/0/02/Rmb.png" width="16" height="16" /></span></span> right clicking allows you to freeze the held object in place regardless of where it is. NPCs cannot be frozen but are capable of being placed. Holding down <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">E</span> (your use key) allows you to rotate the object if grabbed by using the mouse. Your <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;"><span title="Middle mouse button"><img alt="MMB" src="/images/0/0e/Mmb.png" width="16" height="16" /></span></span> mouse wheel, allows you to drag the grabbed target to them or pull it away, although the same thing can be done by holding <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">E</span> and walking forward or backward (<span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">W</span> and <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">S</span> respectively). The Physgun can also straighten out the angles of a prop. It is done by holding down <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">â‡§ Shift</span> and <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">E</span> (your 'use' key) and moving the mouse around. Reloading (<span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">R</span>) with the physgun causes the bones of a highlighted rag doll (Highlight by rolling the crosshair over it) to unfreeze. Reloading twice causes all props you've spawned (so far) to unfreeze. You may want to avoid doing this by accident.
</p><p><br />
</p>
<h3> <span class="mw-headline" id="Physics_Gun_as_a_weapon"> Physics Gun as a weapon </span></h3>
<p>The Physgun is a tool and has no capability as a weapon in itself. As such, it cannot be directly used as one, but rather must be used indirectly. Common means of doing so are grabbing large, heavy objects and whipping them around and into enemies. Other, more trap like methods are taking weighty props such as cars and containers and freezing them above enemies only to unfreeze and let the weight do the work, or holding back motorized or otherwise motion-constrained props until a time where releasing it will kill something.
</p><p><br />
</p>
<h3> <span class="mw-headline" id="Lifting"> Lifting </span></h3>
<p>As mentioned previously, the Physgun can almost grab anything regardless of it's weight. The few things that it cannot pick up are the objects built into the map by default, such as walls, windows and fences, and other players. Beware; sometimes players will suddenly lose control of the grabbed object if among other players, and if bumped or brushed up against, the object may instantly kill a player by accidental release and ricochetting. The best thing to do in this case is to make sure that there is enough space around you to manuver the object.
</p><p><br />
</p>
<h3> <span class="mw-headline" id="Moving_Ragdolls"> Moving Ragdolls </span></h3>
<p>Posing ragdolls with the Physgun can be tricky. Once ready, the best method is to start with the head, then the torso, and lastly the limbs. One can also toggle on slow-movement-noclip by pressing <span class="keyboard-key" style="display: inline-block; background-color: rgba(0, 0, 0, 0.1);border-radius: 4px;text-align: center; color: #000; padding: 0.2em 0.5em; font-size: 0.85em;">Ctrl</span>, which will let them make very precise adjustments.
</p>
