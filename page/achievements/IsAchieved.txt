<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Used in GMod 12 in the achievements menu to show the user if they have unlocked certain achievements.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> AchievementID</span>
</p>
<div style="margin-left: 32px;">
Internal Achievement ID number</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Returns true if the given achievementID is achieved.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints true if you got the achievement.
</p>
<pre><code data-language="lua">print(&quot;Did you get achievement number 7?, &quot; .. achievements.IsAchieved(7))</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">boolean true (assuming you have unlocked achievement 7)</div>
