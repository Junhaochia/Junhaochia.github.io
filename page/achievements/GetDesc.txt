<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Retrieves description of given achievement
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> achievementID</span>
</p>
<div style="margin-left: 32px;">
The ID of achievement to retrieve description of. Note: IDs start from 0, not 1.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a></span>
</p>
<div style="margin-left: 32px;">
Description of an achievement</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Will print achievement descriptions for all of your achievements into console
</p>
<pre><code data-language="lua">for i = 0, achievements.Count() - 1 do
	print( achievements.GetDesc( i ) )
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Prints achievement descriptions for all of your achievements into console.</div>
