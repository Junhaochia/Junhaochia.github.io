<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a list of addons the player have subscribed to on the workshop.
</p><p>This list will also include "Floating" .gma addons that are mounted by the game, but not the folder addons.
</p>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
A table with 6 keys (downloaded, models, title, file, mounted, wsid)</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Will return a list of all the workshop addons you have downloaded / are downloading.
</p>
<pre><code data-language="lua">PrintTable(engine.GetAddons())</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">1:
		downloaded	=	true 
		models		=	0 
		title		=	Title of Addon 
		file		=	addons/title_of_addon_123456789.gma 
		mounted		=	true
		wsid		=	123456789 
</code></pre></div>
