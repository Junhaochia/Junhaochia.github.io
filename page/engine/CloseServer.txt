<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Closes the server and completely exits.
</p><p>This is only functional when running in server test mode (launch option -systemtest). Server test mode is used internally at Facepunch as part of the build process to make sure that the dedicated servers aren't crashing on startup.
