<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Creates a new <a href="/page/Category_colon_DMenu" title="Category:DMenu">DMenu</a> object tied to a <a href="/page/Category_colon_DButton" title="Category:DButton">DButton</a> with the given label on the menu bar.
</p><p>This will create a new menu regardless of whether or not one with the same label exists. To add <b>or</b> get a menu, use <a href="/page/Category_colon_DMenuBar" title="Category:DMenuBar">DMenuBar</a>:<a href="/page/DMenuBar/AddOrGetMenu" title="DMenuBar/AddOrGetMenu">AddOrGetMenu</a>.
</p>
