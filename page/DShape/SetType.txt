<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the shape to be drawn.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> type</span>
</p>
<div style="margin-left: 32px;">
The render type of the DShape. Only rectangles (<i>Rect</i>) work currently. If you don't define a type immediately, the <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/Paint" title="PANEL/Paint">Paint</a> method will generate errors until you do.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DShape and sets the render type.
</p>
<pre><code data-language="lua">local Shape = vgui.Create( &quot;DShape&quot; )
Shape:SetType( &quot;Rect&quot; )
Shape:SetSize(500,500)
Shape:SetPos(10, 10)</code></pre>
