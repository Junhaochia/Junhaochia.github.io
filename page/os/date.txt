<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the date/time as a formatted string or in a table.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> format</span>
</p>
<div style="margin-left: 32px;">
<p>The format string.
</p><p>If this is equal to '*t' or '!*t' then this function will return a <a href="/page/Structures/DateData" title="Structures/DateData">DateData structure</a>, otherwise it will return a string.
</p><p>If this starts with an '!', the returned data will use the UTC timezone rather than the local timezone.
</p><p>See <a rel="nofollow" class="external free" href="https://www.mkssoftware.com/docs/man3/strftime.3.asp">http://www.mkssoftware.com/docs/man3/strftime.3.asp</a> for available format flags.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p><b>Not all flags are available on all operating systems</b> and the result of using an invalid flag is undefined. This currently crashes the game on Windows. Most or all flags are available on OS X and Linux but considerably fewer are available on Windows. See <a rel="nofollow" class="external free" href="https://msdn.microsoft.com/en-us/library/fe06s4ak.aspx">http://msdn.microsoft.com/en-us/library/fe06s4ak.aspx</a> for a list of available flags on Windows. Note that the <b>#</b> flags also crashes the game on Windows.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/329">#329</a></b>
</p>
</td></tr></table></div>
</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This will use the os.time() function, and return it in a friendly way.
os.time() is useful for storing as a date stamp but needs this to make it readable.
</p>
<pre><code data-language="lua">local Timestamp = os.time()
local TimeString = os.date( &quot;%H:%M:%S -&#160;%d/%m/%Y&quot; , Timestamp )
print( &quot;Timestamp:&quot;, Timestamp )
print( &quot;TimeString:&quot;, TimeString )</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">Timestamp:&#9;1583764814
TimeString:&#9;14:40:14 - 09/03/2020</code></pre>
