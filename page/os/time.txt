<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the system time in seconds past the unix epoch. If a table is supplied, the function attempts to build a system time with the specified table members.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> dateData=nil</span>
</p>
<div style="margin-left: 32px;">
Table to generate the time from. This table's data is interpreted as being in the local timezone. See <a href="/page/Structures/DateData" title="Structures/DateData">DateData structure</a></div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
Seconds passed since Unix epoch</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints out the current time, in seconds past the unix epoch.
</p>
<pre><code data-language="lua">print( os.time() )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">1583764814</div>
