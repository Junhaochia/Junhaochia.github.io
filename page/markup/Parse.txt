<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Parses markup into a <a href="/page/Category_colon_MarkupObject" title="Category:MarkupObject"> MarkupObject</a>. Currently, this only supports fonts and colors as demonstrated in the example.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> markup</span>
</p>
<div style="margin-left: 32px;">
The markup to be parsed.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> maxwidth</span>
</p>
<div style="margin-left: 32px;">
The max width of the output</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_MarkupObject" title="Category:MarkupObject">MarkupObject</a></span>
</p>
<div style="margin-left: 32px;">
The parsed markup object ready to be drawn.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Renders a markup string on the HUD.
</p>
<pre><code data-language="lua">local parsed = markup.Parse(&quot;&lt;font=Default&gt;changed font&lt;/font&gt;\n&lt;colour=255,0,255,255&gt;changed colour&lt;/colour&gt;&quot;)

hook.Add(&quot;HUDPaint&quot;, &quot;MarkupTest&quot;, function()
	parsed:Draw(100, 100, TEXT_ALIGN_CENTER, TEXT_ALIGN_CENTER)
end)</code></pre>
