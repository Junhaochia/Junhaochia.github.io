<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the value of a cookie, which is saved automatically by the <a href="/page/Category_colon_sql" title="Category:sql">sql library</a>.
</p><p>These are stored in the *.db files - cl.db for clients, mn.db for menu state and sv.db for servers.
</p>
