<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Adds a new tab.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> name</span>
</p>
<div style="margin-left: 32px;">
Name of the tab</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> pnl</span>
</p>
<div style="margin-left: 32px;">
Panel to be used as contents of the tab. This normally should be a <a href="/page/Category_colon_DPanel" title="Category:DPanel">DPanel</a></div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> icon=nil</span>
</p>
<div style="margin-left: 32px;">
Icon for the tab. This will ideally be a <a href="/page/Silkicons" title="Silkicons">silkicon</a>, but any material name can be used.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> noStretchX=false</span>
</p>
<div style="margin-left: 32px;">
Should <a href="/page/Category_colon_DPropertySheet" title="Category:DPropertySheet">DPropertySheet</a> try to fill itself with given panel horizontally.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_5">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> noStretchY=false</span>
</p>
<div style="margin-left: 32px;">
Should <a href="/page/Category_colon_DPropertySheet" title="Category:DPropertySheet">DPropertySheet</a> try to fill itself with given panel vertically.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_6">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> tooltip=nil</span>
</p>
<div style="margin-left: 32px;">
Tooltip for the tab when user hovers over it with his cursor</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
<p>A table containing the following keys:
</p>
<ul><li> <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> Tab - The created <a href="/page/Category_colon_DTab" title="Category:DTab">DTab</a>.
</li><li> <a href="/page/Category_colon_string" title="Category:string">string</a> Name - Name of the created tab
</li><li> <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a> Panel - The contents panel of the tab</div>
</li></ul>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<pre><code data-language="lua">local frame = vgui.Create(&quot;DFrame&quot;)
frame:SetSize( ScrW() / 2, ScrH() / 2 )
frame:SetTitle( &quot;Test Frame&quot; )
frame:MakePopup()
frame:Center()

local tabs = vgui.Create( &quot;DPropertySheet&quot;, frame )
tabs:Dock( FILL )

local tab1panel = vgui.Create( &quot;DPanel&quot; )

local SheetItem = vgui.Create( &quot;DButton&quot;, tab1panel )
SheetItem:SetText( &quot;Suicide&quot; )
SheetItem:SetConsoleCommand( &quot;kill&quot; )
 
tabs:AddSheet( &quot;Tab 1&quot;, tab1panel, &quot;icon16/user.png&quot;, false, false, &quot;Description of first tab&quot;)</code></pre>
