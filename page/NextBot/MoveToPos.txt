<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>To be called in the behaviour coroutine only! Will yield until the bot has reached the goal or is stuck
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> pos</span>
</p>
<div style="margin-left: 32px;">
The position we want to get to</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> options</span>
</p>
<div style="margin-left: 32px;">
<p>A table containing a bunch of tweakable options.<br /><br />
</p><p><a href="/page/Category_colon_number" title="Category:number">number</a> lookahead - Minimum look ahead distance.<br />
<a href="/page/Category_colon_number" title="Category:number">number</a> tolerance - How close we must be to the goal before it can be considered complete.<br />
<a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> draw - Draw the path. Only visible on listen servers and single player.<br />
<a href="/page/Category_colon_number" title="Category:number">number</a> maxage - Maximum age of the path before it times out.<br />
</p>
<a href="/page/Category_colon_number" title="Category:number">number</a> repath - Rebuilds the path after this number of seconds.</div>
</div>
