<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<p>The spawn icon used for SWEPs and other SENTs, commonly featured as part of the spawn menu. Do note that at least one of your ContentIcon's parents must either be an <a href="/page/Category_colon_EditablePanel" title="Category:EditablePanel">EditablePanel</a> or derived from it (like a <a href="/page/Category_colon_DFrame" title="Category:DFrame">DFrame</a>, for example), else it won't be able to focus and thus be unclickable.
</p><p>This control only exists in Sandbox derived gamemodes.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a button to spawn one or multiple pistols on top of a custom <a href="/page/Category_colon_DFrame" title="Category:DFrame">DFrame</a>.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetSize( 400, 200 )
frame:Center()
frame:SetTitle( &quot;Secret SWEP Spawn Menu&quot; )
frame:MakePopup()

local icon = vgui.Create( &quot;ContentIcon&quot;, frame )
icon:Center()
icon:SetMaterial( &quot;entities/weapon_pistol.png&quot; )
icon:SetName( &quot;Pistol&quot; )
icon.DoClick = function() RunConsoleCommand( &quot;gm_spawnswep&quot;, &quot;weapon_pistol&quot; ) end
icon.OpenMenu = function()
	local menu = DermaMenu()
	menu:AddOption( &quot;Copy to clipboard&quot;, function() SetClipboardText( &quot;weapon_pistol&quot; ) end )
	menu:AddOption( &quot;Spawn 5&quot;, function() for i=1,5 do RunConsoleCommand( &quot;gm_spawnswep&quot;, &quot;weapon_pistol&quot; ) end end )
	menu:AddOption( &quot;Spawn 10&quot;, function() for i=1,10 do RunConsoleCommand( &quot;gm_spawnswep&quot;, &quot;weapon_pistol&quot; ) end end )
	menu:Open()
end</code></pre>
