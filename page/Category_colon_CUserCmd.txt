<p>A class used to store the player inputs, such as mouse movement, view angles, <a href="/page/Enums/IN" title="Enums/IN">IN_ Enums</a> buttons pressed and analog movement, the data from this class is then transfered to a <a href="/page/Category_colon_CMoveData" title="Category:CMoveData">CMoveData</a> during actual movement simulation.
</p><p>Can be modified during <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/CreateMove" title="GM/CreateMove">CreateMove</a>, <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/StartCommand" title="GM/StartCommand">StartCommand</a> and used in read only with <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/SetupMove" title="GM/SetupMove">SetupMove</a> and <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/GetCurrentCommand" title="Player/GetCurrentCommand">GetCurrentCommand</a>.
</p>
