<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a label and fills it with the text "Lua Labels!" parented to a simple frame.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;Frame&quot; )
frame.Label1 = vgui.Create( &quot;Label&quot;, frame )
frame.Label1:SetPos( 50, 50 )
frame.Label1:SetText(&quot;Lua Labels!&quot;)
frame.Label1:SizeToContents()
 
frame:SetSize( ScrW() * 0.25, ScrH() * 0.25 )
frame:Center()
frame:SetVisible( true )
frame:MakePopup()</code></pre>
