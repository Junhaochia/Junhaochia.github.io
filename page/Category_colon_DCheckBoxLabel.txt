<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DCheckBoxLabel, ties it to a convar, and sets its initial value.
</p>
<pre><code data-language="lua">local window = vgui.Create( &quot;DFrame&quot; )
window:SetSize( 300, 200 )
window:Center()
window:MakePopup()

local DermaCheckbox = window:Add( &quot;DCheckBoxLabel&quot; ) -- Create the checkbox
DermaCheckbox:SetPos( 25, 50 )						-- Set the position
DermaCheckbox:SetText(&quot;God Mode&quot;)					-- Set the text next to the box
DermaCheckbox:SetConVar(&quot;sbox_godmode&quot;)				-- Change a ConVar when the box it ticked/unticked
DermaCheckbox:SetValue( true )						-- Initial value
DermaCheckbox:SizeToContents()						-- Make its size the same as the contents</code></pre>
