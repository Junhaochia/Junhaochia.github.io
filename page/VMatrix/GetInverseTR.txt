<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns an inverted matrix without modifying the original matrix. This function will not fail, but only works correctly on matrices that contain only translation and/or rotation.
</p><p>Using this function on a matrix with modified scale may return an incorrect inverted matrix.
</p><p>To get the inverse of a matrix that contains other modifications, see <a href="/page/Category_colon_VMatrix" title="Category:VMatrix">VMatrix</a>:<a href="/page/VMatrix/GetInverse" title="VMatrix/GetInverse">GetInverse</a>.
</p>
