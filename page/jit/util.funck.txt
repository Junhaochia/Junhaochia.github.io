<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets a constant at a certain index in a function.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>This function isn't officially documented on LuJIT wiki, use it at your own risk.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Numbers constants goes from 0 (included) to n-1, n being the value of nconsts in <a href="/page/Category_colon_jit" title="Category:jit">jit</a>.<a href="/page/jit/util.funcinfo" title="jit/util.funcinfo">util.funcinfo</a> in other words, the consts goes from (nconsts-1) to -n
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function only works for Lua defined functions.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This code demonstrates how to get a constant in a function.
</p>
<pre><code data-language="lua">function bob()
    print(&quot;hi&quot;)
end

print(jit.util.funck(bob, -1))
print(jit.util.funck(bob, -2))</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">print<br />
hi</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Demonstrates how a function created inside of the accessed function will be a <a href="/page/Category_colon_proto" title="Category:proto">protos</a> when used with this function.
</p>
<pre><code data-language="lua">local function foo()
    local function bar()
    end 
end

print( type( jit.util.funck( foo, -1 ) ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">proto</div>
