<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Begins a new net message. If another net message is already started and hasn't been sent yet, it will be discarded.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Each net message has a length limit of 64KB!
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>The message name must be pooled with <a href="/page/Category_colon_util" title="Category:util">util</a>.<a href="/page/util/AddNetworkString" title="util/AddNetworkString">AddNetworkString</a> beforehand!
</p>
</td></tr></table>
