<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Appends an integer - a whole number - to the current net message. Can be read back with <a href="/page/Category_colon_net" title="Category:net">net</a>.<a href="/page/net/ReadInt" title="net/ReadInt">ReadInt</a> on the receiving end.
</p><p>Use <a href="/page/Category_colon_net" title="Category:net">net</a>.<a href="/page/net/WriteUInt" title="net/WriteUInt">WriteUInt</a> to send an unsigned number (that you know will never be negative). Use <a href="/page/Category_colon_net" title="Category:net">net</a>.<a href="/page/net/WriteFloat" title="net/WriteFloat">WriteFloat</a> for a non-whole number (e.g. 2.25).
</p>

The integer to be sent.</div>
</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Sends the server the client's age.
</p>
<pre><code data-language="lua">--Client
function SendAge()
     net.Start(&quot;SendAge&quot;)
     net.WriteInt(3, 3)  -- Only 2 bits are needed to store the number '3', but we add one because of the rule.
     net.SendToServer()
end

--Server
util.AddNetworkString(&quot;SendAge&quot;)

local function GetAge(len, ply)
     local age = net.ReadInt(3) -- use the same number of bits that were written
     print(&quot;Player &quot; .. ply:Nick() .. &quot; is &quot; .. age .. &quot; years old.&quot;
end
net.Receive(&quot;SendAge&quot;, GetAge)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Player &lt;name&gt; is 3 years old.</div>
