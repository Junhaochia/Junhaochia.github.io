<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Writes a normalized/direction vector ( Vector with length of 1 ) to the net message.
</p><p>This function uses less bandwidth compared to <a href="/page/Category_colon_net" title="Category:net">net</a>.<a href="/page/net/WriteVector" title="net/WriteVector">WriteVector</a> and will not send vectors with length of &gt; 1 properly.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> normal</span>
</p>
<div style="margin-left: 32px;">
The normalized/direction vector to be send.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Showcases the difference between this function and <a href="/page/Category_colon_net" title="Category:net">net</a>.<a href="/page/net/WriteVector" title="net/WriteVector">WriteVector</a>.
</p>
<pre><code data-language="lua">if ( SERVER ) then
	util.AddNetworkString( &quot;test1&quot; )
	util.AddNetworkString( &quot;test2&quot; )
	
	timer.Simple( 1, function()
		net.Start( &quot;test1&quot; )
			net.WriteVector( Vector( 1.23456789, 2.3456789, 3.456789 ) )
		net.Broadcast()
	
		net.Start( &quot;test2&quot; )
			net.WriteNormal( Vector( 1.23456789, 2.3456789, 3.456789 ) )
		net.Broadcast()

		net.Start( &quot;test2&quot; )
			net.WriteNormal( Vector( 1.23456789, 2.3456789, 3.456789 ):GetNormalized() )
		net.Broadcast()
	
		net.Start( &quot;test2&quot; )
			net.WriteNormal( Vector( 0.5, -0.5, 0.23 ) )
		net.Broadcast()
	end )
else
	net.Receive( &quot;test1&quot;, function( ... )
		print( ... ) 
		print( net.ReadVector() ) 
	end )
	
	net.Receive( &quot;test2&quot;, function( ... )
		print( ... ) 
		print( net.ReadNormal() ) 
	end )
end</code></pre>
