<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Reads a null terminated string from the net stream. The size of the string is 8 bits plus 8 bits for every ASCII character in the string.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>You <b>must</b> read information in same order as you write it.
</p>
</td></tr></table>
