<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Reads a table from the received net message.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Sometimes when sending a table through the net library, the order of the keys may be switched. So be cautious when comparing (See Example 1).
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>You <b>must</b> read information in same order as you write it.
</p>
</td></tr></table>
<p>See <a href="/page/Category_colon_net" title="Category:net">net</a>.<a href="/page/net/WriteTable" title="net/WriteTable">WriteTable</a> for extra info.
</p>
You may get <pre class="inline">net.ReadType: Couldn't read type X</code></pre> during the execution of the function, the problem is that you are sending objects that cannot be serialized/sent over the network.
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
Table recieved via the net message, or a blank table if no table could be read.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This is an example of how the keys order may be switched:
</p>
<pre><code data-language="lua">--Client:
function SendTable()
     local Table = {}
     Table.Type = &quot;Dining&quot;
     Table.Legs = 4
     Table.Material = &quot;Wood&quot;
     net.Start(&quot;TableSend&quot;)
      net.WriteTable(Table)
     net.SendToServer()
     PrintTable(Table) -- Prints the order client side
end

--Server:
function GotTable(len, Player)
     PrintTable(net.ReadTable()) -- Prints the order server side
end
net.Receive(&quot;TableSend&quot;, GotTable)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Client:
<p><code>Type = "Dining"
Legs = 4
Material = "Wood"
</code>
</p><p>Server:
<code>Legs = 4
Material = "Wood"
Type = "Dining"</code>
</p>
</div>
