<h1> <span class="mw-headline" id="Description">Description</span></h1>
<div class="internalfunc"><span>This is an internal function or feature.</span><br />
This means you will be able to use it, but you really shouldn't.<br /><br />Used internally by <a href="/page/Category_colon_net" title="Category:net">net</a>.<a href="/page/net/WriteTable" title="net/WriteTable">WriteTable</a>.</div>
<p>Appends any type of value to the current net message.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>An additional 8-bit unsigned integer indicating the type will automatically be written to the packet before the value, in order to facilitate reading with <a href="/page/Category_colon_net" title="Category:net">net</a>.<a href="/page/net/ReadType" title="net/ReadType">ReadType</a>. If you know the data type you are writing, use a function meant for that specific data type to reduce amount of data sent.
</p>
</td></tr></table>
