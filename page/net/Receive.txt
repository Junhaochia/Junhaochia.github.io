<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Adds a net message handler. Only one receiver can be used to receive the net message.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>The message-name is converted to lower-case so the message-names "<i>BigBlue</i>" and "<i>bigblue</i>" would be equal.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>You <b>must</b> put this function <b>outside</b> of any other function or hook for it to work properly unless you know what you are doing!
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>You <b>must</b> read information in the same order as you write it.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Each net message has a length limit of 64KB!
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> messageName</span>
</p>
<div style="margin-left: 32px;">
The message name to hook to.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> callback</span>
</p>
<div style="margin-left: 32px;">
<p>The function to be called if the specified message was received. Arguments are:
</p><p><a href="/page/Category_colon_number" title="Category:number">number</a> len - Length of the message, in bits
</p>
<a href="/page/Category_colon_Player" title="Category:Player">Player</a> ply - The player that sent the message, works only serverside</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>A simple callback
</p>
<pre><code data-language="lua">net.Receive( &quot;my_message&quot;, function( len, pl )
	if ( IsValid( pl ) and pl:IsPlayer() ) then
		print( &quot;Message from &quot; .. pl:Nick() .. &quot; received. Its length is &quot; .. len .. &quot;.&quot; )
	else
		print( &quot;Message from server received. Its length is &quot; .. len .. &quot;.&quot; )
	end
end )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">When run server-side:
<p><code>Message from John received. Its length is 4.</code>
</p><p>And client-side:
</p>
<code>Message from server received. Its length is 4.</code></div>
