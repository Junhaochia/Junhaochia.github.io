<h1> <span class="mw-headline" id="Description">Description</span></h1>
<div class="internalfunc"><span>This is an internal function or feature.</span><br />
This means you will be able to use it, but you really shouldn't.<br /><br />You might be looking for <a href="/page/Global/RunConsoleCommand" title="Global/RunConsoleCommand">RunConsoleCommand</a> or <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/ConCommand" title="Player/ConCommand">ConCommand</a>.</div>
<p>Used by the engine to run a console command's callback function. This will only be called for commands that were added with <a href="/page/Global/AddConsoleCommand" title="Global/AddConsoleCommand">AddConsoleCommand</a>, which <a href="/page/Category_colon_concommand" title="Category:concommand">concommand</a>.<a href="/page/concommand/Add" title="concommand/Add">Add</a> calls internally. An error is sent to the player's chat if no callback is found.
</p><p>This will still be called for concommands removed with <a href="/page/Category_colon_concommand" title="Category:concommand">concommand</a>.<a href="/page/concommand/Remove" title="concommand/Remove">Remove</a> but will return false.
</p>
