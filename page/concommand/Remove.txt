<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Removes a console command.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This will not always remove the command from auto-complete.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/1183">#1183</a></b>
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p><a href="/page/Category_colon_concommand" title="Category:concommand">concommand</a>.<a href="/page/concommand/Add" title="concommand/Add">Add</a> will fail if the concommand was previously removed with this function in a different realm (creating a command on the client that was removed from the server and vice-versa).
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/1183">#1183</a></b>
</p>
</td></tr></table>
