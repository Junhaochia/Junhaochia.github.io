<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<p>Similarly to <a href="/page/Category_colon_DIconLayout" title="Category:DIconLayout">DIconLayout</a>, this lays out panels in two dimensions as tiles. The difference between this and <a href="/page/Category_colon_DIconLayout" title="Category:DIconLayout">DIconLayout</a> is that <a href="/page/Category_colon_DIconLayout" title="Category:DIconLayout">DIconLayout</a> items all have the same height while <strong class="selflink">DTileLayout</strong> items do not have this enforcement. <strong class="selflink">DTileLayout</strong> will find the best way to "pack" its chidren. For example, in a two column layout, a item of height 2 units will be placed in one column while two items of height 1 unit will be placed in the other column. It is worth noting however that because this panel iterates through its children in an undefined order and lays out while it is iterating, there is no guarentee that this packing will lead to the lowest possible height.
</p><p>This is used by the spawnmenu to arrange spawnicons.
</p><p>The base size defines the smallest a tile can be, and it will resize vertically to accommodate all child panels. The number of elements in each row is determinded by the base size and width.
</p><p>It also optionally permits the rearrangement of these tiles. To enable this functionality, call <a href="/page/Category_colon_DDragBase" title="Category:DDragBase">DDragBase</a>:<a href="/page/DDragBase/MakeDroppable" title="DDragBase/MakeDroppable">MakeDroppable</a> on the DTileLayout with a unique identifier. All panels added following this will be moveable.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DTileLayout within a <a href="/page/Category_colon_DFrame" title="Category:DFrame">DFrame</a>, sets the base size and adds 32 <a href="/page/Category_colon_DLabel" title="Category:DLabel">DLabels</a>.
</p>
<pre><code data-language="lua">local frame = vgui.Create(&quot;DFrame&quot;)
frame:SetSize(300, 300)
frame:SetTitle(&quot;DTileLayout Example&quot;)
frame:MakePopup()
frame:Center()

local layout = vgui.Create(&quot;DTileLayout&quot;, frame)
layout:SetBaseSize(32) -- Tile size
layout:Dock(FILL)

//Draw a background so we can see what it's doing
layout:SetDrawBackground(true)
layout:SetBackgroundColor(Color(0, 100, 100))

layout:MakeDroppable(&quot;unique_name&quot;) -- Allows us to rearrange children

for i = 1, 32 do
	layout:Add(Label(&quot; Label &quot; .. i))
end</code></pre>
