<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Stops particle emission and destroys all particles instantly. Also detaches the particle effect from the entity it was attached to.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function will work identically to <a href="/page/Category_colon_CNewParticleEffect" title="Category:CNewParticleEffect">CNewParticleEffect</a>:<a href="/page/CNewParticleEffect/StopEmission" title="CNewParticleEffect/StopEmission">StopEmission</a>( false, true ) if <a href="/page/Category_colon_CNewParticleEffect" title="Category:CNewParticleEffect">CNewParticleEffect</a>:<a href="/page/CNewParticleEffect/GetOwner" title="CNewParticleEffect/GetOwner">GetOwner</a> entity is not valid.
</p>
</td></tr></table>
<p>Consider using <a href="/page/Category_colon_CNewParticleEffect" title="Category:CNewParticleEffect">CNewParticleEffect</a>:<a href="/page/CNewParticleEffect/StopEmission" title="CNewParticleEffect/StopEmission">StopEmission</a>( false, true ) instead, which has same effect, but doesn't require owner entity, and does't detach the particle system from its entity.
</p>
