<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns entity's map creation ID. Unlike <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/EntIndex" title="Entity/EntIndex">EntIndex</a> or <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetCreationID" title="Entity/GetCreationID">GetCreationID</a>, it will always be the same on same map, no matter how much you clean up or restart it.
</p><p>To be used in conjunction with <a href="/page/Category_colon_ents" title="Category:ents">ents</a>.<a href="/page/ents/GetMapCreatedEntity" title="ents/GetMapCreatedEntity">GetMapCreatedEntity</a>.
</p>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
The map creation ID or -1 if the entity is not compiled into the map.</div>
