<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a specific physics object from an entity with multiple <a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObjects</a> (like ragdolls)
</p><p>See also <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/TranslateBoneToPhysBone" title="Entity/TranslateBoneToPhysBone">TranslateBoneToPhysBone</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> physNum</span>
</p>
<div style="margin-left: 32px;">
The number corresponding to the <a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObj</a> to grab. Starts at 0.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObj</a></span>
</p>
<div style="margin-left: 32px;">
The physics object</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>When run, if the player is dead it will throw their ragdoll up in the air by their head.
</p>
<pre><code data-language="lua">if (&#160;!LocalPlayer():Alive() &amp;&amp; LocalPlayer():GetRagdollEntity() ) then
	local ent = LocalPlayer():GetRagdollEntity()
	local head = ent:GetPhysicsObjectNum( 10 ) // 10 is usually the bone number of the head.
	head:ApplyForceCenter( Vector( 0, 0, 6000 ) )
end</code></pre>

<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Example function that applies force to all physics objects of given entity.
</p>
<pre><code data-language="lua">function ApplySomeForce( ent )
	for i = 0, ent:GetPhysicsObjectCount() - 1 do
		local phys = ent:GetPhysicsObjectNum( i )
		phys:ApplyForceCenter( Vector( 0, 0, 10000 ) )
	end
end</code></pre>
