<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the time (relative to <a href="/page/Global/CurTime" title="Global/CurTime">CurTime</a>) of the current animation frame, which is used to determine <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetCycle" title="Entity/GetCycle">GetCycle</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> time</span>
</p>
<div style="margin-left: 32px;">
The current animation time.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Sets each player's animation time to 1 second in the future, which causes their animations to freeze in place.
</p>
<pre><code data-language="lua">function GM:PrePlayerDraw(ply)

	ply:SetAnimTime(CurTime()+1)

end</code></pre>
