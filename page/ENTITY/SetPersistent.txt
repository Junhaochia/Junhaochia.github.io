<h1> <span class="mw-headline" id="Description">Description</span></h1>
<div class="derivedonly"><span>This function or feature is only available in the Sandbox gamemode and its <a href="/page/Global/DeriveGamemode" title="Global/DeriveGamemode">derivatives</a>.</span></div>
<p>Sets whether or not the given entity is persistent. A persistent entity will be saved on server shutdown and loaded back when the server starts up. Additionally, by default persistent entities cannot be grabbed with the physgun and tools cannot be used on them.
</p><p>In sandbox, this can be set on an entity by opening the context menu, right clicking the entity, and choosing "Make Persistent".
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Persistence can only be enabled with the sbox_persist convar, which works as an identifier for the current set of persistent entities. An empty identifier (which is the default value) disables this feature.
</p>
</td></tr></table>

Whether or not the entity should be persistent.</div>
