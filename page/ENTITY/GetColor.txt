<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the color the entity is set to.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>The returned color will not have the color metatable.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/2407">#2407</a></b>
</p>
</td></tr></table>

<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<pre><code data-language="lua">for key, ply in pairs( player.GetAll( ) ) do -- Loop through all players on the server
 local col = ply:GetColor( ); -- Gets the players color and assigns it to local variable col
 
 print( &quot;Printing &quot; .. ply:Nick() .. &quot;'s color!&quot; ); -- Say we are printing the players name's color
 PrintTable( col ); -- Pass col into PrintTable to print to contents of col
end</code></pre>
