<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Adds a <a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObject</a> to the entity's motion controller so that <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/PhysicsSimulate" title="ENTITY/PhysicsSimulate">PhysicsSimulate</a> will be called for given <a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObject</a> as well.
</p><p>You must first create a motion controller with <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/StartMotionController" title="Entity/StartMotionController">StartMotionController</a>.
</p><p>You can remove added <a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObjects</a> by using <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/RemoveFromMotionController" title="Entity/RemoveFromMotionController">RemoveFromMotionController</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Only works on a scripted <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> of anim type
</p>
</td></tr></table>

<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObj</a> physObj</span>

The <a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObj</a> to add to the motion controller.</div>
