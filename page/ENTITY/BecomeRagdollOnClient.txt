<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Spawns a clientside ragdoll for the entity, positioning it in place of the original entity, and makes the entity invisible. It doesn't preserve flex values (face posing) as CSRagdolls don't support flex.
</p><p>It does not work on players. Use <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/CreateRagdoll" title="Player/CreateRagdoll">CreateRagdoll</a> instead.
</p><p>The original entity is not removed, and neither are any ragdolls previously generated with this function.
</p><p>To make the entity re-appear, run <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetNoDraw" title="Entity/SetNoDraw">SetNoDraw</a>( false )
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Spawns a ragdoll for all NPCs.
</p>
<pre><code data-language="lua">for k,npc in pairs( ents.FindByClass( &quot;npc_*&quot; ) ) do
    npc.RagDoll = npc:BecomeRagdollOnClient()
end</code></pre>
