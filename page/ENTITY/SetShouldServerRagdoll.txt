<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets if entity should create a server ragdoll on death or a client one.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Player ragdolls created with this enabled will have an owner set, see <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetOwner" title="Entity/SetOwner">SetOwner</a> for more information on what effects this has.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is reset for players when they respawn. (<a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/Spawn" title="Entity/Spawn">Spawn</a>)
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> serverragdoll</span>
</p>
<div style="margin-left: 32px;">
Set true if ragdoll should be created on server, false if on client</div>
