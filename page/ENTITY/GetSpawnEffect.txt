<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns if we should show a spawn effect on spawn on this entity.
</p>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
The flag to allow or disallow the spawn effect.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Taken from sandbox's cl_init.lua
</p>
<pre><code data-language="lua">function GM:NetworkEntityCreated( ent )

	--
	-- If the entity wants to use a spawn effect
	-- then create a propspawn effect if the entity was
	-- created within the last second (this function gets called
	-- on every entity when joining a server)
	--

	if ( ent:GetSpawnEffect() &amp;&amp; ent:GetCreationTime() &gt; ( CurTime() - 1.0 ) ) then
	
		local ed = EffectData()
		ed:SetEntity( ent )
		util.Effect( &quot;propspawn&quot;, ed, true, true )

	end

end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Checks if the entity has the spawnEffect flag set to true and checks if it was created in the last second, and then shows the propspawn effect.</div>
