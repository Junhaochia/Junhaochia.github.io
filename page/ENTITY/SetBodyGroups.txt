<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the bodygroups from a string. A convenience function for <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetBodygroup" title="Entity/SetBodygroup">SetBodygroup</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> bodygroups</span>
</p>
<div style="margin-left: 32px;">
Body groups to set. Each single-digit number in the string represents a separate bodygroup. <b>This make it impossible to set any bodygroup to a value higher than 9!</b> For that you need to use <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetBodygroup" title="Entity/SetBodygroup">SetBodygroup</a>.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example of the format
</p>
<pre><code data-language="lua">Entity(1):SetBodyGroups( &quot;021&quot; )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Sets first bodygroup value to 0, second to 2 and third to 1.</div>
