<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a table containing all key values the entity has.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This only includes engine defined key values. "targetname" is not an actual key value in-engine, use <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetName" title="Entity/GetName">GetName</a> for that instead. For custom key values, use <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/EntityKeyValue" title="GM/EntityKeyValue">EntityKeyValue</a> or <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/KeyValue" title="ENTITY/KeyValue">KeyValue</a> to capture and store them.
</p>
