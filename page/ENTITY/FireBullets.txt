<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Fires a bullet.
</p><p>When used in a <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks"> WEAPON</a> hook such as <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a>:<a href="/page/WEAPON/Think" title="WEAPON/Think">Think</a> or <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a>:<a href="/page/WEAPON/PrimaryAttack" title="WEAPON/PrimaryAttack">PrimaryAttack</a>, it will use <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/LagCompensation" title="Player/LagCompensation">LagCompensation</a> internally.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Lag compensation will not work if this function is called in a timer, regardless if the timer was made in a <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a> hook.
</p>
</td></tr></table>
