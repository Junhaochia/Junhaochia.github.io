<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets a networked number at the specified index on the entity.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_any" title="Category:any">any</a> index</span>
</p>
<div style="margin-left: 32px;">
The index that the value is stored in.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> number</span>
</p>
<div style="margin-left: 32px;">
The value to network.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This will set the networked number 'score' on all clients to 3.
</p>
<pre><code data-language="lua">for k, v in pairs( player.GetAll() ) do
    v:SetNetworkedInt( 'score', 3 )
end</code></pre>
