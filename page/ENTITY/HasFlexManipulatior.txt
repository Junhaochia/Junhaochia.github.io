<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns whether or not the the entity has had flex manipulations performed with <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetFlexWeight" title="Entity/SetFlexWeight">SetFlexWeight</a> or <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetFlexScale" title="Entity/SetFlexScale">SetFlexScale</a>.
</p>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
True if the entity has flex manipulations, false otherwise.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Defines server-side function which manipulates all of an entity's flexes and prints true if the entity has flex manipulations.
</p>
<pre><code data-language="lua">function FlexExample(ent)

	if(!IsValid(ent)) then return end
	
	-- Loop through all flexes
	for i = 0, ent:GetFlexNum()-1 do
	
		-- Set each flex to number ranging from 0.0 to 2.0
		ent:SetFlexWeight(i, math.random()*2)
		
	end
	
	-- Print whether or not we have flex manipulations
	print(ent:HasFlexManipulatior())

end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">true</code></pre>
