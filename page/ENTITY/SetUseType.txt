<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the use type of an entity, affecting how often <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/Use" title="ENTITY/Use">Use</a> will be called for Lua entities.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> useType</span>
</p>
<div style="margin-left: 32px;">
The use type to apply to the entity. Uses <a href="/page/Enums/_USE" title="Enums/ USE">_USE Enums</a>.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Makes the <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/Use" title="ENTITY/Use">Use</a> hook only get called once at every use.
</p>
<pre><code data-language="lua">Entity:SetUseType( SIMPLE_USE )</code></pre>
