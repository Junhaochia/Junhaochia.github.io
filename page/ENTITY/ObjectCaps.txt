<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the entity's capabilities as a bitfield.<br />
In the engine this function is mostly used to check the use type, the save/restore system and level transitions flags.
</p><p>Even though the function is defined shared, it is not guaranteed to return the same value across states.
</p>
