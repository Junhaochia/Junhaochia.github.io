<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets the bone the hit box is attached to.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> hitbox</span>
</p>
<div style="margin-left: 32px;">
The number of the hit box.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> group</span>
</p>
<div style="margin-left: 32px;">
<p>The number of the hit box group. This should be 0 in most cases.
</p>
Numbering for these groups start from 0. The total group count can be found with <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetHitBoxGroupCount" title="Entity/GetHitBoxGroupCount">GetHitBoxGroupCount</a>.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
The number of the bone. Will be nil if the hit box index was out of range.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<pre><code data-language="lua">local ply = LocalPlayer()
local numHitBoxGroups = ply:GetHitBoxGroupCount()

for group=0, numHitBoxGroups - 1 do
  local numHitBoxes = ply:GetHitBoxCount( group )
    
  for hitbox=0, numHitBoxes - 1 do
    local bone = ply:GetHitBoxBone(hitbox, group)

    print( &quot;Hit box group &quot; .. group .. &quot;, hitbox &quot; .. hitbox .. &quot; is attached to bone &quot; .. ply:GetBoneName(bone) )
  end
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Hit box group 0, hit box 0 is attached to bone ValveBiped.Bip01_Head1, etc.</div>
