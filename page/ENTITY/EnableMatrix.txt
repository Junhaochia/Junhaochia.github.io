<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Can be used to apply a custom <a href="/page/Category_colon_VMatrix" title="Category:VMatrix">VMatrix</a> to the entity, mostly used for scaling the model by a <a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a>.
</p><p>To disable it, use <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/DisableMatrix" title="Entity/DisableMatrix">DisableMatrix</a>.
</p><p>If your old scales are wrong due to a recent update, use <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetLegacyTransform" title="Entity/SetLegacyTransform">SetLegacyTransform</a> as a quick fix.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>The matrix can also be modified to apply a custom rotation and offset via the <a href="/page/Category_colon_VMatrix" title="Category:VMatrix">VMatrix</a>:<a href="/page/VMatrix/SetAngles" title="VMatrix/SetAngles">SetAngles</a> and <a href="/page/Category_colon_VMatrix" title="Category:VMatrix">VMatrix</a>:<a href="/page/VMatrix/SetTranslation" title="VMatrix/SetTranslation">SetTranslation</a> functions.
</p>
</td></tr></table>

<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> matrixType</span>
</p>
<div style="margin-left: 32px;">
<p>The name of the matrix type.<br />
</p>
The only implemented matrix type is "RenderMultiply".</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_VMatrix" title="Category:VMatrix">VMatrix</a> matrix</span>
</p>
<div style="margin-left: 32px;">
The matrix to apply before drawing the entity.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>To scale a prop's height by 4x
</p>
<pre><code data-language="lua">local scale = Vector(1,1,4)

local mat = Matrix()
mat:Scale(scale)
prop:EnableMatrix(&quot;RenderMultiply&quot;, mat)</code></pre>
