<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sends sequence animation to the view model. It is recommended to use this for view model animations, instead of <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/ResetSequence" title="Entity/ResetSequence">ResetSequence</a>.
</p><p>This function is only usable on view models.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>Sequences 0-6 will not be looped regardless if they're marked as a looped animation or not.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3229">#3229</a></b>
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> seq</span>
</p>
<div style="margin-left: 32px;">
The sequence ID returned by <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/LookupSequence" title="Entity/LookupSequence">LookupSequence</a> or <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SelectWeightedSequence" title="Entity/SelectWeightedSequence">SelectWeightedSequence</a>.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Converting an ACT_VM_* enumeration to a sequence usable by the function.
</p>
<pre><code data-language="lua">local VModel = self.Owner:GetViewModel( )
local EnumToSeq = VModel:SelectWeightedSequence( ACT_VM_PRIMARYATTACK )

VModel:SendViewModelMatchingSequence( EnumToSeq )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Sends the primary attack enumeration sequence to the view model and plays it.</div>
