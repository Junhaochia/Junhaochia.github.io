<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the position an entity's eyes look toward.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> pos</span>
</p>
<div style="margin-left: 32px;">
The world position the entity is looking toward.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Makes an NPC (self) look into a nearby player's eyes.
</p>
<pre><code data-language="lua">for p, ply in pairs(player.GetAll()) do
    if(ply:EyePos():Distance(self:EyePos()) &lt;= 60) then
        self:SetEyeTarget(ply:EyePos())
        break
    end
end</code></pre>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Makes an entity look at a vector the way the eyeposer does it
</p>
<pre><code data-language="lua">local lookat = Vector( 0, 0, 0 )

local attachment = ent:GetAttachment( ent:LookupAttachment( &quot;eyes&quot; ) )
local LocalPos, LocalAng = WorldToLocal( lookat, Angle( 0, 0, 0 ), attachment.Pos, attachment.Ang )
ent:SetEyeTarget( LocalPos )</code></pre>
