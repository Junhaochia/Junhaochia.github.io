<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when another entity uses this entity, example would be a player pressing "+use" this entity.
</p><p>To change how often the function is called, see <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetUseType" title="Entity/SetUseType">SetUseType</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This hook only works for "nextbot", "ai" and "anim" scripted entity types.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Kills any player that uses this entity.
</p>
<pre><code data-language="lua">function ENT:Use( activator )

	if activator:IsPlayer() then 

		activator:Kill()

	end

end</code></pre>
