<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Starts a motion controller in the physics engine tied to this entity's <a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObj</a>, which enables the use of <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/PhysicsSimulate" title="ENTITY/PhysicsSimulate">PhysicsSimulate</a>.
</p><p>The motion controller can later be destroyed via <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/StopMotionController" title="Entity/StopMotionController">StopMotionController</a>.
</p><p>Motion controllers are used internally to control other Entities' <a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObjects</a>, such as the Gravity Gun, +use pickup and the Physics Gun.
</p><p>This function should be called every time you recreate the Entity's <a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObj</a>. Or alternatively you should call <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/AddToMotionController" title="Entity/AddToMotionController">AddToMotionController</a> on the new <a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObj</a>.
</p><p>Also see <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/AddToMotionController" title="Entity/AddToMotionController">AddToMotionController</a> and <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/RemoveFromMotionController" title="Entity/RemoveFromMotionController">RemoveFromMotionController</a>.
</p>
