<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called whenever the entity's position changes. A callback for when an entity's angle changes is available via <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/AddCallback" title="Entity/AddCallback">AddCallback</a>.
</p><p>Like <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/RenderOverride" title="ENTITY/RenderOverride">RenderOverride</a>, this hook works on any entity (scripted or not) it is applied on.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>If EFL_DIRTY_ABSTRANSFORM is set on the entity, this will be called serverside only; otherwise, this will be called clientside only. This means serverside calls of <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetPos" title="Entity/SetPos">SetPos</a> without the EFL_DIRTY_ABSTRANSFORM flag enabled (most cases) will be called clientside only.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>The give concommand will call this hook serverside only upon entity spawn.
</p>
</td></tr></table>
