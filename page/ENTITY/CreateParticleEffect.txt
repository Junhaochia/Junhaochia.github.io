<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Creates a clientside particle system attached to the entity. See also <a href="/page/Global/CreateParticleSystem" title="Global/CreateParticleSystem">CreateParticleSystem</a>
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>The particle effect must be precached with <a href="/page/Global/PrecacheParticleSystem" title="Global/PrecacheParticleSystem">PrecacheParticleSystem</a> and the file its from must be added via <a href="/page/Category_colon_game" title="Category:game">game</a>.<a href="/page/game/AddParticles" title="game/AddParticles">AddParticles</a> before it can be used!
</p>
</td></tr></table>

<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> options=nil</span>
</p>
<div style="margin-left: 32px;">
<p>A table of tables ( IDs 1 to 64 ) having the following structure:
</p>
<ul><li> <a href="/page/Category_colon_number" title="Category:number">number</a> attachtype - The particle attach type. See <a href="/page/Enums/PATTACH" title="Enums/PATTACH">PATTACH_ Enums</a>. <b>Default:</b> PATTACH_ABSORIGIN
</li><li> <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> entity - The parent entity? <b>Default:</b> NULL
</li><li> <a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> position - The offset position for given control point. <b>Default:</b> nil
</li></ul>
This only affects the control points of the particle effects and will do nothing if the effect doesn't use control points.</div>
