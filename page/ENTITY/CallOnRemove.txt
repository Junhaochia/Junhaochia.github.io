<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Causes a specified function to be run if the entity is removed by any means. This can later be undone by <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/RemoveCallOnRemove" title="Entity/RemoveCallOnRemove">RemoveCallOnRemove</a> if you need it to not run.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>Using players with this function will provide a gimped entity to the callback.
</p><p><b>Pull Request: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod/pull/1275">#1275</a></b>
</p>
</td></tr></table>
