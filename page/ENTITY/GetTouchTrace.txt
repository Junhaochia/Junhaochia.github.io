<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the last trace used in the collision callbacks such as <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/StartTouch" title="ENTITY/StartTouch">StartTouch</a>, <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/Touch" title="ENTITY/Touch">Touch</a> and <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/EndTouch" title="ENTITY/EndTouch">EndTouch</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This returns the last collision trace used, regardless of the entity that caused it. As such, it's only reliable when used in the hooks mentioned above
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Dispatches an explosion at the point of impact with another entity.
</p>
<pre><code data-language="lua">function ENT:Touch( otherEntity )
	local tr = self:GetTouchTrace()
	local hitPos = tr.HitPos

	local effectdata = EffectData()
	effectdata:SetOrigin( hitPos )
	util.Effect( &quot;Explosion&quot;, effectdata )

end</code></pre>
