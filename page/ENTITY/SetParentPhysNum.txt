<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the parent of an entity to another entity with the given physics bone number. Similar to <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetParent" title="Entity/SetParent">SetParent</a>, except it is parented to a physbone. This function is useful mainly for ragdolls.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Despite this function being available server side, it doesn't actually do anything server side.
</p>
</td></tr></table>
