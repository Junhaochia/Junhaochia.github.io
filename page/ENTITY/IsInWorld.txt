<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns whether the entity is inside a wall or outside of the map.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Internally this function uses <a href="/page/Category_colon_util" title="Category:util">util</a>.<a href="/page/util/IsInWorld" title="util/IsInWorld">IsInWorld</a>, that means that this function only checks <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetPos" title="Entity/GetPos">GetPos</a> of the entity. If an entity is only partially inside a wall, or has a weird GetPos offset, this function may not give reliable output.
</p>
</td></tr></table>
