<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Used to specify a plane, past which an object will be visually clipped.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a> planeNormal</span>
</p>
<div style="margin-left: 32px;">
The normal of the plane. Anything behind the normal will be clipped.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> planePosition</span>
</p>
<div style="margin-left: 32px;">
The position of the plane.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a blue barrel at Vector(0, 0, 0), freezes it, and will visually clip the barrel's lower half.
</p>
<pre><code data-language="lua">local ent = ents.Create( &quot;prop_physics&quot; )
ent:SetModel( &quot;models/props_borealis/bluebarrel001.mdl&quot; )
ent:SetPos( Vector(0, 0, 0) )
ent:Spawn()
ent:GetPhysicsObject():EnableMotion(false)

local normal = ent:GetUp()
local position = normal:Dot( ent:GetPos() )
ent:SetRenderClipPlaneEnabled( true )
ent:SetRenderClipPlane( normal, position )</code></pre>
