<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets Hammer key values on an entity.
</p><p>You can look up which entities have what key values on the <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/">Valve Developer Community</a> on entity pages.<br />
A list of basic entities can be found <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/List_of_entities">here</a>.
</p><p>Alternatively you can look at the .fgd files shipped with Garry's Mod in the bin/ folder with a text editor to see the key values as they appear in Hammer.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Makes an NPC not drop his weapon and drop a healthkit on death, using <a rel="nofollow" class="external text" href="/page/Enums/SF">SF Enumerations</a> on a Combine Soldier.
</p><p>A list of spawnflags a Combine Soldier has can be found <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Npc_combine_s#Flags">here</a>.
</p>
<pre><code data-language="lua">npc:SetKeyValue( &quot;spawnflags&quot;, bit.bor( SF_NPC_NO_WEAPON_DROP, SF_NPC_DROP_HEALTHKIT ) )</code></pre>
