<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the entity's model sequence.
</p><p>If the specified sequence is already active, the animation will not be restarted. See <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/ResetSequence" title="Entity/ResetSequence">ResetSequence</a> for a function that restarts the animation even if it is already playing.
</p><p>In some cases you want to run <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/ResetSequenceInfo" title="Entity/ResetSequenceInfo">ResetSequenceInfo</a> to make this function run.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This will not work properly if called directly after calling <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetModel" title="Entity/SetModel">SetModel</a>. Consider waiting until the next Tick.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Will not work on players due to the animations being reset every frame by the base gamemode animation system in <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/CalcMainActivity" title="GM/CalcMainActivity">CalcMainActivity</a>. For Players, use in <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/UpdateAnimation" title="GM/UpdateAnimation">UpdateAnimation</a> instead.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> sequenceId</span>
</p>
<div style="margin-left: 32px;">
<p>The sequence to play. Also accepts strings.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>If set to a string, the function will automatically call <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/LookupSequence" title="Entity/LookupSequence">LookupSequence</a> to retrieve the sequence ID as a number.
</p>
</td></tr></table></div>
</div>
