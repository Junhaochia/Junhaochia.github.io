<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Draws the entity or model.
</p><p>If called inside <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/Draw" title="ENTITY/Draw">Draw</a> or <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/DrawTranslucent" title="ENTITY/DrawTranslucent">DrawTranslucent</a>, it only draws the entity's model itself.<br />
If called outside of those hooks, it will call both of said hooks depending on <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetRenderGroup" title="Entity/GetRenderGroup">GetRenderGroup</a>, drawing the entire entity again.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>When drawing an entity more than once per frame in different positions, you should call <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetupBones" title="Entity/SetupBones">SetupBones</a> before each draw; Otherwise, the entity will retain its first drawn position.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is a rendering function that requires a 3D rendering context.<br />This means that it will only work in <a href="/page/Category_colon_3D_Rendering_Hooks" title="Category:3D Rendering Hooks">hooks with a 3D rendering context</a>.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Manually draws a single <a href="/page/Global/ClientsideModel" title="Global/ClientsideModel">ClientsideModel</a> on the specified bone, on the given offset for every player affected by this hook.
</p><p>This is useful in case you want to reuse a single model without having to create one for each player.
</p>
<pre><code data-language="lua">local modelexample = ClientsideModel( &quot;models/thrusters/jetpack.mdl&quot; )
modelexample:SetNoDraw( true )

local offsetvec = Vector( 3, -5.6, 0 )
local offsetang = Angle( 180, 90, -90 )

hook.Add( &quot;PostPlayerDraw&quot; , &quot;manual_model_draw_example&quot; , function( ply )
	local boneid = ply:LookupBone( &quot;ValveBiped.Bip01_Spine2&quot; )
	
	if not boneid then
		return
	end
	
	local matrix = ply:GetBoneMatrix( boneid )
	
	if not matrix then 
		return 
	end
	
	local newpos, newang = LocalToWorld( offsetvec, offsetang, matrix:GetTranslation(), matrix:GetAngles() )
	
	modelexample:SetPos( newpos )
	modelexample:SetAngles( newang )
	modelexample:SetupBones()
	modelexample:DrawModel()
	
end)</code></pre>
