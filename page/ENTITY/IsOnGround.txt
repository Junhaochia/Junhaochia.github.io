<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns whether the entity is on ground or not.
</p><p>Internally, this checks if <a href="/page/Enums/FL" title="Enums/FL">FL_ONGROUND</a> is set on the entity.
</p><p>This function is an alias of <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/OnGround" title="Entity/OnGround">OnGround</a>.
</p>
