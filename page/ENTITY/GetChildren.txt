<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets the children of the entity - that is, every entity whose move parent is this entity.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function returns <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetMoveParent" title="Entity/SetMoveParent">SetMoveParent</a> children, <b>NOT</b> <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetParent" title="Entity/SetParent">SetParent</a>!<br /><a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetParent" title="Entity/SetParent">SetParent</a> however also calls <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetMoveParent" title="Entity/SetMoveParent">SetMoveParent</a>.<br /><br />This means that some entities in the returned list might have a NULL <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetParent" title="Entity/GetParent">GetParent</a>.<br />This also means that using this function on players will return their weapons on the client but not the server.
</p>
</td></tr></table>
