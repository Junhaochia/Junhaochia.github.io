<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns true if the entity has constraints attached to it
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This will only update clientside if the server calls it first. This only checks constraints added through the <a href="/page/Category_colon_constraint" title="Category:constraint">constraint library</a> so this will not react to map constraints.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3837">#3837</a></b>
</p>
</td></tr></table>

<pre><code data-language="lua">function ENT:PhysicsUpdate( physobj )

	-- Don't do anything if the player isn't holding us

		physobj:Sleep()

end</code></pre>
