<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets solid flag(s) for the entity.
</p><p>This overrides any other flags the entity might have had. See <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/AddSolidFlags" title="Entity/AddSolidFlags">AddSolidFlags</a> for adding flags.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> flags</span>
</p>
<div style="margin-left: 32px;">
The flag(s) to set, see <a href="/page/Enums/FSOLID" title="Enums/FSOLID">FSOLID_ Enums</a>.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Mimics <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetTrigger" title="Entity/SetTrigger">SetTrigger</a> call on the entity and adds FSOLID_USE_TRIGGER_BOUNDS flag to it.
</p>
<pre><code data-language="lua">ent:SetSolidFlags( bit.bor( FSOLID_TRIGGER, FSOLID_USE_TRIGGER_BOUNDS ) )</code></pre>
