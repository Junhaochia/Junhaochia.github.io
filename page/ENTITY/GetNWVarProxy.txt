<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns callback function for given NWVar of this entity.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_any" title="Category:any">any</a> key</span>
</p>
<div style="margin-left: 32px;">
The key of the NWVar to get callback of.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a></span>
</p>
<div style="margin-left: 32px;">
The callback of given NWVar, or nil if not found.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints callback function of a NWVar called "Key" of Player 1.
</p>
<pre><code data-language="lua">print( Entity(1):GetNWVarProxy( &quot;Key&quot; ) )
Entity(1):SetNWVarProxy( &quot;Key&quot;, print )
print( Entity(1):GetNWVarProxy( &quot;Key&quot; ) )</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">nil
