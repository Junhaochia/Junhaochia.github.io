<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the color of an entity.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> color=Color(255, 0, 255, 255)</span>
</p>
<div style="margin-left: 32px;">
The color to set. Uses the <a href="/page/Structures/Color" title="Structures/Color">Color structure</a>.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Loop through all players, make them black
</p>
<pre><code data-language="lua">local colBlack = Color( 0, 0, 0, 255 ) -- Creates a black color
for key, ply in pairs(player.GetAll()) do -- Loop through all players on the server
	ply:SetColor(colBlack) -- Sets the players color to colBlack
end</code></pre>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Creates a wooden crate at 0,0,0 and turns it a transparent green
</p>
<pre><code data-language="lua">local ent = ents.Create(&quot;prop_physics&quot;)
ent:SetPos(Vector(0,0,0))
ent:SetModel(&quot;models/props_junk/wood_crate001a.mdl&quot;)
ent:Spawn()

ent:SetColor( Color( 0, 255, 0, 230 ) ) 
ent:SetRenderMode( RENDERMODE_TRANSALPHA ) -- You need to set the render mode on some entities in order for the color to change</code></pre>
