<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called instead of the engine drawing function of the entity. This hook works on any entity (scripted or not) it is applied on.
</p><p>This does not work on "physgun_beam", use <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/DrawPhysgunBeam" title="GM/DrawPhysgunBeam">DrawPhysgunBeam</a> instead.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>As a downside of this implementation, only one RenderOverride may be applied at a time.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Set the entity the player is looking at to not draw if the player is its owner.
</p>
<pre><code data-language="lua">local function DontDrawMe( self )
	if ( self:GetOwner() == LocalPlayer() ) then
		return
	end
	
	self:DrawModel()
end

local pickent = LocalPlayer():GetEyeTrace().Entity

if ( IsValid( pickent ) ) then
	pickent.RenderOverride = DontDrawMe
end</code></pre>
