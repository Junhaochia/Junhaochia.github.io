<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Plays a sound on an entity. If run clientside, the sound will only be heard locally.
</p><p>If used on a player or NPC character with the mouth rigged, the character will "lip-sync". This does not work with all sound files.
</p><p>It is recommended to use sound scripts ( see <a href="/page/Category_colon_sound" title="Category:sound">sound</a>.<a href="/page/sound/Add" title="sound/Add">Add</a> ) over direct file paths. This will allow you to use <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/StopSound" title="Entity/StopSound">StopSound</a> to stop the played sound scripts.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>When using this function with weapons, use the <a href="/page/Category_colon_Weapon" title="Category:Weapon">Weapon</a> itself as the entity, not its owner!
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Do not use this for looping sounds with a filepath: see <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/StopSound" title="Entity/StopSound">StopSound</a> for more details.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG

</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/2651">#2651</a></b>

<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> soundName</span>
</p>
<div style="margin-left: 32px;">
<p>The name of the sound to be played.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>The string will cannot have whitespace at the start or end. You can remove this with <a href="/page/Category_colon_string" title="Category:string">string</a>.<a href="/page/string/Trim" title="string/Trim">Trim</a>.
</p>
</td></tr></table></div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> soundLevel=75</span>
</p>
<div style="margin-left: 32px;">
<p>A modifier for the distance this sound will reach, acceptable range is 0 to 511. 100 means no adjustment to the level. See <a href="/page/Enums/SNDLVL" title="Enums/SNDLVL">SNDLVL_ Enums</a>
</p>
Will not work if a <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Soundscripts">sound script</a> is used.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> pitchPercent=100</span>
</p>
<div style="margin-left: 32px;">
The pitch applied to the sound. The acceptable range is from 0 to 255. 100 means the pitch is not changed.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> volume=1</span>
</p>
<div style="margin-left: 32px;">
The volume, from 0 to 1.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_5">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> channel=CHAN_AUTO, CHAN_WEAPON for weapons</span>
</p>
<div style="margin-left: 32px;">
<p>The sound channel, see <a href="/page/Enums/CHAN" title="Enums/CHAN">CHAN_ Enums</a>.
</p>
Will not work if a <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Soundscripts">sound script</a> is used.</div>
</div>
