<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Move an entity down until it collides with something.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>The entity needs to already have something below it within 256 units.
</p>
</td></tr></table>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Move all props on the server down until they collide with something
</p>
<pre><code data-language="lua">for _, ent in pairs( ents.FindByClass( &quot;prop_physics&quot; ) ) do
	ent:DropToFloor( )
end</code></pre>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Drops players to the ground when they spawn
</p>
<pre><code data-language="lua">hook.Add( &quot;PlayerSpawn&quot;, &quot;DropGround&quot;, function( ply )
	ply:DropToFloor()
end )</code></pre>
