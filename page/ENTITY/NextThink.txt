<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>In the case of a scripted entity, this will cause the next <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/Think" title="ENTITY/Think">Think</a> event to be run at the given time.
</p><p>Does not work clientside! Use <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetNextClientThink" title="Entity/SetNextClientThink">SetNextClientThink</a> instead.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This does not work with SWEPs or Nextbots.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3269">#3269</a></b>
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> timestamp</span>
</p>
<div style="margin-left: 32px;">
The relative to <a href="/page/Global/CurTime" title="Global/CurTime">CurTime</a> timestamp, at which the next think should occur.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints 'Hello, World!' in console and sleeps for a second.
</p>
<pre><code data-language="lua">function ENT:Think()
    print(&quot;Hello, World!&quot;)

    self:NextThink( CurTime() + 1 )
    return true -- Note: You need to return true to override the default next think time
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Hello, World! every second the entity exists in the world.</div>
