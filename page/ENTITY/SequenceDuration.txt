<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns length of currently played sequence.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This will return incorrect results for weapons and viewmodels clientside in thirdperson.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/2783">#2783</a></b>
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> seqid=nil</span>
</p>
<div style="margin-left: 32px;">
A sequence ID to return the length specific sequence of instead of the entity's main/currently playing sequence.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
The length of the sequence</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage for the argument. Retrieve length of animation on specific gesture slot.
</p>
<pre><code data-language="lua">local ply = Entity( 1 )
local seq = ply:SelectWeightedSequence( ACT_GMOD_TAUNT_CHEER )
local len = ply:SequenceDuration( seq )
print( ply, seq, len )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Player [1][Rubat]	303	2.7499999180436</div>
