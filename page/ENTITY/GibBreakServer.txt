<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Causes the entity to break into its current models gibs, if it has any.
</p><p>You must call <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/PrecacheGibs" title="Entity/PrecacheGibs">PrecacheGibs</a> on the entity before using this function, or it will not create any gibs.
</p><p>The gibs will be spawned on the server and be synchronized with all clients.
</p><p>Note, that this function will not remove or hide the entity it is called on.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Large numbers of serverside gibs will cause lag.<br />
You can avoid this cost by spawning the gibs on the client using <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GibBreakClient" title="Entity/GibBreakClient">GibBreakClient</a>
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Despite existing on client, it doesn't actually do anything on client.
</p>
</td></tr></table>
