<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Triggers all outputs stored using <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/StoreOutput" title="ENTITY/StoreOutput">StoreOutput</a>.
</p>

</p>
<div style="margin-left: 32px;">
Name of output to fire</div>
</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>For engine entities you can use <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/Fire" title="Entity/Fire">Fire</a> to hook outputs.
This example hooks all trigger_teleport
</p>
<pre><code data-language="lua">local function SetupMapLua()
    MapLua = ents.Create(&quot;lua_run&quot;)
    MapLua:SetName(&quot;triggerhook&quot;)
    MapLua:Spawn()

    for k, v in pairs(ents.FindByClass(&quot;trigger_teleport&quot;)) do
        print(v)
        v:Fire(&quot;AddOutput&quot;, &quot;OnStartTouch triggerhook:RunPassedCode:hook.Run('OnTeleport'):0:-1&quot;)
    end
end

hook.Add(&quot;InitPostEntity&quot;, &quot;SetupMapLua&quot;, SetupMapLua)
hook.Add(&quot;PostCleanupMap&quot;, &quot;SetupMapLua&quot;, SetupMapLua)
hook.Add(&quot;OnTeleport&quot;, &quot;TestTeleportHook&quot;, function()
    local activator,caller = ACTIVATOR, CALLER
    print(activator, caller)
end)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">When player touches trigger_teleport this will be printed in the console:
<code>Player [1][Player1] Entity [3][trigger_teleport]</code></div>
