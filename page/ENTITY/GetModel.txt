<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets the model of given entity.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This does not necessarily return the model's path, as is the case for brush and virtual models. This is intentional behaviour, however, there is currently no way to retrieve the actual file path.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/2246">#2246</a></b>
</p>
</td></tr></table>

<p>The entity's model. Will be a filesystem path for most models.
