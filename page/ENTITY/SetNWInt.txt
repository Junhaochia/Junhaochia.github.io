<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets a networked integer (whole number) value on the entity.
</p><p>The value can then be accessed with <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetNWInt" title="Entity/GetNWInt">GetNWInt</a> both from client and server.
</p><p>See <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetNWFloat" title="Entity/SetNWFloat">SetNWFloat</a> for numbers that aren't integers.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Running this function clientside will only set it for the client it is called on.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This function will not round decimal values as it actually networks a float internally.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3374">#3374</a></b>
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This will set the networked integer 'money' on all clients to 100.
</p>
<pre><code data-language="lua">for k, v in ipairs( player.GetAll() ) do
    v:SetNWInt( 'money', 100 )
end</code></pre>
