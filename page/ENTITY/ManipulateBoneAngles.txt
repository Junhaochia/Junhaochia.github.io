<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets custom bone angles.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>The repeated use of bone manipulation in multiplayer games is highly discouraged due to the huge produced network traffic.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> boneID</span>
</p>
<div style="margin-left: 32px;">
Index of the bone you want to manipulate</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Angle" title="Category:Angle">Angle</a> ang</span>
</p>
<div style="margin-left: 32px;">
<p>Angle to apply.
</p>
The angle is relative to the original bone angle, not relative to the world or the entity.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This example shows the network usage impact of repeatedly using bone manipulation serverside.
</p><p>To see the difference, type in client's console: <b>net_graph 3</b>
</p><p>The rotation is not smooth when using <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetNWFloat" title="Entity/SetNWFloat">SetNWFloat</a>() because it does not update the value on every frame.
</p>
<pre><code data-language="lua">local server_only = true -- Change the value!

if server_only then
	if SERVER then
		hook.Add( &quot;Think&quot;, &quot;bone_manipulation_test&quot;, function()
			for _,ent in ipairs( ents.FindByModel( &quot;models/buggy.mdl&quot; ) ) do
				ent:ManipulateBoneAngles( 28, Angle( 0,0,RealTime()*180 ) )
			end
		end )
	else
		hook.Add( &quot;Think&quot;, &quot;bone_manipulation_test&quot;, function()
		end )
	end
else
	if SERVER then
		hook.Add( &quot;Think&quot;, &quot;bone_manipulation_test&quot;, function()
			for _,ent in ipairs( ents.FindByModel( &quot;models/buggy.mdl&quot; ) ) do
				ent:SetNWFloat( &quot;bone_manipulation_test&quot;, RealTime()*180 ) -- Entity:SetNW....() do not broadcast new values instantly
			end
		end )
	else
		hook.Add( &quot;Think&quot;, &quot;bone_manipulation_test&quot;, function()
			for _,ent in ipairs( ents.GetAll() ) do
				if ent:GetModel() == &quot;models/buggy.mdl&quot; then
					ent:ManipulateBoneAngles( 28, Angle( 0,0,ent:GetNWFloat( &quot;bone_manipulation_test&quot; ) ) )
				end
			end
		end )
	end
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Rotation of the ammo box of all HL2 buggies.</div>
