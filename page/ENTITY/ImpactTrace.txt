<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when a bullet trace hits this entity and allows you to override the default behavior by returning true.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> traceResult</span>
</p>
<div style="margin-left: 32px;">
The trace that hit this entity as a <a href="/page/Structures/TraceResult" title="Structures/TraceResult">TraceResult structure</a>.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> damageType</span>
</p>
<div style="margin-left: 32px;">
The damage bits associated with the trace, see <a href="/page/Enums/DMG" title="Enums/DMG">DMG_ Enums</a></div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> customImpactName=nil</span>
</p>
<div style="margin-left: 32px;">
<p>The effect name to override the impact effect with.
</p>
Possible arguments are ImpactJeep, AirboatGunImpact, HelicopterImpact, ImpactGunship.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Return true to override the default impact effects.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Hides the original bullet impact effect and dispatches explosions instead.
</p>
<pre><code data-language="lua">function ENT:ImpactTrace(trace,dmgtype,customimpactname)
	local effectdata = EffectData()
	effectdata:SetOrigin( trace.HitPos )
	util.Effect( &quot;Explosion&quot;, effectdata )
	return true
end</code></pre>
