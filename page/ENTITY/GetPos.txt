<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Utility command that would give you the position of the entity you are looking at.
</p>
<pre><code data-language="lua">concommand.Add( &quot;entity_pos&quot;, function( ply )
	local tr = ply:GetEyeTrace()
	if ( IsValid( tr.Entity ) ) then
		print( &quot;Entity position:&quot;, tr.Entity:GetPos() )
	else
		print( &quot;Crosshair position:&quot;, tr.HitPos )
	end
end )</code></pre>
