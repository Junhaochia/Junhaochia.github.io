<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Applies the damage specified by the damage info to the entity.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Calling this function on the victim entity in <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/OnTakeDamage" title="ENTITY/OnTakeDamage">OnTakeDamage</a> can cause infinite loops.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_CTakeDamageInfo" title="Category:CTakeDamageInfo">CTakeDamageInfo</a> damageInfo</span>
</p>
<div style="margin-left: 32px;">
The damage to apply.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Dissolve the target into oblivion.
</p>
<pre><code data-language="lua">function DissolveIt( ent )
	local d = DamageInfo()
	d:SetDamage( ent:Health() )
	d:SetAttacker( ent )
	d:SetDamageType( DMG_DISSOLVE )

	ent:TakeDamageInfo( d )
end

DissolveIt( Entity( 1 ) )</code></pre>
