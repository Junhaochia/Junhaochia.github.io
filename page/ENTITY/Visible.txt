<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns whether the target/given entity is visible from the this entity.
</p><p>This is meant to be used only with NPCs.
</p><p>Differences from a simple trace include:
</p>
<ul><li> If target has <b>FL_NOTARGET</b>, returns false
</li><li> If <b>ai_ignoreplayers</b> is turned on and target is a player, returns false
</li><li> Reacts to <b>ai_LOS_mode</b>:
<ul><li> If 1, does a simple trace with <b>COLLISION_GROUP_NONE</b> and <b>MASK_BLOCKLOS</b>
</li><li> If not, does a trace with <b>MASK_BLOCKLOS_AND_NPCS</b> ( - <b>CONTENTS_BLOCKLOS</b> is target is player ) and a custom LOS filter ( <b>CTraceFilterLOS</b> )
</li></ul>
</li><li> Returns true if hits a vehicle the target is driving
</li></ul>
