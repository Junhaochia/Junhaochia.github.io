<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the position and angle of the given attachment, relative to the world.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This can return the server's position during server lag.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/884">#884</a></b>
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This can return garbage serverside or <a href="/page/Global/Vector" title="Global/Vector">Vector</a>(0,0,0) for v49 models.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3285">#3285</a></b>
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This can return garbage if a trace passed through the target bone during bone matrix access.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3739">#3739</a></b>
</p>
</td></tr></table>
