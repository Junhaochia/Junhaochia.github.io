<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Controls if a property can be used on this entity or not.
</p><p>This hook will only work in Sandbox derived gamemodes that do not have <a href="/page/Category_colon_SANDBOX_Hooks" title="Category:SANDBOX Hooks">SANDBOX</a>:<a href="/page/SANDBOX/CanProperty" title="SANDBOX/CanProperty">CanProperty</a> overridden.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This hook will work on ALL entities, not just the scripted ones (SENTs)
</p>
</td></tr></table>

<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Player" title="Category:Player">Player</a> ply</span>
</p>
<div style="margin-left: 32px;">
Player, that tried to use the property</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">

</p>
<div style="margin-left: 32px;">
Class of the property that is tried to use, for example - bonemanipulate</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">

</p>
<div style="margin-left: 32px;">
<p>Return false to disallow using that property, return true to allow.
</p>
You must return a value. Not returning anything can cause unexpected results.</div>
