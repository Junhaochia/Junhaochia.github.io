<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the collision bounds for the entity, which are used for triggers ( <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetTrigger" title="Entity/SetTrigger">SetTrigger</a>, <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/Touch" title="ENTITY/Touch">Touch</a> ), determining if rendering is necessary clientside, and collision ( If <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetSolid" title="Entity/SetSolid">SetSolid</a> set as <a href="/page/Enums/SOLID" title="Enums/SOLID">SOLID_BBOX</a> ).
</p><p>Input bounds are relative to <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetPos" title="Entity/GetPos">GetPos</a>!
See also <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetCollisionBoundsWS" title="Entity/SetCollisionBoundsWS">SetCollisionBoundsWS</a>.
</p>
