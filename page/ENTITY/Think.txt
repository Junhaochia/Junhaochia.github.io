<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called every frame on the client.
Called every tick on the server.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>By default, it runs about 5-6 times per second on the server, but you can force it to run at servers tickrate using the example below.
</p>
</td></tr></table>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Return true if you used <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/NextThink" title="Entity/NextThink">NextThink</a> to override the next execution time.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Force the think hook to run at the maximum frequency.
This is generally only used for anim entities, if the entity has to play any animations.
</p>
<pre><code data-language="lua">ENT.AutomaticFrameAdvance = true -- Must be set on client

function ENT:Think()
	-- Do stuff

	self:NextThink( CurTime() )
	return true
end</code></pre>
