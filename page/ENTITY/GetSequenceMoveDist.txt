<h1> <span class="mw-headline" id="Description">Description</span></h1>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Experiment which demonstrates that dividing the sequence move distance by <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SequenceDuration" title="Entity/SequenceDuration">SequenceDuration</a> results in a value extremely close to <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetSequenceGroundSpeed" title="Entity/GetSequenceGroundSpeed">GetSequenceGroundSpeed</a>.
</p>
<pre><code data-language="lua">local ply, seq, move_dist, seq_dur, ground_speed = nil, nil, nil, nil, nil

function GM:Think()

	-- Player 1
	ply = Entity(1)
	
	-- Current sequence
	seq = ply:GetSequence()
	
	-- The move distance
	move_dist = ply:GetSequenceMoveDist(seq)
	
	if(move_dist == 0) then return end	-- If it doesn't move, don't bother
	
	-- The sequence duration
	seq_dur = ply:SequenceDuration(seq)
	
	-- Actual sequence ground speed
	ground_speed = ply:GetSequenceGroundSpeed(seq)
	
	-- Compare the calculated value to the actual value
	print(tostring(ground_speed-(move_dist/seq_dur)))

end</code></pre>
