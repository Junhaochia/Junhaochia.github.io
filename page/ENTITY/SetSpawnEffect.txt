<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets whether the entity should use a spawn effect when it is created on the client.
</p><p>See <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetSpawnEffect" title="Entity/GetSpawnEffect">GetSpawnEffect</a> for more information on how the effect is applied.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function will only have an effect when the entity spawns. After that it will do nothing even is set to true.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> spawnEffect</span>
</p>
<div style="margin-left: 32px;">
Sets if we should show a spawn effect.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Simplified code taken from sandbox's commands.lua
</p>
<pre><code data-language="lua">local function MakeRagdoll( Player, Pos, Ang, Model, PhysicsObjects, Data )

	local Ent = ents.Create( &quot;prop_ragdoll&quot; )
	duplicator.DoGeneric( Ent, Data )
	Ent:Spawn()
	
	duplicator.DoGenericPhysics( Ent, Player, Data )
	Ent:Activate()

	Ent:SetSpawnEffect( true )
	return Ent	
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Spawns the ragdoll and then sets the spawnEffect flag to true.</div>
