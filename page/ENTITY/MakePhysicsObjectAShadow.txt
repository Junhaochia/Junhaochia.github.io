<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Turns the <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetPhysicsObject" title="Entity/GetPhysicsObject">GetPhysicsObject</a> into a physics shadow.
It's used internally for the Player's and NPC's physics object, and certain HL2 entities such as the crane.
</p><p>A physics shadow can be used to have static entities that never move by setting both arguments to false.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Unlike <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/PhysicsInitShadow" title="Entity/PhysicsInitShadow">PhysicsInitShadow</a>, this function doesn't remove the current physics object.
</p>
</td></tr></table>
