<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Fires an entity's input. You can find inputs for most entities on the <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Output">Valve Developer Wiki</a>
</p><p>See also <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/Input" title="Entity/Input">Input</a> and <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/AcceptInput" title="GM/AcceptInput">AcceptInput</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> input</span>
</p>
<div style="margin-left: 32px;">
The name of the input to fire</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> param=""</span>
</p>
<div style="margin-left: 32px;">
The value to give to the input, can also be a <a href="/page/Category_colon_number" title="Category:number">number</a> or a <a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a>.</div>
</div>

</p>

Delay in seconds before firing</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>If you are looking at a door, this will lock it
</p>
<pre><code data-language="lua">// Entity(1) is considered a player in this example
local tr = Entity( 1 ):GetEyeTrace()
local ent = tr.Entity
if IsValid( ent ) then
	ent:Fire(&quot;Lock&quot;)
end</code></pre>
