<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a new schedule and assigns it to a scripted NPC.
</p>
<pre><code data-language="lua">local schdTest = ai_schedule.New( &quot;Test Schedule&quot; )
 
schdTest:EngTask( &quot;TASK_GET_PATH_TO_RANDOM_NODE&quot;,  128 )
schdTest:EngTask( &quot;TASK_RUN_PATH&quot;, 		   0   )
schdTest:EngTask( &quot;TASK_WAIT_FOR_MOVEMENT&quot;, 	   0   )
 
 
function ENT:SelectSchedule()
 
	self:StartSchedule( schdTest )
 
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">The scripted NPC will run around when spawned.</div>
