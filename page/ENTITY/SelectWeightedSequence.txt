<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns sequence ID corresponding to given activity ID.
</p><p>Opposite of <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetSequenceActivity" title="Entity/GetSequenceActivity">GetSequenceActivity</a>.
</p><p>Similar to <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/LookupSequence" title="Entity/LookupSequence">LookupSequence</a>.
</p><p>See also <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SelectWeightedSequenceSeeded" title="Entity/SelectWeightedSequenceSeeded">SelectWeightedSequenceSeeded</a>.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Use this hook to check if the model has a certain ACT_* enumeration, and if it does, play it.
</p>
<pre><code data-language="lua">local VModel = self.Owner:GetViewModel()

if ( self:Clip1() == 0 and VModel:SelectWeightedSequence( ACT_VM_RELOAD_EMPTY ) ) then

	local SEQ = self:LookupSequence( ACT_VM_RELOAD_EMPTY )

	if ( SEQ == -1 ) then

		print( &quot;reload&quot; )

		local EnumToSeq = VModel:SelectWeightedSequence( ACT_VM_RELOAD )

		--	Play the normal reload animation
		VModel:SendViewModelMatchingSequence( EnumToSeq )

	else

		print( &quot;reload empty&quot; )

		local EnumToSeq = VModel:SelectWeightedSequence( ACT_VM_RELOAD_EMPTY )

		-- Play the empty reload animation
		VModel:SendViewModelMatchingSequence( EnumToSeq )
		
	end

end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Will play the ACT_VM_RELOAD_EMPTY enumeration if the model has it. If not, it will play the normal reload enum.</div>
