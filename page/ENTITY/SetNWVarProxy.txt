<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets a function to be called when the NWVar changes.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_any" title="Category:any">any</a> key</span>
</p>
<div style="margin-left: 32px;">
The key of the NWVar to add callback for.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> callback</span>
</p>
<div style="margin-left: 32px;">
<p>The function to be called when the NWVar changes. It has 4 arguments:
</p>
<ul><li> <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent - The entity
</li><li> <a href="/page/Category_colon_string" title="Category:string">string</a> name - Name of the NWVar that has changed
</li><li> <a href="/page/Category_colon_any" title="Category:any">any</a> oldval - The old value
</li><li> <a href="/page/Category_colon_any" title="Category:any">any</a> newval - The new value</div>
</li></ul>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints all changes to a NWVar called "Key" of Player 1.
</p>
<pre><code data-language="lua">Entity( 1 ):SetNWVarProxy( &quot;Key&quot;, print )
Entity( 1 ):SetNWString( &quot;Key&quot;, &quot;Value&quot; )
Entity( 1 ):SetNWString( &quot;Key&quot;, &quot;Table&quot; )</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">Player [1][Player1]	Key	nil	Value
Player [1][Player1]	Key	Value	Table</code></pre>
