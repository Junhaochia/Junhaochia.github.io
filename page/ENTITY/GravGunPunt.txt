<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when this entity is about to be punted with the gravity gun (primary fire).
</p><p>Only works in Sandbox derived gamemodes and only if <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/GravGunPunt" title="GM/GravGunPunt">GravGunPunt</a> is not overridden.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Player" title="Category:Player">Player</a> ply</span>
</p>
<div style="margin-left: 32px;">
The player pressing left-click with the gravity gun at an entity</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Return true or false to enable or disable punting respectively.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Enables a scripted entity to be punted even when frozen.
</p>
<pre><code data-language="lua">function ENT:GravGunPunt( ply )
	self:GetPhysicsObject():EnableMotion( true )
	return true
end</code></pre>
