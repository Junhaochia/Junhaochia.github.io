<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Stops emitting the given sound script from the entity, especially useful for looping sound scripts.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>This only works when the sound you're trying to stop is a sound script (see <a href="/page/Category_colon_sound" title="Category:sound">sound</a>.<a href="/page/sound/Add" title="sound/Add">Add</a>), using a file path will not work!
</p>
</td></tr></table>

</p>
<div style="margin-left: 32px;">
The name of the sound script to stop playback of.</div>
