<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the entity's physics object, if the entity has physics.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Entities don't have clientside physics objects by default, so this will return [NULL PHYSOBJ] on the client in most cases.
</p>
</td></tr></table>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObj</a></span>
</p>
<div style="margin-left: 32px;">
The entity's physics object.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Gets the mass of an entity.
</p>
<pre><code data-language="lua">local phys = ent:GetPhysicsObject()
if ( IsValid( phys ) ) then -- Always check with IsValid! The ent might not have physics!
	return phys:GetMass()
else
	return 0
end</code></pre>
