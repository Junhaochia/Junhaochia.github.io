<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Scales the model of the entity, if the entity is a <a href="/page/Category_colon_Player" title="Category:Player">Player</a> or an <a href="/page/Category_colon_NPC" title="Category:NPC">NPC</a> the hitboxes will be scaled as well.
</p><p>For some entities, calling <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/Activate" title="Entity/Activate">Activate</a> after this will scale the collision bounds and <a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObj</a> as well; be wary as there's no optimization being done internally and highly complex collision models might crash the server.
</p><p>This is the same system used in TF2 for the Mann Vs Machine robots.
</p><p>To resize the entity along any axis, use <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/EnableMatrix" title="Entity/EnableMatrix">EnableMatrix</a> instead.
</p><p>If your old scales are wrong due to a recent update, use <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetLegacyTransform" title="Entity/SetLegacyTransform">SetLegacyTransform</a> as a quick fix.
</p>

<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> scale</span>
</p>
<div style="margin-left: 32px;">
A float to scale the model by. 0 will not draw anything. &lt; 0 will draw the model inverted.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> deltaTime=0</span>
</p>
<div style="margin-left: 32px;">
Transition time of the scale change, set to 0 to modify the scale right away.</div>
</div>
