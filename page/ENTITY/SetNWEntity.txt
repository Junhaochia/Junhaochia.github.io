<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets a networked entity value on the entity.
</p><p>The value can then be accessed with <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetNWEntity" title="Entity/GetNWEntity">GetNWEntity</a> both from client and server.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Running this function clientside will only set it for the client it is called on.
</p>
</td></tr></table>

</p>

</div>

</p>
<div style="margin-left: 32px;">
The value to set</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This will set the networked entity 'owner' on all clients to themselves.
</p>
<pre><code data-language="lua">for k, v in pairs( player.GetAll() ) do
    v:SetNWEntity( &quot;owner&quot;, v )
end</code></pre>
