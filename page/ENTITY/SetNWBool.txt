<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets a networked boolean value on the entity.
</p><p>The value can then be accessed with <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetNWBool" title="Entity/GetNWBool">GetNWBool</a> both from client and server.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Running this function clientside will only set it for the client it is called on.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Sets a networked boolean with a key of "VIP" and a value of true on a player.
</p>
<pre><code data-language="lua">player:SetNWBool( &quot;VIP&quot;, true )
print( player:GetNWBool( &quot;VIP&quot; ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">true</div>
