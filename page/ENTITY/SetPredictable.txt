<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets whether an entity should be predictable or not.
When an entity is set as predictable, its DT vars can be changed during predicted hooks. This is useful for entities which can be controlled by player input.
</p><p>Any datatable value that mismatches from the server will be overridden and a prediction error will be spewed.
</p><p>Weapons are predictable by default, and the drive system uses this function to make the controlled prop predictable as well.
</p><p>Visit <a href="/page/Category_colon_Predicted_Hooks" title="Category:Predicted Hooks">Predicted Hooks</a> for a list of all predicted hooks, and the <a href="/page/Prediction" title="Prediction">Prediction</a> page.
For further technical information on the subject, visit <a rel="nofollow" class="external text" href="https://developer.valvesoftware.com/wiki/Prediction">valve's wiki</a>.
</p>
