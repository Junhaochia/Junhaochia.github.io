<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called from the Entity's motion controller to simulate physics.
</p><p>This will only be called after using <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/StartMotionController" title="Entity/StartMotionController">StartMotionController</a> on a scripted entity of "anim" type.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This hook can work on the CLIENT if you call <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/StartMotionController" title="Entity/StartMotionController">StartMotionController</a> and use <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/AddToMotionController" title="Entity/AddToMotionController">AddToMotionController</a> on the physics objects you want to control
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Do not use functions such as <a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObj</a>:<a href="/page/PhysObj/EnableCollisions" title="PhysObj/EnableCollisions">EnableCollisions</a> or <a href="/page/Category_colon_PhysObj" title="Category:PhysObj">PhysObj</a>:<a href="/page/PhysObj/EnableGravity" title="PhysObj/EnableGravity">EnableGravity</a> in this hook as they're very likely to crash your game. You may want to use <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/PhysicsUpdate" title="ENTITY/PhysicsUpdate">PhysicsUpdate</a> instead.
</p>
</td></tr></table>
