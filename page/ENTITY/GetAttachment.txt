<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Gets the orientation and position of the attachment by its ID, returns nothing if the attachment does not exist.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>The update rate of this function is limited by the setting of ENT.AutomaticFrameAdvance for <a href="/page/Scripted_Entities" title="Scripted Entities">Scripted Entities</a>!
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This will return improper values for viewmodels if used in <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/CalcView" title="GM/CalcView">CalcView</a>.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/1255">#1255</a></b>
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> attachmentId</span>
</p>
<div style="margin-left: 32px;">
The internal ID of the attachment.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
The angle and position of the attachment. See the <a href="/page/Structures/AngPos" title="Structures/AngPos">AngPos structure</a>. Most notably, the table contains the keys "Ang" and "Pos".</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Grabs the muzzle position of a player's view model
</p>
<pre><code data-language="lua">local vm = ply:GetViewModel()
local obj = vm:LookupAttachment( &quot;muzzle&quot; )
local muzzlepos = vm:GetAttachment( obj )</code></pre>
