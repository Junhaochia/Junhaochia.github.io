<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>This is the spawn function. It's called when a player spawns the entity from the spawnmenu.
</p><p>If you want to make your SENT spawnable you need this function to properly create the entity.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Unlike other ENTITY functions, the "self" parameter of this function is not an entity but rather the table used to generate the SENT. This table is equivalent to <a href="/page/Category_colon_scripted_ents" title="Category:scripted ents">scripted_ents</a>.<a href="/page/scripted_ents/GetStored" title="scripted ents/GetStored">GetStored</a>("ent_name").
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This is how it is defined in sent_ball
</p>
<pre><code data-language="lua">function ENT:SpawnFunction( ply, tr, ClassName )

	if (&#160;!tr.Hit ) then return end

	local SpawnPos = tr.HitPos + tr.HitNormal * 16

	local ent = ents.Create( ClassName )
	ent:SetPos( SpawnPos )
	ent:SetBallSize( math.random( 16, 48 ) )
	ent:Spawn()
	ent:Activate()

	return ent

end</code></pre>

<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>This is how base_edit spawns (also rotates the entity to face the player, remove * 10 if it spawns in the air)
</p>
<pre><code data-language="lua">function ENT:SpawnFunction( ply, tr, ClassName )

	if (&#160;!tr.Hit ) then return end
	
	local SpawnPos = tr.HitPos + tr.HitNormal * 10
	local SpawnAng = ply:EyeAngles()
	SpawnAng.p = 0
	SpawnAng.y = SpawnAng.y + 180
	
	local ent = ents.Create( ClassName )
	ent:SetPos( SpawnPos )
	ent:SetAngles( SpawnAng )
	ent:Spawn()
	ent:Activate()
	
	return ent
	
end</code></pre>
