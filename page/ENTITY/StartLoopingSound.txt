<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Starts a "looping" sound. As with any other sound playing methods, this function expects the sound file to be looping itself and will not automatically loop a non looping sound file as one might expect.
</p><p>This function is almost identical to <a href="/page/Global/CreateSound" title="Global/CreateSound">CreateSound</a>, with the exception of the sound being created in the STATIC channel and with normal attenuation.
</p><p>See also <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/StopLoopingSound" title="Entity/StopLoopingSound">StopLoopingSound</a>
</p>
