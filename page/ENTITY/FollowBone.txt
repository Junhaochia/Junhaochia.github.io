<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Makes an entity follow another entity's bone.
</p><p>Internally this function calls <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetParent" title="Entity/SetParent">SetParent</a>( parent, boneid ) and <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/AddEffects" title="Entity/AddEffects">AddEffects</a>( EF_FOLLOWBONE ).
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>If the entity vibrates, you probably need to run <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetPredictable" title="Entity/SetPredictable">SetPredictable</a>( false ) clientside.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>This function will not work if the target bone's parent bone is invalid or if the bone is not used by VERTEX LOD0
</p>
</td></tr></table>
