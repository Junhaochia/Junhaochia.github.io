<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when the entity should set up its <a href="/page/Networking_Entities" title="Networking Entities"> Data Tables</a>.
</p><p>This is a much better option than using Set/Get Networked Values.
</p><p>This hook is called after <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/OnEntityCreated" title="GM/OnEntityCreated">OnEntityCreated</a> and <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/NetworkEntityCreated" title="GM/NetworkEntityCreated">NetworkEntityCreated</a>.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Sets up networked variables, adds functions SetAmount, GetAmount, GetBloodPos, SetBloodPos, GetUrinePos, SetUrinePos.
</p><p>This function only supports 64 data tables per type (#0-63), except for strings which only supports 4.
</p>
<pre><code data-language="lua">function ENT:SetupDataTables()

	self:NetworkVar( &quot;Float&quot;, 0, &quot;Amount&quot; )
	self:NetworkVar( &quot;Vector&quot;, 0, &quot;BloodPos&quot; )
	self:NetworkVar( &quot;Vector&quot;, 1, &quot;UrinePos&quot; )

	if SERVER then
		self:SetAmount( 3 )
		self:SetBloodPos( Vector( 0, -32, 0 ) )
		self:SetUrinePos( Vector( 0, 0, -16 ) )
	end

end</code></pre>
