<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns whether this entity has the specified spawnflags bits set.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> spawnFlags</span>
</p>
<div style="margin-left: 32px;">
The spawnflag bits to check, see <a href="/page/Enums/SF" title="Enums/SF">SF_ Enums</a>.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Whether the entity has that spawnflag set or not.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>As seen in sandbox's PhysgunPickup hook.
</p>
<pre><code data-language="lua">function GM:PhysgunPickup( ply, ent )

	-- Don't move physboxes if the mapper logic says no
	if ( ent:GetClass() == &quot;func_physbox&quot; &amp;&amp; ent:HasSpawnFlags( SF_PHYSBOX_MOTIONDISABLED ) ) then return false end

	return true
	
end</code></pre>
