<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>This forces an entity to use the bone transformation behaviour from versions prior to <b>2014-07-08</b>.<br />
This behaviour affects <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/EnableMatrix" title="Entity/EnableMatrix">EnableMatrix</a> and <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetModelScale" title="Entity/SetModelScale">SetModelScale</a> and is incorrect, therefore this function be used exclusively as a quick fix for old scripts that rely on it.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> enabled</span>
</p>
<div style="margin-left: 32px;">
Whether the entity should use the old bone transformation behaviour or not.</div>
