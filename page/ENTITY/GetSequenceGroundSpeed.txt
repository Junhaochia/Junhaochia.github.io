<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the ground speed of the entity's sequence.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> sequenceId</span>
</p>
<div style="margin-left: 32px;">
The sequence ID.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
The ground speed of this sequence.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Move the <a href="/page/Category_colon_NextBot" title="Category:NextBot">NextBot</a> based on the ground speed of its walking animation (within its <a href="/page/Category_colon_coroutine" title="Category:coroutine">coroutine</a>).
</p>

<tr>

</p>
</td></tr></table>
<pre><code data-language="lua">local sequence = self:LookupSequence( &quot;walk_all&quot; )

if ( sequence ) then
	self:StartActivity( ACT_WALK )
	self:SetSequence( sequence )
	self.loco:SetDesiredSpeed( self:GetSequenceGroundSpeed( sequence ) )
end</code></pre>
