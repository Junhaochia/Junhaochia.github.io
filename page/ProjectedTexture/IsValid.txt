<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns true if the projected texture is valid (i.e. has not been removed), false otherwise.
</p><p>Instead of calling this directly it's a good idea to call <a href="/page/Global/IsValid" title="Global/IsValid">IsValid</a> in case the variable is nil.
</p>
<pre><code data-language="lua">IsValid( ptexture )</code></pre>
<p>This not only checks whether the projected texture is valid - but also checks whether it's nil.
</p>
