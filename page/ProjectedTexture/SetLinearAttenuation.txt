<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the linear attenuation of the projected texture.
</p><p>See also <a href="/page/Category_colon_ProjectedTexture" title="Category:ProjectedTexture">ProjectedTexture</a>:<a href="/page/ProjectedTexture/SetConstantAttenuation" title="ProjectedTexture/SetConstantAttenuation">SetConstantAttenuation</a> and <a href="/page/Category_colon_ProjectedTexture" title="Category:ProjectedTexture">ProjectedTexture</a>:<a href="/page/ProjectedTexture/SetQuadraticAttenuation" title="ProjectedTexture/SetQuadraticAttenuation">SetQuadraticAttenuation</a>.
</p><p>The default value of linear attenuation when the projected texture is created is 100. (others are 0, as you are not supposed to mix them)
</p><p>You must call <a href="/page/Category_colon_ProjectedTexture" title="Category:ProjectedTexture">ProjectedTexture</a>:<a href="/page/ProjectedTexture/Update" title="ProjectedTexture/Update">Update</a> after using this function for it to take effect.
</p>
