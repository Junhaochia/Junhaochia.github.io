<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Starts a new block of data that you can write to inside current block. Blocks <b>must</b> be ended with <a href="/page/Category_colon_ISave" title="Category:ISave">ISave</a>:<a href="/page/ISave/EndBlock" title="ISave/EndBlock">EndBlock</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> name</span>
</p>
<div style="margin-left: 32px;">
Name of the new block. Used for determining which block is which, returned by <a href="/page/Category_colon_IRestore" title="Category:IRestore">IRestore</a>:<a href="/page/IRestore/StartBlock" title="IRestore/StartBlock">StartBlock</a> during game load.</div>
</div>

<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage
</p>
<pre><code data-language="lua">saverestore.AddSaveHook( &quot;HookNameHere&quot;, function( save )
	save:StartBlock( &quot;Test&quot; )
		save:WriteString( &quot;myawesomestring&quot; )
	save:EndBlock()
end )</code></pre>
