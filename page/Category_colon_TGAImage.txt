<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a TGAImage panel and loads the image defined by line 4.
</p>
<pre><code data-language="lua">local panel = vgui.Create( &quot;DFrame&quot; )
panel:SetSize( 500, 500 )
panel:MakePopup()
panel:Center()

local image = vgui.Create( &quot;TGAImage&quot;, panel )
image:SetSize( 32, 32 )
image:SetPos( 50, 50 )
image:LoadTGAImage( &quot;materials/addon.tga&quot;, &quot;&quot; )</code></pre>
