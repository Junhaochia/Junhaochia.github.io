<h1> <span class="mw-headline" id="Description">Description</span></h1>
<div class="markedfordeletion"><span>This page has been nominated for deletion.</span><br />Reason for deletion: This function was removed due to security concerns.</div>
<p>Sets a local variable's value.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_thread" title="Category:thread">thread</a> thread=Current Thread</span>
</p>
<div style="margin-left: 32px;">
The thread</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> level</span>
</p>
<div style="margin-left: 32px;">
<p>The level above the thread.
0 is the function that was called (most always this function)'s arguments
</p><p>1 is the thread that had called this function.
</p>
2 is the thread that had called the function that started the thread that called this function.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> index</span>
</p>
<div style="margin-left: 32px;">
<p>The variable's index you want to get.
</p><p>1 = the first local defined in the thread
</p>
2 = the second local defined in the thread</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_any" title="Category:any">any</a> value=nil</span>
</p>
<div style="margin-left: 32px;">
The value to set the local to</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a></span>
</p>
<div style="margin-left: 32px;">
The name of the local variable if the local at the index exists, otherwise nil is returned.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints the local variables, sets them, then prints the variables again.
</p>
<pre><code data-language="lua">local var1 = &quot;Luke, I am not your father.&quot;
local var2 = &quot;PMFPMF&quot;

(function()
	print(&quot;Getting the locals now!&quot;)
	PrintTable({debug.getlocal(2, 1)})
	PrintTable({debug.getlocal(2, 2)})

	print(&quot;\nSetting the locals now!&quot;)
	print(debug.setlocal(2, 1, &quot;I'm actually your mother.&quot;))
	print(debug.setlocal(2, 2, &quot;Chemo-chi&quot;))
	print(debug.setlocal(2, 3, &quot;nil should be returned here!&quot;))

	print(&quot;\nHere are the locals after being set!&quot;)
	PrintTable({debug.getlocal(2, 1)})
	PrintTable({debug.getlocal(2, 2)})
end)()</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">
Getting the locals now!
1	=	var1
2	=	Luke, I am not your father.
1	=	var2
2	=	PMFPMF

Setting the locals now!
var1
var2
nil

Here are the locals after being set!
1	=	var1
2	=	I'm actually your mother.
1	=	var2
2	=	Chemo-chi
</code></pre></div>
