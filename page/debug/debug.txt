<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Enters an interactive mode with the user, running each string that the user enters. Using simple commands and other debug facilities, the user can inspect global and local variables, change their values, evaluate expressions, and so on. A line containing only the word cont finishes this function, so that the caller continues its execution.
</p>
<ul><li> Commands for debug.debug are not lexically nested within any function, and so have no direct access to local variables.
</li><li> This only works on the source dedicated server.
</li><li> To exit this interactive mode, you can press Ctrl + Z then Enter OR type the word 'cont' on a single line and press enter.
</li></ul>
