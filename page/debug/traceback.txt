<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a full execution stack trace.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_thread" title="Category:thread">thread</a> thread=current thread</span>
</p>
<div style="margin-left: 32px;">
Thread (ie. error object from xpcall error handler) to build traceback for.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> message=nil</span>
</p>
<div style="margin-left: 32px;">
Appended at the beginning of the traceback.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> level=1</span>
</p>
<div style="margin-left: 32px;">
Which level to start the traceback.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a></span>
</p>
<div style="margin-left: 32px;">
A dump of the execution stack.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints the traceback into console.
</p>
<pre><code data-language="lua">print(debug.traceback())</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">&gt; print(debug.traceback())...
<p>stack traceback:
</p>
lua_run:1: in main chunk</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Defines two functions that are later visible in the traceback. Enter "lua_run TracebackTest()" into the development console to achieve exact results.
</p>
<pre><code data-language="lua">function TracebackTest()
     AnotherTracebackFunction()
end

function AnotherTracebackFunction()
     print(debug.traceback())
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">stack traceback:
<pre><code data-language="lua">       lua_run:1: in function 'AnotherTracebackFunction'
       lua_run:1: in function 'TracebackTest'
</code></pre>
lua_run:1: in main chunk</div>
