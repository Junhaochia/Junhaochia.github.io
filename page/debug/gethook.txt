<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the current hook settings of the passed thread. The thread argument can be omitted. This is unrelated to <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">gamemode hooks</a>. More information on hooks can be found at <a rel="nofollow" class="external free" href="https://www.lua.org/pil/23.2.html">http://www.lua.org/pil/23.2.html</a>
</p>
