<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Defines a new Derma control with an optional base.
</p><p>This calls <a href="/page/Category_colon_vgui" title="Category:vgui">vgui</a>.<a href="/page/vgui/Register" title="vgui/Register">Register</a> internally, but also does the following:
</p>
<ul><li> Adds the control to <a href="/page/Category_colon_derma" title="Category:derma">derma</a>.<a href="/page/derma/GetControlList" title="derma/GetControlList">GetControlList</a>
</li><li> Adds a key "Derma" - This is returned by <a href="/page/Category_colon_derma" title="Category:derma">derma</a>.<a href="/page/derma/GetControlList" title="derma/GetControlList">GetControlList</a>
</li><li> Makes a global table with the name of the control (This is technically deprecated and should not be relied upon)
</li><li> If reloading (i.e. called this function with name of an existing panel), updates all existing instances of panels with this name. (Updates functions, calls <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/PreAutoRefresh" title="PANEL/PreAutoRefresh">PreAutoRefresh</a> and <a href="/page/Category_colon_PANEL_Hooks" title="Category:PANEL Hooks">PANEL</a>:<a href="/page/PANEL/PostAutoRefresh" title="PANEL/PostAutoRefresh">PostAutoRefresh</a>, etc.)
</li></ul>

function PANEL:OnChange()

end
