<h1> <span class="mw-headline" id="Not_a_function">Not a function</span></h1>
<p>This is NOT a function, it's a variable containing all derma controls, registered with <a href="/page/Category_colon_derma" title="Category:derma">derma</a>.<a href="/page/derma/DefineControl" title="derma/DefineControl">DefineControl</a>.
</p><p>Use <a href="/page/Category_colon_derma" title="Category:derma">derma</a>.<a href="/page/derma/GetControlList" title="derma/GetControlList">GetControlList</a> to retrieve this list.
</p><p>It's a list of tables, each having 3 keys, all from <a href="/page/Category_colon_derma" title="Category:derma">derma</a>.<a href="/page/derma/DefineControl" title="derma/DefineControl">DefineControl</a> arguments:
</p>
<ul><li> <a href="/page/Category_colon_string" title="Category:string">string</a> ClassName - The class name of the panel
</li><li> <a href="/page/Category_colon_string" title="Category:string">string</a> Description - The description of the panel
</li><li> <a href="/page/Category_colon_string" title="Category:string">string</a> BaseClass - The base class of the panel
</li></ul>
