<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Adds a horizontal line spacer.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a DMenu with 3 options and adds a spacer before the third.
</p>
<pre><code data-language="lua">local m = DermaMenu()

m:AddOption( &quot;Kill yourself&quot;, function() RunConsoleCommand( &quot;kill&quot; ) end ) -- Add first option
m:AddOption( &quot;Disconnect&quot;, function() RunConsoleCommand( &quot;disconnect&quot; ) end ) -- Add second option

m:AddSpacer() -- Add a spacer

m:AddOption( &quot;Say hi&quot;, function() RunConsoleCommand( &quot;say&quot;, &quot;Hi!&quot; ) end ) -- Add third option

m:Open() -- Show our menu</code></pre>
