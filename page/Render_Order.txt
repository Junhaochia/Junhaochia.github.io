<h1> <span class="mw-headline" id="Render_Order"> Render Order </span></h1>
<p>This is the call order for <a href="/page/Category_colon_Render_Hooks" title="Category:Render Hooks">render hooks</a>. The lower a hook is on the list, the higher priority it has for drawing. This list is only to be used for priority reference; in reality, while rendering a frame, the engine might call certain groups of hooks multiple times to achieve reflective properties, draw render targets, and fulfill <a href="/page/Category_colon_render" title="Category:render">render</a>.<a href="/page/render/RenderView" title="render/RenderView">RenderView</a> or <a href="/page/Category_colon_render" title="Category:render">render</a>.<a href="/page/render/RenderHUD" title="render/RenderHUD">RenderHUD</a> calls.

<h2> <span class="mw-headline" id="DrawOpaqueRenderables"> DrawOpaqueRenderables </span></h2>
<p>This block specifies two boolean arguments that point to what PreDraw*Renderables and PostDraw*Renderables will be called with. The first arg is true if the rendering is writing to the depth buffer, and the second is true if the rendering is being done for skybox objects.
</p><p><a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PrePlayerDraw" title="GM/PrePlayerDraw">PrePlayerDraw</a> and <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PostPlayerDraw" title="GM/PostPlayerDraw">PostPlayerDraw</a> will be called for every visible, opaque player, just as <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/RenderOverride" title="ENTITY/RenderOverride">RenderOverride</a> will be for every visible, opaque entity. The order in which entities are drawn is based on their distance from the viewport in the world, and thus the entity type hook pairings can be called in any order. In other words, the entity hooks can be called before the player hooks depending on the object render order. With that, the only guarenteed hook orderings are:
</p>
<ul><li> (<a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/ShouldDrawLocalPlayer" title="GM/ShouldDrawLocalPlayer">ShouldDrawLocalPlayer</a> is checked first if the player entity being drawn is the local player) -&gt; <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/RenderOverride" title="ENTITY/RenderOverride">RenderOverride</a> -&gt; <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PrePlayerDraw" title="GM/PrePlayerDraw">PrePlayerDraw</a> -&gt; <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PostPlayerDraw" title="GM/PostPlayerDraw">PostPlayerDraw</a> for players
</li><li> (<a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/Draw" title="ENTITY/Draw">Draw</a> if the entity is scripted and animated) -&gt; <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/RenderOverride" title="ENTITY/RenderOverride">RenderOverride</a> for entities
</li><li> (<a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a>:<a href="/page/WEAPON/DrawWorldModel" title="WEAPON/DrawWorldModel">DrawWorldModel</a> if the weapon is scripted) -&gt; <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/RenderOverride" title="ENTITY/RenderOverride">RenderOverride</a> for weapons
</li></ul>
<p>Opaque rendergroups include RENDERGROUP_OPAQUE_HUGE, RENDERGROUP_OPAQUE, RENDERGROUP_BOTH, and RENDERGROUP_OPAQUE_BRUSH.
