<h1> <span class="mw-headline" id="About"> About </span></h1>
<p>This tutorial will explain the basics of creating a custom scoreboard in Garry's Mod.
</p><p>I will only show how to start it. Customize it to your liking based off of what you learn here. You will require some knowledge of GMod LUA and Client/Server side differences.
</p><p>Creating a scoreboard is simpler than one would think.
</p>
<h1> <span class="mw-headline" id="Getting_started"> Getting started </span></h1>
<p>To start, create a client lua file to overwrite the default scoreboard. Then you will want to create the hooks to do so.
</p>
<h2> <span class="mw-headline" id="Required_hooks"> Required hooks </span></h2>
<p>You should only need 2 hooks for the scoreboard:
</p>
<ul><li> <i><a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/ScoreboardShow" title="GM/ScoreboardShow">ScoreboardShow</a></i>
</li><li> <i><a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/ScoreboardHide" title="GM/ScoreboardHide">ScoreboardHide</a></i>
</li></ul>
<p>ScoreboardShow and ScoreboardHide will Show and Hide the scoreboard, respectively.
</p>
<h2> <span class="mw-headline" id="Basic_template"> Basic template </span></h2>
<p>We'll use an template, that Niandra and others use, to show the information on the scoreboard, you are expected to know how to set up the information on this page.
</p>
<pre><code data-language="lua">
scoreboard = scoreboard or {}
function scoreboard:show()
	--Create the scoreboard here, with an base like DPanel, you can use an DListView for the rows.
	
	function scoreboard:hide()
		-- This is where you hide the scoreboard, such as with Base:Remove()
	end
end

function GM:ScoreboardShow()
	scoreboard:show()
end

function GM:ScoreboardHide()
	scoreboard:hide()
end
</code></pre>
<p>The code is self explanatory, it sets up the basic functions, and it calls the Hide &amp; Show functions, the comments also show how to make a scoreboard, like using the <a href="/page/Category_colon_DListView" title="Category:DListView">DListView</a> for creating the rows etc, and a <a href="/page/Category_colon_DPanel" title="Category:DPanel">DPanel </a> for the base.
</p>
<h1> <span class="mw-headline" id="Misc"> Misc </span></h1>
<p>If you use the DListView, and want to paint it, there are multiple ways. For example, I used this for the headers &amp; rows:
</p>
<pre><code data-language="lua">
	--[[ Line painting ]]--
for _, v in pairs (player.GetAll()) do
	local line = playerlist:AddLine(v:Name(), v:Frags(), v:Deaths(), v:Ping())
	function line:Paint(w, h)
		--Paint the LINEs here
	end
end

	--[[ Column header painting ]]--
for _, v in pairs(playerlist.Columns) do
	function v.Header:Paint(w, h)
		-- Paint the HEADERs here
	end
	v.Header:SetTextColor(Color(255, 255, 255, 0)) -- Set its text alpha with this in case you paint the text manually
end
</code></pre>
