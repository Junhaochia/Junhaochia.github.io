<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the vehicle parameters for given vehicle.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Not all variables from the <a href="/page/Structures/VehicleParams" title="Structures/VehicleParams">VehicleParams structure</a> can be set.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>Because this method uses miles per hour but <a href="/page/Category_colon_Vehicle" title="Category:Vehicle">Vehicle</a>:<a href="/page/Vehicle/GetVehicleParams" title="Vehicle/GetVehicleParams">GetVehicleParams</a> returns Hammer units per second, this method incorrectly modifies the vehicle engine's "boostMaxSpeed", "maxRevSpeed" and "maxSpeed" even when not explicitly set in a call to this method.
</p><p><b>Workaround</b>: In order to retain the original values for these fields, call <a href="/page/Category_colon_Vehicle" title="Category:Vehicle">Vehicle</a>:<a href="/page/Vehicle/GetVehicleParams" title="Vehicle/GetVehicleParams">GetVehicleParams</a> yourself, convert the mentioned fields from Hammer units per second to miles per hour (1 MPH &#8776; 17.6 HU/s in this case) and add them to the table passed into the call to this setter. Avoid doing this repeatedly to avoid floating point inaccuracies over time (store the ready-calculated value for next time if possible).
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/2625">#2625</a></b>
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Adds 25 horsepower to every vehicle spawned.
</p>
<pre><code data-language="lua">hook.Add(&quot;PlayerSpawnedVehicle&quot;, &quot;VehicleParamsExample&quot;, function(ply, ent)

    local params = ent:GetVehicleParams()


    params.engine.horsepower = params.engine.horsepower + 25


    ent:SetVehicleParams(params)

end)</code></pre>
