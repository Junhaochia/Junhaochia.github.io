<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the vehicle class name. This is only useful for Sandbox spawned vehicles or any vehicle that properly sets the vehicle class with <a href="/page/Category_colon_Vehicle" title="Category:Vehicle">Vehicle</a>:<a href="/page/Vehicle/SetVehicleClass" title="Vehicle/SetVehicleClass">SetVehicleClass</a>.
</p>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a></span>
</p>
<div style="margin-left: 32px;">
The class name of the vehicle.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints the spawn menu name of the vehicle the player is currently in.
</p>
<pre><code data-language="lua">local c = Entity(1):GetVehicle():GetVehicleClass()
if (&#160;!list.Get( &quot;Vehicles&quot; )[ c ] ) then return end
local t = list.Get( &quot;Vehicles&quot; )[ c ]
print( t.Name )</code></pre>
