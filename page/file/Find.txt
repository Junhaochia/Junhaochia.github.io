<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a list of files and directories inside a single folder.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> name</span>
</p>
<div style="margin-left: 32px;">
The wildcard to search for. "models/*.mdl" will list .mdl files in the models/ folder.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> path</span>
</p>
<div style="margin-left: 32px;">
The path to look for the files and directories in. See <a href="/page/File_Search_Paths" title="File Search Paths">this list</a> for a list of valid paths.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> sorting="nameasc"</span>
</p>
<div style="margin-left: 32px;">
<p>The sorting to be used, optional.
</p>
<ul><li> "nameasc" sort the files ascending by name
</li><li> "namedesc" sort the files descending by name
</li><li> "dateasc" sort the files ascending by date
</li><li> "datedesc" sort the files descending by date</div>
</li></ul>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
A table of found files, or nil if the path is invalid</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns_2">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
A table of found directories, or nil if the path is invalid</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints the first file and the first directory in the <i>data</i> folder.
</p>
<pre><code data-language="lua">local files, directories = file.Find(&quot;*&quot;, &quot;DATA&quot;)

print( &quot;File: &quot; .. files[1], &quot;Folder: &quot; .. directories[1] )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><br /> File: helloworld.txt<br />
Folder: ctp</div>
