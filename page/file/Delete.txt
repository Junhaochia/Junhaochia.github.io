<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Deletes a file or <i>empty</i> folder that is relative to the <b>data</b> folder.
You can't remove any files outside of <b>data</b> folder.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">

</p>
<div style="margin-left: 32px;">
The file name</div>
</div>
