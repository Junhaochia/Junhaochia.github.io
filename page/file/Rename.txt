<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Attempts to rename a file with the given name to another given name.
</p><p>This function is constrained to the data/ folder.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> orignalFileName</span>
</p>
<div style="margin-left: 32px;">
<p>The original file or folder name. See <a href="/page/Category_colon_file" title="Category:file">file</a>.<a href="/page/file/Write" title="file/Write">Write</a> for details on filename restrictions when writing to files.
</p>
This argument will be forced lowercase.</div>
</div>

</p>
This argument will be forced lowercase.</div>
