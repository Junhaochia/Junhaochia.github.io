<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Writes the given string to a file. Erases all previous data in the file. To add data without deleting previous data, use <a href="/page/Category_colon_file" title="Category:file">file</a>.<a href="/page/file/Append" title="file/Append">Append</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> fileName</span>
</p>
<div style="margin-left: 32px;">
<p>The name of the file being written into. The path is relative to the <b>data/</b> folder.
</p><p>This argument will be forced lowercase.
</p><p>The filename <b>must</b> end with one of the following:
</p>
<ul><li> .txt
</li><li> .jpg
</li><li> .png
</li><li> .vtf
</li><li> .dat
</li><li> .json
</li></ul>
<br />Restricted symbols are: <pre class="inline">&quot;&#160;:</code></pre></div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> content</span>
</p>
<div style="margin-left: 32px;">
The content that will be written into the file.</div>
</div>

</p>
