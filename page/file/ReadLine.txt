<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the contents of the file from the current position up until the end of the current line.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
This function will look specifically for <i>Line Feed</i> characters <pre class="plain inline">\n</code></pre> and will <b>completely ignore <i>Carriage Return</i> characters</b> <pre class="plain inline">\r</code></pre>.
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function will not return more than 8192 characters.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Open a file in read only mode, reads a line, tells where the current file pointer is at and then closes the file handle.
</p>
<pre><code data-language="lua">local f = file.Open( &quot;cfg/mapcycle.txt&quot;, &quot;r&quot;, &quot;MOD&quot; )
print( f:ReadLine() )
print( f:ReadLine() )
print( f:Tell() )
f:Close()</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">//

// Default mapcycle file for Garry's Mod.

45</code></pre>
