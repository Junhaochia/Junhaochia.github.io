<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the file's size in bytes. If the file is not found, returns -1.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> fileName</span>
</p>
<div style="margin-left: 32px;">
The file's name.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> path</span>
</p>
<div style="margin-left: 32px;">
<p>The path type.
</p>
<ul><li> "GAME" Structured like base folder (garrysmod/), searches all the mounted content (main folder, addons, mounted games etc)
</li><li> "LUA" or "lsv" - All Lua folders (lua/) including gamesmodes and addons
</li><li> "DATA" Data folder (garrysmod/data)
</li><li> "MOD" Strictly the game folder (garrysmod/), ignores mounting.</div>
</li></ul>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints the size of <i>helloworld.txt</i>.
</p>
<pre><code data-language="lua">print( file.Size(&quot;helloworld.txt&quot;, &quot;DATA&quot;) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">8</div>
