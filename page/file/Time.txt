<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns when the file or folder was lasted modified in Unix time.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> path</span>
</p>
<div style="margin-left: 32px;">
The <b>file</b> or <b>folder</b> path.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> gamePath</span>
</p>
<div style="margin-left: 32px;">
<p>The game path to be used.
</p>
<ul><li> "GAME" Structured like base folder (garrysmod/), searches all the mounted content (main folder, addons, mounted games etc)
</li><li> "LUA" or "lsv" - All Lua folders (lua/) including gamesmodes and addons
</li><li> "DATA" Data folder (garrysmod/data)
</li><li> "MOD" Strictly the game folder (garrysmod/), ignores mounting.</div>
</li></ul>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a></span>
</p>
<div style="margin-left: 32px;">
Seconds passed since Unix epoch.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints out the last modified date of <b>file</b> helloworld.txt
</p>
<pre><code data-language="lua">print( os.date(&quot;%d.%m.%Y&quot;, file.Time(&quot;helloworld.txt&quot;, &quot;DATA&quot;) ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">04.08.2012</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Prints out the last modified date of <b>folder</b> lua
</p>
<pre><code data-language="lua">print( os.date( &quot;%d.%m.%Y&quot;, file.Time( &quot;lua&quot;, &quot;GAME&quot; ) ) )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">31.01.2016</div>
