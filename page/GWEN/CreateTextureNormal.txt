<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Used in derma skins to create a rectangle drawing function from an image. The texture of the rectangle will be scaled. The texture is taken from SKIN.GwenTexture
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> x</span>
</p>
<div style="margin-left: 32px;">
The X coordinate on the texture</div>
</div>

The Y coordinate on the texture</div>

<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>

<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a></span>

<div style="margin-left: 32px;">
<p>The drawing function. Arguments are:
</p>
<ul><li><a href="/page/Category_colon_number" title="Category:number">number</a> x - X coordinate for the box
</li><li><a href="/page/Category_colon_number" title="Category:number">number</a> y - Y coordinate for the box
</li><li><a href="/page/Category_colon_number" title="Category:number">number</a> w - Width of the box

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Draws a huge checked checkbox in the top left corner of the screen using current skins texture.
</p>
<pre><code data-language="lua">local paintBox = GWEN.CreateTextureNormal( 448, 32, 15, 15 )
hook.Add( &quot;HUDPaint&quot;, &quot;PaintStuff&quot;, function()
    paintBox( 0, 0, 100, 100 )
end )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">If using default skin, it will be a 100x100px checked checkbox in top left corner of the screen.</div>
