<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<p>A very versatile text display element that's used to power the default chat and console.
</p><p>Rich Text panels allows multicolored, highlight-able, and interactive text using individual text segment markup (segments are defined by the <a href="/page/Category_colon_Panel" title="Category:Panel">Panel</a>:<a href="/page/Panel/AppendText" title="Panel/AppendText">AppendText</a> method).
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Creates a Rich Text frame with 3 text segments.
</p>
<pre><code data-language="lua">-- Window frame for containing text
local frame = vgui.Create(&quot;DFrame&quot;)
frame:SetSize(200, 150)
frame:Center()
frame:MakePopup()

-- Rich Text panel
local richtext = vgui.Create( &quot;RichText&quot;, frame )
richtext:Dock( FILL )

-- Text segment #1 (grayish color)
richtext:InsertColorChange(192, 192, 192, 255)
richtext:AppendText(&quot;This \nRichText \nis \n&quot;)

-- Text segment #2 (light yellow)
richtext:InsertColorChange(255, 255, 224, 255)
richtext:AppendText(&quot;AWESOME\n\n&quot;)

-- Text segment #3 (red ESRB notice localized string)
richtext:InsertColorChange(255, 64, 64, 255)
richtext:AppendText(&quot;#ServerBrowser_ESRBNotice&quot;)</code></pre>
