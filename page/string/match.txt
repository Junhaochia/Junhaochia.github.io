<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Finds a <a href="/page/Patterns" title="Patterns">Pattern</a> in a string.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> string</span>
</p>
<div style="margin-left: 32px;">
String which should be searched in for matches.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> pattern</span>
</p>
<div style="margin-left: 32px;">
The pattern that defines what should be matched.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> startPosition=1</span>
</p>
<div style="margin-left: 32px;">
The start index to start the matching from, can be negative to start the match from a position relative to the end.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_vararg" title="Category:vararg">vararg</a></span>
</p>
<div style="margin-left: 32px;">
Matched text(s)</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<pre><code data-language="lua">local toMatch = &quot;this is a sample text&quot;
print( string.match( toMatch, &quot;sample&quot; ) )
-- regex works
print( string.match( toMatch, &quot;^[a-z]&quot; ) )
print( string.match( toMatch, &quot;^this&quot; ) )
print( string.match( toMatch, &quot;^..is&quot; ) )
print( string.match( toMatch, &quot;text$&quot; ) )
-- entire string
print( string.match( toMatch, &quot;^.*$&quot; ) )
-- nil
print( string.match( toMatch, &quot;this$&quot; ) )
print( string.match( toMatch, &quot;nil&quot; ) )</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">sample  
t  
this  
this  
text  
this is a sample text  
nil  
nil</code></pre>
