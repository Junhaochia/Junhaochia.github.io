<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Escapes all special characters within a string, making the string safe for inclusion in a Lua pattern.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> str</span>
</p>
<div style="margin-left: 32px;">
The string to be sanitized</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a></span>
</p>
<div style="margin-left: 32px;">
The string that has been sanitized for inclusion in Lua patterns</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Replaces all occurrences of "100%" in a string with "0%" and prints it.
</p><p>We call <a href="/page/Category_colon_string" title="Category:string">string</a>.<strong class="selflink">PatternSafe</strong> here as '%' is a special character when used in Lua patterns.
</p>
<pre><code data-language="lua">local result = string.gsub( &quot;You scored 100%!&quot;, 
							string.PatternSafe( &quot;100%&quot; ), 
							string.PatternSafe( &quot;0%&quot; ) )

print( result )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">You scored 0%!</div>
