<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a sub-string, starting from the character at position <i>StartPos</i> of the string (inclusive), and optionally ending at the character at position <i>EndPos</i> of the string (also inclusive). If EndPos is not given, the rest of the string is returned.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> string</span>
</p>
<div style="margin-left: 32px;">
The string you'll take a sub-string out of.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> StartPos</span>
</p>
<div style="margin-left: 32px;">
The position of the first character that will be included in the sub-string.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> EndPos=nil</span>
</p>
<div style="margin-left: 32px;">
The position of the last character to be included in the sub-string. It can be negative to count from the end.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a></span>
</p>
<div style="margin-left: 32px;">
The substring.</div>
</div>
