<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Escapes special characters for JavaScript in a string, making the string safe for inclusion in to JavaScript strings.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> str</span>
</p>
<div style="margin-left: 32px;">
The string that should be escaped.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a></span>
</p>
<div style="margin-left: 32px;">
The escaped string.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Executes JavaScript on a panel using user input safely.
</p>
<pre><code data-language="lua">local user_input = &quot;The user's input lives in this variable&quot;

local Panel = vgui.Create( &quot;DHTML&quot; )
Panel:SetURL( &quot;example.com/something.html&quot; )
Panel:Center()
Panel:SetSize( 100, 100 )
Panel:RunJavascript( &quot;MyJavaScriptFunction('&quot; .. string.JavascriptSafe( user_input ) .. &quot;')&quot; )</code></pre>
