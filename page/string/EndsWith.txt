<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns whether or not the second passed string matches the end of the first.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> str</span>
</p>
<div style="margin-left: 32px;">
The string whose end is to be checked.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> end</span>
</p>
<div style="margin-left: 32px;">
The string to be matched with the end of the first.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
<i>true</i> if the first string ends with the second, or the second is empty, otherwise <i>false</i>.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Looks for arguments at the end of a string.
</p>
<pre><code data-language="lua">local endswith = string.EndsWith(&quot;Supercalifragilisticexpialidocious&quot;, &quot;docious&quot;)
if endswith then
	print(&quot;Marry Poppins&quot;)
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><pre><code data-language="lua">Marry Poppins</code></pre></div>
