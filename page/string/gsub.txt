<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>This functions main purpose is to replace certain character sequences in a string using <a href="/page/Patterns" title="Patterns">Patterns</a>.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> string</span>
</p>
<div style="margin-left: 32px;">
String which should be modified.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> pattern</span>
</p>
<div style="margin-left: 32px;">
The pattern that defines what should be matched and eventually be replaced.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> replacement</span>
</p>
<div style="margin-left: 32px;">
<p>In case of a string the matches sequence will be replaced with it.
</p><p>In case of a table, the matched sequence will be used as key and the table will tested for the key, if a value exists it will be used as replacement.
</p>
In case of a function all matches will be passed as parameters to the function, the return value(s) of the function will then be used as replacement.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> maxReplaces=nil</span>
</p>
<div style="margin-left: 32px;">
Maximum number of replacements to be made.</div>
</div>

<h1 class="first_ret"> <span class="mw-headline" id="Returns_2">Returns</span></h1>

</p>

replaceCount</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Replaces "hello" with "hi" in the string "hello there!"
</p>
<pre><code data-language="lua">string.gsub(&quot;hello there!&quot;, &quot;hello&quot;, &quot;hi&quot;)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">hi there!</div>
