<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the time as a formatted string or as a table if no format is given.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> float</span>
</p>
<div style="margin-left: 32px;">
The time in seconds to format.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> format=nil</span>
</p>
<div style="margin-left: 32px;">
An optional formatting to use. If no format it specified, a table will be returned instead.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a></span>
</p>
<div style="margin-left: 32px;">
<p>Returns the time as a formatted string only if a format was specified.
</p><p>Returns a table only if no format was specified. The table will contain these fields:<br />
<a href="/page/Category_colon_number" title="Category:number">number</a> ms - milliseconds<br />
<a href="/page/Category_colon_number" title="Category:number">number</a> s - seconds<br />
<a href="/page/Category_colon_number" title="Category:number">number</a> m - minutes<br />
<a href="/page/Category_colon_number" title="Category:number">number</a> h - hours<br />
</p>
<br /></div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Formats the time in seconds
</p>
<pre><code data-language="lua">local time = string.FormattedTime( 90, &quot;%02i:%02i:%02i&quot; )
print( time )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">01:30:00</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Returns a table with the time separated by units.
</p>
<pre><code data-language="lua">local time = string.FormattedTime( 90 )
PrintTable( time )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><br />ms = 0<br />
<p>m = 1<br />
s = 30<br />
</p>
h = 0<br /></div>
