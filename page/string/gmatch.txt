<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Using <a href="/page/Patterns" title="Patterns">Patterns</a>, returns an iterator which will return either one value if no capture groups are defined, or any capture group matches.
</p>

</div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Explodes the string for each space and comma in the string
</p>
<pre><code data-language="lua">str = &quot;qwe,a cde&quot;
for s in string.gmatch(str, &quot;[^%s,]+&quot;) do
    print(s)
end</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">qwe
a
cde</code></pre>
