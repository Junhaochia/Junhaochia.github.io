<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Adds a language item. Language placeholders preceded with "#" are replaced with full text in Garry's Mod once registered with this function.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> placeholder</span>
</p>
<div style="margin-left: 32px;">
The key for this phrase, without the preceding "#".</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> fulltext</span>
</p>
<div style="margin-left: 32px;">
The phrase that should be displayed whenever this key is used.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Small excerpt from a STOOL named cooltool. It has been registered as cooltool.
</p>
<pre><code data-language="lua">language.Add(&quot;Tool_cooltool_name&quot;, &quot;The really cool tool&quot;)
language.Add(&quot;Tool_cooltool_desc&quot;, &quot;Do some random cool stuff.&quot;)
language.Add(&quot;Tool_cooltool_0&quot;, &quot;Left-click: Cool Stuff. Right-click: Nothing.&quot;)
language.Add(&quot;Undone_cooltool&quot;, &quot;Cool stuff has been undone.&quot;)</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">When the player presses undo, "Cool stuff has been undone" will be shown. The cooltool's name will be "The really cool tool". Below that, where the description is shown, "Do some random cool stuff." will be shown. Below that, where the instructions or additional notes for the tool are shown, it will display "Left-click: Cool Stuff. Right-click: Nothing.".</div>
