<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when a variable is edited on an Entity (called by Edit Properties... menu), to determine if the edit should be permitted.
</p><p>See <a href="/page/Editable_Entities" title="Editable Entities">Editable entities</a> for more details about the system.
</p>
