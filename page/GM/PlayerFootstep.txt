<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called whenever a player steps. Return true to mute the normal sound.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Disables default player footsteps and plays custom ones.
</p>
<pre><code data-language="lua">function GM:PlayerFootstep( ply, pos, foot, sound, volume, rf ) 
	ply:EmitSound(&quot;NPC_Hunter.Footstep&quot;) -- Play the footsteps hunter is using
	return true -- Don't allow default footsteps
end</code></pre>
