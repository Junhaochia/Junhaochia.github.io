<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when a player has been hurt by an explosion. Override to disable default sound effect.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Player" title="Category:Player">Player</a> ply</span>
</p>
<div style="margin-left: 32px;">
Player who has been hurt</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_CTakeDamageInfo" title="Category:CTakeDamageInfo">CTakeDamageInfo</a> dmginfo</span>
</p>
<div style="margin-left: 32px;">
Damage info from explsion</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Disables the high pitched ringing sound effect.
</p><p>Note that this hook does not have a return value, and instead by default it calls <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/SetDSP" title="Player/SetDSP">SetDSP</a>( 35, false ) in the base gamemode.
</p>
<pre><code data-language="lua">hook.Add( &quot;OnDamagedByExplosion&quot;, &quot;DisableSound&quot;, function()
	return true
end )</code></pre>
