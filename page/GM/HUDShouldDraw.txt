<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when the Gamemode is about to draw a given element on the client's HUD (heads-up display).
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This hook is called HUNDREDS of times per second (more than 5 times per frame on average). You shouldn't be performing any computationally intensive operations.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> name</span>
</p>
<div style="margin-left: 32px;">
The name of the HUD element. You can find a full list of HUD elements for this hook <a href="/page/HUD_Element_List" title="HUD Element List">here</a>.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Return false to prevent the given element from being drawn on the client's screen.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Hides the default health and battery (armor) HUD elements, while still allowing the display of other elements to be controlled by other addons.
</p>
<pre><code data-language="lua">local hide = {
	[&quot;CHudHealth&quot;] = true,
	[&quot;CHudBattery&quot;] = true
}

hook.Add( &quot;HUDShouldDraw&quot;, &quot;HideHUD&quot;, function( name )
	if ( hide[ name ] ) then return false end

	-- Don't return anything here, it may break other addons that rely on this hook.
end )</code></pre>
