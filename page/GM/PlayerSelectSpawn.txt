<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called to determine a spawn point for a player to spawn at.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Player" title="Category:Player">Player</a> ply</span>
</p>
<div style="margin-left: 32px;">
The player who needs a spawn point</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> transition</span>
</p>
<div style="margin-left: 32px;">
If true, the player just spawned from a map transition. You probably want to not return an entity for that case to not override player's position.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a></span>
</p>
<div style="margin-left: 32px;">
The spawnpoint entity to spawn the player at</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Find a random spawn point
</p>
<pre><code data-language="lua">function GM:PlayerSelectSpawn( pl )
    
    local spawns = ents.FindByClass( &quot;info_player_start&quot; )
    local random_entry = math.random( #spawns )
    
    return spawns[ random_entry ]
	
end</code></pre>
