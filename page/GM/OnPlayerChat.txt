<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called whenever a player sends a chat message. For the serverside equivalent, see <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PlayerSay" title="GM/PlayerSay">PlayerSay</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>The text input of this hook depends on <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PlayerSay" title="GM/PlayerSay">PlayerSay</a>. If it is suppressed on the server, it will be suppressed on the client.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Code from base gamemode. See garrysmod/gamemodes/base/gamemode/cl_init.lua#L139
</p>
<pre><code data-language="lua">function GM:OnPlayerChat( player, strText, bTeamOnly, bPlayerIsDead )
 
	--
	-- I've made this all look more complicated than it is. Here's the easy version
	--
	-- chat.AddText( player, Color( 255, 255, 255 ), &quot;: &quot;, strText )
	--

	local tab = {}

	if ( bPlayerIsDead ) then
		table.insert( tab, Color( 255, 30, 40 ) )
		table.insert( tab, &quot;*DEAD* &quot; )
	end

	if ( bTeamOnly ) then
		table.insert( tab, Color( 30, 160, 40 ) )
		table.insert( tab, &quot;(TEAM) &quot; )
	end

	if ( IsValid( player ) ) then
		table.insert( tab, player )
	else
		table.insert( tab, &quot;Console&quot; )
	end

	table.insert( tab, Color( 255, 255, 255 ) )
	table.insert( tab, &quot;: &quot;..strText )

	chat.AddText( unpack(tab) )

	return true
 
end</code></pre>

<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>How you could create a clientside only chat command.
</p>
<pre><code data-language="lua">hook.Add( &quot;OnPlayerChat&quot;, &quot;HelloCommand&quot;, function( ply, strText, bTeam, bDead ) 
    if ( ply&#160;!= LocalPlayer() ) then return end

	strText = string.lower( strText ) -- make the string lower case

	if (strText == &quot;/hello&quot;) then -- if the player typed /hello then
		print(&quot;Hello world!&quot;) -- print Hello world to the console
		return true -- this suppresses the message from being shown
	end

end )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Prints "Hello world!" to the console when you type /hello in the chat.</div>
