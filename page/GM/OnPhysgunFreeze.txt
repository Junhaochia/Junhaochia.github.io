<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when a player freezes an entity with the physgun.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This is not called for players or NPCs being held with the physgun.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/723">#723</a></b>
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Only allows admins to freeze things.
</p>
<pre><code data-language="lua">hook.Add( &quot;OnPhysgunFreeze&quot;, &quot;PhysFreeze&quot;, function( weapon, phys, ent, ply )

  	if (&#160;!ply:IsAdmin() ) then

		return false

	end

end )</code></pre>
