<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called to allow override of the default Derma skin for all panels.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This hook is only called on Lua start up, changing its value (or adding new hooks) after it has been already called will not have any effect.
</p>
</td></tr></table>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a></span>
</p>
<div style="margin-left: 32px;">
<p>A <b>case sensitive</b> Derma skin name to be used as default, registered previously via <a href="/page/Category_colon_derma" title="Category:derma">derma</a>.<a href="/page/derma/DefineSkin" title="derma/DefineSkin">DefineSkin</a>.<br />
</p>
Returning nothing, nil or invalid name will make it fallback to the "Default" skin.</div>
</div>

<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example on how to use this hook. This will make that all panels
</p>
<pre><code data-language="lua">hook.Add( &quot;ForceDermaSkin&quot;, &quot;my_new_skin&quot;, function()
	return &quot;some_skin&quot;
end )</code></pre>
