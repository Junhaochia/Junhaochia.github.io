<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Runs when a bind has been pressed. Allows to block commands.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>By using the "alias" console command, this hook can be effectively circumvented
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>To stop the user from using +attack, +left and any other movement commands of the sort, please look into using <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/StartCommand" title="GM/StartCommand">StartCommand</a> instead
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>The third argument will always be true.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/1176">#1176</a></b>
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This does not run for function keys binds (F1-F12).
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/2888">#2888</a></b>
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prevents players from using flashlight.
</p>
<pre><code data-language="lua">hook.Add( &quot;PlayerBindPress&quot;, &quot;PlayerBindPressExample&quot;, function( ply, bind, pressed )
	--To block more commands, you could add another line similar to
	--the one below, just replace the command
	if ( string.find( bind, &quot;impulse 100&quot; ) ) then return true end
end )</code></pre>
