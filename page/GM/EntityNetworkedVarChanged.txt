<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when an NWVar is changed.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Currently this hook only works for the NW2Var system (accessed by adding a 2 in between NW and Var for most NWVar functions), which will replace the original one at some point in the future
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This hook is fired before the client value is actually changed. Calling the GetNW function for the specified variable name within this hook will return the old value, not the current/updated one.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example usage of the hook. Prints out all NWVar changes.
</p>
<pre><code data-language="lua">hook.Add(&quot;EntityNetworkedVarChanged&quot;,&quot;printchange&quot;, print )

-- Trigger a change!
Entity(1):SetNW2String( &quot;UserGroup&quot;, &quot;owner&quot; )</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">Player [1][Player1]	UserGroup	superadmin	owner</code></pre>
