<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when an addon from the Steam workshop begins downloading. Used by default to place details on the workshop downloading panel.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> id</span>
</p>
<div style="margin-left: 32px;">
Workshop ID of addon.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> imageID</span>
</p>
<div style="margin-left: 32px;">
<p>ID of addon's preview image.<br />
</p><p>For example, for <b>Extended Spawnmenu</b> addon, the image URL is
<samp>http://cloud-4.steamusercontent.com/ugc/702859018846106764/9E7E1946296240314751192DA0AD15B6567FF92D/</samp>
</p>
So, the value of this argument would be <b>702859018846106764</b>.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> title</span>
</p>
<div style="margin-left: 32px;">
Name of addon.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> size</span>
</p>
<div style="margin-left: 32px;">
File size of addon in bytes.</div>
</div>
