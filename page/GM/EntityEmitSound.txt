<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called whenever a sound has been played. This will not be called clientside if the server played the sound without the client also calling <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/EmitSound" title="Entity/EmitSound">EmitSound</a>.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This is not called for scripted sequences.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/1021">#1021</a></b>
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> data</span>
</p>
<div style="margin-left: 32px;">
Information about the played sound. Changes done to this table can be applied by returning true from this hook.<br />See <a href="/page/Structures/EmitSoundInfo" title="Structures/EmitSoundInfo">EmitSoundInfo structure</a>.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Return true to apply all changes done to the data table.<br />Return false to prevent the sound from playing.<br />Return nil or nothing to play the sound without altering it.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Slows down all sounds to reflect <a href="/page/Category_colon_game" title="Category:game">game</a>.<a href="/page/game/SetTimeScale" title="game/SetTimeScale">SetTimeScale</a>.
</p>
<pre><code data-language="lua">hook.Add( &quot;EntityEmitSound&quot;, &quot;TimeWarpSounds&quot;, function( t )
	
	local p = t.Pitch
	
	if ( game.GetTimeScale()&#160;!= 1 ) then
		p = p * game.GetTimeScale()
	end
	
	if ( GetConVarNumber( &quot;host_timescale&quot; )&#160;!= 1 &amp;&amp; GetConVarNumber( &quot;sv_cheats&quot; ) &gt;= 1 ) then
		p = p * GetConVarNumber( &quot;host_timescale&quot; )
	end
	
	if ( p&#160;!= t.Pitch ) then
		t.Pitch = math.Clamp( p, 0, 255 )
		return true
	end
	
	if ( CLIENT &amp;&amp; engine.GetDemoPlaybackTimeScale()&#160;!= 1 ) then
		t.Pitch = math.Clamp( t.Pitch * engine.GetDemoPlaybackTimeScale(), 0, 255 )
		return true
	end
	
end )</code></pre>
