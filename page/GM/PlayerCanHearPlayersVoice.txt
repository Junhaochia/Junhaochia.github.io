<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Decides whether a player can hear another player using voice chat.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This hook is called several times a tick, so ensure your code is efficient.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Player" title="Category:Player">Player</a> listener</span>
</p>
<div style="margin-left: 32px;">
The listening player.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Player" title="Category:Player">Player</a> talker</span>
</p>
<div style="margin-left: 32px;">
The talking player.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Return true if the listener should hear the talker, false if they shouldn't.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns_2">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
3D sound. If set to true, will fade out the sound the further away listener is from the talker, the voice will also be in stereo, and not mono.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p><b>Inefficient</b> implementation where players can only hear each other if they are within 500 units.
</p>
<pre><code data-language="lua">hook.Add(&quot;PlayerCanHearPlayersVoice&quot;, &quot;Maximum Range&quot;, function(listener, talker)
    if listener:GetPos():Distance(talker:GetPos()) &gt; 500 then return false end
end)</code></pre>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p><b>Efficient</b> implementation where players can only hear each other if they are within 500 units. This is more efficient, because <a href="/page/Category_colon_Vector" title="Category:Vector">Vector</a>:<a href="/page/Vector/Distance" title="Vector/Distance">Distance</a> adds an additional square root operation, which isn't necessary and rather expensive.
</p>
<pre><code data-language="lua">hook.Add(&quot;PlayerCanHearPlayersVoice&quot;, &quot;Maximum Range&quot;, function(listener, talker)
    if listener:GetPos():DistToSqr(talker:GetPos()) &gt; 250000 then return false end
end)</code></pre>
