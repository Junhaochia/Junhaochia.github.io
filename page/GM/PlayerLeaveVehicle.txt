<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when a player leaves a vehicle.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>For vehicles with exit animations, this will be called <b>at the end</b> of the animation, <b>not at the start</b>!
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This is not called when a different vehicle is immediately entered with <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/EnterVehicle" title="Player/EnterVehicle">EnterVehicle</a>.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/2619">#2619</a></b>
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Make the vehicle continue running (or more precisely start its engine again) when a player exists it, unless the player holds their Reload key.
</p>
<pre><code data-language="lua">hook.Add( &quot;PlayerLeaveVehicle&quot;, &quot;PlayerLeaveVehicleTurnOn&quot;, function( ply, veh )
	if ( ply:KeyDown( IN_RELOAD ) ) then return end

	-- We need to wait a few frames for the vehicle to stop working
	timer.Create( &quot;magic timer&quot; .. veh:EntIndex(), 0, 2, function()
		-- Only run on the last time the timer is fired
		if ( timer.RepsLeft( &quot;magic timer&quot; .. veh:EntIndex() ) &gt; 0 ) then return end

		veh:StartEngine( true ) -- Start the engine back up
		veh:ReleaseHandbrake() -- Release the handbrake
	end )
end )</code></pre>
