<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when a player tries to pick up something using the "use" key, return to override.
</p><p>See <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/GravGunPickupAllowed" title="GM/GravGunPickupAllowed">GravGunPickupAllowed</a> for the Gravity Gun pickup variant.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Player" title="Category:Player">Player</a> ply</span>
</p>
<div style="margin-left: 32px;">
The player trying to pick up something.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> ent</span>
</p>
<div style="margin-left: 32px;">
The Entity the player attempted to pick up.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Allow the player to pick up the entity or not.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Allows only admins to pick up things
</p>
<pre><code data-language="lua">local function up( ply, ent )
    return ply:IsAdmin()
end
hook.Add( &quot;AllowPlayerPickup&quot;, &quot;some_unique_name&quot;, up )</code></pre>
