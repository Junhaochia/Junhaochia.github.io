<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called before the view model has been drawn. This hook by default also calls this on weapons, so you can use <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a>:<a href="/page/WEAPON/PreDrawViewModel" title="WEAPON/PreDrawViewModel">PreDrawViewModel</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is a rendering hook with a 3D rendering context.<br />This means that the only rendering functions will work in it are <a href="/page/Category_colon_3D_Rendering_Functions" title="Category:3D Rendering Functions">functions with a 3D rendering context</a>.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This is also called once a frame with no arguments.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3024">#3024</a></b>
</p>
</td></tr></table>
