<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when a key-value pair is set on an entity, either by the engine (for example when map spawns) or <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetKeyValue" title="Entity/SetKeyValue">SetKeyValue</a>.
</p><p>See <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/KeyValue" title="ENTITY/KeyValue">KeyValue</a> for a hook that works for scripted entities.<br />
See <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a>:<a href="/page/WEAPON/KeyValue" title="WEAPON/KeyValue">KeyValue</a> for a hook that works for scripted weapons.
</p>
