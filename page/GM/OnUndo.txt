<h1> <span class="mw-headline" id="Description">Description</span></h1>

</p>

</p>
<div style="margin-left: 32px;">

</div>

<div class="argument">

The custom text for the undo, set by <a href="/page/Category_colon_undo" title="Category:undo">undo</a>.<a href="/page/undo/SetCustomUndoText" title="undo/SetCustomUndoText">SetCustomUndoText</a></div>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Print a message when the player undoes something.
</p>
<pre><code data-language="lua">function GM:OnUndo( name, customText )
     if customText&#160;!= nil then
          MsgN( &quot;Undone &quot; .. customText )
     else
          MsgN( &quot;Undone &quot; .. name )
     end
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Undone &lt;action&gt;</div>
