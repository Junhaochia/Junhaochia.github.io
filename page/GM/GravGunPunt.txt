<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when an entity is about to be punted with the gravity gun (primary fire).
</p><p>By default this function makes <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/GravGunPunt" title="ENTITY/GravGunPunt">GravGunPunt</a> work in Sandbox derived gamemodes.
</p>
