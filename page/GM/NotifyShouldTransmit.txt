<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called whenever this entity changes its transmission state for this <a href="/page/Global/LocalPlayer" title="Global/LocalPlayer">LocalPlayer</a>, such as exiting or re entering the PVS.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is the best place to handle the reset of <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetPredictable" title="Entity/SetPredictable">SetPredictable</a>, as this would be usually called when the player lags and requests a full packet update
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>When the entity stops transmitting, <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/IsDormant" title="Entity/IsDormant">IsDormant</a> will only return true <b>after</b> this hook
</p>
</td></tr></table>

</p>

</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
