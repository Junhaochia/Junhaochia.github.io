<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns whether or not a player is allowed to pick up a weapon.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Player" title="Category:Player">Player</a> ply</span>
</p>
<div style="margin-left: 32px;">
The player attempting to pick up the weapon</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Weapon" title="Category:Weapon">Weapon</a> wep</span>
</p>
<div style="margin-left: 32px;">
The weapon entity in question</div>
</div>

<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Allowed pick up or not</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Disallows picking up a weapon if player already has this weapon.
( Prevents ammo pickups from lying guns )
</p>
<pre><code data-language="lua">hook.Add( &quot;PlayerCanPickupWeapon&quot;, &quot;noDoublePickup&quot;, function( ply, wep )
    if ( ply:HasWeapon( wep:GetClass() ) ) then return false end
end )</code></pre>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Players can only pick up the HL2 Pistol.
</p>
<pre><code data-language="lua">function GM:PlayerCanPickupWeapon(ply, wep)
    return (wep:GetClass() == &quot;weapon_pistol&quot;)
end</code></pre>
<p><br />
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples_3">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_3">Example</span></h2>
<p>How you could give a player an alternate weapon to the one they picked up (such as an RPG Launcher rather than a pistol)
</p>
<pre><code data-language="lua">hook.Add( &quot;PlayerCanPickupWeapon&quot;, &quot;NoPistolGiveFists&quot;, function( ply, wep )
	if wep:GetClass() == &quot;weapon_pistol&quot; then -- if the weapon they are trying to pick up is a pistol
		ply:Give( &quot;weapon_rpg&quot; ) -- give them an RPG
		wep:Remove() -- remove the one they were trying to pick up
		return false -- don't give them a pistol
	end
end )</code></pre>
