<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when the player spawns for the first time.
</p><p>See <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PlayerSpawn" title="GM/PlayerSpawn">PlayerSpawn</a> for a hook called every player spawn.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This hook is called before the player has fully loaded, when the player is still in seeing the "Starting Lua" screen. For example, trying to use the <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetModel" title="Entity/GetModel">GetModel</a> function will return the default model ("player/default.mdl")
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Due to the above note, sending <a href="/page/Category_colon_net" title="Category:net">net library</a> messages to the spawned player in this hook is highly unreliable, and they most likely won't be received. See <a rel="nofollow" class="external free" href="https://github.com/Facepunch/garrysmod-requests/issues/718">https://github.com/Facepunch/garrysmod-requests/issues/718</a>. A quick and dirty work-around is to delay any sending using <a href="/page/Category_colon_timer" title="Category:timer">timer</a>.<a href="/page/timer/Simple" title="timer/Simple">Simple</a> with at least 5 seconds delay.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints the name of the player joining.
</p>
<pre><code data-language="lua">function GM:PlayerInitialSpawn(ply)
	print( ply:GetName()..&quot; joined the server.\n&quot; )
end

-- That way you are overriding the default hook
-- you can use hook.Add to make more functions get called when this event occurs
local function spawn(ply)
	print( ply:GetName()..&quot; joined the game.\n&quot;)
end
hook.Add( &quot;PlayerInitialSpawn&quot;, &quot;some_unique_name&quot;, spawn )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Player1 joined the game</div>
