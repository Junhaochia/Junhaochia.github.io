<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when a player is killed by <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/Kill" title="Player/Kill">Kill</a> or any other normal means.
</p><p>This hook is <b>not</b> called if the player is killed by <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/KillSilent" title="Player/KillSilent">KillSilent</a>. See <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PlayerSilentDeath" title="GM/PlayerSilentDeath">PlayerSilentDeath</a> for that.
</p>
<ul><li> <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/DoPlayerDeath" title="GM/DoPlayerDeath">DoPlayerDeath</a> is called <b>before</b> this hook.
</li><li> <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PostPlayerDeath" title="GM/PostPlayerDeath">PostPlayerDeath</a> is called <b>after</b> this hook.
</li></ul>
<p>See <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/LastHitGroup" title="Player/LastHitGroup">LastHitGroup</a> if you need to get the last hit hitgroup of the player.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>If the player suicides (he is the killer and the victim (ply)), then it will print a message to console. If someone else kills him, it will print a different message to console.
</p>
<pre><code data-language="lua">function GM:PlayerDeath( victim, inflictor, attacker )
    if ( victim == attacker ) then
        PrintMessage( HUD_PRINTTALK, victim:Name() .. &quot; committed suicide.&quot; )
    else
        PrintMessage( HUD_PRINTTALK, victim:Name() .. &quot; was killed by &quot; .. attacker:Name() .. &quot;.&quot;)
    end
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">If suicide:
<p>Player1 has committed suicide.
</p><p>Else:
</p>
Player1 was killed by Player2.</div>
