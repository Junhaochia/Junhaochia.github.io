<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>This hook is used to calculate animations for a player.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>This hook must return the same values at the same time on both, client <b>and</b> server. On client for players to see the animations, on server for hit detection to work properly.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This can return the incorrect velocity when on a moving object.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3322">#3322</a></b>
</p>
</td></tr></table>
