<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called whenever the HUD should be drawn. Called right before <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/HUDDrawScoreBoard" title="GM/HUDDrawScoreBoard">HUDDrawScoreBoard</a> and after <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/HUDPaintBackground" title="GM/HUDPaintBackground">HUDPaintBackground</a>.
</p><p>Not called when the Camera SWEP is equipped. See also <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/DrawOverlay" title="GM/DrawOverlay">DrawOverlay</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Only be called when r_drawvgui is enabled and the game is not paused
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is a rendering hook with a 2D rendering context.<br />This means that the only rendering functions will work in it are <a href="/page/Category_colon_2D_Rendering_Functions" title="Category:2D Rendering Functions">functions with a 2D rendering context</a>.
</p>
</td></tr></table>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Draws a transparent black box in the top left corner of the screen.
</p>
<pre><code data-language="lua">hook.Add(&quot;HUDPaint&quot;, &quot;HUDPaint_DrawABox&quot;, function()
	surface.SetDrawColor( 0, 0, 0, 128 )
	surface.DrawRect( 50, 50, 128, 128 )
end )</code></pre>
