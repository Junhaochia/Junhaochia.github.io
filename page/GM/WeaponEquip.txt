<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called as a weapon entity is picked up by a player.
</p><p>See also <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PlayerDroppedWeapon" title="GM/PlayerDroppedWeapon">PlayerDroppedWeapon</a>.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>At the time when this hook is called <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/GetOwner" title="Entity/GetOwner">GetOwner</a> will return NULL. The owner is set on the next frame
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This will not be called when picking up a weapon you already have as the weapon will be removed and <a href="/page/Category_colon_WEAPON_Hooks" title="Category:WEAPON Hooks">WEAPON</a>:<a href="/page/WEAPON/EquipAmmo" title="WEAPON/EquipAmmo">EquipAmmo</a> will be called instead
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Drops the player's weapons as soon as they pick one up.
</p>
<pre><code data-language="lua">hook.Add( &quot;WeaponEquip&quot;, &quot;WeaponEquipExample&quot;, function( wep, ply )
	timer.Simple( 0, function() ply:DropWeapon( wep ) end )
end )</code></pre>
