<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Runs when the user tries to open the chat box.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>Returning true won't stop the chatbox from taking VGUI focus.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/855">#855</a></b>
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> isTeamChat</span>
</p>
<div style="margin-left: 32px;">
Whether the message was sent through team chat.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Return true to hide the default chat box.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<pre><code data-language="lua">hook.Add( &quot;StartChat&quot;, &quot;HasStartedTyping&quot;, function( isTeamChat )
	if ( isTeamChat ) then
		print( &quot;Player started typing a message in teamchat.&quot; )
	else
		print( &quot;Player started typing a message.&quot; )
	end
end )</code></pre>
