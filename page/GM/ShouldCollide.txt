<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called to decide whether a pair of entities should collide with each other. This is only called if <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/SetCustomCollisionCheck" title="Entity/SetCustomCollisionCheck">SetCustomCollisionCheck</a> was used on one or both entities.
</p><p>Where applicable, consider using <a href="/page/Category_colon_constraint" title="Category:constraint">constraint</a>.<a href="/page/constraint/NoCollide" title="constraint/NoCollide">NoCollide</a> instead - it is considerably easier to use.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>This hook <b>must</b> return the same value consistently for the same pair of entities. If an entity changed in such a way that its collision rules change, you <b>must</b> call <a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a>:<a href="/page/Entity/CollisionRulesChanged" title="Entity/CollisionRulesChanged">CollisionRulesChanged</a> on that entity immediately - <b>not in this hook.</b>
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This hook can cause all physics to break under certain conditions.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/642">#642</a></b>
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This hook is <a href="/page/Prediction" title="Prediction">predicted</a>. This means that in singleplayer, it will <i>not</i> be called in the Client realm.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This should always return true unless you have a good reason for it not to.
</p>
<pre><code data-language="lua">function GM:ShouldCollide( ent1, ent2 )

    -- If players are about to collide with each other, then they won't collide.
    if ( IsValid( ent1 ) and IsValid( ent2 ) and ent1:IsPlayer() and ent2:IsPlayer() ) then return false end 

    -- We must call this because anything else should return true.
    return true

end</code></pre>
