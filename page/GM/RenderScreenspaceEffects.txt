<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Used to render post processing effects.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This is a rendering hook with a 2D rendering context.<br />This means that the only rendering functions will work in it are <a href="/page/Category_colon_2D_Rendering_Functions" title="Category:2D Rendering Functions">functions with a 2D rendering context</a>.
</p>
</td></tr></table>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Renders color modify and sobel effects to create a cartoon effect.
</p>
<pre><code data-language="lua">local tab = {
	[&quot;$pp_colour_addr&quot;] = 0,
	[&quot;$pp_colour_addg&quot;] = 0,
	[&quot;$pp_colour_addb&quot;] = 0,
	[&quot;$pp_colour_brightness&quot;] = -0.04,
	[&quot;$pp_colour_contrast&quot;] = 1.35,
	[&quot;$pp_colour_colour&quot;] = 5,
	[&quot;$pp_colour_mulr&quot;] = 0,
	[&quot;$pp_colour_mulg&quot;] = 0,
	[&quot;$pp_colour_mulb&quot;] = 0
}
function GM:RenderScreenspaceEffects()
	DrawColorModify( tab ) --Draws Color Modify effect
	DrawSobel( 0.5 ) --Draws Sobel effect
end</code></pre>
