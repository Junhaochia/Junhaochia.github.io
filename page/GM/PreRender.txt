<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called before the renderer is about to start rendering the next frame.
</p>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a></span>
</p>
<div style="margin-left: 32px;">
Return true to prevent all rendering. This can make the whole game stop rendering anything.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Fills the draw buffer with black pixels, removing tearing when looking at a world leak. Achieves the same effect as using gl_clear. This is no longer needed as GMod does this by default.
</p>
<pre><code data-language="lua">hook.Add(&quot;PreRender&quot;, &quot;ResetBuffer&quot;, function()
	cam.Start2D()
		surface.SetDrawColor(0, 0, 0, 255)
		surface.DrawRect(0, 0, ScrW(), ScrH())
	cam.End2D()
end)</code></pre>
