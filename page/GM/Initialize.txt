<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called after the gamemode loads and starts.
</p>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>"hi" will be printed to the console when the gamemode initializes.
</p>
<pre><code data-language="lua">function GM:Initialize()
	print(&quot;hi&quot; )
end

-- That way you are overriding the default hook
-- you can use hook.Add to make more functions get called on initialization
local function init()
	print(&quot;Initialization hook called&quot;)
end
hook.Add( &quot;Initialize&quot;, &quot;some_unique_name&quot;, init )</code></pre>
<p><b>Output:</b>
</p>
<pre class="plain">Initializtion hook called
hi</code></pre>
