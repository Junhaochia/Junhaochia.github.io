<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called to determine if a player should be able to pick up an entity with the Physics Gun.
</p><p>See <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/OnPhysgunPickup" title="GM/OnPhysgunPickup">OnPhysgunPickup</a> for a hook which is called when a player has successfully picked up an entity.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This hook is <a href="/page/Prediction" title="Prediction">predicted</a>. This means that in singleplayer, it will <i>not</i> be called in the Client realm.
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Allows Admins to pick up players.
</p>
<pre><code data-language="lua">hook.Add( &quot;PhysgunPickup&quot;, &quot;AllowPlayerPickup&quot;, function( ply, ent )
	if ( ply:IsAdmin() and ent:IsPlayer() ) then
		return true
	end
end )</code></pre>
