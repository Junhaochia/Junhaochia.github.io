<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called when a player leaves the server. See the <a href="/page/Game_Events" title="Game Events">player_disconnect gameevent</a> for a shared version of this hook.
</p>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This is not called in single-player or listen servers for the host.
</p><p><b>Issue Tracker: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod-issues/issues/3523">#3523</a></b>
</p>
</td></tr></table>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Print a message to the chatbox upon player disconnect
</p>
<pre><code data-language="lua">function GM:PlayerDisconnected( ply )
     PrintMessage( HUD_PRINTTALK, ply:Name().. &quot; has left the server.&quot; )
end</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Garry&#160;:D has left the server.</div>
