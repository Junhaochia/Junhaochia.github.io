<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Called whenever an entity becomes a clientside ragdoll.
</p><p>See <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/CreateEntityRagdoll" title="GM/CreateEntityRagdoll">CreateEntityRagdoll</a> for serverside ragdolls.

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>A way of fade out a ragdoll easily. Idea from <a rel="nofollow" class="external text" href="https://github.com/ValveSoftware/source-sdk-2013/blob/0d8dceea4310fde5706b3ce1c70609d72a38efdf/mp/src/game/client/c_baseanimating.cpp#L533">here</a>.
</p>
<pre><code data-language="lua">hook.Add( &quot;CreateClientsideRagdoll&quot;, &quot;fade_out_corpses&quot;, function( entity, ragdoll )

	ragdoll:SetSaveValue( &quot;m_bFadingOut&quot;, true ) -- Set the magic internal variable that will cause the ragdoll to immediately start fading out

end )</code></pre>
