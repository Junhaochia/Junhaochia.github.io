<p>As noted in the welcome tutorial Lua is plain text - so you can edit it in notepad. But you'll probably find you want something a bit better.
</p><p>The editor you use is a personal preference - so it's up to you to choose one for yourself. The editors below are listed because they're the most popular editors in the Garry's Mod community.
</p>

<h1> <span class="mw-headline" id="Notepad.2B.2B"> <a rel="nofollow" class="external text" href="https://notepad-plus-plus.org/">Notepad++</a> </span></h1>
<p>A text editor with a number of features. Like Sublime Text, it also has syntax highlighting, a minimap and a number of hotkeys and a Garry's Mod specific Syntax Highlighter which can be found <a rel="nofollow" class="external text" href="https://web.archive.org/web/20190406231524/https://forum.facepunch.com/gmoddev/oarh/Notepad-GLua-Highlighter/1/">here</a>.
</p>
<div class="center"><div class="floatnone"><a href="/page/File:Notepadplusplus.png" class="image"><img alt="Notepadplusplus.png" src="/images/8/81/Notepadplusplus.png" width="456" height="345" /></a></div></div>
<h1> <span class="mw-headline" id="Sublime_Text"> <a rel="nofollow" class="external text" href="https://sublimetext.com/">Sublime Text</a> </span></h1>
<p>A very good text editor in terms of looks and effectiveness, it's highly customizable and easy to use. Has a number of features to aid in development, such as syntax highlighting, a minimap and a number of hotkeys. It also has a great <a rel="nofollow" class="external text" href="https://packagecontrol.io/">package manager</a> for installing plugins. The following are a few recommended packages to install:
</p>
<ul><li> <a rel="nofollow" class="external text" href="https://packagecontrol.io/packages/GMod%20Lua">GMod Lua Syntax Highlighting</a> (<a rel="nofollow" class="external text" href="https://web.archive.org/web/20160907203732/https://facepunch.com/threads/1038951a">Facepunch thread</a>)
</li><li> <a rel="nofollow" class="external text" href="https://packagecontrol.io/packages/SublimeLinter-contrib-glua">GMod Lua Linter</a>
</li></ul>
<p><br />
</p>

<h1> <span class="mw-headline" id="Visual_Studio"> <a rel="nofollow" class="external text" href="https://msdn.microsoft.com/en-us/vstudio/">Visual Studio</a> </span></h1>
<p>If you use Visual Studio then it might be more comfortable using it to edit Lua too. The free versions of Visual Studio are called 'Express'. There's a good page about Lua plugins on the <a rel="nofollow" class="external text" href="http://www.wowwiki.com/Lua_editors/Visual_Studio">World Of Warcraft Wiki</a>.
</p>
<div class="center"><div class="floatnone"><a href="/page/File:VisualStudioLua.png" class="image"><img alt="VisualStudioLua.png" src="/images/thumb/0/0a/VisualStudioLua.png/700px-VisualStudioLua.png" width="700" height="437" /></a></div></div>
<h1> <span class="mw-headline" id="IntelliJ_IDEA"> <a rel="nofollow" class="external text" href="http://www.jetbrains.com/idea/">IntelliJ IDEA</a> </span></h1>
<p>If you're a Java developer, then you've probably heard of IntelliJ. IntelliJ is an opensource(ish) IDE for Java and many other languages including Lua. You can get the community edition for <a rel="nofollow" class="external text" href="http://www.jetbrains.com/idea/download/index.html">free</a> or <a rel="nofollow" class="external text" href="http://www.jetbrains.com/idea/buy/index.jsp">purchase</a> the ultimate edition for as low as $99.
</p>
<div class="center"><div class="floatnone"><a href="/page/File:intellij.png" class="image"><img alt="intellij.png" src="/images/thumb/8/8b/intellij.png/700px-intellij.png" width="700" height="425" /></a></div></div>
<h1> <span class="mw-headline" id="Eclipse"> <a rel="nofollow" class="external text" href="http://http://www.eclipse.org/">Eclipse</a> </span></h1>
<p>Eclipse is an IDE primarily focused on Java. If you've ever messed around with Android or Java in general, then you're probably familiar with it. Eclipse has additional support for many languages including Lua via optional plugins such as <a rel="nofollow" class="external text" href="https://www.eclipse.org/koneki/ldt/">LDT</a>.
</p>
