<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Sets the lock on the X axis.
</p><p>For example the value 0.5 will lock the draggable panel to half the width of the slider's panel.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> lockX=nil</span>
</p>
<div style="margin-left: 32px;">
<p>Set to nil to reset lock.
</p>
The value range is from 0 to 1.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Example of a slider that can be dragged on the X and Y axis.
</p>
<pre><code data-language="lua">local frame = vgui.Create( &quot;DFrame&quot; )
frame:SetSize( 500, 300 )
frame:Center()
frame:MakePopup()

local Slider = vgui.Create( &quot;DSlider&quot;, frame )
Slider:SetPos( 50, 50 )
Slider:SetSize( 100, 100 )
Slider:SetLockX()
Slider:SetLockY()</code></pre>
