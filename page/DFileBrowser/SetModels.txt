<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Enables or disables the model viewer mode. In this mode, files are displayed as <a href="/page/Category_colon_SpawnIcon" title="Category:SpawnIcon">SpawnIcons</a> instead of a list.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This should only be used for <i>.mdl</i> files; the spawn icons will display error models for others. See <a href="/page/Category_colon_DFileBrowser" title="Category:DFileBrowser">DFileBrowser</a>:<a href="/page/DFileBrowser/SetFileTypes" title="DFileBrowser/SetFileTypes">SetFileTypes</a>
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> showModels=false</span>
</p>
<div style="margin-left: 32px;">
Whether or not to display files using <a href="/page/Category_colon_SpawnIcon" title="Category:SpawnIcon">SpawnIcons</a>.</div>
</div>
