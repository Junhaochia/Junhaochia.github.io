<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Performs a query on the local SQLite database, returns a table as result set, nil if result is empty and false on error.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> query</span>
</p>
<div style="margin-left: 32px;">
The query to execute.</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns">Returns</span></h1>
<div class="return" style="counter-increment: returns;">
<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
false is returned if there is an error, nil if the query returned no data.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Functions that are examples of saving and creating information into the database.
</p>
<pre><code data-language="lua">function CreateTable()
     sql.Query(&quot;CREATE TABLE player_data(SteamID TEXT, Money INTEGER)&quot;)
end

function NewPlayerToDataBase(ply)
     sql.Query(&quot;INSERT INTO player_data(SteamID, Money) VALUES('&quot;..ply:SteamID()..&quot;', 0)&quot;)
end

function SavePlayerToDatabase(ply, Money)
     sql.Query(&quot;UPDATE player_data SET Money=&quot;..Money..&quot; WHERE SteamID='&quot;..ply:SteamID()..&quot;'&quot;)
end</code></pre>
