<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Escapes dangerous characters and symbols from user input used in an SQLite SQL Query.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>this function is not meant to be used with external database engines such as <i>MySQL</i>. Escaping strings with inadequate functions is dangerous!
</p>
</td></tr></table>

<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> string</span>
</p>

</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>

</p>
<div style="margin-left: 32px;">
