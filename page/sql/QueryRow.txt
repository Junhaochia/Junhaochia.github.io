<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Performs the query like <a href="/page/Category_colon_sql" title="Category:sql">sql</a>.<a href="/page/sql/Query" title="sql/Query">Query</a>, but returns the first row found.<br />
</p>
Basically equivalent to<pre><code data-language="lua">sql.Query(&quot;*query* LIMIT 1&quot;)</code></pre>

</p>
<div style="margin-left: 32px;">
The input query</div>
</div>
