<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Applies a "halo" glow effect to one or multiple entities.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>Using this function outside of the PreDrawHalos hook can cause instability or crashes.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> entities</span>
</p>
<div style="margin-left: 32px;">
A table of entities to add the halo effect to</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> color</span>
</p>
<div style="margin-left: 32px;">
The desired color of the halo. See <a href="/page/Structures/Color" title="Structures/Color">Color structure</a></div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> blurX=2</span>
</p>
<div style="margin-left: 32px;">
The strength of the halo's blur on the x axis.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_4">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> blurY=2</span>
</p>
<div style="margin-left: 32px;">
The strength of the halo's blur on the y axis.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_5">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> passes=1</span>
</p>
<div style="margin-left: 32px;">
The number of times the halo should be drawn per frame. Increasing this may hinder player FPS.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_6">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> additive=true</span>
</p>
<div style="margin-left: 32px;">
Sets the render mode of the halo to additive.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_7">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_boolean" title="Category:boolean">boolean</a> ignoreZ=false</span>
</p>
<div style="margin-left: 32px;">
Renders the halo through anything when set to true.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Adds a halo around all props in the map using an O(n) operation and iterating through unseen objects which can be extremely expensive to process.
</p>
<pre><code data-language="lua">hook.Add( &quot;PreDrawHalos&quot;, &quot;AddPropHalos&quot;, function()
	halo.Add( ents.FindByClass( &quot;prop_physics*&quot; ), Color( 255, 0, 0 ), 5, 5, 2 )
end )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;"><br />
<a href="/page/File:halo_example.png" class="image" title="halo_example.png"><img alt="halo_example.png" src="/images/5/50/halo_example.png" width="400" height="205" /></a><br />All the props on the map will be rendered with a red halo, a blur amount of 5, and two passes.</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples_2">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example_2">Example</span></h2>
<p>Adds a green halo around all admins.
</p>
<pre><code data-language="lua">hook.Add( &quot;PreDrawHalos&quot;, &quot;AddStaffHalos&quot;, function()
	local staff = {}
	local i = 0

	for _, ply in ipairs( player.GetAll() ) do
		if ( ply:IsAdmin() ) then
			i = i + 1
			staff[i] = ply
		end
	end

	halo.Add( staff, Color( 0, 255, 0 ), 0, 0, 2, true, true )
end )</code></pre>
