<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the entity the halo library is currently rendering the halo for.
</p><p>The main purpose of this function is to be used in <a href="/page/Category_colon_ENTITY_Hooks" title="Category:ENTITY Hooks">ENTITY</a>:<a href="/page/ENTITY/Draw" title="ENTITY/Draw">Draw</a> in order not to draw certain parts of the entity when the halo is being rendered, so there's no halo around unwanted entity parts, such as lasers, 3D2D displays, etc.
</p>

If set, the currently rendered entity by the halo library.</div>
