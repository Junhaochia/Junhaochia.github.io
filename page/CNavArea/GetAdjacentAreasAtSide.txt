<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a table of all the <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavAreas</a> that have a ( one and two way ) connection <b>from</b> this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a> in given direction.
</p><p>If an area has a one-way incoming connection to this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>, then it will <b>not</b> be returned from this function, use <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>:<a href="/page/CNavArea/GetIncomingConnections" title="CNavArea/GetIncomingConnections">GetIncomingConnections</a> to get all incoming connections.
</p><p>See <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>:<a href="/page/CNavArea/GetAdjacentAreas" title="CNavArea/GetAdjacentAreas">GetAdjacentAreas</a> for a function that returns all areas from all sides/directions.
</p>

The direction, in which to look for <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavAreas</a>, see <a href="/page/Enums/NavDir" title="Enums/NavDir">NavDir_ Enums</a>.</div>
