<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a table of all the <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavAreas</a> that have a one-way connection <b>to</b> this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>.
</p><p>If a <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a> has a two-way connection <b>to or from</b> this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a> then it will not be returned from this function, use <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>:<a href="/page/CNavArea/GetAdjacentAreas" title="CNavArea/GetAdjacentAreas">GetAdjacentAreas</a> to get outgoing ( one and two way ) connections.
</p><p>See <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>:<a href="/page/CNavArea/GetIncomingConnectionsAtSide" title="CNavArea/GetIncomingConnectionsAtSide">GetIncomingConnectionsAtSide</a> for a function that returns one-way incoming connections from only one side/direction.
</p>

</p>
<div style="margin-left: 32px;">
A table of all <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavAreas</a> with one-way connection <b>to</b> this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>.<br />Returns an empty table if there are no one-way incoming connections <b>to</b> this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>.</div>
