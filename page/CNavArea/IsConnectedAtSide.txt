<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns whether this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a> has an outgoing ( one or two way ) connection <b>to</b> given <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a> in given direction.
</p><p>See <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>:<a href="/page/CNavArea/IsConnected" title="CNavArea/IsConnected">IsConnected</a> for a function that checks all sides.
</p>
