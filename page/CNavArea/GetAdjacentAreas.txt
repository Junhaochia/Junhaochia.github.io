<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a table of all the <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavAreas</a> that have a ( one and two way ) connection <b>from</b> this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>.
</p><p>If an area has a one-way incoming connection to this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>, then it will <b>not</b> be returned from this function, use <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>:<a href="/page/CNavArea/GetIncomingConnections" title="CNavArea/GetIncomingConnections">GetIncomingConnections</a> to get all one-way incoming connections.
</p><p>See <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>:<a href="/page/CNavArea/GetAdjacentAreasAtSide" title="CNavArea/GetAdjacentAreasAtSide">GetAdjacentAreasAtSide</a> for a function that only returns areas from one side/direction.
</p>
