<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns a table of all the <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavAreas</a> that have a one-way connection <b>to</b> this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a> from given direction.
</p><p>If a <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a> has a two-way connection <b>to or from</b> this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a> then it will not be returned from this function, use <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>:<a href="/page/CNavArea/GetAdjacentAreas" title="CNavArea/GetAdjacentAreas">GetAdjacentAreas</a> to get outgoing ( one and two way ) connections.
</p><p>See <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>:<a href="/page/CNavArea/GetIncomingConnections" title="CNavArea/GetIncomingConnections">GetIncomingConnections</a> for a function that returns one-way incoming connections from all sides/directions.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> navDir</span>
</p>
<div style="margin-left: 32px;">
The direction, from which to look for <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavAreas</a>, see <a href="/page/Enums/NavDir" title="Enums/NavDir">NavDir_ Enums</a>.</div>
</div>

<p><span class="ret_number" style="counter-increment: returns -1;"></span> <span class="ret_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a></span>
</p>
<div style="margin-left: 32px;">
A table of all <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavAreas</a> with one-way connection <b>to</b> this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a> from given direction.<br />Returns an empty table if there are no one-way incoming connections <b>to</b> this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a> from given direction.</div>
