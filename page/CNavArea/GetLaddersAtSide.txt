<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns all <a href="/page/Category_colon_CNavLadder" title="Category:CNavLadder">CNavLadders</a> that have a ( one or two way ) connection <b>from</b> ( one and two way ) this <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a> in given direction.
</p><p>See <a href="/page/Category_colon_CNavArea" title="Category:CNavArea">CNavArea</a>:<a href="/page/CNavArea/GetLadders" title="CNavArea/GetLadders">GetLadders</a> for a function that returns <a href="/page/Category_colon_CNavLadder" title="Category:CNavLadder">CNavLadder</a> from/in all sides/directions.
</p>

<p>The direction, in which to look for <a href="/page/Category_colon_CNavLadder" title="Category:CNavLadder">CNavLadders</a>.

</p>
1 = Down ( LadderDirectionType::LADDER_DOWN )</div>
