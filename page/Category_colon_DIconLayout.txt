<h1> <span class="mw-headline" id="DescriptionPanelPage">Description<span style="display: none;"><a href="/page/Form:PanelPage" title="Form:PanelPage">PanelPage</a></span></span></h1>
<p>DIconLayout is what replaced <a href="/page/Category_colon_DPanelList" title="Category:DPanelList">DPanelList</a> in Garry's Mod 13. DPanelList still exists in GMod but is deprecated and does not support the new GWEN skin.
</p><p><strong class="selflink">DIconLayout</strong> is used to make a list of panels. Unlike DPanelList, DIconLayout does not automatically add a scroll bar - the example below shows you how you can do this.
</p>

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<pre><code data-language="lua">local Frame = vgui.Create( &quot;DFrame&quot; ) -- Create a Frame to contain everything.
Frame:SetTitle( &quot;DIconLayout Example&quot; )
Frame:SetSize( 365, 240 )
Frame:Center()
Frame:MakePopup()

local Scroll = vgui.Create( &quot;DScrollPanel&quot;, Frame ) -- Create the Scroll panel
Scroll:Dock( FILL )

local List = vgui.Create( &quot;DIconLayout&quot;, Scroll )
List:Dock( FILL )
List:SetSpaceY( 5 ) -- Sets the space in between the panels on the Y Axis by 5
List:SetSpaceX( 5 ) -- Sets the space in between the panels on the X Axis by 5

for i = 1, 20 do -- Make a loop to create a bunch of panels inside of the DIconLayout
	local ListItem = List:Add( &quot;DPanel&quot; ) -- Add DPanel to the DIconLayout
	ListItem:SetSize( 80, 40 ) -- Set the size of it
	-- You don't need to set the position, that is done automatically.
end

local ListLabel = List:Add( &quot;DLabel&quot; ) -- Add a label that will be the only panel on its row
ListLabel.OwnLine = true -- The magic variable that specifies this item has its own line all for itself
ListLabel:SetText( &quot;Hello World!&quot; )

for i = 1, 5 do
	local ListItem = List:Add( &quot;DPanel&quot; )
	ListItem:SetSize( 80, 40 )
end</code></pre>
