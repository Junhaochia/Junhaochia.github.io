<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Called when the page inside the <a href="/page/Category_colon_DHTML" title="Category:DHTML">DHTML</a> window runs <i>console.log</i>. This can also be called within the Lua environment to emulate <i>console.log</i>. If the contained message begins with <i>RUNLUA:</i> the following text will be executed as code within the Lua environment (this is how Lua is called from DHTML windows).
</p>
