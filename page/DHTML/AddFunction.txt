<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Defines a Javascript function that when called will call a Lua callback.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Must be called after the HTML document has fully loaded.
</p>
</td></tr></table>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> library</span>
</p>
<div style="margin-left: 32px;">
Library name of the JS function you are defining.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> name</span>
</p>
<div style="margin-left: 32px;">
Name of the JS function you are defining.</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_3">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> callback</span>
</p>
<div style="margin-left: 32px;">
Function called when the JS function is called. Arguments passed to the JS function will be passed here.</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Prints text from Javascript to the console in color.
</p>
<pre><code data-language="lua">-- Create the frame
local f = vgui.Create(&quot;DFrame&quot;)
f:SetSize(800, 600)
f:Center()

-- Create a green color variable
local color_green = Color(0, 255, 0)

-- Define the Javascript function in the DHTML element
local DHTML = vgui.Create(&quot;DHTML&quot;, f)
DHTML:Dock(FILL)
DHTML:OpenURL(&quot;http://wiki.garrysmod.com/page/VGUI/Elements/DHTML&quot;)
DHTML:AddFunction(&quot;console&quot;, &quot;luaprint&quot;, function(str)
	MsgC(color_green, str) -- Print the given string
end)

--This runs our function. Our function could also be called from Javascript on the DHTML's page.
DHTML:RunJavascript(&quot;console.luaprint('Hello from Javascript!');&quot;)</code></pre>
