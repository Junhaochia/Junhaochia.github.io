<h1> <span class="mw-headline" id="DescriptionPanelFunc">Description<span style="display: none;"><a href="/page/Form:PanelFunc" title="Form:PanelFunc">PanelFunc</a></span></span></h1>
<p>Runs/Executes a string as JavaScript code.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function does <b>NOT</b> evaluate expression (i.e. allow you to pass variables from JavaScript (JS) to Lua context).<br />Because a return value is nil/no value (a.k.a. void).<br />If you wish to pass/return values from JS to Lua, you may want to use <a href="/page/Category_colon_DHTML" title="Category:DHTML">DHTML</a>:<a href="/page/DHTML/AddFunction" title="DHTML/AddFunction">AddFunction</a> function to accomplish that job.
</p>
</td></tr></table>

<div style="margin-left: 32px;">

<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>Shows how to change <a rel="nofollow" class="external text" href="https://www.w3schools.com/jsref/prop_html_innerhtml.asp">document.body.innerHTML</a> property by calling this function on <a href="/page/Category_colon_DHTML" title="Category:DHTML">DHTML</a> panel.
</p>
<pre><code data-language="lua">-- First we create a container, in this case it is a full-screen Derma Frame window.
local dframe = vgui.Create( 'DFrame' )
dframe:SetSize( ScrW(), ScrH() )
dframe:SetTitle( &quot;Garry's Mod Wiki&quot; )
dframe:Center()
dframe:MakePopup() -- Enable keyboard and mouse interaction for DFrame panel.

-- Create a new DHTML panel as a child of dframe, and dock-fill it.
local dhtml = vgui.Create( 'DHTML', dframe )
dhtml:Dock( FILL )
-- Navigate to Garry's Mod wikipedia website.
dhtml:OpenURL( '/index.php' )
-- Run JavaScript code.
dhtml:QueueJavascript( [[document.body.innerHTML = 'HTML changed from Lua using JavaScript!';]] )

-- This does not throw an error/exception, but instead returns nil/no value.
-- That means you can't pass/return values from JavaScript back to Lua context using this function.
local number = dhtml:QueueJavascript( '22;' )
print( number )</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">Inner HTML of document body in DHTML panel is now set to "HTML changed from Lua using JavaScript!".</div>
