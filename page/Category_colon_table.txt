<p>The table <i>type</i> is an object that can store multiple values.
</p><p>The table <i>library</i> is a standard Lua library which provides functions to manipulate tables. In Garry's Mod there are several extra useful functions added to this library.
</p><p>This category lists the functions available in the table <i>library</i>.
