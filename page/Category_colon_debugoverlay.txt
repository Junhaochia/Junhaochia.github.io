<p>The debugoverlay library is mainly useful for 3D debugging, it can be used to draw shapes on the screen for debug purposes.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>These functions will not do anything unless the <b>developer</b> console variable is set to non 0.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>These function appear to not work in singleplayer
</p>
</td></tr></table>
