<p>Adds a control to the control panel.
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> type</span>
</p>
<div style="margin-left: 32px;">
<p>The control type to add. The complete list is:
</p>
<ul><li> header
</li><li> textbox
</li><li> label
</li><li> checkbox/toggle
</li><li> slider
</li><li> propselect
</li><li> matselect
</li><li> ropematerial
</li><li> button
</li><li> numpad
</li><li> color
</li><li> combobox
</li><li> listbox
</li><li> materialgallery</div>
</li></ul>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_table" title="Category:table">table</a> controlinfo</span>
</p>
<div style="margin-left: 32px;">
<p>Each control takes their own table structure. You may search "AddControl" on GitHub for examples.
</p><p>Here is a full list of each type and the table members it requires:
</p>
<ul><li>header
<ul><li>description
</li></ul>
</li></ul>
<ul><li>textbox:
<ul><li>label (def: "Untitled")
</li><li>command
</li></ul>
</li></ul>
<ul><li>label:
<ul><li>text
</li></ul>
</li></ul>
<ul><li>checkbox, toggle (same thing):
<ul><li>label (def: "Untitled")
</li><li>command
</li><li>help (boolean, if true assumes label is a language string ("#tool.toolname.stuff") and adds ".help" at the end)
</li></ul>
</li></ul>
<ul><li>slider:
<ul><li>type (optional string, if equals "float" then 2 digits after the decimal will be used, otherwise 0)
</li><li>label (def: "Untitled")
</li><li>command
</li><li>min (def: 0)
</li><li>max (def: 100)
</li></ul>
</li><li>help (boolean, see above)
</li></ul>
<ul><li>propselect:
<ul><li>(data goes directly to PropSelect's&#160;:ControlValues(data))
</li></ul>
</li></ul>
<ul><li>matselect:
<ul><li>(data goes directly to MatSelect's&#160;:ControlValues(data))
</li></ul>
</li></ul>
<ul><li>ropematerial:
<ul><li>convar (notice: NOT called command this time!)
</li></ul>
</li></ul>
<ul><li>button:
<ul><li>label / text (if label is missing will use text. Def: "No Label")
</li><li>command
</li></ul>
</li></ul>
<ul><li>numpad:
<ul><li>command
</li><li>command2
</li><li>label
</li><li>label2
</li></ul>
</li></ul>
<ul><li>color:
<ul><li>label
</li><li>red (convar)
</li><li>green (convar)
</li><li>blue (convar)
</li><li>alpha (convar)
</li></ul>
</li></ul>
<ul><li>combobox:
<ul><li>menubutton (if doesn't equal "1", becomes a listbox)
</li><li>folder
</li><li>options (optional, ha)
</li><li>cvars (optional)
</li></ul>
</li></ul>
<ul><li>listbox:
<ul><li>height (if set, becomes <a href="/page/Category_colon_DListView" title="Category:DListView">DListView</a>, otherwise is <a href="/page/Category_colon_CtrlListBox" title="Category:CtrlListBox">CtrlListBox</a>)
</li><li>label (def: "unknown")
</li><li>options (optional)
</li></ul>
</li></ul>
<ul><li>materialgallery:
<ul><li>width (def: 32)
</li><li>height (def: 32)
</li><li>rows (def: 4)
</li><li>convar
</li><li>options</div>
</li></ul>
</li></ul>
</div>
