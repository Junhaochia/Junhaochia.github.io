<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Adds a function to call when the current undo block is undone
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_function" title="Category:function">function</a> func</span>
</p>
<div style="margin-left: 32px;">
The function to call</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_vararg" title="Category:vararg">vararg</a> arguments</span>
</p>
<div style="margin-left: 32px;">
Arguments to pass to the function (after the undo info table)</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This example creates a prop_physics, and adds it to the players undo list. A message will be printed to console about it.
</p>
<pre><code data-language="lua">prop = ents.Create(&quot;prop_physics&quot;)
prop:SetModel(&quot;models/props_junk/wood_crate001a.mdl&quot;)
prop:Spawn()
undo.Create(&quot;prop&quot;)
 undo.AddEntity(prop)
 undo.AddFunction(function(tab, arg2)
 print(tab.Owner:GetName()..&quot; removed prop &quot;..tab.Entities[1]:GetModel()..&quot;, code: &quot;..arg2)
 end, 556)
 undo.SetPlayer(ply)
undo.Finish()</code></pre>
<p><b>Output:</b>
</p>
<div style="margin:2em;">"PlayerName removed prop models/props_junk/wood_crate001a.mdl, code: 556" will be printed</div>
