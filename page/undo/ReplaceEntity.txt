<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Replaces any instance of the "from" reference with the "to" reference, in any existing undo block. Returns true if something was replaced
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> from</span>
</p>
<div style="margin-left: 32px;">
The old entity</div>
</div>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments_2">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_Entity" title="Category:Entity">Entity</a> to</span>
</p>
<div style="margin-left: 32px;">
The new entity to replace the old one</div>
</div>

</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>When an entity is ragdolled, this will replace any instances of the entity with it's ragdoll.
</p>
<pre><code data-language="lua">function GM:CreateEntityRagdoll( entity, ragdoll )
 // Replace the entity with the ragdoll in cleanups etc
 undo.ReplaceEntity( entity, ragdoll )
 cleanup.ReplaceEntity( entity, ragdoll )
end</code></pre>
