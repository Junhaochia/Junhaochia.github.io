<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets a custom undo text for the current undo block
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> customText</span>
</p>
<div style="margin-left: 32px;">
The text to display when the undo block is undone</div>
</div>
<h1 class="first_example"> <span class="mw-headline" id="Examples">Examples</span></h1>
<h2 class="examples_number"> <span class="mw-headline" id="Example">Example</span></h2>
<p>This example creates a prop_physics, adds it to the players undo list, and sets a custom undo text
</p>
<pre><code data-language="lua">local prop = ents.Create( &quot;prop_physics&quot; )
prop:SetModel( &quot;models/props_junk/wood_crate001a.mdl&quot; )
prop:Spawn()
undo.Create( &quot;prop&quot; )
 undo.AddEntity( prop )
 undo.SetPlayer( Player )
 undo.SetCustomUndoText(&quot;Undone a crate prop&quot;)
undo.Finish()</code></pre>
