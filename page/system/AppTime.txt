<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the total uptime of the current application as reported by Steam.
</p><p>This will return a similar value to <a href="/page/Global/SysTime" title="Global/SysTime">SysTime</a>.
</p>

<tr>
<td>NOTE
