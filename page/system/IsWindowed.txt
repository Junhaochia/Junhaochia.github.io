<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns whether the game is being run in a window or in fullscreen (you can change this by opening the menu, clicking 'Options', then clicking the 'Video' tab, and changing the Display Mode using the dropdown menu):
</p><p><a href="/page/File:DisplayModeDropdown.jpeg" class="image"><img alt="DisplayModeDropdown.jpeg" src="/images/thumb/f/f0/DisplayModeDropdown.jpeg/300px-DisplayModeDropdown.jpeg" width="300" height="240" /></a>
</p>
