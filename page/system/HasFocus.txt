<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns whether or not the game window has focus.
</p><p>Returns true 100% of the time on OS X.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function does not work on dedicated servers and will instead return no value.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
