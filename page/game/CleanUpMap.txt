<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>If called serverside it will remove ALL entities which were not created by the map(not players or weapons held by players).
</p><p>On the client it will remove decals, sounds, gibs, dead NPCs, and entities created via <a href="/page/Category_colon_ents" title="Category:ents">ents</a>.<a href="/page/ents/CreateClientProp" title="ents/CreateClientProp">CreateClientProp</a>.
</p><p>This function calls <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PreCleanupMap" title="GM/PreCleanupMap">PreCleanupMap</a> before cleaning up the map and <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/PostCleanupMap" title="GM/PostCleanupMap">PostCleanupMap</a> after cleaning up the map.
</p>
