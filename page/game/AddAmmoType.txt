<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Adds a new ammo type to the game.
</p><p>You can find a list of default ammo types <a href="/page/Default_Ammo_Types" title="Default Ammo Types">here</a>.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>This function must be called shared or you will have unexpected problems.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>This function must be called in <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/Initialize" title="GM/Initialize">Initialize</a>.
</p>
</td></tr></table>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>There is a limit of 128 ammo types, including the default ones.
</p>
</td></tr></table>
<table class="gmodwiki_note bug_note">
<tr>
<td>BUG
</td>
<td>
<p>This will have unpredictable results when registering ammo types with the same name but different capitalisation.
</p><p><b>Pull Request: <a rel="nofollow" class="external text" href="https://github.com/Facepunch/garrysmod/pull/1487">#1487</a></b>
</p>
</td></tr></table>

<pre><code data-language="lua">game.AddAmmoType({
	name = &quot;ammo_9mm_max&quot;,

	tracer = TRACER_LINE,

	npcdmg = 0,

	minsplash = 10,
