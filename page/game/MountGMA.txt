<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Mounts a GMA addon from the disk. Any error models currently loaded that the mounted addon provides will be reloaded.
</p><p>Can be used with <a href="/page/Category_colon_steamworks" title="Category:steamworks">steamworks</a>.<a href="/page/steamworks/DownloadUGC" title="steamworks/DownloadUGC">DownloadUGC</a>
</p>
<h1 class="first_arg"> <span class="mw-headline" id="Arguments">Arguments</span></h1>
<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_string" title="Category:string">string</a> path</span>
</p>
<div style="margin-left: 32px;">
Location of the GMA file to mount, relative to the garrysmod directory</div>
</div>

</p>
<div style="margin-left: 32px;">
success</div>
</div>
<h1 class="first_ret"> <span class="mw-headline" id="Returns_2">Returns</span></h1>

If successful, a table of files that have been mounted</div>
</div>
