<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Returns the map load type of the current map.
</p><p>After changing the map with the console command <i>changelevel</i>, "newgame" is returned. With <i>changelevel2</i> (single player only), "transition" is returned.
</p>
