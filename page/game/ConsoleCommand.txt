<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Runs a console command.
Make sure to add a newline ("\n") at the end of the command.
</p>
<table cellspacing="0" class="gmodwiki_note warning_note">
<tr>
<td>WARNING
</td>
<td>
<p>If you use data that were received from a client, you should avoid using this function because newline and semicolon (at least) allow the client to run arbitrary commands!
</p><p>For safety, you are urged to prefer using <a href="/page/Global/RunConsoleCommand" title="Global/RunConsoleCommand">RunConsoleCommand</a> in this case.
</p>
</td></tr></table>
