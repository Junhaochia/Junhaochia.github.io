<h1> <span class="mw-headline" id="Description">Description</span></h1>
<p>Sets the time scale of the game.
</p><p>This function is supposed to remove the need of using the host_timescale convar, which is cheat protected.
</p><p>To slow down or speed up the movement of a specific player, use <a href="/page/Category_colon_Player" title="Category:Player">Player</a>:<a href="/page/Player/SetLaggedMovementValue" title="Player/SetLaggedMovementValue">SetLaggedMovementValue</a> instead.
</p>
<table cellspacing="0" class="gmodwiki_note">
<tr>
<td>NOTE
</td>
<td>
<p>Like host_timescale, this method does not affect sounds, if you wish to change that, look into <a href="/page/Category_colon_GM_Hooks" title="Category:GM Hooks">GM</a>:<a href="/page/GM/EntityEmitSound" title="GM/EntityEmitSound">EntityEmitSound</a>.
</p>
</td></tr></table>

<div class="argument">
<p><span class="arg_number"></span> <span class="arg_chunk"><a href="/page/Category_colon_number" title="Category:number">number</a> timeScale</span>
</p>
<div style="margin-left: 32px;">
The new timescale, minimum value is 0.001 and maximum is 5.</div>
